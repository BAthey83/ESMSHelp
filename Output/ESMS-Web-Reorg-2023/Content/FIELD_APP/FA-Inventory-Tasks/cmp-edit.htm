Edit Component

To edit a component,

  1. Select any one of the top tabs.
  2. In the navigation pane (open it if it is auto-hidden), activate the Systems tab.
  3. In the asset tree, select the component to be edited.
  4. Activate Edit Component in the ribbon. The View Component tab will appear in the content area.
  5. Edit information as desired in the fields. More detail about these fields is given in the topic Component Data.
  6. Activate the Save button (located above the Basic Properties set of fields).

For more information about accessing and configuring the navigation pane and content pane, see Manipulating Navigation and Content Panes.
