View a Direct Rating

You can view a direct rating any time from the navigation pane. The navigation pane is viewable from any of the top tabs (except for the File tab at the left), as long as no subtask has been selected at the ribbon. If the navigation pane has been auto-hidden, bring it into view using the Navigation tab at the left of the screen.

 

Productivity Tip: For how to resize the navigation pane, or use Auto Hide to use limited screen space efficiently, see the topic Manipulating Navigation and Content Panes.

To view a direct rating, <--{{afdo - consider making this phrase bold in all contexts, INCLUDING when there is only one step and therefore no step numbers.}}

  1. Select any one of the top tabs except for the File tab.
  2. Mark the Direct Rating checkbox in the navigation pane toolbar. A dropdown field will appear. {{afdo - show screenshot with first entry of the dropdown. Redbox it.}}
  3. Single-click the desired rating date in the dropdown list. This results in the Direct Rating column showing the assessment status of assets as of that date. {{ screenshot. }}
