Project Details - Assets

The Assets tab of the Project Details Screen is for viewing and maintaining the list of assets and WBS items managed by the project. The ability to edit the project is available to users with Update Project privileges for at least one of the project sites.

The list of assets and WBS items is editable only when the project is in the Project Development state.

Maintaining the Asset List

Engage the Add Project Assets button to select the assets to add to the project.

The Select Assets dialog displays a paginated searchable list of assets that belong to the project sites.

The Options button allows assets to be searched by Name, Facility Number, RPUID or CAT Code. The dialog provides a way to select one or multiple assets.

Once the assets are selected they are displayed in the list of assets on the Project Assets tab.

Engage the Remove Asset button to remove the asset from the list.

Maintaining the WBS Item List

To view or maintain the WBS item list, select a row in the Assets list.

Engage the Add WBS Items button to select from the list of available Work Items to add to the project.

The Add WBS Items dialog displays a paginated filterable list of Work Items. The Work Items listed are for the components that belong to the selected asset and are not associated with any other project.

The listed Work Items can be filtered by the Work Item source type: Work Analysis, Assessment and Work Package.

When a source type is selected, the subsequent dialog shows available Work Item sources of the selected type. For example, if Work Package is selected, the Select Work Package Source dialog displays all available Work Packages that contain Work Items that can be managed by the project. Picking an item from the list of Packages filters the available Work Item list to the ones that belong to the selected Package only.

Check the check box in the Work Item list to indicate the Work Items to be managed by the project. A WBS is created for each selected Work Item.

To remove a WBS and disassociate an individual Work Item, engage the Delete button in the WBS list row.

To remove a WBS and disassociate Work Items in batches, engage the Delete button above the WBS list.

The Remove WBS Items dialog displays a paginated filterable list of WBS items for the selected asset.

The listed WBS items can be filtered by the Work Item source type in the same way as in the Add WBS Items dialog. Check the check box in the list to indicate which WBS items must be removed.
