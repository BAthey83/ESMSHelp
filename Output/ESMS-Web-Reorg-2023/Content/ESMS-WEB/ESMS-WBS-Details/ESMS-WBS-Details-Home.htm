WBS Details Screen

This Screen shows attributes of a WBS item selected from the Project Details - Assets tab of the Project Details Screen.

The WBS Details tab displays the following attributes:

  • Component Name
  • Activity Type
  • Estimated Cost
  • Actual Cost
  • Status
  • Work Classification
  • Modification Number
  • Work Order Number
  • Year
  • CLIN
  • Created Date
  • RTO/RTI Planned Date
  • RTO/RTI Actual Date
  • Technically Complete Date
  • Reference
  • Source
  • Comment

For the WBS items in the Fuels domain, the following attributes are displayed in the Attributes expandable section of the tab:

  • Current Situation
  • Impact
  • Scope Of Work
  • Regulatory Requirement
  • Classification

For users with Update Project privileges, this screen allows them to Edit WBS Details Dialog.
