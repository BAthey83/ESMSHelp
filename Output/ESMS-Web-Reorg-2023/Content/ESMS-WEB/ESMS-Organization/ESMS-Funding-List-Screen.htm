Funding List Screen

This screen displays the list of available Funding for the selected organization to view and/or edit funding. The list is sorted in ascending order by funding precedence. It is accessible from the Funding tab of Organization Screen to users with Funding privileges.

The screen includes the following actions:

  • Adding New Funding
  • Funded Project List
  • Edit Funding Details
  • Delete Funding
  • Checking the box allows the user to change funding precedence by dragging and dropping a funding card.
