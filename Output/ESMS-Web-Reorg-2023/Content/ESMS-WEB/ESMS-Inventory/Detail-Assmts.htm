Assessment Properties

At the Assessment Details screen, you can see the following properties for the selected group, if the properties have been entered into Enterprise SMS Web:

  • name
  • component
  • facility
  • group
  • date
  • user
  • date updated
  • type
  • non-representative
  • quantity
  • future date
  • sample location
