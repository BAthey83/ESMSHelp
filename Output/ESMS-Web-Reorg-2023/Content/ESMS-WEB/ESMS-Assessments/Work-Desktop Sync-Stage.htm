Desktop Sync / Assessment Stage

After the Review stage is complete, the work flow will transfer to the ESMS field application (Desktop or Tablet) for the assessment plan.

This topic is not intended as comprehensive instructions for assessment, but simply to provide a high-level glimpse at what happens in the interim where the action passes temporarily outside of ESMS-Web.

Pre-Sync

To perform pre-assessment plan synchronization,

  1. On the main menu in ESMS Web, activate the dropdown of user options.
  2. Select Application Sync Settings from the dropdown.
  3. In the popup, select the Workflow Plan to be made accessible to ESMS Desktop or Tablet for the assessment plan that has been prepared for with asset selection, group selection, assessment plan team selection, and Assessment Plan Questionnaire.
  4. Mark the Sync Images checkbox if you want images as well as data to be synchronized.
  5. Next, log in to ESMS Desktop or Tablet and activate the Sync button to synchronize.

Assessment Plan Questionnaire

After synchronization, execute the assessment plan questionnaire, either in the field with a tablet or else in ESMS Desktop using field notes.

Work Item Generation

Generate work items in ESMS Desktop, based on answers to the assessment plan questionnaire. After re-synchronization, these work items will appear in the Work Items table at the Post-Assessment Plan Stage stage in ESMS Web.

Re-Sync

When the assessment plan has been completed, synchronize again using the Sync button to make the assessment plan results available to ESMS Web.
