Project Details Screen - Workflow-Enabled Project Status

Project status updates for workflow-enabled projects are done according to the workflow scheme selected for the project.

The current project workflow state is displayed on the Project Details Screen.

It is possible to change the project to the next or previous state according to the project workflow scheme by engaging the State button and then engaging the Approve button for that state.

The user must be granted workflow team permissions for the selected project to be able to approve the state change.

The status update history, along with the approval comments, can be viewed by engaging the History button.
