Change Role Name (Security Permissions)

This task requires that you have permission to administer a team.

To change the name of a security permissions role,

  1. At the Teams ("Team Editor") screen, select a team.
  2. Activate the Roles tab. The Roles associated with the team will appear.
  3. In the list of roles, activate the role you want to change the name of. The Role Details should appear in the content area to the right.
  4. In the Name field, do one of the following:
    1. Edit the old role name,
    2. Delete the old role name and replace it with the new name, or
    3. Select the old name and overwrite it with the new name.
  5. Activate the Save button at the bottom of the screen.
