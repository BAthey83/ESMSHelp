You can assign security permissions to the team you administer. The scope of these permissions are defined by selected sites and/or groups. Assigning a permissions role is done in the "Team Workspace" area at the Teams tab.

Select Site

  1. In the "Add Site Role" popup, activate the button next to "Site". This might have an existing site name, or it might be labeled None Selected.
  2. In the "Select Site" popup that appears, locate the site you wish to select.

    Note: If there are a large number of sites, you can use (1) the search option available via the "Site name" field and/or (2) the page navigation buttons at the bottom of the "Select Site" popup to help you find it. For more details about the search process, see "Select Site" in the topic Enterprise SMS Entry-Level Tasks.

  3. Activate the Select button at the bottom of the popup. After selecting the site, you will be returned to the "Add Site Role" popup.

Select Site Role and Finalize

  1. In the "Add Site Role" popup, activate the dropdown arrow in the "Role" field.
  2. Select from the available roles.
  3. Note that new roles can be designed and created, if desired. This is described in the topic Add Role (Security Permissions) .
  4. Once you've selected the desired role, activate the Create button. (Alternatively, activate Cancel to cancel the operation without assigning a role to the team.) The newly assigned role should appear in the Team Workspace area.

    5. The steps for assigning a site role can be repeated to create additional Target/Role pairs in the Team Workspace area.

Assign Group Role

To assign the team a permissions role that is defined in scope by one or more groups, you will

  • bring up the "Add Group Role" popup
  • select a group
  • select a role
  • finalize

Bring up the "Add Group Role" Popup

To bring up the popup that coordinates assigning a permissions role,

  1. Navigate to the Teams tab on the Teams screen.
  2. Select the desired team. The Team Details should appear in the content area to the right.
  3. Activate the Groups tab in the "Team Workspace".
  4. Activate the Add Groups Role button at the bottom of the Team Workspace.

Select Group

  1. In the "Add Group Role" popup, activate the button next to "Group". This might have an existing site name, or it might be labeled None Selected.

  2. In the "Select Group" popup that appears, locate the group you wish to select. You can use (1) the search option available via the "Group Name" field and/or (2) the page navigation buttons at the bottom of the popup to help you find it.
  3. Activate the desired group (once activated, it will be highlighted).
  4. Activate the Select button at the bottom of the popup.

Select Group Role and Finalize

  1. In the "Add Group Role" popup, activate the dropdown arrow in the "Role" field.
  2. Select from the available roles.
  3. Note that new roles can be designed and created, if desired. This is described in the topic Add Role (Security Permissions) .
  4. Once you have selected the desired role, activate the Create button. (Alternatively, activate Cancel to cancel the operation without assigning a role to the team.)

The steps for assigning a group role can be repeated to create additional Target/Role pairs in the Team Workspace area.
