Assessment Plan Navigation

Navigate to an Assessment Plan

From Outside of an Assessment Plan

To navigate to an assessment plan from the menu, start with step 1 below. If you are already at the assessment plans screen, start with step 2:

  1. To start using Workflow, activate the Assessment Plans tile at the ESMS menu.
  2. Select an assessment plan from the table. You will be brought to the Charter stage for the assessment plan.
  3. Alternatively, to create a new assessment plan, see the instructions at Add Assessment Plans.

From Inside an Assessment Plan

  1. Once you are in one of the six stages of the plan creation, you can activate the browser's back arrow until you return to the table of assessment plans on the initial assessment plans screen.
  1. There you can select a different assessment plan from the table.

Navigate in an Assessment Plan

Once in an assessment plan, you can activate the Workflow button/flyout tab to open the Workflow navigation tool.

The Workflow navigation tool is located in one of two places:

  • On a wide desktop screen, this is activated via a flyout tab at the upper left of the screen.
  • On a narrower screen, activate the navy blue bar with the word "Workflow" appearing in white, below the title/menu bar.

After you activate, the Workflow navigation tool will expand as shown below:

Note: What you see and what you can activate in the Workflow menu depends on multiple factors. Depending on your permissions, you might not see the full selection of navigation buttons shown here. If approvals are outstanding, some of the options may be grayed out. Also, if Workflow was not activated for the assessment plan, this navigation tool will show only Charter / Details and Assessment Plan / Draft Work Item.

Using the expanded navigation tool, you can jump directly to a tab in one of the Workflow stages. (Scroll down as needed if not all of the stages are visible.) Then when you click outside the tool, it will close.
