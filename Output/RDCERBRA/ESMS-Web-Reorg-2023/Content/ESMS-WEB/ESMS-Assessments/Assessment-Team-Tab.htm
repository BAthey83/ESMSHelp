Assessment Team Tab

This topic describes the Assessment Team tab of an Enterprise SMS assessment plan. This tab is to be selected after the Assessor Pre-Assessment Team Tab.

— Quick Links —

Tasks at this stage are to:

  • Add Assessment Team Members
  • Add Full Desktop Access
  • Identify and Manage Disciplines

Add Assessment Team Members

To add team members to your assessment,

  • Select the Add Team Member button (the plus sign button highlighted above in red).
  • The User Search popup menu will appear.
  • Search for the user you want to add to your assessment team.
  • Click on the user, followed by Select to add the team member.
  • If you want to discard the addition, choose the Close button.

Add Full Desktop Access

To add Full Desktop Access to a team member's profile, simply select the checkbox under the "Allow Full Desktop" column.

Identify and Manage Disciplines

  • Select the Manage Disciplines button (the wrench icon highlighted above in red).
  • The Manage Disciplines popup menu appears.
  • Select the disciplines that apply to the user. Click Save.
  • The chosen disciplines will be added to the user's profile.
