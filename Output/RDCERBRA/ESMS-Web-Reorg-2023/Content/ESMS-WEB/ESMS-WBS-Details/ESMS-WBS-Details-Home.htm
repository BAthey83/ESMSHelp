Edit WBS Details Dialog

The Edit WBS dialog is activated by the Edit button on the WBS Details page.

The dialog provides a way to update the following attributes of the selected WBS item:

  • Status
  • Classification
  • Notification Number
  • Work Order Number
  • CLIN
  • Actual Cost
  • RTO/RTI Planned Date
  • RTO/RTI Actual Date
  • Technically Complete Date
  • Comment

WBS Details Screen

This Screen shows attributes of a WBS item selected from the Project Details - Assets tab of the Project Details Screen.

The WBS Details tab displays the following attributes:

  • Component Name
  • Activity Type
  • Estimated Cost
  • Actual Cost
  • Status
  • Work Classification
  • Modification Number
  • Work Order Number
  • Year
  • CLIN
  • Created Date
  • RTO/RTI Planned Date
  • RTO/RTI Actual Date
  • Technically Complete Date
  • Reference
  • Source
  • Comment
