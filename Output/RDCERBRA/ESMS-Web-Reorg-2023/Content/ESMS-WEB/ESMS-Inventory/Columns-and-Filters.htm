Introduction to Columns and Filters

At each tab in the Inventory screen, the selection of columns is identical to the selection of fields. The difference is in how the information is selected and applied: Fields will affect the presentation of search results at the Inventory screen, whereas filters affect what content is returned by a search. After they are selected, fields in "Select Columns" can optionally be configured (using Sort options) to order the search results in different ways. By contrast, the fields presented in the "Select Columns" button are designed to be filled in by the user to narrow the search. Many of the numeric filter options are divided into entry boxes for both a minimum value and a maximum value, to specify a range of values.

Select Columns

>>>>>>> master

The Select Columns button at the Inventory screen allows you to configure the presentation of search results. Checkboxes that are marked are those that will appear on the results screen.

Open the Select Columns Button

To open the dropdown for selecting columns, activate the Select Columns button at the top of the Asset Search screen under any Inventory page.

Note: In the Select Columns dropdown, you are not limited to one option; you can mark as many checkboxes as you wish before activating the Update button.

Change Column Selection

>>>>>>> master

If you want to clear all the columns and start from scratch,

  1. Activate the Select Columns button.
  2. Mark the desired checkboxes.
  3. Activate the Update button.

Alternatively, you can use the following procedure for changing column selection:

  1. If there are columns you want to see in the search results that are not already marked, mark the checkboxes for these.
  2. If any fields you don't care to see in the search results are marked, unmark the checkboxes for these.
  3. Activate the Update button.

WARNING: If you don't activate the Update button, your changes will not be saved.

Note: If you've made no changes since activating the "Select Columns" popup or since last activating Update, the Update button will be grayed out.

Sort Search Results (Optional)

You have the option to sort your search results by selected filters. Selecting sort filters and sort direction is done by toggling the - / ↑ / ↓ button located to the left of the Filter button.

Sort Direction

>>>>>>> master

Choose just one of the following for each selected field:

  • the minus sign (no sort)
  • down arrow (sort order from high to low, or Z to A)
  • up arrow (sort order from low to high, or A to Z)

Note: Only those fields with marked checkboxes will offer the option to sort.

Note: If you've made no changes since activating the "Fields" dropdown or since last activating Apply Fields, the Apply Fields button will be grayed out.

Filters

>>>>>>> master

The Filter buttons by each column on each Inventory screen allow you to set parameters for the contents of your search results.

The selection of Filters is the same as the selection of Fields for each column. In the case of Filters, however, you have the option to restrict search results to a particular value or range of values for one or more parameters.

Open the Filter Popup

To open the dropdown for selecting filters, activate the Filter button next to any column for which you want to see data.

Set Filters

To set filters,

  1. Enter the value into the search box or Select Items in the dropdown. If both Min and Max fields are displayed for that parameter, you can enter the same number in the Min and Max fields in order to specify the single value you are searching for.
  2. For a parameter you want to restrict to a range of values, enter the lowest acceptable value in the Min field, and the highest acceptable value in the Max field.

Note: For any field that you leave blank, all values of that field will be included in the search.

Clear Filters

To clear any filter, delete the values in its field, and/or activate the X button.

Close Filters Dropdown

To close the Filters dropdown, activate the Filter button again.
