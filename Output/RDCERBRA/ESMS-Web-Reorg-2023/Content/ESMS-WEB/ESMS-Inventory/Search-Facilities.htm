Search

To perform a search at the Facilities tab,

  1. If not already selected, select one or more domains.
  2. If desired and if not already selected, select a single site to search in.
  3. Activate the Facilities tab if necessary. (The Facilities tab is the default tab for the Search screen, so it should already be selected.)
  4. Configure what data fields you will want to see in the results. See Set Up Columns and Filters for instructions.
  5. To narrow the scope of the search by properties such as status, year constructed, CI Condition Index, FCI Facility Condition Index., FI Functionality Index., PI Performance Index. The Performance Index is a weighted combination of the CI value and the FI value, with the lower of the two values being given two-thirds of the weight and the higher of the two values being given one-third of the weight. If no functional assessment has been performed, the FI is assigned a value of 100., or MDI Mission Dependency Index, follow instructions for setting up filters in Set Up Columns and Filters.
  6. Finally, enter a search term in the search field on the main menu bar and press Enter.

Export Results

You can export your results to CSV by activating the Download to CSV button at the top of the screen:

The CSV file can be opened with Excel. It will be exported to your standard downloads location, and the file name will incorporate the download date.

Details

Click anywhere in a row of data in the results table to see more details about that item. For more information, see the topic Facility Details.
