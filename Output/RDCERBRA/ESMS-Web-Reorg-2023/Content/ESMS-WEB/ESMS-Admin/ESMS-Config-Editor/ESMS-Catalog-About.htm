Understanding the Configuration Manager (Catalog)

This topic lists tasks you can perform at the configuration manager, and details tasks associated with creating (1) custom component types and (2) custom group types. Configuration manager tasks associated with templates for assessment questionnaires and other types of questionnaires are introduced starting at Understanding Questionnaire Templates.

Note: The Configuration Manager screen is viewable by ESMS administrators only; the Catalog tile will not appear on the Dashboard of a non-administrator.

What Can I Do Here?

1. At this screen, you can add custom component and group types to the ESMS catalog.

2. You also have the option here to design templates used to create assessment questionnaires: these can be for the new types you create, or for existing component and group types.

Permissions Note: These tasks require ESMS admin privilege.

Custom Component Type

Navigation

To work with custom component types in the configuration manager, make sure that the Component Types tab is activated.

In the left pane, a list of existing component types will appear. You can browse the list to select one, or use the search box at the top of the list. In the right pane, the tabs General, Custom Attributes, and Ageism Mapping are available.

Add Component Type

To create a new component type at the Component Types tab,

  1. Activate the Component Types tab on the Configuration Manager screen.
  2. Activate the "+" button in the footer toolbar. The "Create component type" popup will appear.
  3. At the General tab, fill in the Name field.
  4. Specify whether the component is a Root Type (if yes, mark the checkbox) and/ or a System Type (if yes, mark the checkbox).
  5. Select domain from the dropdown at the Domain field.
  6. Select or add Parent Components if applicable.
  7. Select applicable Subcomponents. See the next documentation header (Add Subcomponents) for instructions.
  8. Select attributes for the new component type at the Custom Attributes tab. See Add Properties/Attributes for instructions.
  9. When finished, activate the Create button, located at the lower right in the popup. Alternatively, activate Cancel to exit the popup without creating a new component type.

Add Subcomponents

To add subcomponents to a custom component type,

  1. Make sure you are at the General tab for the desired component type. If it is an existing type, you can open the desired component type in the listing at the left, and the General tab should be selected by default in the right-hand pane. If you are still in the process of creating a new component type, make sure you are at the General tab in the "Create component type" popup.
  2. In the Subcomponents section at the General tab, activate the Add More button. The "Select Subcomponent to Add" popup will appear.
  3. In the popup, select one or more subcomponents to be added. You can use the Shift or Ctrl keys to select multiple subcomponents at one time.
  4. Activate Select. Alternatively, activate Close to close the popup without adding a component. You will be returned to the General tab.
  5. You can repeat the previous three steps if needed.
  6. When finished adding subcomponents, use the Close button to return to the General tab, if not already there.

Add Attributes

Assign attributes to a custom component type or custom group type at the Custom Attributes tab.

To create a new, user-defined attribute,

  1. Make sure you are at the Custom Attributes tab for the desired component or group type. If it is an existing type, you can open the desired type in the listing at the left, and then select the Custom Attributes tab. If you are still in the process of creating a new component or group type, make sure you are at the Custom Attributes tab in the "Create component type" or "Create group type" popup.

  2. In the Attributes toolbar, you can add a root attribute (activate the "+" button) or an attribute category (activate the folder button). Once you have created a category, you can add attributes to it.
  3. To name the new attribute or category, activate it in the listing, then fill in the name in the Attribute Name or Category Name field that appears below.

    Note: The active attribute/category will appear on a blue bar, instead of pink.

  4. For a category, type a description in the Category Description field.
  5. For an attribute, select a type from the dropdown in the Attribute Type field. If any additional fields appear as a result of the attribute type selected (for example, Length field for String type), fill them in.
  6. Repeat the steps above as desired.
  7. When finished, activate the Save button, or else return to the last step of creating a new component or group type. Alternatively, activate Cancel to not save or to exit the popup without creating it.

Custom Group Type

Navigation

To work with custom group types in the configuration manager, make sure that the Group Types tab is activated.

In the left pane, a list of existing group types will appear. You can browse the list to select one, or use the search box at the top of the list.

Add Group Type

To create a new group type at the Group Types tab,

  1. Activate the Group Types tab on the Configuration Manager screen.
  2. Activate the "+" button in the footer toolbar. The "Create group type" popup will appear.
  3. At the General tab, fill in the Name field.
  4. Specify whether the group is a system type (if yes, mark the checkbox).
  5. Select domain from the dropdown at the Domain field.
  6. Create custom attributes at the Custom Attributes tab. See for instructions.
  7. When finished, activate the Create button. Alternatively, activate Cancel to exit the popup without creating a new component type.

Troubleshooting tip: If required fields have not been filled in, the Create button will be grayed out.
