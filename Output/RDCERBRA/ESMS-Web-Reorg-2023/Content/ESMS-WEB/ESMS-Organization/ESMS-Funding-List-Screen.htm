Funding List Screen

This screen displays the list of available Funding for the selected Organization to view and/or edit funding. The list is sorted in ascending order by funding precedence. It is accessible from the Funding tab of the Organization Details Screen to users with Funding privileges.

The screen includes the following actions:

  • Adding New Funding
  • Edit Funding Details
  • Delete Funding
  • Selecting the icon allows the user to change funding precedence by dragging and dropping the funding row.
