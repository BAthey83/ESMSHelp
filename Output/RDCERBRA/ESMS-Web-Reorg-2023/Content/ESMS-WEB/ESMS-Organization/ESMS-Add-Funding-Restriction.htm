Add Funding Restriction

The button on the Funding Restriction List screen brings up the Add Funding Restriction dialog. When the Site or Domain option is selected, Funding availability will be restricted to a particular Site or Domain.

Add Funding Restriction Dialog

The dialog includes the following input options:

  • Category: This drop-down allows to select "Site" or "Domain" to apply funding restriction.
  • Property: This field is not used.
  • Condition: This drop-down allows to select "equal to" or "not equal to" relational operator to apply funding restriction.
