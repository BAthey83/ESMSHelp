Old-Understanding the Import Screen

This topic introduces tasks you can perform at the Import screen and provides links to greater detail for each.

What Can I Do Here?

At the Import screen you can import data contained in an appropriately formatted text file (specifications will be listed in the topic Old-Check Format of Import File) into Enterprise SMS Web.

Search Tasks

Tasks you can perform at the Import screen are:

  • Check format of import file – it must be a text file that (a) contains only comma-delimited data without a header row and (b) is named with the schema and table name
  • Navigate to the import file
  • Activate the import.

Navigating the Import Screen

The "Import Data" Popup

When you activate the Import tile at the Enterprise SMS dashboard, the "Import Data" popup will appear:

 

The components of the "Import Data" popup are

  • CSV Bar (Non-Active) – this is not an active button, simply a reminder that the data to be imported must be in CSV format, even though the file itself doesn't need a .csv extension
  • Choose File button – activate this button to browse to the location of the text file to be imported
