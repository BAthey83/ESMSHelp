Understanding the Enterprise SMS Menu

This topic introduces the purpose of the Enterprise SMS Menu and provides a brief overview of each of the task areas.

What Can I Do Here?

  • The menu serves as the central hub connecting all of the task areas in Enterprise SMS Web, so you can transfer from one task area to another quickly and easily.
  • You can initiate a search from the menu without having to navigate to the Inventory screen first. (Before searching, you must select one or more domains and, if desired, select a site.)
  • Via the user dropdown menu in the upper right of the screen, you can select from the following options: (a) announcements, (b) user preferences, (c) save default workspace, (d) download the application, (e) application sync settings, (f) clear the application cache, (g) about, (h), SMS support, or (i) log out.

Note: If you don't have permission to administer a team, the Administration, Users, Teams, and Permissions icons won't appear on your menu.

Overview of Task Areas

Each task area in Enterprise SMS Web is represented by an icon on the main menu. Below is a brief description of each of the task areas.

Inventory

At the Inventory button, you can search for organizations, facilities, groups, and components that meet your search criteria. The Inventory process is driven by:

  • filters (selected in a context-sensitive popup)
  • your organization search (typed in the search field on the organization screen).

Results will show dynamically in the content area of the page.

Starting from the Inventory button, you can navigate to pages showing detailed information about assessment plans, work items, individual assets, FCIs Facility Condition Index. by domain, associated sites, performance records, associated groups, cost modifiers, properties, and inspection schedules.

For more detail about the Inventory screen, see Navigating the Asset Search Screen and Understanding the Inventory Screen.

Organizations

At the Organizations screen, you can:

  • search organizations
  • view, add, or edit funding
  • define multiple funding sources
  • create funding restrictions
  • review FCI Facility Condition Index. data by domain
  • find associated sites
  • check performance records

Sites

At the Sites screen, you can:

  • add a site
  • edit site inventory data
  • generate site reports
  • delete a site
  • view site maps
  • view graphs related to site CI Condition Index distribution and system summaries
  • download site data in CSV format

Facilities

At the Facilities screen, you can:

  • search facilities at a given site
  • search facilities at a single domain
  • see the properties of a given facility
  • review domain-specific facility properties
  • create components, assessments and work items
  • upload documents
  • search the cost modifier book
  • review performance records

Groups

At the Groups screen, you can:

  • find group associations
  • search components
  • explore domain-specific properties
  • create and review child groups
  • review assessments
  • create and review work items
  • search the cost modifier book
  • review performance records

Components

At the Components screen, you can:

  • explore custom attributes
  • review properties
  • review assessments
  • create and review work items
  • review the inspection schedule
  • look for child components
  • search the cost modifier book
  • review performance records

Assess

At the Assess screen, you can see the following properties:

  • date of assessment
  • date updated
  • site name
  • assessor name
  • result
  • completion percentage

Assessments

At the Assessments screen, you can see the following properties:

  • assessment name
  • component name
  • facility name
  • date of assessment
  • assessment type
  • assessment cost

Assessment Plans

At the Assessment Plans screen, you can:

  • create an assessment
  • review and edit assessment details
  • select and review assets
  • select groups
  • add and delete assessment team members

Work

At the Work screen, you can perform work analyses, as well as create and update existing work plans. This screen also allows you to search and update projects, create and update workflow schemes and review tasks related to your work plans.

Work Analysis

At the Work Analysis screen, you can:

  • choose, add, or delete a policy
  • duplicate or edit selected policies
  • publish policies
  • add or delete policy standards
  • add, duplicate, or delete prioritization schemes
  • input prioritization scheme data
  • draft and publish prioritization schemes
  • select, add, duplicate, or delete a profile
  • configure a work plan
  • review your queue
  • compare reports

Work Items

At the Work Items screen, you can:

  • create, search, and edit work items
  • show or hide maps
  • view related graphs
  • customize category columns for your work items

Projects

At the Projects screen, you can:

  • filter and view a list of available projects
  • export the filtered list of projects
  • add projects
  • view and edit project details

The Projects screen is accessible from the main menu to users with Project privileges.

Workflows

At the Workflows screen, you can:

  • view, add, or edit workflow schemes to be used with assessments
  • manage scheme parameters
  • view scheme details

Permissions Note: Only users with "WorkflowRole" permission set at an ESMS Administrator's Security screen can access the Workflow editor screen.

Tasks

At the Tasks screen, you can:

  • generate work items
  • specify threshold levels for CI Condition Index and RSL Remaining service life.

Catalog

At the Catalog screen, you can:

  • manage the inventory catalog
  • create assessment templates
  • search assessment templates for components and assets
  • create and update PowerBi reports
  • create and delete ArcGis endpoints
  • create and configure mapping sets
  • import or export configurations

Inventory Catalog

At the Inventory Catalog screen, you can:

  • create and search component types, attributes, and references
  • create and search group types, and custom attributes
  • search component specifications

Assessment Templates

At the Assessment Templates screen, you can:

  • create and search assessment templates
  • create and search category templates

PowerBI Reports

At the PowerBI Reports screen, you can:

  • create and update PowerBI reports
  • delete PowerBI reports

Import/Export Configurations

At the Import/Export Configurations screen, you can:

  • create and delete ArcGIS endpoints
  • create and configure mapping sets
  • import or export configurations

Reference Books

At the Reference Books screen, you can:

  • select or create new CII Component Importance Index books
  • select or create new service life books
  • select or create new inflation books
  • add, edit, duplicate, or delete cost books
  • add, edit, or delete cost book modifiers
  • add, edit, delete, or duplicate work action books

Component Importance

At the Component Importance screen, you can:

  • create a new CII Component Importance Index book
  • edit or delete existing CII Component Importance Index books
  • edit an existing CII Component Importance Index book name

Service Life

At the Service Life screen, you can:

  • create a new service life book
  • edit or delete existing service life books
  • assign a service life book to an organization
  • copy existing service life data to another service life book
  • edit an existing service life book name

Inflation

At the Inflation screen, you can:

  • create a new inflation book
  • edit or delete existing inflation books
  • assign an inflation book to an organization
  • copy existing inflation book data to another inflation book
  • edit an existing service life book name

Cost

At the Cost screen, you can:

  • create a new cost book
  • edit or delete existing cost books
  • assign a cost book to an organization
  • copy existing cost book data to another cost book
  • edit an existing cost book name

Cost Modifiers

At the Cost Modifiers screen, you can:

  • create a new cost modifier book
  • edit or delete existing cost modifier books
  • assign a cost modifier book to an organization
  • copy existing cost modifier book data to another cost modifier book
  • edit an existing cost modifier book name

Work Actions

At the Work Actions screen, you can:

  • select a work activity
  • edit work actions
  • filter work action data for each work activity

Administration

Note: the Administration screen is only viewable by ESMS administrators. It will not appear for users who are not administrators.

At the Administration screen, you can:

  • manage users and roles
  • create and delete teams
  • add and delete permissions and roles
  • search and configure sites
  • search and configure organizations

Users

At the Users screen, you can:

  • view user details
  • add a user
  • edit user details
  • delete a user
  • assign a role to a user
  • create a role or copy an existing role

Teams

At the Teams tab on the Teams screen, you can:

  • add and delete teams
  • add and delete team members
  • assign roles (security permissions) to teams
  • rename teams

Permissions

At the Permissions screen, you can:

  • add a security permissions role
  • delete a security permissions role
  • edit a security permissions role
  • change a role's name

Permissions Note: These tasks require permission to administer a team.

Import/Export

At the Import/Export screen, you can:

  • import Organization configurations
  • import or export ArcGIS and CSV data from external sources

Note: the import/export screens are only visible to users with administrator privileges.

Generalized Imports

At the Generalized Imports screen, you can:

  • select and view an Organization's import configurations

Import

At the Import screen, you can:

  • open ArcGIS import requests
  • request ArcGIS imports

Export

At the Export screen, you can:

  • open CSV export requests
  • request CSV exports

Documents

At the Documents screen, you can:

  • search for a document
  • search for documents with a particular tag
  • view a document
  • download a document

Reports

At the Reports screen, you can:

  • (Optional) narrow the list of available reports to those tailored for just one of the Enterprise SMS domains
  • select a report to be generated
  • view the report

Map Search

At the Map Search screen, you can:

  • select a site map
  • choose between grayscale, satellite view or street view
  • select the option to add CI Condition Index indicators, separated by color
  • (Optional) color pavement components based on PCI or CI Condition Index

Note: Some icons such as Catalog will only appear if you have ESMS administrator-level permissions.
