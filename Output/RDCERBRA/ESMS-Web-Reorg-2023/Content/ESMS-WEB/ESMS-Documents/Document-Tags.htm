Tag Documents

This topic explains how to:

  • associate a search tag with a document stored in Enterprise SMS ("add tag association")
  • delete the tag association

For how to add a tag to or delete a tag from the pool of available tags, see the topic Manage Search Tags.

About Tagging

To help Enterprise SMS users locate a document, you can associate one or more search tags with it at the Tag Editor button on the Documents screen.

Note: You can create additional tags as needed at the Tag Editor button on the Documents screen. Instructions, as well as tips for selecting tag names, are at the topic Manage Search Tags.

When a single document is selected for tagging, an already-selected tag will appear as a rounded button. When multiple documents are selected for tagging, all available tags will appear as rounded buttons.

Associate Search Tag(s) with a Document

Single Document - Associate Tag(s)

To add tag(s) to just one document,

  1. Make sure the desired document is showing on the Documents screen. (Use View Documents as needed).
  2. Activate the Tag Editor button if that section isn't already open.
  3. Checkmark the desired document. The headers "Associations" and "Tags" will appear in the Document Associations and Tags Editoron the right.
  4. If an appropriate rounded tag button appears, activate it, but do NOT activate the "x" on the button. (The "x" will cause the button to disappear, and you will have to use the alternative method described below.)
  5. Select the desired tag from the available options.
  6. Select Close.
  7. Refresh the browser to see the update. (In Windows, Ctrl-R will refresh).

*If you don't see a tag you want to use, you can Manage Search Tags using the + Tag button.

Note: Your choice of tags at the Document Associations and Tags Editor will be limited to those tags already existing at the Tag Editor section. To expand the pool of search tags to choose from, open the Document Associations and Tags Editor section and see the instructions.

Multiple Documents - Associate Tag(s)

To add tag(s) to multiple documents,

  1. Make sure the desired documents are showing on the Documents screen. (Use View Documents as needed.)
  2. Mark the checkbox for each desired document.
  3. Activate the Tags tab if that section isn't already open. A list of available tags should appear under the heading "Add Tags to Selected Documents".*
  4. Activate the tag(s) you want added to the selected document but do NOT activate the "X" on the button(s). (However, if this happens, don't worry that the tag disappears. It will reappear after a browser refresh.)
  5. Refresh the browser to see the update (In Windows, Ctrl-R will refresh).

*If you don't see a tag you want to use, you can Manage Search Tags using the Tag Editor button.

Delete Search Tag Association(s)

Single Document - Delete Tag Association(s)

To delete search tag(s) from a single document,

  1. In the list of documents, mark the checkbox for the desired document.
  2. Activate the Tag Editor button if that section isn't already open.
  3. Activate the "X" on each tag you want deleted from the selected document. (The "X" will enlarge when you hover over it.)
  4. To see the results, refresh the browser (In Windows, Ctrl-R will refresh).

Multiple Documents - Delete Tag(s)

Caution: Currently, the tag(s) will be deleted from only one of the selected documents.

To delete search tag(s) from multiple documents,

  1. In the list of documents, mark the checkbox for each desired document.
  2. Activate the Document Associations and Tags Editor if that section isn't already open. A list of all tags should appear.
  3. Activate the "X" portion of each tag you want deleted from the selected documents. (The "X" will enlarge when you hover over it.)
  4. To see the results, refresh the browser (In Windows, Ctrl-R will refresh).
