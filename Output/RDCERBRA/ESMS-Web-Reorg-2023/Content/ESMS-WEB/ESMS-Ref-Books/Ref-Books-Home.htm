Reference Books

This topic introduces users to the Reference Books found within Enterprise SMS, providing detailed descriptions for each Reference Book type.

What Can I Do Here?

Reference Books allow Enterprise SMS users to calculate current and future replacement values, estimate a building's Remaining Service Life (RSL Remaining service life.), and apply higher or lower importance rankings to selected Components.

Component Importance Index

The Component Importance Index (CII) reference book is used to specify the importance level for individual Components.

Service Life

The Service Life reference book specifies the expected service life for Components. This data, as well as data from inspections, is used to calculate the Remaining Service Life and produce deterioration curves.

Inflation

The Inflation reference book specifies yearly inflation rates into the future. Enterprise SMS uses these rates to calculate future replacement costs.

Cost

The Cost reference book specifies the replacement costs for Components and Assets

Cost Modifiers

The Cost Modifiers reference book specifies the multipliers or adders that can be applied at multiple Enterprise SMS levels in order to adjust Section replacement costs. Modified costs will roll up through the different levels, and will also affect CI Condition Index roll-up weights, work plan item costs, and costs output by Scenario simulations.

Work Actions

The Work Actions reference book specifies the costs required to carry a work item through to completion. Work Action costs are calculated using the Component Importance Index, the work item's service life, the Inflation reference book, the Components cost reference book and cost modifiers.
