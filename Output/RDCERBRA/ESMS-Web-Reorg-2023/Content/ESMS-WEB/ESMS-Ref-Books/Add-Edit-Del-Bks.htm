Add, View, Edit, or Delete Books

This topic details how to add, view, edit, or delete a Book in Reference Books.

Note: These processes require Administrative permissions. Consult your system administrator for details.

Before you edit or delete a Book:

  1. Choose an Organization from the Current Organization button.
  2. Select the Reference Books dropdown, followed by the button for your chosen book type. In this example, we will be referring to Cost Books.
  3. Pick a book from the subsequent Search window.
  4. The book's window will display, with the name of the book at the top. You can now proceed with editing or deleting books.

Add a Book

To add a book,

  1. Click the "+" icon that appears above the search bar. The Create Book window will appear.


  1. Complete the fields in the window as directed in the View and Edit a Book topic.
  2. Once you have named your book in the Book Name column, select a Source, which identifies the source of the cost information.
  3. If you wish to import existing book data to your new book, select a file from the Copy Existing Book dropdown menu.

    4. Note: Leave the Copy Existing Book field blank if you wish to create a custom book from scratch.

  4. Click Create to add your new custom book, or select Cancel to stop the process.

View and Edit a Book

To view a book,


  1. Open a book from the Search window.
  2. Your book data will appear in the above window.

To edit a book,

  1. Click the "pen-and-paper" icon on the far right, as highlighted above.
  2. Complete the Name and Source information, as shown above. Select Update to save your information, or click Cancel to delete your changes.

Delete a Book

To delete a book,

  1. Click on the book you want to delete in the Search menu.
  1. Select the trashcan icon just above the Search menu.
  1.  The Confirm Action popup will appear. Click the Confirm button to delete your book. Alternatively, select Cancel to keep your book.
