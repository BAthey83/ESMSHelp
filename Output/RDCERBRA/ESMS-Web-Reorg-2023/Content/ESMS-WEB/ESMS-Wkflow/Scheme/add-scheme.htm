Add a Scheme

To add a scheme,

  1. On the Scheme menu bar, activate the New Scheme button. The "Create New Scheme" popup will appear.
  2. Enter desired scheme name in the Scheme Name field.
  3. Activate OK. (Alternatively, activate the Close button to not add the scheme.)

As soon as you have created a new scheme, you should see a left-hand column with some boxes, and a wider blank area on the right where you will place elements into a work flow diagram. The elements of a diagram are states and transitions, as defined below:

A scheme is comprised of states and transitions, and is presented as a diagram. The states appear as boxes in the diagram; the transitions are represented by lines connecting the boxes. Each state and each transition should be named.

A state represents the condition that the process is in at a specific time.

A transition defines how the process will move from one state to the next.

Add a State to the Diagram

To add a state to the diagram, drag one of the state boxes from the left-hand pane into the main content pane:

  • The green box represents the state that is the first state in the work flow.
  • The white box represents a middle state.
  • The $$ box represents the state that is the last state in the work flow.

Add a Transition to the Diagram

Once you have added at least two state boxes to the diagram area, you can drag from one state box to the next one in the work flow sequence to create a transition, which will appear as a connecting line.

State and Transition Details

For details about the attributes of the states and transitions, see the topics Edit Scheme States and Edit Scheme Transitions.
