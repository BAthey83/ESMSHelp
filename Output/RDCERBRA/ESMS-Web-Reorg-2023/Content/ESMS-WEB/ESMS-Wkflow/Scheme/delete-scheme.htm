Delete a Scheme

Permissions Note: Deleting a scheme typically requires elevated permissions.

To delete a scheme,

  1. On the Scheme menu bar, activate the Delete Scheme button.
  2. From the dropdown list, select the scheme you wish to delete.
  3. Confirm the deletion in the popup box.

About Workflow Stages

If you enable Workflow with an assessment, you can design your own stages for the workflow to pass through. ESMS also provides a default workflow that provides the following stages:

  • Charter Stage
  • Pre-Assessment Stage
  • Review Stage
  • Desktop Sync / Assessment Stage
  • Post-Assessment Stage
  • Packages Stage
