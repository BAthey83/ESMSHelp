Getting Started with Workflow

ESMS Workflow allows you to design and/or navigate specific steps in creating an assessment. For more detail about the Workflow / assessment profile stages, see About Designing and Using a Work Flow.

To get started, activate the Assessments tile on the ESMS Dashboard. A table will appear, containing any already-existing assessments that you have permission for. You can either open an existing assessment, or create a new one.

Note: You will see in the table only those assessments you have permission for.

Option A: Open a Work Profile

To work with an existing work profile, activate its line in the table. You will be brought to the Charter stage for that assessment.

Option B: Add an Assessment

To enter pertinent data for a assessment,

  1. Activate the Create Assessment button, then fill out the fields described below.

  2. Site. To initiate selecting a site, first activate the link icon () at the left side of the field.

    Note: Only one site may be selected for an Assessment.

  3. At the "Organization Selectors" popup, select one or more organizations to narrow the search field for sites.
  4. Activate Next.
  5. At the "Site Selectors" popup, use the search box and Find button to select a site.
  6. Activate Next. Alternatively, you can return to Organization Selectors using the Prev button, or cancel by activating the Close button.
  7. Domain. Select a domain from the dropdown list.
  8. Budget. Enter the budget amount for the assessment here.
  9. Name. Specify a name for the assessment.
  10. Charter. Enter the text of the assessment's charter here.
  11. Planned Assessment Date. Either fill in the planned assessment date here, or navigate into the field box and select the larger down arrow to bring up a calendar tool. To clear the date, navigate into the field box and activate the "x"

    Note: the "x" will not appear if no date value has been entered or selected.

  12. To continue, in order to select Scheme, Pre-Assessment Template, or Assessment Template, you must first mark the Enabled checkbox. (Shaded fields will become white.)
  13. Scheme. Select a scheme from the dropdown.
  14. Pre-Assessment Template. Select a pre-assessment template from the dropdown.
  15. Assessment Template. Select a assessment template from the dropdown.
  16. Permissions. Before you can set permissions, all the previous fields must be filled in. See the following list of instructions for how to set permissions.
  17. When you have finished specifying permissions and closed the "Set Permissions" popup box, activate the Create button to create the assessment you have specified.

To select users that will be associated with the assessment, set permissions for the assessment as follows.

  1. Activate the Permissions button at the bottom of the "Create Assessment" popup. The "Set Permissions" popup should appear.

    Note: Before you can set permissions, all the previous fields must be filled in.

  2. In the "Set Permissions" popup, activate the Add More button in a category you want to add a user to. A popup will appear. You use the page ladder at the bottom of the popup to step through the list of users. Alternatively, you can use the Name search field and the Go button to narrow down the selection list.
  3. When you find a user you want to add to that category, activate the Select button. You will be returned to the "Set Permissions" popup, and you may repeat the previous step and this one as many times as needed.
  4. When you have designated a permissions level for all desired users, activate the close ($$Done) button in the "Set Permissions" popup. The popup will close.

    IMPORTANT: All permissions must be set at this stage before saving.

  5. Next, be sure to save the assessment (or cancel, if desired) as specified in the next set of instructions.

To save the assessment,

  1. When you have finished specifying permissions and closed the "Set Permissions" popup box, activate the Create button to create the assessment you have specified.
  2. Alternatively, activate close/Cancel to clear all of the information you have entered and cancel the operation.

Navigating to and in an Assessment Profile

Navigate to anAssessment Profile

  1. To start using Workflow, activate the Assessments tile at the ESMS Dashboard.
  2. Select an assessment from the table. You will be brought to the Charter stage for the assessment.
  3. Alternatively, to create a new assessment, see the instructions at Option B: Add an Assessment.
  4. Once you are in one of the six stages of the assessment profile, you can use the browser's back arrow to return to the initial "Assessments" screen if desired.

Navigate in an Assessment Profile

Once in an assessment profile, you can activate the Workflow navigation tool. On a wide desktop screen, this is activated via a flyout tab at the upper left of the screen.

On a narrower screen, activate the navy blue bar with the word "Workflow" appearing in white on a dark navy bar, below the main menu.

After you activate, the Workflow navigation tool will expand as shown below:

Note: Depending on your permissions, you might not see the full selection of navigation buttons shown here.

Using the expanded navigation tool, you can jump directly to a tab in one of the six Workflow stages. Scroll down as needed if not all of the stages are visible.

Next Step

After the assessment has been set up, you use the Workflow navigation tool to navigate to the Details tab of the Charter Stage.
