Edit Assessment

To edit an existing assessment,

  1. Navigate to the table of assessments by one of these means: (a) from the Dashboard, activate the Assessments tile; or (b) from within a different assessment, activate the browser's back arrow button until you reach the Assessments screen with the table of assessments.
  2. In the table of assessments, activate the line for the desired assessment. You will be brought to the Charter stage for that assessment.

Assessment level data such as the assessment name, budget, etc. can be edited at the Details tab in the Charter stage. For explanations of the assessment data fields, see Add Assessment.

Use the Assessments button/flyout menu at the upper left to navigate to other tabs and stages. (For instructions, see Assessment Navigation.)

More information about each stage provided in the workflow packaged with ESMS can be found here:

  • Charter Stage
  • Pre-Assessment Stage
  • Review Stage
  • Desktop Sync / Assessment Stage
  • Post-Assessment Stage
  • Packages Stage

To edit the underlying workflow states and transitions, refer to Understanding the Workflow Editor .
