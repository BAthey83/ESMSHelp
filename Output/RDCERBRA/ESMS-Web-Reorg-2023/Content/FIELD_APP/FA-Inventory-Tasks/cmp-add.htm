Add Component

To add a component,

  1. Select the Assets tab above the ribbon.
  2. In the navigation pane (open it if it is auto-hidden), activate the Systems tab.
  3. In the asset tree, select an asset on the first or second level below the site level. The new component will be created below the level you have selected, either as a component of that asset, or as a subcomponent if you have selected a component.
  4. Activate Add Component in the ribbon (see image below). The "Add Component" popup will appear.

  5. Enter the requested information in the fields. More detail is given in the topic Component Data.

For more information about accessing and configuring the navigation pane and content pane, see Manipulating Navigation and Content Panes.
