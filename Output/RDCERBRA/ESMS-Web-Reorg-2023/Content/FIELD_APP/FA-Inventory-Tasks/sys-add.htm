Add System

Note: When you add a system you will need to rebalance the Importance Factor percentages for the list of systems at the Group Editor tab.

CAUTION: Adding a system is a high-impact operation that should be carefully planned in context of design decisions concerning how real property assets are to be organized.

To add a system,

  1. At the Tools tab, activate Group Editor in the ribbon. A Group Editor tab will open in the content pane.
  2. In the list of group types (located at the left side of the content pane), activate "System". A listing of all systems will appear to the right, along with the importance factor percentage for each.

  3. In the blank field above the list of systems, type a name for the new system.

  4. Activate the Add Group button directly right of the field where you entered the new system name. The newly created system will appear at the bottom of the list of systems.
  5. In the column labeled "Importance Factor (%)" type a number for the importance factor of the new system. This number is a percentage, where the total of all the numbers in the "Importance Factor (%)" column should add to 100.
  6. Adjust one or more numbers in other rows downwards to compensate for the percentage number inserted for the new system. This step is not finished until the total of all the numbers in the "Importance Factor (%)" column add to exactly 100.
  7. IMPORTANT: Before exiting, activate the Save Systems button.
