    Screen Layout

    ========

    Screen Layout

    >>>>>>>> master:Output/RDCERBRA/ESMS-Web-Reorg-2023/Content/FIELD_APP/0First-Steps/nav-esms-desk.htm

    The layout of each Enterprise SMS Desktop page contains the following areas:

    1. Top Tabs. The tabs at the top of the screen—top tabs—will bring you to the different areas of Enterprise SMS.
    2. Ribbon (Main Toolbar). Directly below the top tabs is the ribbon that shows the actions available at the current page.
    3. Navigation Pane. In the default screen layout, you will find the navigation pane on the left below the ribbon, with the content pane to its right.

      Productivity Tip: You can change your screen to display the content pane only, with the navigation pane available on demand, as described in Manipulating Navigation and Content Panes.

    4. Content Pane. Below the ribbon you will find the content pane on the right. As you work in Enterprise SMS, appropriate tabs will appear in the content pane as needed.

    Expand/Collapse

    ========

    Expand/Collapse

    >>>>>>>> master:Output/RDCERBRA/ESMS-Web-Reorg-2023/Content/FIELD_APP/0First-Steps/nav-esms-desk.htm

    In the navigation pane toolbar you can use dedicated buttons to expand and collapse the inventory tree.

    Expand or Collapse the Asset Tree in the Navigation Pane
    Expand Activate the green plus button to expand all assets down to the component level.
    Collapse Activate the red minus button to collapse the asset tree to the system level.

    Search

    To search for a term in the asset tree, type it into the navigation pane search box. The asset tree will dynamically show results as you type; no need to click "Go" or a magnifying glass.

    To cancel the search and return to the full asset tree, click in the search box.

    Select Assets

    In the asset tree table, a dark blue highlight bar indicates where mouse focus is.

    A lighter blue highlight bar will indicate which inventory item is currently selected. When the mouse focus is also on the selected item, the highlight bar will be dark blue.

    View

    To view a component,

    1. In the navigation pane, select the Systems tab.
    2. Double-click it in the asset tree on the component you want to view. Component data will appear at the View Component tab in the content pane.

    After the first double-click, only a single click should be needed to view subsequently selected components.

    Note: If you select an asset that not a component, such as a system, the most recently viewed component will remain at the View Component tab.
