Remove Work Item(s) from a Work Plan

Remove One Work Item

To remove one work item from a work plan,

  1. View the work plan as described in View Work Plan.
  2. In the bottom section of the content pane, view the list of "Work Plan Work Items" on the right.
  3. Select the work item you wish to remove from the work plan. (When you select the work item, it will highlight.)

  4. Between the "Component Work Items" column on the left and the "Work Plan Work Items" column on the right, activate the left-facing single arrow.

  5. The selected item will be moved to the "Component Work Items" column on the left.

Remove All Work Items

To remove all listed work items from a work plan,

  1. View the work plan as described in View Work Plan.
  2. In the bottom section of the content pane, view the list of "Work Plan Work Items" on the right.
  3. Between the "Component Work Items" column on the left and the "Work Plan Work Items" column on the right, activate the double left-facing single arrow.

  4. All of the work items will be moved out of the work plan. If you wish to add one or more of the work items back into the plan, see Add Work Item(s) to a Work Plan.
