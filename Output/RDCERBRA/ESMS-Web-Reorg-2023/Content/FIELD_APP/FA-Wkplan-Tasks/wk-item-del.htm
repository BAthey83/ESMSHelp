Delete a Cost Modifier Library

Note: this process requires elevated permissions.

To delete a cost modifier library,

  1. Select the Reference Books dropdown menu.
  2. Click the Cost Modifiers button.
  3. Choose the Cost Modifier library you wish to delete from the Search CM Books window.
  4. Click the trashcan icon to delete the selected cost modifier library.
  5. In the confirmation popup window, activate the Confirm button to finalize the deletion (alternatively, select Close to abandon the deletion).

    Delete Work Item

    To delete a work item,

    1. In the navigation pane, select the component that the work item is associated with.

      Note: You can select the component either at the Systems tab or at the Facilities tab.

    2. At the assessment plans tab, activate Work Item in the ribbon. A Work Item tab will appear in the content pane.
    3. Navigate to the work item in the listing you wish to delete, and hover over the "X" at the right side of that work item entry. The "X" should turn red.

    4. Activate the red "X".
    5. In the "Delete Confirmation" popup, select Yes.
