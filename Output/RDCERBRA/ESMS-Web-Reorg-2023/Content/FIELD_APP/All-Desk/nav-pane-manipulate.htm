Manipulating Navigation and Content Panes

This topic helps you configure your display screen for comfort and efficiency.

Display options discussed are:

  • Auto Hide the navigation pane (take me there)
  • Resize the navigation pane (take me there)

In default configuration, the navigation and content panes are located side by side. The Auto Hide option allows you to layer the panes and easily switch between them.

Auto Hide ("Pin") the Navigation Pane

To make efficient use of limited screen space, you may want to activate Auto Hide to open and close the navigation pane with one click. This allows more width for both the navigation pane and the content pane.

About the Pin icon

The pin icon in the upper right corner of the navigation pane is the icon that toggles Auto Hide on and off. The pin position indicates whether you are in the default configuration (pin is pointing down) or using Auto Hide (pin is pointing left when the navigation pane is visible).

Turn Auto Hide On

To turn Auto Hide on, click the pin. The navigation pane will reduce to a tab (Navigation) at the left side of the screen.

 

In Auto Hide mode, the content pane will fill the screen width. The navigation pane no longer intrudes on the size of the content pane, so you can resize it as desired.

Navigating in Auto Hide Mode

View the navigation pane

Whenever you want to view the navigation pane, position the mouse cursor on the Navigation tab (shown in the screenshot above) to make that pane appear. The navigation pane will continue visible as long as the mouse cursor is in or above that pane.

View the content pane

When you click in the content pane, the navigation pane will shrink back to a tab.

Turn Auto Hide Off

To turn Auto Hide off, click the pin. The navigation and content panes will return to a side-by-side view. The size of the navigation pane will be as you last left it in the standard configuration.

Note: When the navigation pane is in Auto Hide mode, the pin in the navigation pane shows with the point to the left instead of down.

Resize the Navigation Pane

For better viewing, you may want to widen the navigation pane. The procedure for resizing the navigation pane will vary slightly depending on whether you are using Auto Hide or not.

Resize in Standard Mode

To resize the navigation pane in standard mode,

  1. Navigate the mouse cursor to between the navigation pane and the content pane.
  2. Watch for the appearance of the double-headed arrow; at the same time, the pale line between the borders of the navigation pane and the content pane should change to a dark vertical bar.
  3. Click the left mouse button and drag right or left to make the navigation pane larger or smaller.

Resize in Auto Hide Mode

To resize the navigation pane in Auto Hide mode,

  1. Make the navigation pane visible if it is not already (Hover over the Navigation button shown above).
  2. Keeping the mouse cursor in the navigation pane, bring it to the right hand border of the navigation pane.
  3. When the mouse cursor changes to a double headed arrow, click the left mouse button and drag right or left to make the navigation pane larger or smaller.
