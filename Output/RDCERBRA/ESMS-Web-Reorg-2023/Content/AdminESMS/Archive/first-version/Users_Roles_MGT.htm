APPENDIX A: Manage Users and Roles

In most cases users will be permitted to use the Enterprise SMS User Manager to edit their own basic user information, to change their password, or to retrieve a forgotten password. Enterprise SMS Administrators have additional privileges such as registering new users and setting their roles, privileges, and scope.

User Tasks

Before logging in, you have these options:

  • Login
  • Register User (Enterprise SMS Administrator only)
  • Reset/Forgot Password

Login Note: On the default entry page to the Enterprise SMS User Manager utility, the word "Login" appears in two locations: (1) in the navigation list at the left, and (2) as a button below the "Username" and "Password" fields. THESE ARE DIFFERENT. Activating the Login button will evaluate the username and password entered in the fields, whereas "Login" in the navigation list will effectively do nothing at this page (it navigates to the Login page, and you are already there).

After logging in, you have these options:

  • Register User (Enterprise SMS Administrator only)
  • Change Password
  • Logout

Set Up Username, Password, and Email

IMPORTANT: Be aware that there is a one-to-one correspondence between a username and its associated email. To be able to successfully reset a password using email, if you have more than one username you should designate a different email address for each one of them.

Your Enterprise SMS Administrator will either provide you with a username and password or designate some other procedure for obtaining these.

Login

First, to access the User Manager using the browser bar,

  1. Delete "Enterprise SMS" from the end of the pathname to your instance of the Enterprise SMS.
  2. In its place, append the word "usermanager" (no space between the words) after the slash character.

Example: https://app1.sms.erdc.dren.mil/usermanager.

The User Manager login screen is shown by default when the Enterprise SMS User Manager loads.

Next, at the login screen, enter your username and password. Then activate the Login button.

Note: Logging in does not log you into Enterprise SMS. It brings up your basic user information so that you can edit if desired, and allows you access to where you can change your password.

Password

The User Manager lets you (a) reset a forgotten password or (b) change your password.

Reset Forgotten Password

You can reset a forgotten password from the Enterprise SMS User Manager screen without having to log in:

  1. At the login screen, select either "Reset/Forgot Password" from the navigation options on the left, or "Forgot Password?" below the Username and Password fields. The User Manager password reset page will appear.
  2. Enter your email in the "Email" field at the password reset page.
  3. Activate the Submit button.
  4. Check your inbox for an email with your password.

Note: Your organization's security measures may dictate that this feature be inactivated in your version of Enterprise SMS.

Change Password

To change your password for the Enterprise SMS,

  1. You need to first login to the Enterprise SMS User Manager. This will bring you to the Edit User page.
  2. At the Edit User page, do NOT enter any information. Instead, select "Change Password" in the navigation list at the left.

  3. At the Change Password page, enter your existing password in the "Current Password" field.
  4. Enter a new password in the blank provided.
  5. Confirm the password in the next blank.
  6. Activate Change Password.

Note: Your organization may have procedures in place for password change that differ from these instructions.

Edit User Information

To edit user information such as email and phone number,

  1. Login to the Enterprise SMS User Manager. This will bring you to the Edit User page.

  2. Edit your user information as desired.
  3. Activate Save Changes.

Note: Your organization's security measures may dictate that this feature be restricted to the Enterprise SMS Administrator.

Logout

If you are logged into the Enterprise SMS User Manager, selecting "Logout" in the navigation at the left will log you out and return you to the default User Manager login screen.

 

Enterprise SMS Administrator Tasks

By default, only the "Register User" task is reserved for Enterprise SMS Administrators. If your Enterprise SMS installation also restricts other tasks to Administrators only, look in the sections above for how to complete those tasks.

Note: The "Register User" task is still under development.
