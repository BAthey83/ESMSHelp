Add a Custom Cost Book

This topic covers how to add a custom cost book. The topic Edit Cost Book Dataon page 1 explains how to add or edit data in a cost book.

Permissions Note: Adding a custom cost book requires Work Planner permission level or above at the target Site or Organization.

To add a cost book,

  1. Select the Cost button within the Reference Books dropdown menu. (Be sure to select a Site or Organization from the Current Organization menu first).
  2. Click the "+" icon that appears above the "Search Cost Books" search bar. The Create Cost Book window will appear.
  3. Complete the fields in the window as directed in the Edit Cost Book Dataon page 1 topic.
