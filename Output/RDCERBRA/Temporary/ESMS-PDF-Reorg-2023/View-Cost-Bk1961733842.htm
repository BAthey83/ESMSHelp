View a Cost Book

This topic covers how to view the default cost book (Reference) or a custom cost book. The topic Edit Cost Book Dataon page 1 explains how to add or edit data in a cost book.

Permissions Note: Depending on the operation (e.g. adding vs. deleting a book), configuring reference data requires either Work Planner or Master Planner permission for the Organization applied to the reference book. The ability to assign a custom reference book to an Organization depends on your permissions for that Organization.

To view a cost book,

  1. Select an Organization to assign to the Cost Book in the Current Organization menu.
  2. Open the Reference Books drop-down menu.
  3. Select the Cost button.
  4. Open a Cost Book from the Search Cost Books window.
  5. Your Cost Book data will appear. For details on how to edit your Cost Book, see Edit Cost Book Dataon page 1. If you need to add a Cost Book, see the Add a Custom Cost Bookon page 1 topic.
