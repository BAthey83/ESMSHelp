Add Role (Security Permissions)

This topic explains designing and creating a new security permissions role in Enterprise SMS Web. These tasks require that you have permission to administer a team.

Custom Security Permissions Roles

You can design customized permissions roles by selecting from a set of permission options provided by Enterprise SMS. To implement creation of a customized role, you need team leader permission level yourself.

The sequential steps in adding a customized role are to (1) create and name the role; (2) specify the permission design by selecting from a set of permission options provided; and (3) save the new role.

Create Role

To create a role,

  1. At the Role Editor screen, select the Roles tab.
  2. Activate the "+" button in the colored bar at the bottom of the screen.
  3. In the "Create Role" popup, enter a name for the new role in the "Role Name" field.
  4. To confirm, activate the Create button.
  5. Alternatively, activate the Cancel button to cancel adding the role.
