Assign a Cost Book to an Organization

This section discusses how to assign a Cost Book to an Organization, with detailed steps on the process.

You can create multiple cost books, and assign a separate book to any Organization within your permissions scope, as desired. This is done as follows:

  1. Go to Current Organization and select the Organization to assign to the cost book.
  2. Open the Reference Books drop-down menu.
  3. Select the Cost button.
  4. Choose or create a Cost Book for that particular Organization.
  5. Click the Save icon to retain your revisions/updates.
