View a Cost Modifier Library

Note: this process requires elevated permissions.

To view a cost modifier library,

  1. Open the Reference Books dropdown menu.
  2. Select the Cost Modifiers button.
  3. Go to the Search CM Books window and select a Cost Modifier book from the available options.

The following functions are available on the Cost Modifiers page:

  • Add Book. Select the Add Book icon to create a new Cost Modifier book in the library.
  • Delete Book. If the Delete Book icon is activated, it will delete the cost modifier book.
  • Edit Book Name. Select the Edit Book Name icon to modify the Cost Modifier Book Name and/or the Cost Modifier Book Source.

The table of cost modifiers has the following columns:

  • Name. Identifying name assigned to the modifier.
  • Description. This field is provided for descriptive text.
  • Multiplier? Indicates whether a multiplier has been applied.
  • Scope. Shows the scope where the cost modifier applies.
  • Times Assigned. The number of times the modifier has been assigned.
  • Value. For an adder, the value represents a dollar amount. For a multiplier, it is what the cost should be multiplied by.
  • PRV. The Plant Replacement Value. For an inventory item above the Building level, the number will be the aggregate of the PRVPlant Replacement Value. This number will be the aggregate of the PRV of all inventory items contained within the current inventory item. of all Buildings included in that item.
  • Clear. This field lets you clear any or all filters applied to the cost modifier.

Each of the columns allows you to apply a filter on any or all of the columns to potentially reduce the number of cost modifier records displayed on the screen.
