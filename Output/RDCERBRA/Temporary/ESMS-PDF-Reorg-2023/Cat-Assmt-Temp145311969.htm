Assessment Templates

This topic introduces tasks you can perform at the Assessment Templates screen and provides links to greater detail for each.

Assessment Templates Tab

What Can I Do at the Assessment Templates Tab?

At the Assessment Templates tab, you can create different kinds of assessment questions, and organize them using question categories and nesting (nesting and question categories are both optional). How to do this is explained in the Add Questionnaire Templateon page 1 topic. The completed set of questions is called an assessment set template, or template set. The Enterprise SMS program automatically assigns the correct numbering of questions within the template set.

If one or more template sets have been created, you can select an existing one using the menu option Choose Template Set.

When viewing a template set, you may find navigating through it to be more efficient if you activate Collapse All in the menu, to view just the individual questions without details about answers. Next, you can expand individual questions as needed, or activate Expand All to expand all of the questions simultaneously.

Navigating at the Assessment Templates Tab

To activate the Assessment Templates tab menu,

  1. Select the Catalog screen in the main menu, followed by the Assessment Templates button in the dropdown menu.
  2. Activate the Assessment Templates tab. A list of component types will appear in the left pane.
  3. To make the menu appear, activate a component type in the list. You can use the search field (labeled "Search component types here") at the top of the list to help find a desired component type. Once a component type is activated, the menu bar will appear below the Assessment Templates tab:

The menu bar at the Assessment Templates tab allows you to work with template sets and to view and add questions in a template.

Note: The menu will not appear until a component type is selected in the list at the left.

Menu options include:

  • Add. After you select an assessment set, this button allows you to add questions or question categories to the template.
  • Add Assessment Set Template. Activating this button introduces a popup for creating a new template.
  • Expand All. If the content area has been collapsed to show just each question category (where one has been assigned) or question in the content area.
  • Collapse All. Activate Collapse All to see only the question category (where one has been assigned) or question in the content area.
  • Choose Template Set. Activate the dropdown arrow to display a menu of template sets to select.

Assessment Set Templates Tab

At the Assessment Set Templates tab, you can create new set templates, as well as edit and search for existing set templates. You can also delete your assessment set template(s) as needed.

Value Categories Tab

The Value Categories tab lets users add, edit, or delete value categories, as well as add Modifiers for them. Select the plus sign to add a value category, select the pencil icon to edit a value category, or choose the trashcan icon to delete a value category.

Modifiers

The Modifiers page allows you to insert data into the following fields:

  • Name
  • Comment
  • Choice Value
  • Numeric Value
  • Text Value
  • Value Group

If the available Modifiers do not apply to your Value Category, you can add a new modifier by clicking the plus button. To ensure your changes stay in the Value Category, select the floppy disk icon to save your revisions.

Category Templates Tab

The Category Templates tab lets you add a new category type (using the plus sign) or delete an existing category type (using the X button).

The following items can be edited in a Category Template:

  • Domain: Select any of the 22 Domains available in Enterprise SMS.
  • Category: Choose one of the following Assessment Category Templates: Comprehensive, Desk Survey, Direct Rating, Modified, and Validation.
  • Category/Type: Choose one of the following Assessment Category Types: DOD/Design Criteria, Environmental, Functional, Maintenance, Operational, and Regulatory.
  • Scheme: Choose any of the available Schemes to complete your template.
  • Use Operator/Assessor Pre-Assessments: Select the checkbox to include Operator/Assessor Pre-Assessments.
  • Operator Pre-Assessment Template: Choose any of the available Operator Pre-Assessment Templates for your Category Template.
  • Assessor Pre-Assessment Template: Choose any of the available Assessor Pre-Assessment Templates for your Category Template.
  • Assessment Template: Choose any of the available Assessment Templates for your Category Template.
