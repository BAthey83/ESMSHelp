About Cost Books

Enterprise SMS has an embedded cost module, used in the critical areas of work planning and condition assessment, that assigns replacement values to inventory. The data used by the cost module is contained in one or more cost books.

A default cost book is provided and is all that is required. Optionally, a customized inflation book can be used in place of the default, or multiple customized cost books can be associated with different Organizations. This topic explains how to do this.

Permissions Note: Depending on the operation (e.g., adding vs. deleting a book), configuring reference data requires either Work Planner or Master Planner. A user with Master Planner Role has all the rights of a Work Planners, and can also generate multi-year work plan scenarios, and edit any and all data saved by other users of the program. Master Planner is the highest level of user privileges. permission for the Organization that the reference book applies to. The ability to assign a custom reference book to an Organization depends on your permissions for that Organization.

Enterprise SMS's embedded cost module contributes to multiple Enterprise SMS functions:

  • Inventory: Provides inventory replacement costs for all levels of the Enterprise SMS hierarchy.
  • Condition index calculations: Replacement costs are used as weighing factors in rolling up CICondition Indexs in the inventory hierarchy.
  • Work planning: For work planning, the cost module provides initial programming cost estimates for proposed maintenance, repair, and rehabilitation projects.

The cost data used by the cost module is contained in one or more cost books. A given Organization can have many defined cost books. Multiple cost books can also be associated with different Organizations. A single book can be assigned to an Organization for rollup purposes, while the remaining books can be used for the Work Analysis process.

If an Organization does not have a cost book assigned to it, the Organization's assigned cost book is used for rollups. If no book is found to be assigned to an Organization, the process will run the course of the organizational hierarchy until a parent Organization with an assigned book is found.

Cost Data

The cost data used by Enterprise SMS's cost module is contained in one or more cost books. Multiple cost books can be associated with different Organizations as desired (see Assign a Cost Book to an Organizationon page 1).

Default Cost Book

The Default Cost Book establishes default costs for any combination of any Specifications or Work Action Types not overridden in the Cost Book.

Custom Cost Book(s)

You can manage cost data by using custom cost books. To create custom cost data, you can either:

  1. Edit an existing cost book (except for Reference), or
  2. Create one or more new cost books. You can use an existing cost book, as the starting point for the new cost book.

By assigning a custom cost book to an Organization, you can control the scope of that cost book, applying custom values to either all or part of the cost book.

Features and Search Bar

  • Specification: The item selected for editing within the Cost Book.
  • Activity: The Global Repair, Replace or Demolish action required for the specified item.
  • Level: The Building or Component-Section required for the specified item.
  • Parent: The parent Organization for the specified item.
  • Current Cost: The most recent value of the specified item.
  • Original Cost: The earliest value of the specified item.
  • Cost Year: The year used to calculate a specified item's inflation data.
  • UoM Unit of Measure: The Unit of Measurement for the selected item(s).
  • Clear: Select this option to clear all Filters.
