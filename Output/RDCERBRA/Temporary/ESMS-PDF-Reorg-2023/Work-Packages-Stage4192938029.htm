Packages Stage

This topic describes the Packages stage of designing an ESMS assessment plan. This is the last of five stages, to be performed after any necessary approvals have been obtained to progress from the Post-Assessment Stageon page 1.

To complete the Packages stage, assign work items to appropriate packages. You can also have individual work items that are not assigned to any package if that is acceptable under the procedures followed by your organization. A work item can be in zero packages or in one package. The same work item cannot be in more than one package.

Add Work Package

Available work packages are listed to the left of the table of work items. If you have work items in the table that need to be assigned to one or more work packages that do not appear in the list of work packages, you can create one or more new work packages to be added to the list.

To add a work package to the list of work packages at the left,

  1. Activate the Create Work Package icon (represented as a + sign, as shown below) located above the list of work packages.
  2. The "Package Editor" popup will appear.
  3. Select the Work Package tab at the top.
  4. Enter a name for the new work package in the Package Name field.
  5. Fill out all applicable remaining fields.
  6. Activate Save.

Edit Work Package

To edit an existing work package,

  1. Select a package name in the list of work packages.
  2. Click the Edit Work Package button above the list.
  3. Edit fields as desired.
  4. Activate Save.

Assign Work Item to a Package

This section describes how to assign a work item to a package that is listed at the left side of the screen. If you don't see an appropriate package, you can add a new one to the list as described in Add Work Packageon page 1.

To assign a work item to an existing package,

  1. Click the right drop-down menu and select Unassigned.
  2. Select the work item's entry in the table of work entries.
  3. Click the corresponding + sign button under Add/Remove on the right side.

Note: The Work Package field is the only editable field here.

  1. If there are no work packages listed, or if none of the packages listed apply to the selected work item, exit the popup and follow instructions at Add Work Packageon page 1 to add one or more packages to the list of available work packages. Then repeat the steps above.
  2. From the list of available packages, select the one you want to assign the work item to.
  3. Activate the corresponding button under Add/Remove on the right-hand side.
