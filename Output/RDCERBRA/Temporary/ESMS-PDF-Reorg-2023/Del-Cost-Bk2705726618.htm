Delete a Cost Book

This topic explains how to delete a custom cost book. The original Reference book cannot be deleted.

Permissions Note: Deleting a cost book requires Master Planner permission for the Organization or Site applied to the cost book.

To delete a cost book,

  1. Select the Reference Books dropdown menu. Choose the Cost button.
  2. Click on the custom cost book you want to delete in the Search Cost Books menu.
  3. Select the trashcan icon that is just above the Search Cost Books menu.
  4.  The Confirm Action popup will appear. Click the Yes button to delete your custom cost book.
