Edit CII Books

This topic describes how to add or edit information in a custom Component Importance Index (CII) book. If the custom book has not been created yet, see the topic Add a Custom CII Bookon page 1.

Permissions Note: Editing a custom CII book requires Master Planner permission level or above at the target Organization.

Open a Custom CII Book

To open a custom CII book,

  1. Go to the Current Organization button and select an Organization.
  2. Select Reference Books from the Main Menu and choose Component Importance from the subsequent drop-down.
  3. Choose a custom CII book by selecting an option from the Search CI Books search results.

Edit Data

You can edit data for the Component Importance Index reference books when you create a new CII book. For additional information on this topic, see Add a Custom CII Bookon page 1.

Save Changes

When you are finished making changes,

  1. Activate the Save button, located below your CII Book title.
  2. You will see a "Changes Saved" confirmation after your save is successful.
  3. You will then return to the main Component Importance Index page.
