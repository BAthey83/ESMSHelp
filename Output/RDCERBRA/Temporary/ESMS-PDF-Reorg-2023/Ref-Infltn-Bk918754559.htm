About Inflation Books

Enterprise SMS has an embedded inflation module that is used in tandem with the cost module for work planning cost estimates and for inventory. The data used by the inflation module is contained in one or more inflation books.

A default inflation book is provided and is all that is required. Optionally, a customized inflation book can be used in place of the default, or multiple customized inflation books can be created in association with different Organizations. This topic explains how to begin this process.

Permissions Note: Depending on the operation (such as adding vs. deleting a book), configuring reference data requires either Work Planner or Master Planner permission for the Organization that utilizes the reference book. The ability to assign a custom reference book to an Organization depends on your permissions for that Organization.

Inflation Data

The inflation data used by the inflation module is contained in one or more inflation books. Multiple custom inflation books can be associated with different Organizations.

Default Inflation Book

The default inflation book provided with Enterprise SMS is named "Reference." It cannot be edited, and the scope of its data spans the entire site.

You can choose to depend exclusively on the data in Reference, or you can create one or more custom inflation books.

Custom Inflation Book(s)

To create custom inflation data, you can either:

  1. Edit an existing inflation book (except for Reference), or
  2. Create one or more new inflation books. You can use Reference, or another existing inflation book, as the starting point for the new inflation book.

By assigning a custom inflation book to an Organization, you can control scope of that inflation book, applying custom values to all or part of the site.

Note: The only data that you can change in the inflation book is the inflation rate for each year, from the present forward.

Inflation Factor Used in Cost Books

The inflation factor used in cost books is computed from the base year of the cost data and the inflation rates given in the default Reference inflation book. If the cost data corresponds to the data in the Reference cost book, then the cost module uses a fixed year for the base year. If you add your own cost book and enter unit cost different from the Reference set, then the cost module saves the year in which you input that data as the base year for inflation computations.

Assign an Inflation Book to an Organization

You can create multiple inflation books, and assign a separate book to any Organization within your permissions scope, as needed. This can be done with the following steps:

  1. Select the Current Organization that you wish to assign to your inflation book.
  2. Go to the Reference Books drop-down menu. Click the Inflation button.
  3. Select an Inflation Book from the Search Inflation Books box. If no Inflation Books appear, or if you need to create a new custom reference book, view the Add a Custom Inflation Bookon page 1 topic.
