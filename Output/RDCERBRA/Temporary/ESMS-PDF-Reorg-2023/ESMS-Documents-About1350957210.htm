Understanding the Documents Screen

This topic introduces tasks you can perform at the Documents screen and provides links to greater detail for each. It also provides an overview of navigating the documents screen.

What Can I Do Here?

At the Documents screen, documents associated with components in the Enterprise SMS database can be added, tagged, and searched for independently of the components they may be associated with.

Document Tasks

Consuming Documents

At the Documents screen, you can

  • search for a document
  • search for documents with a particular tag
  • view a document
  • download a document

The tasks above are described in View Documentson page 1.

Managing Documents

If you are actively engaged in uploading documents and adding or grooming search tags, you can also do the following:

  • add documents (in bulk)*
  • associate document(s) with a component*
  • delete a document
  • associate search tags with documents
  • add or delete tags from the pool of search tags

*This can also be performed at the relevant Components Details screen, which is accessed via the Search screen.

Productivity Tip: proficiency in View Documentson page 1 will help for most of these tasks.

Navigating the Documents Screen

When you select Documents at the Enterprise SMS dashboard, you'll be brought to a screen that lists all documents associated with your permitted sites.

  • To narrow the list to a single site, use Select Site on the main menu.
  • To narrow the domains, use Domains on the main menu.

The major components of the Documents screen are illustrated and explained below.

  1. Search Fields (1). The "Document Name" and "Tag" search fields at the top of the Documents screen allow you to search in multiple ways. See the topic View Documentson page 1 for more detail.
  2. Table of Documents (2). The table of documents (#2 in the illustration) lists documents. "Next" and "Previous" links located between the table and the accordion buttons provide navigation through multiple screens of documents. Each line shows the following for its document: (a) name and extension; (b) search tags associated with the document; and (c) the date it was uploaded. Icons allow viewing, download, or deletion.
  3. Document Associations and Tags Editor Window (3). has two subsections. The first shows the component(s) (1) associated to the document(s) selected in the table of documents. The second allows you to add or delete search tags.

    4. WARNING: Deleting a tag from the search tag pool will delete this search tag from every image (or any other item that uses these search tags) that it is associated with.

  4. Tag Editor (4). The Tag Editor button allows you to add to or delete from the pool of available search tags.
  5. Upload (5).The Upload Files button allows you to add images and other permissible types of documents into the Enterprise SMS by browsing for them and initiating the upload.
