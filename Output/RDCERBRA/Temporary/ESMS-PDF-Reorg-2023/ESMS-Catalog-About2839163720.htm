Understanding the Catalog Screen

This topic introduces tasks you can perform at the Catalog screen and provides links to greater detail for each.

What Can I Do Here?

At the Catalog screen, you can add custom component and group types to the Enterprise SMS catalog. You have the option to design an assessment template for each new type. In addition, you can open and edit PowerBI Reports and import or export configurations.

Catalog Tasks

Custom Component Type

Tasks involved in creating a custom component type are:

  • Add the new component type to the catalog
  • Assign subcomponents to it as needed
  • List the relevant properties for the new component type.

Custom Group Type

  • Add the new group type to the catalog
  • Assign subcomponents to it as needed
  • List the relevant properties for the new group type.

Assessment Template

  • Design an assessment template for any of the custom components as desired, using the assessment template editor

Navigating the Catalog Screen

The Inventory Catalog screen is where you add a new component or group type. A popup will gather information such as the name of the new type.

On the Inventory Catalog screen, you can also select the new type by name and flesh out the design by assigning properties and subcomponents. Separate tabs are provided for subcomponent assignment and for listing properties.

Also within the Catalog screen is where you can design an assessment template for the new type, if desired, by selecting the Assessment Template Editor tab.

In addition, you can review and edit PowerBI Reports, and Import or Export Configurations via the Import Catalog Manager.
