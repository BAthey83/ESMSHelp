Edit WBS Details Dialog

The Edit WBS dialog is activated by the Edit button on the WBS Details page.

The dialog provides a way to update the following attributes of the selected WBS item:

  • Status
  • Classification
  • Notification Number
  • Work Order Number
  • CLIN
  • Actual Cost
  • RTO/RTI Planned Date
  • RTO/RTI Actual Date
  • Technically Complete Date
  • Comment

For WBS items in the Fuels domain, the following attributes can also be edited:

  • Current Situation
  • Impact
  • Scope Of Work
  • Regulatory Requirement
