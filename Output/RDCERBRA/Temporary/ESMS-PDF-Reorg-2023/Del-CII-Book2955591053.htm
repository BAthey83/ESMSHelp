Delete a CII Book

This topic explains how to delete a custom Component Importance Index (CII) book. The original Reference CII book cannot be deleted.

Permissions Note: Deleting a CII book requires Master Planner permissions level or above at the target Organization.

To delete a CII book,

  1. Select the CII book you wish to delete.
  2. Click the trashcan icon that appears next to the "+" icon.
  3. The "Confirm Action" popup window will appear.
  4. Select Yes if you wish to delete the CII book, or select No if you wish to keep the CII book.
