Navigating the Facility Details Screen

When you select a row in the facility results table at the Search screen (a single-click suffices), the detail screen for that facility will appear. For example:

The screen layout consists of (1) a listing of Facility Propertieson page 1, followed by (2) a series of Accordion Buttonson page 1 providing access to additional aspects of facility management.

Facility Properties

At the Facility Details screen, you can see the following properties for the selected facility, if the properties have been entered into Enterprise SMS Web:

  • Facility Number
  • Facility Name
  • Parent Site
  • Installation Code
  • Installation Name
  • RPUID Real Property Unique Identifier. This is a facility-level property, described in JIRB 2004-07-19, Department of Defense Real Property Inventory (RPI) Initiative, dated July 19, 2004. (Real Property Unique Identifier)
  • RPSUID (Real Property Site Unique Identifier)
  • Construction Date
  • Date Rebuilt
  • PRV Plant Replacement Value. This number will be the aggregate of the PRV of all inventory items contained within the current inventory item. (Plant Replacement Value)
  • Quantity
  • UoM (Units of Measure)
  • MDI Mission Dependency Index (Mission Dependency Index)
  • Domain
  • Category Code
  • Type
  • Operational Status
  • PoC (Point of Contact)
  • CI Condition Index (Condition Index)
  • FI Functionality Index.  (Functionality Index)
  • FCI Facility Condition Index.  (Facility Condition Index)
  • EFCI (Extended Facility Condition Index)
  • PI Performance Index. The Performance Index is a weighted combination of the CI value and the FI value, with the lower of the two values being given two-thirds of the weight and the higher of the two values being given one-third of the weight. If no functional assessment has been performed, the FI is assigned a value of 100. (Performance Index)
  • Comments

Note: The unit of measure (UoM) for the Quantity property is listed below Description, without a label.

Accordion Buttons

At the bottom of the screen you can activate any of the accordion buttons to toggle the section open or closed. An explanation of information available at each button follows.

Components

Shows what components are in the current facility.

Assessment Plans

Shows assessment plans associated with the current facility.

Work Items

Shows work items associated with the current facility.

Documents

Shows documents associated with the current facility.

Cost Modifiers

Shows cost modifiers associated with the current facility.

Performance Records

Shows performance records associated with the current facility.
