Adding a Project

The Add New Project button on the Project List Screenon page 1 brings up the Create Project dialog. The button is accessible only to those users with Create Project privileges.

Create Project Dialog

The dialog includes the following required project attributes:

  • Title
  • Type
  • Year
  • Number
  • Construction Agent
  • Sites (multiple sites can be selected)
  • Domain
  • Assigned PM (The lookup list includes all the users with the Project Manager privileges for at least one of the selected sites)
  • Workflow check box

When the project is Workflow-enabled (Workflow is checked), the following additional required attributes are available:

  • Scheme (The scheme selected must be appropriate for a Project, for example Project Scheme)
  • Permissions, if the selected scheme requires any
