Project Details - Funding

The Funding tab of the Project Details Screenon page 1 is for viewing and maintaining the list of Funding sources available for the project. The ability to edit the project is available to users with Update Project privileges for at least one of the project sites.

The Funding list is editable regardless of the project status.

Maintaining the Funding List

Engage the Add New Funding button to open the Add Funding dialog.

The Add Funding dialog allows the following Funding attributes to be specified:

  • Acceptance Category (required)
  • Assets (required). The Add Assets command brings up a searchable list of assets associated with the project. Assets can be searched by name, facility number, RPUID or CAT Code
  • Purchase Request (required)
  • Fiscal Year (required)
  • Fund Account (required). The list of funds contains funding available for the project sites and domain for the selected fiscal year
  • MIPR Number (required)
  • GT&C Code
  • Purchase Order (PO)
  • PO Date
  • PO Amount

Once a Funding list item is created, it can be edited or removed by engaging the Edit or Remove button respectively.

For a quick view of the assets associated with a Funding list item, engage the View Assets button.

Maintaining the list of Funding Modifications

To view or maintain the list of Funding Modifications, select a row in the Funding list.

Engage the button to add a new Modification for the selected Funding.

The Add Funding Modification dialog allows the following attributes to be specified:

  • Amendment Number (required)
  • Amount (required)
  • PR Committed Date
  • PR Amount
  • MIPR Signed Date
  • MIPR Accepted Date
  • PO Date
  • PO Amount
  • Contract Award Date
  • Contract Amount

Once a Funding Modification is created, it can be edited or removed by engaging the Edit or Remove button respectively.

Engage the Comment button to view or edit Modification comment history.
