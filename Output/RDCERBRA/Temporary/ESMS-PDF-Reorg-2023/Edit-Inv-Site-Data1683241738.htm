Edit Inventory Site Data

This topic describes the information that can be viewed and edited at the Site data screen. The data will be in the content area of the screen. A type of data unique to Sites is a set of location factors including (1) Area Cost Factor, (2) seismic zone, and (3) climate zone. These factors can be assigned manually, or you can take advantage of the option to auto-apply factors from an existing geographic location.

To edit the inventory data for a specific Site, first select the Site from the Site Search menu.

Information about the Site will be presented on an ID line. You can use the to perform actions such as saving changes, or adding or editing comments.

If you want to change the lookup location for the Site, that is also done at the content area toolbar at the Site level.

Toolbar

  • Download as CSV. Download the site data as a chart in Excel format.
  • Show/Hide Map. Toggle this button to show or hide the Site Map.
  • Show/Hide Graphs. Toggle this button to show or hide the Site Graphs.
  • Filter. Activate this switch to filter your site data search results as needed.
  • Select Columns. Here, you can choose which columns you want to appear in your site data.

    • Selectable Columns include:

    • RPSUID - Real Property Site Unique Identifier.
    • Site Code - An alphanumeric code to aid in identifying sites. Limit: 12 alphanumeric characters.
    • Name - The name of the Site's point of contact. Limit: 30 alphanumeric characters.
    • Installation Name - The identifier for the installation site. Limit: 50 alphanumeric characters.
    • Installation Code - A four- or five-digit identifying code for the installation.
    • Organization Name - The identifier for the Organization. Limit: 50 alphanumeric characters.
    • CONUS - Indicates whether the Site is part of the Continental United States. Select "Yes," "No," or "N/A."
    • Country - The Site's country of location.
    • State - The Site's state location. Limit: 2 alphanumeric characters.
    • Zip Code - The Site's zip code. Limit: 10 alphanumeric characters.
    • CI Condition Index - The Site's Condition Index rating.
    • PI Performance Index. The Performance Index is a weighted combination of the CI value and the FI value, with the lower of the two values being given two-thirds of the weight and the higher of the two values being given one-third of the weight. If no functional assessment has been performed, the FI is assigned a value of 100. - The Site's Performance Index rating.
    • EFCI - The Site's Estimated Facility Condition Index rating.
    • Replacement Value - The aggregate value of all the Buildings within a Site.
    • ACF - Area Cost Factor, a multiplier that adjusts the national average labor/material/equipment costs to local costs.
    • Climate - The Site's Climate rating is an integer from 1 to 11. You can set your Site's Climate rating by selecting a minimum-to-maximum data range.
    • Seismic - The Site's Seismic Zone rating is an integer from 1 to 7 that follows the Federal Emergency Management Agency's (FEMA's) U.S. seismic zoning. You can limit your report's seismic results by selecting a minimum-to-maximum data range.
