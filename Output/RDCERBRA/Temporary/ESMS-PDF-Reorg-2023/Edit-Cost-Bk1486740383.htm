Edit Cost Book Data

This topic covers how to add or edit data in a custom cost book. If the custom book still needs to be created, refer to the Add a Custom Cost Bookon page 1 topic.

Permissions Note: Editing a custom cost book requires Master Planner permission level or above at the target Organization.

Tasks:

  1. Open a Custom Cost Bookon page 1
  2. Navigate Data Fieldson page 1
  3. Edit Dataon page 1
  4. Save Changeson page 1

Steps for each of the tasks are listed below.

Open a Custom Cost Book

To open a custom cost book,

  1. Select the Reference Books dropdown menu. Click the Cost button. The available custom cost books will appear.
  2. Choose a custom cost book from the menu by clicking and highlighting it with your mouse. If you do not see the cost book you want, type your book title in the Search Cost Books window.

Navigate Data Fields

You can use one or a combination of the navigation options described below to bring cost book records into view: Level: Activity, filters, and/or page navigation.

Activity

The Activity field lets you filter whether to add a Global Repair, Replace or Demolish action to the cost book.

Level

The Level field can be filtered to display a Building or Component-Section in the table.

Filters

You can apply a Filter in as many or as few of the cost book fields as you wish.

Note: The columns displayed depend on the Activity and Level selected.

To use filters to reduce the view of the data,

  1. Select the Filter button (highlighted above) to hide or show Filters above the data table.
  2. Enter a search term or number in one or several of the fields above the data table.
  3. Filtered data for the selected fields will appear in the data table.
  4. To further restrict the results using filters, you can type a search term or number into the filter field of any one of the remaining empty filterable columns, repeating this list of steps.

Edit Data

Current Cost Data

To change the current cost data for an item,

  1. Select the item you wish to modify.
  2. The Cost Record window will appear.
  3. Add the new Unit Cost value to the Unit Cost field.
  4. In addition, you can modify the Cost Year, Minimum Cost, Material Cost, Labor Cost, Equipment Cost, Bare Total Cost, Cost Source, Frequency, Crew, Daily Output, and Total Work Output (not pictured) fields.

Save Changes

  1. To save your changes to the various cost fields, click the Update button.

Note: If you wish to remove your cost data changes, select the Clear button at the top of the field.
