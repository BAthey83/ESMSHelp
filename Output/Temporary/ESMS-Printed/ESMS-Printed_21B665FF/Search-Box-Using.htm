Using the Search Box

This topic explains how to use the search box located on the main menu bar. You can use the search box to perform searches even when not at the Asset Inventory screen.

Search Box Parameters

Scope

The Search box is available from all task areas in Enterprise SMS Web except for Import.

If you are not at the Asset Inventory screen and you initiate a search from the search box (by pressing Enter when the cursor is showing in the search box),

  • You will be brought to the Asset Inventory screen.
  • Your site and domain selections will be preserved.

    • Tip: When using the search box from a location other than the Asset Inventory screen, selecting a site is required.

Persistence

At the Asset Inventory screen, your search will persist through domain changes, site changes, and change in which Inventory tab is currently displayed.

Site Change Example

If you enter "tank" in the search box, then select sites one at a time from the main menu bar while at the Asset Inventory screen, you will see the search results for each of the sites in turn without having to re-enter your search term into the box.

How to Search

Prerequisite

To use the Search box, you must first select at least one domain at the main menu.

Enter a Search Term

Search terms are not case-sensitive; "fence" "Fence" and "FENCE" will all obtain the same result.

When your search term is entered, press Enter while the cursor is showing in the search box.

View Results

By Tab
 When you perform a search, the number of search results present at each tab on the Asset Inventory screen is shown after the tab name.

You can browse the results in each tab desired.

By Site

If you change your selected site in the main menu while viewing results on the Asset Inventory screen, you will automatically see results for the newly selected site.

Troubleshooting Search Results

No Results
  • Check whether a domain is selected.
  • If initiating the search from a location other than the Asset Inventory screen, selecting a site is required.
