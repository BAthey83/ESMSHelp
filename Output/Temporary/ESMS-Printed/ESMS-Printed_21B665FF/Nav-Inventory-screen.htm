Navigating the Asset Inventory Screen

This topic provides a navigation overview of the information available on the Enterprise SMS Asset Inventory screen. Separate topics (linked below at the end of this topic) show the information available at each screen at the Details level.

After you have selected one or more domains and, if applicable, a site, the initial display at the Asset Inventory screen displays

  • the number of facilities, groups, and components that are in the active site(s)
  • what site and domain(s) are selected (on the main menu)
  • a data table of assets, at the facilities level

Once you understand the components of the Asset Inventory screen and how they work, you will be able to

  • control the kind of data that is displayed by designating which fields are active for each asset level
  • narrow searches using filters
  • navigate to pages that show detailed information about assessment plans, work items, and individual assets

Components of the Asset Inventory Screen

The major components of the Asset Inventory screen, as numbered in the illustration below, are:

  1. main menu
  2. search results title bar (this appears only after a search has been performed)
  3. tab bar, where you can select whether to view facilities, groups, components, assessment plans, or work items
  4. content area, which will display the results of the most recent search
  5. fields selection (activate the Fields button to change what fields are displayed as headers in the content area)
  6. filters selection (activate the Filters button to designate what filters to apply)
  7. command bar at the bottom of the screen, which provides additional options via icon buttons.

Main Menu

The main menu is persistent throughout Enterprise SMS. At the main menu, you can

  • return to the home page, using the Enterprise SMS logo button (site and domains will be cleared)
  • select one or more domains to work with (Required)
  • search the site, Using the Search Boxon page 1
  • select from options in the user menu:(a) save your current workspace as the default; (b) change sync settings for synchronizing with ESMS Desktop; (c) download a Desktop version of ESMS; or (d) logout.
  • logout (from the user options menu at the far right)

Search Results Title Bar

This title bar (in a paler blue) confirms the current search term, if one is active.

Tab Bar

Directly below the Main Menu on the Inventory screen is the tab bar that allows you to select, via the following tabs, what to view:

  • Facilities
  • Groups
  • Components
  • Assessment Plans
  • Work (work items)

Note: The number in parentheses after each tab label indicates how many facilities/groups/etc. there are in the site(s) that are active for your current search session.

Content Area

The content area of the Asset Inventory screen is below the tab bar, and shows one of the following:

  • Depending on which tab is selected on the tab bar, the content area shows a table listing the facilities, groups, components, assessment plans, or work items that are (1) associated with either your selected site or all of your permitted sites (depending on whether a site has been selected) and (2) within your selected domain(s). The column headings in the table are determined by the fields selected via the Fields button.
  • When you perform a search, a results table showing the search results is displayed in the content area. The column headings for the results table are shown directly below the task selection bar. These column headings present are determined by the fields selected via the Fields button.

Fields Selection

To control which data fields will be displayed in the content area,

  1. Activate the vertical Fields button (outlined in red below).
  2. Select and/or deselect checkboxes.
  3. Activate the Apply Fields button.
  4. Activate the vertical Fields button again to hide the Fields selection box.

Note: The options shown in the Fields selection box will depend on what is selected on the tab bar (Facilities, Groups, Components, Assessment Plans, or Work items).

Expanded information about setting up fields is in the topic Set Up Fields and Filterson page 1.

Filters Selection

Filter selection can be accessed using the Filters button at the top left of the content area (directly below the Fields button).

To apply filters to your search,

  1. Activate the vertical Filters button directly left of the content area.
  2. In the popup, fill in the desired filter information.
  3. Activate the Apply Filters button.
  4. Activate the vertical Filters button again to hide the Filters selection box.

Note: The options shown in the Filters selection box will depend on what is selected on the tab bar (Facilities, Groups, Components, Assessment Plans, or Work items).

Expanded information about setting up filters is in the topic Set Up Fields and Filterson page 1.

Command Bar

Icons on the command bar provide you with additional options:

  • download the search results in .csvComma-separated values format/Excel
  • switch to map view
