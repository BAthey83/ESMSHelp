Import Request Details Screen

View import details including status. Update, re-submit or clone an existing request

Note: BUILDER environments are configured by the ESMS administrators.

Prerequisites

Before starting to import data from a BUILDER environment

  • Obtain user credentials for that BUILDER environment
  • Mapping BUILDER Sites to ESMS Siteson page 1

To view request details click on the item of interest in the list on the Buildings Import Screen. The Import Request Details dialog is displayed.

Information Displayed in the Dialog

  • Environment
  • Date of request
  • Request author
  • Status
  • Queue status
  • Sites selected

Functionality Available in the Dialog

  • If the request is New, the site selection can be edited.
  • If the request is Sent, the processing status on the BUILDER side can be queried by clicking the Check Status button
  • If the request is Sent, In Progress or in Error, the request can be restarted
  • If the request is Done, the import log is displayed
  • Any request can be cloned
