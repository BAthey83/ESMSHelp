Navigating the Work Item Details Screen

When you select a row in the work item results table at the Search screen (a single click suffices), the detail screen for that work item will appear. For example:

The screen layout consists of a listing of work item properties.

General Work Item Properties

At the Work Item Details screen, you can see the following general properties for the selected work item, if the properties have been entered into Enterprise SMS Web:

  • component name
  • current CICondition Index
  • component type
  • current RSLRemaining service life.
  • created date
  • direct rating assessment date
  • updated date
  • projected CICondition Index
  • reference
  • projected RSLRemaining service life.
  • year

Work Item Description

At the Work Item Details screen, you can see the following descriptive properties for the selected work item, if the description has been entered into Enterprise SMS Web:

  • description
  • source
  • activity type
  • estimated cost
  • profile
  • modernization
  • funding source
  • priority
  • work plan
  • classification
  • work package
  • category
  • completion date
  • RPUID Real Property Unique Identifier. This is a facility-level property, described in JIRB 2004-07-19, Department of Defense Real Property Inventory (RPI) Initiative, dated July 19, 2004. (real property unique identifier)
  • status
