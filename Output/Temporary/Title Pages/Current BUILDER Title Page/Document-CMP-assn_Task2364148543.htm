Associate Document(s) with a Component

Enterprise SMS Web allows documents such as images, Word documents, and PDFs to be associated with individual components. There are two methods for associating a document stored in Enterprise SMS with the component it's related to. It can be done

  1. Associate Documents at the Component Details Screenon page 1 for the component, or
  2. by using Selection Bulk Edit options Associate Documents at the Documents screenon page 1.

See Which Method To Use?on page 1 immediately below for advice which method to use.

Which Method To Use?

If there is only one or a small number of related documents, and if they are available at the time of creating the component, it's usually best to do the association at that time, at the Component Details screen, because the documents will be automatically associated with the component.

However, if there are a large number of documents, it's better to use Selection Bulk Edit at the Documents screen so that multiple documents can be selected at one time using the checkboxes.

Associate Documents at the Component Details Screen

Note: This section contains instructions for uploading a document and associating it with a component at the component's Details screen. The other method for associating a document with a component is done Associate Documents at the Documents screenon page 1.

In this method you will start at the desired component and search for documents to be associated with it, one at a time.

To associate documents (images, etc.) with a component using this method,

  1. Navigate to the Search screen.
  2. Select the Components tab.
  3. As needed, use Filters, the navigation buttons below the results table, and/or the Search box to display the relevant component in the results table.
  4. Select the component. You will be brought to the component's Component Details screen.
  5. Activate the Documents accordion button if that section isn't already open.
  6. Activate the Choose File button.
  7. Navigate to and select a document to be associated with the component. Repeat this step as many times as needed. Associated files will appear under "Name", above the Choose File button.
  8. When finished, you can select the Documents accordion button to close it.

Associate Documents at the Documents screen

Note: This section contains instructions for associating a document with a component by using Selection Bulk Edit at the Documents screen. The other method for associating a document with a component is done Associate Documents at the Component Details Screenon page 1.

In this method you will start with one or more documents selected at the Documents screen, then search for components to be associated with the document(s). This method will only work for documents that have already been uploaded.

To associate documents (images, etc.) with a component using this method,

  1. Navigate to the Documents screen.
  2. As needed, use document search methods (name, tags) to display relevant documents in the results table.
  3. Using the checkboxes in the table of documents, select one or more of the documents listed.
  4. Select the Selection Bulk Edit accordion button to open it.
  5. In the field labeled "No component selected", enter a search term that will help you find the desired component to associate.
  6. Tip: If the search term is not one you've used at this location recently, clicking outside of the field and then back in it might help trigger the search results to appear.
  7. When a list of search results appears, select the desired component. The illustration below shows results for the search term "fuel", with one of the results selected.
  8. Note that after you select a component, its name will appear in the field above the Associate button. At the right side of the field you will see a blue "x" and a down arrow. You can activate the "x" to delete your selection from the field; the down arrow will re-display the components search results.
  9. When the desired component is showing in the field, activate the Associate button under "Component Associations". The component name will appear under "Component Associations".
