Set Up Fields and Filters

This topic shows how to (1) set up Fieldson page 1 to control what kinds of data you want to see in your search results and (2) set up Filterson page 1 to refine your searches.

Introduction to Fields and Filters

At each tab in the Inventory screen, the selection of filters is identical to the selection of fields. The difference is in how the information is selected and applied: Fields will affect the presentation of search results at the Inventory screen, whereas filters affect what content is returned by a search. After they are selected, fields in the "Fields" dropdown can optionally be configured (using Sort options) to order the search results in different ways. By contrast, the fields presented in the "Filters" dropdown are designed to be filled in by the user to narrow the search. Many of the numeric filter options are divided into entry boxes for both a minimum value and a maximum value, to specify a range of values.

Fields

The Fields button at the Inventory screen allows you to configure the presentation of search results. The example screenshot below shows the fields available for configuring information presented at the Work tab. Checkboxes that are marked are those that will appear on the results screen.

Open the Fields Dropdown

To open the dropdown for selecting fields, activate the Fields button at the left side of the Inventory screen.

Note: In the Fields dropdown, you are not limited to one option; you can mark as many checkboxes as you wish before activating the Apply Fields button.

Change Field Selection

If you want to clear all the fields and start from scratch,

  1. Activate the Clear Fields button.
  2. Mark desired checkboxes.
  3. Activate the Apply Fields button.

Alternatively, you can use the following procedure for changing field selection:

  1. If there are fields you want to see in the search results that are not already marked, mark the checkboxes for these.
  2. If any fields you don't care to see in the search results are marked, unmark the checkboxes for these.
  3. Activate the Apply Fields button.

WARNING: If you don't activate the Apply Fields button, your changes will not be saved.

Note: If you've made no changes since activating the "Fields" dropdown or since last activating Apply Fields, the Apply Fields button will be grayed out.

Close the Fields Dropdown

To close the Fields dropdown, activate the Fields button again.

Sort Search Results (Optional)

You have the option to sort your search results by selected fields. Selecting sort fields and sort direction is done by toggling the Sort button located to the right of the field.

Sort Direction

Choose just one of the following for each selected field:

  • the minus sign (no sort)
  • down caret (sort order from high to low, or Z to A)
  • up caret (sort order from low to high, or A to Z)

Note: Only those fields with marked checkboxes will offer the option to sort.

Sorting Example

To sort by one or more fields,

  1. Open the Fields Dropdownon page 1 if it's not already open.
  2. Activate the Sort button for the primary field you want to sort by. When you do so, the minus sign to the right of the word "Sort" will change to a down caret, as shown in the screenshot below.



  3. Toggle until you see your desired sort order for the field. Selecting the down caret will sort from high to low or Z to A; selecting the up caret will sort from low to high or A to Z. Selecting the minus line will cancel any sorting activity for that field.
  4. If you want to sort by more than one field, proceed to the second tier field you want to sort by, and activate its Sort button.
  5. Continue for as many levels as you wish.The screenshot below shows three levels.
  6. Activate the Apply Fields button.

WARNING: If you don't activate the Apply Fields button, your changes will not be saved.

Note: If you've made no changes since activating the "Fields" dropdown or since last activating Apply Fields, the Apply Fields button will be grayed out.

  1. To change the sort order, either (a) start from scratch with the sorting process, or (b) experiment with adding and deleting sort fields while paying careful attention to the relative order in which you add or delete. Note: If you've made no changes since activating the "Fields" dropdown or since last activating Apply Fields, the Apply Fields button will be grayed out.

Filters

The Filters button at the Inventory screen allows you to set parameters for the contents of your search results.

The table below on the next page shows the filter selection for each of the five tabs on the Inventory screen. The selection of Filters is the same as the selection of Fields for each tab. In the case of filters, however, you have the option to restrict search results to a particular value or range of values for one or more parameters.

Work

Open the Filters Dropdown

To open the dropdown for selecting filters, activate the Filters button at the left side of the screen.

Set Filters

To set filters,

  1. For a parameter you want to restrict to a particular numerical value or text string, enter the value or text into the field. If both Min and Max fields are displayed for that parameter, you can enter the same number in the Min and Max fields in order to specify the single value you are searching for.
  2. For a parameter you want to restrict to a range of values, enter the lowest acceptable value in the Min field, and the highest acceptable value in the Max field.

Note: For any field that you leave blank, all values of that field will be included in the search.

Clear Filters

To clear one filter, delete the values in its fields.

To clear all of the filters, activate the Clear Filters button.

Close Filters Dropdown

To close the Filters dropdown, activate the Filters button again.
