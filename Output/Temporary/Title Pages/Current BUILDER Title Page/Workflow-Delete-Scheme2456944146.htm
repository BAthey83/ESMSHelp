Delete Scheme

Permissions Note: Deleting a scheme typically requires elevated permissions.

WARNING: If you delete a scheme on which you have based one or more assessments, these assessments will technically still work. However, you would no longer be able to refer to the scheme on which they are based, which could cause difficulties.

To delete a scheme,

  1. Open the scheme you wish to delete, using the Open Scheme icon button on the Workflow editor menu bar.
  2. On the Workflow editor menu bar, activate the Delete Scheme icon button (far left).
  3. Confirm the deletion in the popup box.

Delete a State or Transition From a Scheme

To delete a state or transition within a scheme, select the state or transition in the diagram and activate the Delete key on the keyboard. Make sure to save your change(s) by activating the Save icon button on the Workflow editor menu bar.
