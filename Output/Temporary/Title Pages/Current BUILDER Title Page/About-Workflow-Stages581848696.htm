Workflow Stages

If you enable Workflow with an assessment, you can design your own stages for the workflow to pass through. ESMS also provides a default workflow that provides the following stages:

  • Charter Stageon page 1
  • Pre-Assessment Plan Stage
  • Review Stageon page 1
  • Desktop Sync / Assessment Stageon page 1
  • Post-Assessment Plan Stageon page 1
  • Packages Stageon page 1
