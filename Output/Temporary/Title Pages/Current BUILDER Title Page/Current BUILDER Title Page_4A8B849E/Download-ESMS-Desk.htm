Download Enterprise SMS Desktop or Tablet

This topic provides instructions for downloading the Desktop or Tablet version of Enterprise SMS from Enterprise SMS Web. The procedure is the same for first time use and for updating to a more recent version.

Note : Not all users need the Desktop version of Enterprise SMS, which is for making changes to asset information and entering assessment plans. Using Enterprise SMS Web, you can (1) view inventory, assessment, and work item information; and (2) generate work plans. If this is sufficient, you might not need to download the Desktop or Tablet version.

Prerequisites

Before downloading and running the Desktop or Tablet version of Enterprise SMS, you need

  1. Access to the Web version of Enterprise SMS.

    Note: If you don't already have this, see instructions for Web access at Get Username and Password for Enterprise SMS Webon page 1 below.

  2. User name and password for the Desktop version.

    Note: More detail about this is provided in the Enterprise SMS Desktop documentation topic Get Started with Enterprise SMS Webon page 1.

Get Username and Password for Enterprise SMS Web

If you already have a username and password for Enterprise SMS Web, skip to Log In to Enterprise SMS Webon page 1 below.

If not, your Enterprise SMS Administrator should be able to set you up with the following for Enterprise SMS Web access:

  1. Username and password. Username and password need to be the same for Web, Desktop, and Tablet sign-ins (whichever of the three you use).
  2. Designated permissions roles and the scope for each.

Download Steps

Log In to Enterprise SMS Web

Note: If needed, instructions for logging in are under the "Log In" header at Get Started with Enterprise SMS Webon page 1.

Once you're logged in to Enterprise SMS Web, follow the steps below to download the Desktop or Tablet version.

Download the Zip File

To download the zip file,

  1. Activate the user menu dropdown (on the menu bar at the far right).
  1. Select "Download Application" from the dropdown. A popup will appear, showing the available versions.
  1. This will initiate the download.
  2. Once the EnterpriseDesktop.zip file has loaded and appeared in the lower left of the screen, click the up arrow to show options.

Note: The up arrow will change to a down arrow when the list of options is open.

  1. Select "Open".

Extract the Zip File

To extract the zip file,

  1. Activate "Extract All" in the Downloads folder. (The image shown here is from Internet Explorer.)
  1. Designate a destination for the extracted files. In the popup (illustrated below), do one of the following:
  1. Enter the pathname.
  2. Alternatively, browse to the desired folder location for the extracted files, then confirm (to confirm in Internet Explorer, be sure to click on the folder to make the Select Folder button highlight; then activate the Select Folder button).
  3. Activate the Extract button at the bottom of the popup to initiate the extraction.

Open and Run

When the file has finished extracting,

  1. Activate Enterprise SMS.FieldApp.If this is your first use of the program, the "Sync" popup will appear.
  1. Enter username and password in the "Sync" popup, then activate the Sync button.

Tip: Sorting files in the folder by Type makes it easier to find the executable file. The message "Authenticating" will appear in the popup.

Note: Sync will not always be requested in future logins.
