Search for a Document

At the Documents screen, accessed from the dashboard, you can search for one or more documents by (either or both):

  • name or name fragment
  • associated search tag

A sample of the Documents screen, before any of the search fields are filled in, is shown below.

Types of Document Search

Search by Document Name

To search by document name,

  1. Type the name or name fragment in the "Document Name" field.
  2. Activate the Search button.

The display will change to list only those documents meeting the search requirement.

Note: if the "Next" link appears between the list of documents shown and the accordion buttons, this means there is more than one page of search results. You can use the "Next" and "Previous" links to navigate through the results

Search by Document Type

To search by document type, you can type just its extension (for example, .png) in the "Document Name" field.

Search by Tag

To search by tag,

  1. Type the name of the desired search tag in the "Tag" field.
  2. Activate the Search button.

The display will change to list only those documents meeting the search requirement.

Note: if the "Next" link appears between the list of documents shown and the accordion buttons, this means there is more than one page of search results. You can use the "Next" and "Previous" links to navigate through the results

Combination Search

If you fill in both the "Document Name" and the "Tag" fields, the search will find only those documents meeting both criteria.

Progressive Search

You can effectively search by more than one name (for example) by progressively refining your results using progressive search.

To use progressive search, perform the first iteration described below, once. Follow up with subsequent iterations as many times as desired.

First Iteration:

  1. Enter your first search term in the "Name" field under the "Document Search" header.
  2. Activate the Search button.
  3. Do NOT press the Clear button.

Subsequent Iterations:

  1. Erase the search term in the Name field, then enter your next search term. The results of the previous search(es) will be trimmed down by the criterion in the current search.
  2. Activate the Search button.
  3. Do NOT press the Clear button.

Clear a Search

To clear a search and return to the full listing of documents, activate the Clear button.

View a Document

Once you have found a relevant document via Search for a Documenton page 1, you can view it by activating its view icon

If the document record has no view icon, follow these steps:

  1. Activate the download icon.
  2. Select the document at the bottom of the Documents screen. (This step might not be needed if you have already selected the option to "Always open files of this type" for its file type.)

View Component Associations

To ascertain whether a document is associated with one or more components,

  1. Checkmark the document in the list of documents under "Document Search".
  2. Activate the Selection Bulk Edit accordion button if that section isn't already open.
  3. Look in the "Component Associations" section of Selection Bulk Edit.
  4. If the document is not associated with any components, you will see a message "Selected documents have no associated components".
  5. If there are component associations, they will be listed under the header "Name".

Productivity Tip: You can leave Selection Bulk Edit open while you uncheck and check different documents.

Caution: If you select more than one document at a time at the table of documents, there won't be any indication of which document is associated with what component. You will need to select documents one at a time to see the precise matching.

Download a Document

To download a document, activate its download icon:

The document will be in your designated downloads folder, and its name will also be listed on a button at the bottom of the Documents screen.

Productivity Tip: You can use the "Show in folder" option on the document's button to jump to its download location.
