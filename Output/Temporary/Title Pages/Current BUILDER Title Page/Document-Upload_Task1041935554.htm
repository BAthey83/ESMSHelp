Upload Document(s)

There are two methods for uploading documents (such as images, PDFs, etc.) into Enterprise SMS Web. It can be done

  1. Upload Documents at the Documents Screenon page 1 in the Bulk Upload section, or
  2. Upload Document at Component Details Screenon page 1; this will automatically associate the document with that component.

See Which Method To Use?on page 1 immediately below for advice which method to use.

Which Method To Use?

If you want to associate the documents to be uploaded with one or a few already existing components, and there are only a couple documents per component to upload, you may find it most convenient to do upload and association in one step at the Component Details screen for each of the components involved. The main disadvantage with this method is that you can upload only one document at a time.

For a large number of documents, it's better to use Bulk Upload Upload Documents at the Documents Screenon page 1.

Upload Documents at the Documents Screen

Note: This section contains instructions for uploading documents using Bulk Upload at the Documents screen. The other method for associating a document with a component is done Upload Document at Component Details Screenon page 1.

CAUTION: When uploading documents according to instructions in this section, association with a component does not automatically take place. This must be done in a separate step, described in the topic Associate Document(s) with a Componenton page 1. An alternative is to Upload Document at Component Details Screenon page 1 of a component, in which case they are automatically associated with that component.

Select and Upload File(s)

To select one or more files to be uploaded,

  1. At the Documents screen, activate the Bulk Upload accordion button if that section isn't already open.
  2. Activate the Choose Files button.
  3. As prompted, navigate in the file system to the file location of one or more documents/images you want to upload.

    Productivity Tip: At Bulk Upload, you have the standard file selection options available, such as (a) CTRL-click to add an additional single document to the selection, or (b) click-SHIFT-click to select a contiguous range of files, etc.

  4. Select the files representing the images or other documents you want to upload. A "Queue progress:" bar will show status of the upload. When the bar is solid blue, the upload is complete. In the list of documents in the Bulk Upload section, fully uploaded documents will be so indicated by a checkmark in the "Status" column.

    Troubleshooting Tip: The name of the newly uploaded document(s) might not appear in the list of documents right away. If this is the case, and you want to view or perform an action on the document(s) right away, activate the "Next" link between the list of documents and the accordion buttons, then navigate back using the "Previous" link. (Refreshing the browser will not necessarily work.)

  5. To clear the list of uploaded documents, activate the Clear Queue button.

Associate with Component(s)

Best Practice: For any uploaded documents that should be associated with one or more components, it's best to do this association promptly at the time of upload.

See the topic Associate Document(s) with a Componenton page 1 for how to associate uploaded documents with their respective components.

Upload Document at Component Details Screen

Note: This section contains instructions for uploading a document and associating it with a component at the component's Details screen. The other method for associating a document with a component is done Upload Documents at the Documents Screenon page 1.

At the Component Details screen, you can upload documents to be associated with the selected component.

Notes:

  • At the Component Details screen, documents can be uploaded only one at a time. To upload more than one at a time, use Bulk Upload at the Documents screen (but then you will have the extra step of associating the documents with the correct component).
  • Uploading a document at a component's Component Details screen, following the instructions in this section, automatically associates the document with that component.

To upload a file,

  1. Navigate to the Component Details screen for the component you want the document associated with.
  2. Open the Documents section at the bottom of the screen by clicking on it.

     

  3. Activate the Choose File button.
  4. As prompted, navigate in the file system to the file location of the document/image you want to upload.
  5. Select the file (for example, in Chrome, by double-clicking on it). The file will appear under the "Name" header in the Documents section.
