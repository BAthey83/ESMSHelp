Edit Questionnaire Template

This topic shows how to edit an existing questionnaire template.

Note: The screen for this functionality is viewable by ESMS administrators only. It will not appear on the Dashboard of a non-administrator.

What Can I Edit?

In the template, you can change question categories, questions, or answers:

  • add question categories
  • rename question categories
  • change composition of question categories
  • delete question categories
  • add questions
  • edit questions
  • delete questions
  • add single-select answers
  • edit answers
  • delete answers

Warning: One thing that can not be changed is the hierarchy of question categories. Hierarchy levels in the questionnaire template cannot be changed, and the questions located at a particular hierarchical level in the questionnaire template cannot be moved to another level.

How Do I Edit?

To edit a questionnaire template, first you will need to navigate to and open it as shown in the next section, Open a Questionnaire Templateon page 1. Next, you will activate individual question categories or questions to expand them for editing. How to make different types of edits is explained under the headings Question Categorieson page 1, Questionson page 1, and Answerson page 1 later in this topic.

Open a Questionnaire Template

To open a questionnaire template, first activate the Assessment Templates tab menu:

  1. If not already in the configuration manager, activate the Catalog tile on the Enterprise SMS Dashboard to reach the configuration manager.
  2. Activate the Assessment Templates tab. A list of component types will appear in the left pane.
  3. To make the menu appear, activate a component in the left hand pane.

Tip: You will get the fastest results if you activate the component type in the list that is associated with the template you plan to view.

You can use the search field (labeled "Search component types here") at the top of the list to help find a desired component type. Once a component type is activated, the menu bar will appear below the Assessment Templates tab:

After the Assessment Templates tab menu (above) appears, continue as follows:

  1. Activate the Choose Template Set button in the menu.
  2. Select the desired template set from the dropdown.
  3. If you get the message "The selected managed object type doesn't have an assessment template associated with it. Click Add to start adding questions," this means that the selected template set is not associated with the activated component type. In the list of component types in the left pane, activate the one associated with the template set you just selected, and the questionnaire template should appear.

Tip: As long as you click only within the visible list of component types, you can keep trying different component types in the list until you get the right one.

  1. If you get the message shown in the previous step, but have clicked elsewhere in the meantime (unfortunately this includes using the search box above the list of component types), you will need to back up and try again.
  2. Once you have the correct component type activated and the corresponding template set selected in the dropdown, the template set should appear in the content pane. It will initially appear in fully collapsed mode, showing one line for each top-level question or question category.

Question Categories

You can make the following changes to question categories:

Action Instructions
Add question category See Question Categoryon page 1 in Add Questionnaire Templateon page 1
Rename question category

To rename a question category,

  1. In the template, activate the shaded title line containing the question category. If the category was collapsed, it will expand. The category name will appear in the Category field.

  2. In the Category field, edit the category name as desired.

  3. To lock in the change, click anywhere outside of the Category field but within the template.
Change the order of questions/categories

To move template elements (questions or question categories) around within a given level of the template, use an up or down arrow at the right end of the title line for each question or question category you want to move. The up arrow will move the element above the element before it. The down arrow will move the element below the element after it.

Note: You can change the order of template elements only within the same hierarchical level.
Delete category

To delete a category, activate the "x" at the right end of the category's title line.

WARNING: Deleting a category will delete any and all questions contained in it.

Questions

You can make the following changes to questions:

Action Instructions
Add question See Question Typeson page 1 in Add Questionnaire Templateon page 1
Add followup question See Followup Questionson page 1 in About Nesting and Presentation Orderon page 1
Edit question

To edit a question,

  1. In the template, activate the shaded title line containing the question . If the question was collapsed, it will expand. The question will appear in the Question field.
  2. In the Question field, edit the question as desired.
  3. To edit answer options, or add one or more answer options in the case of a single-select question, see the Answerson page 1 table below.
  4. To lock in the change, click anywhere outside of the Question field but within the template.
Change order of questions

To move template elements (questions or question categories) around within a given level of the template, use an up or down arrow at the right end of the title line for each question or question category you want to move. The up arrow will move the element above the element before it. The down arrow will move the element below the element after it.

Note: You can change the order of template elements only within the same hierarchical level.
Delete question To delete a question, activate the "x" at the right end of the question's title line.

Answers

You can make the following changes to answers:

Action Instructions
Add single-select answer

A single-select question can have an unlimited number of answer options for the user to choose from. To add an answer,

  1. Navigate to the desired single-select question or followup question.
  2. If the question is collapsed, activate the question's shaded title bar to expand it.
  3. Activate the Add Answer button. An answer block will appear below the question block, after any other answer options that are already in place for that question.
  4. Fill in the Answer field, and, if desired, the Points field and/or the Comment field.
  5. (Optional) If you wish to add a followup question that will be posed to the user if she or he selects the answer you are now entering, then follow the instructions in Followup Questionson page 1 in About Nesting and Presentation Orderon page 1.
  6. To add an additional answer option, you can repeat the steps above, starting with Step 3. There's no limit to the number of single-select answers.
  7. Activate the Save button when finished.
Edit single-select answer option

To edit an answer option for a single-select question,

  1. Navigate to the desired single-select question or followup question.
  2. If the question is collapsed, activate the question's shaded title bar to expand it.
  3. Scroll to the answer option you wish to edit.
  4. Edit the Answer field, and, if desired, the Points field and/or the Comment field.
  5. Activate the Save button.
Delete single-select answer option To delete an answer option, activate the "x" at the right end of the answer option's title line.
Edit multi-select answer option

The designation of Yes, No, and N/A as the answer options for a multi-select question are fixed. What you can change is (1) the number of points assigned to each of the answer options, and (2) the presence and contents of any followup questions (unless the question is already a followup question).

To edit an answer option for a single-select question,

  1. Navigate to the desired multi-select question or followup question.
  2. If the question is collapsed, activate the question's shaded title bar to expand it.
  3. Scroll to the answer option you wish to edit.
  4. Edit the Points field if desired.
  5. (Optional) If the question is not already a followup question and you wish to add a followup question that will be posed to the user if she or he selects the answer you are now entering, then follow the instructions in Followup Questionson page 1 in About Nesting and Presentation Orderon page 1.
  6. If you wish to edit an existing followup question associated with any of the answer options, either (a) follow the instructions in this list if the followup question is multi-select, or (b) if it is a single-select followup question, follow instructions for editing a single-select answer option as described above in this table.
  7. Activate the Save button.
Delete multi-select answer To delete an answer option, activate the "x" at the right end of the answer option's title line.
Delete followup question from an answer To delete a followup question from an answer, activate the "x" at the right end of the followup question's title line.
