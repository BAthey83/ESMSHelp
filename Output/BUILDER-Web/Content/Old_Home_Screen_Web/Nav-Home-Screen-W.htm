Navigating Enterprise SMS Web

The Enterprise SMS Dashboard

You can reach the Enterprise SMS dashboard at anytime by clicking on the Enterprise SMS logo in the upper left of the screen. The following areas are accessible through the dashboard:

WARNING: Navigating to the dashboard will obliterate your site and domain selections.

 

Search Screen

The major components of the search screen are, from top to bottom on the screen:

  • Main Menu
  • Tab Bar
  • Fields
  • Content Area
  • Filters

Main Menu

The main menu is persistent throughout Enterprise SMS. From it, you can do any of the following:

  • Select a site to work with
  • Select one or more domains to work with
  • Use a tool (Import, Role Permission, Work Generation)
  • Search the site
  • Specify user preferences
  • Log out

Tab Bar

Directly below the Main Menu on the search screen is the tab bar that allows you to select which of the following to view:

  • Facilities
  • Groups
  • Components
  • assessment plans
  • Work (work items)

Fields

Directly below the task selection bar, the Fields line displays column headings for the content area. You control which data fields will be displayed as follows:

  1. Activate the vertical Fields button (outlined in red below).
  2. Select and/or deselect checkboxes.
  3. Activate the Apply Fields button.
  4. Activate the vertical Fields button again to hide the Fields selection box.

Note: The options shown in the Fields selection box will depend on what is selected on the tab bar (Facilities, Groups, Components, assessment plans, or Work items).

Content Area

Immediately below the column headings in the Fields area is the content area where data is shown.

Filters

Filter selection can be accessed using the Filters button at the top left of the content area (directly below the Fields button).

Filters can be applied as follows:

  1. Activate the vertical Filters button directly left of the content area.
  2. In the popup, fill in the desired filter information.
  3. Activate the Apply Filters button.
  4. Activate the vertical Filters button again to hide the Filters selection box.

Note: The options shown in the Filters selection box will depend on what is selected on the tab bar (Facilities, Groups, Components, assessment plans, or Work items).
