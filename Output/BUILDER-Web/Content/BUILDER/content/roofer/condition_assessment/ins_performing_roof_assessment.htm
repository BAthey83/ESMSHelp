Performing Roof Condition Assessments

In ROOFER, condition assessments are performed at the roof section level. To initiate a new condition assessment (inspection), you will start at the Inspections tab at the roof section level of ROOFER inventory.

The screen at this tab provides a quick overview of past inspections for the selected roof section. To see details about what is included, refer to the header "Inspections Tab" in Edit Roof Section Inventory Data.

Overview of the Inspection/Assessment Process

The roof inspection process will consist of the following steps:

  1. Navigate in the ROOFER hierarchy tree to the roof section to be inspected.
  2. Select the roof section.
  3. Activate the Inspections tab at the screen for that roof section.
  4. Add a new roof inspection. This process is described in Add a Roof Inspection.
  5. Enter inspection data. This process is described in Roof Inspection Data and Add Roof Distresses.

Roof Inspection Types

The two types of inspection that you might perform are:

  1. Visual Inspection. For a visual inspection, you enter roofing distresses from options provided, along with their severity and density. Performing a visual inspection is described in Edit Roof Inspection Data at Visual Inspection, and in Add Roof Distresses.
  2. Insulation Inspection. Insulation inspection is done for membrane roofs only. Performing an insulation inspection is described in Edit Roof Inspection Data at Insulation Inspection.

Non-Inspectable Roof Types

If the Add button above the table listing inspections is grayed out, that means that roof section type is non-inspectable.

 

As of May 4, 2016, the roof types listed below do not have inspection criteria or distresses entered into ROOFER. Visual Inspection is not selectable for these roof types.

Steep Roof, Non-Metal: Non-Inspectable Types

Non-inspectable types of non-metal steep roof are as follows:

  • Asphalt Rolled
  • Synthetic roofing: Plastic
  • Synthetic roofing: Wood Fiber
  • Synthetic roofing: Metal Shingles
  • Synthetic roofing: Fiber Cement
  • Wood Shakes
  • Wood Shingles
  • Slate
  • Metal panel (this metal panel covered steep roof type is different from the metal panel roof type)
  • Concrete Tile: Pan and Cover
  • Concrete Tile: S Shape
  • Concrete Tile: Flat
  • Clay Tile: Pan and Cover
  • Clay Tile: S Shape
  • Clay Tile: Flat
  • Other

Metal Panel: Non-Inspectable Types

Non-inspectable types of metal panel roof are as follows:

  • Metal Panel - Shingles
  • Flat Soldered Seam
  • Other

Membrane: Non-Inspectable Types

Non-inspectable types of membrane roof are as follows:

  • Liquid Application: Neoprene/Hypalon
  • Liquid Application: Acrylic Elastomer
  • Liquid Application: Butyl
  • Liquid Application: Polysulfide
  • Liquid Application: Urethane
  • Liquid Application: Silicone
  • Liquid Application: Type Unknown
  • PUF: Silicone
  • PUF: Urethane
  • PUF: Catal. Urethane
  • PUF: Acrylic
  • PUF: Coating Unknown
