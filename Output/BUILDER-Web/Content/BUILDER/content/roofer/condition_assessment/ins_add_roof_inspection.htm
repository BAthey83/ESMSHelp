Add a Roof Inspection

This topic covers how to add a new roof inspection and enter the minimum identifying data required.

Create a New Inspection

To add a new inspection,

  1. Navigate to the desired roof section on the inventory tree.
  2. Activate the Inspections tab in the content area. At the Inspections tab, basic information will be displayed about past inspections, for example the FCIFlashing Condition Index. In ROOFER the FCI, as determined by inspection and/or algorithm, is used in the calculation of the Roof Condition Index (RCI). and ICI Insulation Condition IndexInsulation Condition Index.
  3. Activate the Add button (above the Inspection Date column header).

    The "New Roof Inspection" popup window will appear:

Enter Required Information about the Inspection

In the "New Roof Inspection" popup window shown above,

  1. Enter the inspection date by doing one of the following:
    1. Enter the date of the inspection in the Inspection Date field. ROOFER will accept a variety of different date entry formats.
    2. Alternatively, you can use the calendar button to select the date.
  2. Identify the Inspection Type by selecting from the dropdown list. This will be either a Visual Inspection or an Insulation Inspection.

    Note: Insulation Inspection will not be available for some roof section types.

  3. The additional fields that need to be filled in at the time you create the new inspection will vary, depending on the roof section type.

Confirm or Cancel

Once the required minimum roof inspection data is entered,

  1. Activate the Add button on the popup toolbar. This will add the inspection to the roof section. The popup window should disappear, and you will be returned to the Inspections tab for the roof section.
  2. Alternatively, activate Cancel to close the popup window without adding a new inspection.

Continue Data Entry

To begin data entry of inspection results, follow the instructions in Edit Roof Inspection Data.
