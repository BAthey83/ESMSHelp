Edit Roof Inspection Data

This topic covers how to access an existing roof inspection, and explains the data that can be added or edited.

Access an Existing Inspection

To add or edit data in an existing inspection, including one that has just been created, first access the inspection as follows:

  1. Navigate to the Inspections tab of the roof section.
  2. Activate the date of the inspection in the Inspection Date column.

Next, follow instructions in Roof Inspection Data.

Roof Inspection Data

The list of inspections for a given roof section will indicate not only the date of the inspection, but also the inspection type, which will be one of the following:

  • Initial Visual Inspection
  • Visual Inspection
  • Insulation Inspection (This option is not displayed for some roof section types.)

When you select an inspection by its date, a screen will appear that has two tabs: an Inspection tab and an Attachments tab. The Attachments tab will indicate the number of attachments associated with the inspection. Each inspection type and the Attachments tab are discussed below.

Inspection Tab

The appearance of the Inspection tab screen will vary depending on the type of inspection. Information at the headers below will describe how to enter inspection data.

Initial Visual Inspection

This inspection entry is automatically created by ROOFER for the date last replaced, or for the date of construction if the roof has not been replaced. It is not generally accessible.

Visual Inspection

Note: There are certain roof types for which Visual Inspections are not allowed in ROOFER. For a listing of these types, see Non-Inspectable Roof Types.

The Inspection tab for a visual inspection presents selected inspection and roof section data, then below that it contains a table where you add distresses found during the section condition assessment. Note that "Visual Inspection" will appear in the breadcrumb list at the top of the content area.

The following inspection data appears at the Inspection tab:

  • Inspection Date. This field is initially set to the current date. However, you can change the date either by activating the field (the date will highlight) and changing the date, or by using the calendar button with scroll arrows to select from the calendar.
  • Roof Surface (Read-Only). The area of the roof section's surface is listed here.
  • Roof data such as Perimeter Flashing, Curb Flashing, and Membrane Type or Roof Covering Type. The data types will vary according to the type of roof section inspected.
  • FCI (Read-Only). This field displays the calculated Flashing Condition Index for the selected roof section.
  • MCI Membrane Condition IndexMembrane Condition Index (Read-Only). For a membrane roof, this field will appear and display the calculated Membrane Condition Index for the selected roof section.
  • SCI System Condition Index - a condition rating for a Building System. For each System, the SCI is computed by taking the average of its Components' CIs, weighted by replacement cost.System Condition Index - a condition rating for a Building System. For each System, the SCI is computed by taking the average of its Components' CIs, weighted by replacement cost. (Read-Only). For a shingle roof, this field will appear and display the calculated Shingle Condition Index for the selected roof section.
  • PCI (Read-Only). For a metal roof, this field will appear and display the calculated Shingle Condition Index for the selected roof section.
  • Est. RSL Remaining Service LifeRemaining Service Life (ERSL) w/o Repair (Read-Only). This field displays the estimated remaining service life without repairs.
  • Est. RSL Remaining Service LifeRemaining Service Life (ERSL) with Repairs(Read-Only). This field displays the estimated remaining service life with repairs.

Insulation Inspection

Note: Insulation inspections apply only to membrane roofs.

The Inspection tab for an insulation inspection displays the number of test cores obtained; then for each of the cores it lists the wet area and percent water.

The following roof section will be used as an example to illustrate an insulation inspection:

Selecting the Inspections tab for the roof section, then Add, causes the "New Roof Inspection" popup to open (formerly named "Add Roof Inspection").

Tip: Before attempting an insulation inspection, make sure that an insulation type has been checkmarked at the Insulation tab for the roof section. If no insulation type has been selected, you will not be given the option in the dropdown list to select an insulation inspection.

From the Inspection Type dropdown, select "Insulation Inspection."

The next screen shows that both a Visual Inspection and an Insulation Inspection were added on the same date.

 

The Inspection tab contains the relevant inspection data. The following inspection data appears when Insulation Inspection is selected:

  • Inspection Date. This field is initially set to the current date. However, it can be changed by activating the field (the date will highlight) and using the calendar button with scroll arrows to select from the calendar the date for the new inspection.
  • Number of Cores. Enter the number of cores take during the inspection
  • Section Area (Read-Only). This is a read-only field that reflects the specified area of the selected roof section.
  • Number of Wet Areas(Read-Only). This is a read-only field that shows the number of wet areas entered for the inspection. The wet areas are entered in the inspection data table.
  • ICI Insulation Condition IndexInsulation Condition Index(Read-Only). This field displays the calculated Insulation Condition Index for the selected roof section.

The Inspection Data table appears in the lower half of the content area once the Number of Cores is entered. Select a core and edit the Wet Area and the % Water values for that core.

Once the inspection is saved by activating the Save button at the top of the content area, ROOFER will calculate the Insulation Condition Index (ICI Insulation Condition IndexInsulation Condition Index) and display it in a color-coded field on the inspection screen. The color will be red, yellow, or green depending on the value of the condition index.

Attachments Tab

The Attachments tab allows files such as an inspection worksheet or an image file to be attached to the inspection record.

To add an attachment,

  1. Activate the Add button in the toolbar. The "Add Image" popup window will appear.

  2. In the popup window, enter a Title and Description for the image you will attach. Also, use the radio buttons to designate whether the image is an inspection worksheet.

  3. Next, activate the Select button.

  4. Browse to the file to be attached, and either (a) double-click it, or (b) select it and activate the Open button.

The attachment may be in any of the following formats:

  • .jpg
  • .jpeg
  • .png
  • .zip
  • .doc
  • .pdf

Note: As of January 31, 2022, .doc format is allowed, but not .docx.

  1. If you select a file with a nonsupported format, you will see a red dot. Activate "x Remove" and try again.

  2. If you select a file with a supported format, you will see a green dot.

  3. Once you have a valid file selected, activate Save on the toolbar.
  4. Alternatively, if you activate Cancel, the file will not be attached and the popup window will close.
