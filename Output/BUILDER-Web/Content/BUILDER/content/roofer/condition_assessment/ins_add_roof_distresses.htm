Add Roof Distresses

To initiate adding distresses found during the condition inspection, activate the Add button above the Distress table.

The "Visual Roof Inspection Input" popup window will present the option to enter the defects using one of the following two methods:

  1. Standard Entry, which presents drop-down list boxes where you select the distress type and severity; or
  2. Rapid Entry, which requires the user to enter an abbreviation to identify each distress.

The popup window will remain open for distress entries until either the Cancel or Add & Close button is activated.

Add Roof Distresses Using Standard Entry

To add roof distresses using Standard Entry, first select the Standard Entry radio button.

Note: The first field in the popup window is the ID Number, which is automatically generated for the individual defect and is read-only.

Enter Roof Distress Data

To enter roof distress data using Standard Entry, select entries for the following fields:

  1. Distress. Select from the dropdown list of available ROOFER distresses for the selected roof section type.
  2. Severity. Select Low, Medium, or High severity for the distress.

    Note: Depending on the distress selected, there may be only a subset of Low, Medium, or High available to select from.

  3. Defect. Use the dropdown list to select the particular defect for the distress.
  4. Quantity. Enter the number (if the unit is "each") or area affected by the Distress.

Progress to the Next Distress or Close

When you complete entries for a distress, activate one of the following buttons at the bottom of the popup screen:

  1. Add. Activate the Add button if there are still more distresses to be entered. The "Visual Roof Inspection Input" popup window will re-appear with an incremented ID number.
  2. Add & Close. If you have entered the last distress, select the Add & Close button instead.
  3. Cancel. To close the popup window without adding the entry to the Distress table, activate the Cancel button.

SAVE Distress Entries

WARNING: Even though you see the defects entered in the Distress table, you must activate Save in the toolbar at the top of the content area to secure the information. Even an action as seemingly innocuous as resizing your software window can cause the data to disappear if you have not saved.

Save the distress information by activating Save in the toolbar at the top of the content area.

Attachments

To add attachments pertaining to the distresses entered, proceed to the Attachments Tab for the visual inspection. The Attachments tab is to the right of the Inspection tab.

Add Roof Distresses Using Rapid Entry

To add roof distresses using Rapid Entry, first select the Rapid Entry radio button.

Note: The first field in the popup window is the ID Number, which is automatically generated for the individual defect and is read-only.

Enter Roof Distress Data

To enter roof distress data using Rapid Entry, fill in the following fields:

  1. Distress. Enter the two-digit code for the distress in the Distress field. A tool tip help will appear, listing the codes. It is not necessary to use capital letters when making an entry.

  2. Severity. Enter L, M, or H, corresponding to Low, Medium, or High severity for the distress.

    Note: Depending on the distress selected, there may be only a subset of Low, Medium, or High available to select from.

  3. Defect. Enter the defect number. The tool tip will list the valid options. If the field is not active, that means there are no valid options for the Defect field.
  4. Quantity. Enter the number (if the unit is "each") or area affected by the distress.

Progress to the Next Distress or Close

When you complete entries for a distress, activate one of the following buttons at the bottom of the popup screen:

  1. Add. Activate the Add button if there are still more distresses to be entered. The "Visual Roof Inspection Input" popup window will re-appear with an incremented ID number.
  2. Add & Close. If you have entered the last distress, select the Add & Close button instead.
  3. Cancel. To close the popup window without adding the entry to the Distress table, activate the Cancel button.

SAVE Distress Entries

WARNING: Even though you see the defects entered in the Distress table, you must activate Save in the toolbar at the top of the content area to secure the information. Even an action as seemingly innocuous as resizing your software window can cause the data to disappear if you have not saved.

Save the distress information by activating Save in the toolbar at the top of the content area.

Attachments

To add attachments pertaining to the distresses entered, proceed to the Attachments Tab for the visual inspection. The Attachments tab is to the right of the Inspection tab.

Edit a Distress Entry

To edit distress information after entry ,

  1. Select the distress in the Distress table.
  2. Activate the pencil icon in the toolbar at the top of the Distress table. This will return you to the "Visual Roof Inspection Input" popup window for that entry, where you can make your changes.
  3. Change the information in one or more of the following fields: Severity, Defect, or Quantity.
  4. If the Update button is still grayed out, click or tap in the background area of the popup window.
  5. Activate the Update button at the bottom of the popup window.
  6. Alternatively, activate Cancel to close the popup window without saving the change.

Delete a Distress Entry

To delete a distress entry,

  1. Select the distress in the Distress table.
  2. Activate the Delete icon (a red "X") in the toolbar at the top of the Distress table.

Complete the Inspection

Once the inspection data is completed and all distresses have been entered, activate the Save button at the top of the content area to commit the inspection to the database. ROOFER will then calculate and display the appropriate condition indexes for the roof section.
