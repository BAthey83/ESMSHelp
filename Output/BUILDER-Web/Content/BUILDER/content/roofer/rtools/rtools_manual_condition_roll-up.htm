Perform Manual Condition Rollup

**Note: this topic has not yet been completed to reflect ROOFER operation.

The About Condition Assessments topic contains a description of ROOFER's process for calculating current CI Condition IndexCondition Index values for all levels of its inventory. The process "rolls up" the condition index values from one level of the inventory hierarchy to the next in a methodical manner, beginning with roof sections and ending with the Site (or parent Organization if one is included in the inventory). ROOFER is designed to perform the rollup recalculation on your command. You should perform this rollup calculation frequently, but especially when:

  • You have not performed the rollup within the last 6 months
  • You have entered a large number of new condition assessments
  • You need the most current condition index measures to develop a new work plan.

To initiate the manual rollup process, select Tools > Administration > Manual Condition Rollup from the main menu.

The rollup process will begin for all inventory in the database and the Progress window will appear.

The following processes will occur during the rollup:

  • Retrieve the latest condition assessment records and estimate a current condition index for all roof sections.
  • Estimate the current remaining service life (RSL Remaining Service LifeRemaining Service Life) for all roof sections. The RSL Remaining Service LifeRemaining Service Life is adjusted by the rate of deterioration in the condition indices.
  • Estimate the current condition index for all Buildings by calculating the average of their roof sections' condition indices weighted by replacement cost.
  • Estimate the current condition index for all Complexes, Sites, and Organizations by calculating the average of their Buildings' condition indices weighted by replacement cost.

After the rollup process completes, the Progress window will close and you will be returned to the main ROOFER screen. Building inventory windows will display recalculated current condition indices for all elements.
