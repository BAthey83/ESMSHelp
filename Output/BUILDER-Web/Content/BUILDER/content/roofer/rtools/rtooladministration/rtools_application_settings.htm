When you create a new inventory database, you should set the preferences in the Application Settings window before proceeding with any additions to inventory.

To edit the application settings, select Tools > Administration > Application Settings starting at the main menu.

The "Application Settings" popup window will appear. Subsequent sections explain each tab in the Application Settings window.

Settings Tab

Tip: If you do not see a screen similar to the one above, try selecting a different tab and then coming back to the Settings tab.

At the Settings tab, you can set the following preferences for your database:

User Category

After you designate a user category, ROOFER will present an agency-specific list of Building Use categories when inventory is being added.

Selecting the All category allows you to enter Building Use categories across multiple categories.

Note: Once your inventory contains Buildings from more than one category, the only choice here is All.

User's Default Unit of Measure

Here is where you select the default unit of measure (English or Metric) to be used for the inventory for all users. This is the unit of measure that will initially be shown for all users, but each user can change his or her units of measure at any time with the User Preferences tool when they are logged in.

Fiscal Year Start Date

At this tab, enter the fiscal year start date. When running simulations using Scenarios, the fiscal start date determines:

  1. When items are estimated to be completed.
  2. When new work items will be evaluated.

License Tab

(This is a pending capability) The License Tab provides a location where you can copy and paste your security key.

Advanced Tab

At the Advanced tab you can do any of the following:

  • Designate an email address for support administration.
  • Designate the URI of the RSS feed to display on the login page.

Email Tab

At the Email tab are options that configure automated response to users who have difficulty logging in and activate the request to change their password or to have their account unlocked.

The large text box is the location where you input the text to be sent in automated email to the user. Variables that can be used in the text are as follows:

  • <%ApplicationName%> = name of the application (BUILDER or ROOFER)
  • <%Url%> = link for the user to follow to complete the password action
  • <%SupportEmail%> = email address of the application support representative

Session Information Tab

At the Session Information tab (abbreviated Session Info), you can see session information such as number of users logged in.
