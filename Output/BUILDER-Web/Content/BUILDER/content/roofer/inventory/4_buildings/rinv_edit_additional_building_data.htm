Edit Additional Building Inventory Data

In addition to data available at the General Information tab, further Building information is displayed and can be edited at the tabs listed below.

Note: All of the additional Site information is optional, or else generated by ROOFER.

Additional Information Tab

The additional Building information is accessed from the general Building information screen by selecting the Additional Info tab:

All of the additional Building information is optional, and includes:

Contact Information

Point of contact (POC) information for the Site may be viewed and edited by selecting the Contact Information (Contact Info.) tab. The available fields are as follows:

  • Name. Enter the name of the Site point of contact. Limit: 30 alphanumeric characters.
  • Phone Number. Enter the phone number of the Site POC. Limit: 20 alphanumeric characters.
  • Email Address. Enter the email address of the Site POC. Limit: 75 alphanumeric characters.
  • Street. Enter the official street address of the Site. Limit: 30 alphanumeric characters).
  • City. Enter the city where the Site is located. Limit: 15 alphanumeric characters.
  • State. Enter the state where the Site is located. Limit: 2 alphanumeric characters.
  • Zip Code. Enter the zip code of the Site. Limit: 10 alphanumeric characters.

Building Design

  • Architect. Lists the architects of the Building. Add and delete names by using the adjacent buttons. Each architect's name is limited to 30 characters.
  • Contractor. Lists the contractors of the Building. Add and delete names by using the adjacent buttons. Each contractor's name is limited to 30 characters.
  • Documents. Lists the documents related to the Building. Add and delete document names by using the adjacent buttons. Each document's name is limited to 24 characters.

Building Dimensions

  • Building Perimeter. Enter the linear measure of the perimeter of the Building.
  • Description (Table). Enter a description of the distinct areas of the Building that you wish to add dimensions for. For example, you may list "East Wing/West Wing," or "Offices/Library/Warehouse," or "Public Areas/Secure Areas." Each description is limited to 50 characters. Use the Add Dimension and Delete Dimension buttons to add and delete items from the list.

    To make entries in the dimension description table, follow these steps:

Assessment History Tab

The Assessment History Trends tab displays a graph showing how the Roofing Condition Index has changed over time. The graph displays the RCI Roof Condition IndexRoof Condition Index from the year the Building was constructed to the current fiscal year.

Work Item History Tab

The Work Item Sustainment Management SystemA discrete work activity applied to a single managed object such as a Component-Section, intended to improve its life cycle performance. History tab shows a list of the work items that have been performed and/or are scheduled to be performed in the Building. The work items can be filtered by fiscal year (FY), Status, Cost, or Activity using the drop down lists on the screen. For more information on work items, see the Work Planning Overview.

Attachments Tab

The Attachments tab allows files such as an image file to be attached to the Building's record. An image might be a picture of the Building, or it could be an electronic image of written notes or diagrams made during inventory.

The attachment may be in any of the following formats:

  • .jpg
  • .jpeg
  • .png
  • .zip
  • .doc
  • .pdf

Note: As of January 31, 2022, .doc format is allowed, but not .docx.

Attach a File

To attach a file,

  1. At the Attachments tab, activate the Add button in the toolbar below the row of tabs.

    This will open the "Add Image" popup window:

  2. Enter Title and Description information for the attachment.
  3. Activate the Select button to the right of the "Image" field to browse to the desired image or other file.
  4. If you are adding an attachment at the Building level, use the Image Type radio buttons to indicate whether the attachment is a Key Plan, a Section Plan, or "Other."
  5. Once the required attachment data is entered, activate the Save button on the popup toolbar. Alternatively, to close the popup window without attaching the selected file or image, activate the Cancel button.
