Remember that Sites should be location-specific (namely, confined to a relatively limited geographic area), because ROOFER inputs pertaining to location are stored at the Site level.

To add a new Site to an Organization, navigate to that Organization and do one of the following:

  1. Select the Organization, then activate the Add Site icon on the main toolbar.
  2. Or, right-click the Organization, then select "Add Site" from the options.

The "Add Site" popup window will appear:

Enter Required Site Information

This documentation segment describes the minimum data to be entered when creating a new Site. Other data that can be entered is described in Edit Site Inventory Data.

In the popup window shown above, enter the following data:

  1. Site Number -Enter the identifying number for the Site. The Site Number field may be left blank if you wish to supply only a name. You must enter at least a Site name or a Site number, and the combination of the two must be unique in the navigation tree.

    Specifications: The Site The ID Number is limited to 12 alphanumeric characters.

  2. Site Name -Enter the identifying name for this Site. The Site Name field may be left blank if you wish to supply only a number. You must enter at least a Site name or Site number, and the combination of the two must be unique in the navigation tree.

    Specifications: The name is limited to 50 alphanumeric characters. Best Practice Recommendation: Avoid using single quotes, double quotes, and ampersands. These characters can interfere with query formation.

Copy Factors from a Listed Location (Optional)

In the popup window, you are also given the opportunity to copy Site location factors (Area Cost Factor A multiplier that adjusts national average labor/material/equipment costs to local costs. For example, in a remote area with high shipping costs and a tight labor market, the Area Cost Factor might be 1.3. The ROOFER database contains this data for most major cities in the United States as well as Department of Defense installations. To copy factors, mark the Use factors from following location checkbox and select a location from the dropdown lists.

Copying factors from an existing location can also be done later by selecting Lookup Location from the Site toolbar.

Note: Until you set location factors, they will all be set by default to 1.

CAUTION: Selecting the checkbox Use Location Code and Name for Site Information will overwrite the Site information with that of the location that you are copying factors from.

Confirm or Cancel

Once the required minimum Site data is entered,

  1. Activate the Add button on the popup toolbar to create the Site. Additional Site details can then be added as described in Edit Site Inventory Data
  2. Alternatively, to close the popup window without creating the Site, activate the Cancel button.

Continue Data Entry

To enter additional details about the Site, follow the instructions in Edit Site Inventory Data.
