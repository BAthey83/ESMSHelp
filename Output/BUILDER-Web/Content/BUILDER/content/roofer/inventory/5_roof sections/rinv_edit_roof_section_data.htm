Edit Roof Section Data

Data for a given roof section can be viewed and edited by selecting the roof section in the inventory tree.

Toolbar

The following buttons and information are available at the content screen toolbar:

  • Save. Saves changes made.
  • Comment. Allows you to add, edit, and view comments about the section.
  • Reports. Launches the Report Selection tool, which gives you access to a list of standard reports relevant to section inventory. See Using the Report Viewer.
  • Breadcrumbs. The breadcrumbs shown on the toolbar provide both orientation and an alternative method of navigation:

ID Line

On the roof section ID line you can view, but not edit, the following:

  • Building Number.
  • Building Name.
  • Section Name.

General Roof Section Information - General Info. Tab

When you select a roof section in the inventory navigation tree, the General Information tab (abbreviated General Info.) is shown:

To edit the information, type the updated information into the available fields, then activate Save on the content pane toolbar. The fields are as follows:

Identification:

  • Section Name. This is the same as the Roof Section ID. The ID/name of the roof section is limited to one or two alphabetic characters, which must be capitalized.
  • Section Description. Here you can give a more extensive description of the roof section.
  • Occupancy. The IBC occupancy classification type may be entered here as historical information, if desired, but it is not a requirement to produce outputs from ROOFER.
  • Roof Type (Read-Only). This field displays the roof type: (a) Membrane; (b) Metal Panel; or (c) Steep Roof: Non-metal.

Important Dates:

  • Date Constructed.
  • Date Last Replaced.
  • Inventory Date.
  • Last Inspection Date (Read-Only).
  • Age (Read-Only). This is calculated by subtracting the Date Last Replaced from the current date.

Condition Summary (Read-Only):

  • Remaining Service Life. The expected remaining service life is calculated using the Date Last Replaced, the current date, and the Service Life Book selected at the Site or Organization level.
  • Roof Condition Index (RCI Roof Condition IndexRoof Condition Index). The RCI Roof Condition IndexRoof Condition Index for the roof section is calculated using the values of the indexes listed below.
  • Flashing Condition Index (FCI).
  • MCI Membrane Condition IndexMembrane Condition Index / PCI / SCI System Condition Index - a condition rating for a Building System. For each System, the SCI is computed by taking the average of its Components' CIs, weighted by replacement cost.System Condition Index - a condition rating for a Building System. For each System, the SCI is computed by taking the average of its Components' CIs, weighted by replacement cost.. Depending on the roof type, either the Membrane Condition Index, Panel Condition Index, or Substrate Condition Index will be displayed.
  • Insulation Condition Index (ICI Insulation Condition IndexInsulation Condition Index). This index results from the insulation inspection, which is a core sample inspection of the roof section's insulation.

Roofing Details Tabs

In addition to data available at the General Information tab, further roof section information is displayed and can be edited at the tabs listed below.

Layout Tab

  • Roof Dimension. In the Area field, enter the area of the roof. The unit of measure is shown to the right of the Area field.
  • Perimeter (Membrane Roof). In the fields provided, enter the length of each applicable perimeter type for the roof section. The unit of measure is shown to the right of each of the fields. The perimeter types for a membrane roof are as follows:
    • Parapet.
    • Roof Edge.
    • Adjacent Wall.
    • Expansion Joint.
    • Area Divider.
    • Other. Use this category for any perimeter type not listed above.
  • Perimeter (Steep Roof: Non-metal). In the fields provided, enter the length of each applicable perimeter type. The unit of measure is shown to the right of each of the fields. The perimeter types for a metal panel roof are as follows:
    • Parapet.
    • Roof Edge.
    • Adjacent Wall.
    • Fire Wall.
    • Hip.
    • Valley.
    • Ridge.
    • Other. Use this category for any perimeter type not listed above.
  • Roof Access. From the dropdown list, select the method by which the roof section is accessed.
  • Adjacent Sections. The other roof sections for the Building are listed here. Mark the checkbox for any of them that are adjacent to the current roof section.
  • Roof Types. Check all that apply. Does not apply to membrane roofs.

Structure Tab

Here you can enter or view the following information:

  • Structural Frame Type. Select from the dropdown list.
  • Roof/Structural Deck Type. Select from the dropdown list.
  • Attachment Surface. Select from the dropdown list. Does not apply to membrane roofs.
  • Roof Slope. Select from the dropdown list.
  • Drainage Types. Mark the checkbox for each option that applies.
  • Vapor Retarder Type (Membrane Roof Only). Select from the dropdown list.

Insulation Tab (Membrane Roof Only)

  • Insulation Types. Check all that apply. Then, if there is Board Stock, fill in the number of layers and total thickness; also, use the radio button to indicate whether or not it is tapered.
  • Total R-Value. Fill in the R-value of the insulation.
  • Attachment. From the dropdown list, select the method of attachment for the insulation.

Membrane Tab (Membrane Roof Only)

At this tab, the fields for Manufacturer, Spec. No., Description, Warranty, and Warranty Expiration are for reference and information storage; they are not used by ROOFER for calculations.

  • Manufacturer. Enter the manufacturer name.
  • Spec No. Enter either (a) the UNIFORMAT specification number or (b) the Unified Facilities Guide Specifications (UFGS) specification number that the roof section was built to.
  • Description. Enter a descriptive phrase or sentence for the roof section.
  • Protected.
  • Warranty. Enter warranty information such as warranty terms, deductible, and dollar limit. It is a good practice to attach an electronic copy of the warranty (at the roof section's Attachments TAB, not at the "Attachment" field at the Membrane tab).
  • Warranty Expiration. Enter the expiration date for the warranty.
  • Type. Select from the dropdown list.
  • Attachment. From the dropdown list, select the method of attachment for the membrane.
  • Reinforcement. Select from the dropdown list.
  • Surfacing. Select from the dropdown list.
  • Walkways. Check all that apply.

Flashing Tab (Membrane Roof Only)

  • Base. Check all that apply.
  • Adhesive. Select from the dropdown list.
  • Counter-Flashing. Check all that apply.
  • Flashing Types. Check all that apply.

Covering & Flashing Tab (Steep Roof and Metal Panel Roof Only)

Here you can enter or view the following information:

  • Underlayment Type(s). Check all that apply.
  • Roof Covering Type. Select from the dropdown list. For steep roof, this field will be locked.If you activate the lock icon, the following message will display in a popup:

  • Shingle Reinforcement (Steep Roof Only). Select from the dropdown list.
  • Base Metal Type (Metal Panel Roof Only). Select from the dropdown list.
  • Panel Finish (Metal Panel Roof Only). Select from the dropdown list.
  • Attachment. Select from the dropdown list.
  • Flashings/Flashing Types. Do one of the following:
    1. For metal panel roof: Fill in the linear measure of each flashing type that applies.
    2. For steep roof: Mark the checkbox for each flashing type that applies.

Miscellaneous Tab (Steep Roof and Metal Panel Roof Only)

At the Misc tab, you can enter or view the following information:

  • Accessories. Check all that apply.
  • Roof Overlaid.
    1. Select the Yes or No radio button.
    2. If Yes, enter the number of layers in the "Layers" field to the right.

Attachments Tab

The Attachments Tab allows files such as an image file to be attached to the roof section's record. An image might be a picture of the roof section, or it could be an electronic image of written notes or diagrams made during inventory.

The attachment may be in any of the following formats:

  • .jpg
  • .jpeg
  • .png
  • .zip
  • .doc
  • .pdf

Note: As of January 31, 2022, .doc format is allowed, but not .docx.

Attach a File

To attach a file,

  1. At the Attachments tab, activate the Add button in the toolbar below the row of tabs.

    This will open the "Add Image" popup window:

  2. Enter Title and Description information for the attachment.
  3. Activate the Select button to the right of the "Image" field to browse to the desired image or other file.
  4. If you are adding an attachment at the Building level, use the Image Type radio buttons to indicate whether the attachment is a Key Plan, a Section Plan, or "Other."
  5. Once the required attachment data is entered, activate the Save button on the popup toolbar. Alternatively, to close the popup window without attaching the selected file or image, activate the Cancel button.

Inspections Tab

At the Roof Section display screen, the Inspections tab provides the following information for each inspection performed:

  • Inspection Date (Read-Only).
  • Inspection Type (Read-Only). This is a read-only field that reflects the type of inspection: Initial, Visual, or Insulation. Inspection types are discussed in the Roof Section Condition Assessment section in Roof Inspection Types.
  • MCI Membrane Condition IndexMembrane Condition Index (Read-Only). This field displays the calculated Membrane Condition Index for the selected roof section.
  • FCIFlashing Condition Index. In ROOFER the FCI, as determined by inspection and/or algorithm, is used in the calculation of the Roof Condition Index (RCI). (Read-Only). This field displays the calculated Flashing Condition Index for the selected roof section.
  • ICI Insulation Condition IndexInsulation Condition Index (Read-Only). This field displays the calculated Insulation Condition Index for the selected roof section.
  • Est. RSL Remaining Service LifeRemaining Service Life (ERSL) w/o Repair(Read-Only). This field displays the Estimated Remaining Service Life without Repairs.
  • Est. RSL Remaining Service LifeRemaining Service Life (ERSL) with Repairs(Read-Only). This field displays the Estimated Service Life with Repairs.

The Inspections tab at the roof section screen is also where you add an inspection or open an existing one. For how to perform a roof inspection in ROOFER, see Performing Roof Condition Assessments.
