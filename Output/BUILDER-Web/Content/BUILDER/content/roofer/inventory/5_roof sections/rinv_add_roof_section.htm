Add a Roof Section

To add a new roof section to an existing Building, navigate to the Building in the inventory tree, then do one of the following:

  1. Select the Building. Then activate the Add Roof Section icon in the main toolbar.

  2. Or, right-click the Building, then select "Add Roof Section" from the options.

The "Add Roof Section" popup will appear.

Note: The last field (belowRoof Section Type) will vary, depending on the type of roof selected in the field above it.

Enter Required Roof Section Information

This documentation segment describes the minimum data to be entered when creating a new roof section. Other data that can be entered is described in Edit Roof Section Data.

In the popup window shown above, enter the following required data:

  1. Roof Section ID. Enter the ID of the roof section. The ID is limited to one or two alphabetic characters, which must be capitalized.

    Note: A roof section is described by a combination of (1) its ID, (2) its roof section type, and (3) its membrane type or roof covering type (depending on the roof section type). You may have multiple sections of the same roof section type and membrane/roof covering type, but each such roof section must have a unique ID.

  2. Roof Section Area. Enter the area of the roof section you are adding.
  3. Inventory Date (Read-Only). This date is populated with the date on which the roof section was added into ROOFER.
  4. Date Constructed. Enter the date that the roof section was constructed.
  5. Date Last Replaced. Enter the date that the roof section was last replaced, if applicable.
  6. Roof Section Type. From the dropdown list, choose the correct roof section type.
  7. Membrane/Roof Covering Type. From the dropdown list, choose the correct roof membrane or roof covering type.

    Note: the Membrane/Roof Covering Type field will not appear until after the Roof Section Type has been selected from the dropdown list.

Confirm or Cancel

Once the required minimum roof section data is entered,

  1. Activate the Add button on the popup toolbar to create the roof section. Additional details can then be added as described in Edit Roof Section Inventory Data
  2. Alternatively, to close the window without creating the roof section, activate the Cancel button.
