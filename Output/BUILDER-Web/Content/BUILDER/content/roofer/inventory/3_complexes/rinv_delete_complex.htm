Delete a Complex

Permissions Note: This action requires Administrator A user with Administrator Role can administer users, assign users to Roles, and change software configuration settings.A user with BUILDER or ROOFER Administrator Role can administer users, assign users to Roles, and change software configuration settings. permission for the relevant location in inventory.

To delete a Complex, navigate to it in the inventory tree and do one of the following:

  1. (1) Right-click the Complex, then (2) select "Delete Complex" from the options. Or,
  2. (1) Select the Complex, then (2) activate the Delete Complex icon on the menu toolbar.

WARNING: Deleting a Complex deletes all inventory (Buildings, Systems, Components, and roof sections) in the Complex. This is a significant step to take and should only be done when you are certain that you wish to clear the entire inventory of the Complex you have selected.

Best Practice Recommendation: Making frequent backups of your inventory database will protect you from significant data losses if mistakes in deleting are made.
