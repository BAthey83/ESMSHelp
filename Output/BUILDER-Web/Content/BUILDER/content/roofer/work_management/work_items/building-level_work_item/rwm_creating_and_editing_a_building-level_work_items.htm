Add or Edit a ROOFER Work Item

In ROOFER, work items automatically generated at either the Site or Building level. Work items can also be entered manually; however, automatic work item generation must always be done first.

Add a ROOFER Work Item

Generate ROOFER Work Items Automatically (Option A)

To automatically generate work items,

  1. Navigate to the work plan by selecting Work Plan Sustainment Management SystemA multi-year collection of work items and/or projects planned for a Site or Organization. from the main toolbar.

  1. Next, depending on the scope of work items to be generated (for a Building, or for a Site), navigate to the Building or Site containing the roof section(s) that work items are to be generated for.
  2. On the main toolbar, activate Generate Items.

  3. In the "Generate" popup box, after reading the warning, activate the Proceed button. You will then be at the Work PlanA multi-year collection of work items and/or projects planned for a Site or Organization. Items tab.

When generation is complete, you will be at the Work PlanA multi-year collection of work items and/or projects planned for a Site or Organization. Items tab, and see that ROOFER has made designations as to which roof sections should be replaced, and which should be repaired.

Add a ROOFER Work Item Manually (Option B)

Currently in ROOFER, manual entry is available only for Building-level work items. Building-level work items concern modernization, demolition, or replacement of a Building.

To manually create a Building-level work item,

  1. Activate the Add button on the Work ItemA discrete work activity applied to a single managed object such as a Component-Section, intended to improve its life cycle performance. screen.

The "New Work ItemA discrete work activity applied to a single managed object such as a Component-Section, intended to improve its life cycle performance." popup window will appear:

  1. Select the Building Work ItemA discrete work activity applied to a single managed object such as a Component-Section, intended to improve its life cycle performance. radio button option.
  2. Select, from the dropdown provided, the Building associated with the work item.
  3. Activate the Proceed button.

The Building Work ItemA discrete work activity applied to a single managed object such as a Component-Section, intended to improve its life cycle performance. window will appear:

On this screen, the following data for the Building-level work item can be recorded:

  • Description of Work ItemA discrete work activity applied to a single managed object such as a Component-Section, intended to improve its life cycle performance.. Enter a description of the work item. Limit: 255 alphanumeric characters.
  • Work Request ID. Enter your agency's work request number for this work item. Limit: 50 alphanumeric characters. This item of data is not used by ROOFER in any analysis.
  • Status. Select the current status of the work item from the drop-down list of choices. The status will change over time as the work item moves from planning to completion. The values of status are:
    • Awaiting Contract Award
    • Awaiting Funds
    • Budgeted
    • Canceled
    • Completed
    • Deferred
    • Funded
    • In Design
    • In Progress
    • On hold
  • Funding FY. Enter the four-digit fiscal year that the work item was or will be funded.
  • Must Complete as Planned. Check this box if the work item falls into the "must do" category. As discussed above, using this checkbox is one of the ways to ensure the work items is funded and completed.
  • Completion FY. Enter the four-digit fiscal year that the work item was or will be completed.
  • Fund Source. Select the funding source to be used for this work item.
  • Date Completed. Enter the date the work item was completed.
  • Estimated Cost. Enter the estimated cost of the work item. ROOFER provides a means of estimating the initial cost once you have chosen the Cost Book using the Calculate button next to this field.
  • Score. Work items to demolish buildings are not scored by prioritization schemes in ROOFER, so the score is defaulted to 0.000. Special steps to ensure the work item is completed are discussed above at "Must Complete as Planned."
  • Override Automatic Cost Estimate. Check this box if you wish to override the cost estimated using the Calculate button and enter your own estimated cost of the work item.
  • Work Code. Select the work code of the work item. The available work codes are:
    • Sustainment
    • Modernization
    • Demolition
    • New Footprint
  • Activity. Select the activity of the work item from the dropdown list. The available activities are Demolish, Modernize, and Replace.
  • Actual Cost. Enter the actual cost of the work item after completion.
  • Cost Book. Select the cost book from the drop down list to be used in estimating the cost of the work item.
  • ROI Information. Displays information from the cost analysis tab. For more information see Work Item Cost Analysis.

After entering the data for the work item, activate the Save button in the toolbar to save the changes to the work item. If you do not wish to save the changes made to the work item, activate the Cancel button on the toolbar.

Edit a ROOFER Work Item

To edit a Building-level work item, activate either the Repair or Replace link next to the work item, or the Details link next to a Building-level work item on the Work Items screen.

The Building Work Items window will appear and the data described above can be edited as desired.
