Work Configuration Overview

After inventory is completed and roof section condition assessments data are in place, the next step is to perform work configuration to prepare for using ROOFER's work planning tools.

Work configuration is a necessary prerequisite to work planning. During work configuration, you will create prioritization schemes that enable automatic prioritizing of work items according to objectives and measures that you specify in the prioritization schemes. The next step in work configuration is to enter and define funding streams that will be available for funding work items.

Prioritization Schemes

ROOFER offers you the option of automatically prioritizing your work items by specifying prioritization schemes that will automatically rank outstanding work items so that higher priority tasks receive funding first. See About Work Prioritization.

Funding Streams

ROOFER allows you to enter funding sources and the amount of funding from each source. After these funding streams have been entered and defined, you can then activate "Fund Items" during work planning, and all of the work items listed will be labeled with a status of "Awaiting Funds," "Budgeted," "Funded" (for completed work items), or other designation as appropriate.
