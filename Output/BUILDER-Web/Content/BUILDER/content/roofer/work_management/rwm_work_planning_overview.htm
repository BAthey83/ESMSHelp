Work Planning Overview

After inventory is completed, roof section condition assessments data are in place, and appropriate work configuration has been performed, you can begin to truly manage the work using ROOFER's work planning tools.

Regardless of the type of work planned, a work item record is generated for each work activity. A work item stores basic information about what work is planned, including the type of work; description; estimated cost; funding year; and quantity. Groups of work items can be viewed in several useful ways, such as a list of all past, present, and future work, or in a list filtered by aspects such as year or status.

Adding Work Items

There are two ways to create work items. Each method is briefly described here at the headers below, and a link is provided to further information about how to use the automatic work generator in ROOFER.

Using the Automatic Work Item Generator

ROOFER provides an automated process that creates work items for your inventory. For best results, this requires you to specify one or more prioritization schemes in order to rank the work items by score. See Generate ROOFER Work Items Automatically (Option A) in the Add or Edit a ROOFER Work Item topic for instruction on using the automatic work generator.

Manually Entering Work Items

Currently in ROOFER, manual entry is available only for Building-level work items. Building-level work items concern modernization, demolition, or replacement of a Building.
