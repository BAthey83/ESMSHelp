Getting Started with ROOFER

Setting up Your Account

A ROOFER Administrator A user with Administrator Role can administer users, assign users to Roles, and change software configuration settings.A user with BUILDER or ROOFER Administrator Role can administer users, assign users to Roles, and change software configuration settings. will set you up with a user account that has a particular level of user privileges.

Once you are logged into ROOFER, you can access the User Preferences (the link is at the upper right hand corner of the main screen) to do any of the following:

  • change your user email address
  • change your password
  • register your smart card or CAC to be able to use it to log in (optional)
  • change the default view you wish to see when you start the program
  • set your preferred unit of measure (English vs. metric).

To do any of these things, refer to the User Preferences topic.
