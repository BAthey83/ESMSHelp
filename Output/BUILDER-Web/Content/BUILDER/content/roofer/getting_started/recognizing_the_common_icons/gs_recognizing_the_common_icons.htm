Recognizing Common Icons

Listed below are the common command, toolbar, and menu buttons you will encounter when using ROOFER. It's a good idea to become familiar with these icons before using the program.

Inventory Icons

  • - ADD ORGANIZATION: Activate to add a new Organization to the inventory.
  • - DELETE ORGANIZATION: Activate to delete a selected Organization from the navigation tree.
  • - ORGANIZATION: Activate this icon on the navigation tree to access data about a particular Organization. The icon color (Gray, Green, Yellow, Red) reflects the condition of the Organization.
  • - ADD SITE: Activate to add a new Site under an Organization in the inventory.
  • - DELETE SITE: Activate to delete a selected Site from the navigation tree.
  • - SITE: Activate this icon on the navigation tree to access data about a particular Site. The icon color (Gray, Green, Yellow, Red) reflects the condition of the Site.
  • - ADD COMPLEX: Activate to add a Complex to a Site.
  • - DELETE COMPLEX: Activate to delete a selected Complex from the navigation tree.
  • - COMPLEX: Activate this icon on the navigation tree to access data about a particular Complex. The icon color (Gray, Green, Yellow, Red) reflects the condition of the complex.
  • - ADD BUILDING: Activate to add a new Building to a Site.
  • - DELETE BUILDING: Activate to delete a selected Building from a Site.
  • - BUILDING: Activate this icon on the navigation tree to access data about a particular Building. The icon color (Gray, Green, Yellow, Red) reflects the condition of the Building.
  • - ADD ROOF SECTION: Activate to add a roof section to a Building.
  • - DELETE ROOF SECTION: Activate to delete a selected roof section from a Building.
  • - ROOF SECTION: Type: Membrane. Activate this icon on the navigation tree to access data for a particular membrane type roof section.The icon color (Gray, Green, Yellow, Red) reflects the condition of the roof section.
  • - ROOF SECTION: Type: Metal Panel. Activate this icon on the navigation tree to access data for a particular metal-panel type roof section. The icon color (Gray, Green, Yellow, Red) reflects the condition of the roof section.
  • - ROOF SECTION: Type: Steep Roof (Non-Metal). Activate this icon on the navigation tree to access data for a particular steep-roof type roof section. The icon color (Gray, Green, Yellow, Red) reflects the condition of the roof section.
  • - ROLLUP: Activate to manually roll up performance metrics for a selected Building, Complex, Site, or Organization.
  • - REFRESH: Activate to refresh the inventory tree to reflect the latest data edits. Note that refreshing the tree collapses all expanded nodes.

Task and Form Icons

General

  • - SAVE: Activate to permanently save and store changes to the data in the content area or popup window.
  • - COMMENT: Activate to add a comment about a particular inventory item.
  • - NEW: Activate to create a new record. The type of record to be created is determined by where the NEW button is located.
  • - EDIT: Activate to edit a selected record. This icon is available in multiple contexts within BUILDER.
  • - DELETE: Activate to delete a selected record. The record to be deleted is determined by where the DELETE button is located.

  • - CANCEL: Activate to cancel the defined operation.

  • - PROCEED: Activate to proceed with the defined operation.
  • - CLOSE: Activate to close a particular popup window or form.
  • - HELP: Activate to access context specific help within a particular window or form.
  • - CALENDAR: Activate to select a date from a calendar to populate a date type field.

Reports

  • - REPORTS: Activate to access reports available for the item in the content area.

Work Configuration

  • - EVALUATE: Activate to assign weighting points to the various values of a measure. (A measure is a type of objective.)
  • - OBJECTIVE: Objective or sub-objective in a prioritization schema. Activate to access the Objective data.
  • - MEASUREMENT: Measurement belonging to an objective or sub-objective in a prioritization schema. Activate to evaluate a measurement.

Work Plan

  • - GENERATE ITEMS: Activate to generate analysis results. This may be for inspection schedules, work items, or scenarios.
  • - PRIORITIZE: Activate to prioritize work items according to the active prioritization scheme. The prioritization scheme created in the Work Configuration > Work Prioritization area will prioritize and rank your work plan in a quick, objective, repeatable, and representative method with the touch of this button.
  • - RANKINGS: Activate to view the rankings of the work items in your work plan.
  • - FUND ITEMS: Activate to apply funding to work items according to their priority.
  • - EXPORT: Activate to export work items to an Excel spreadsheet.
