Navigating ROOFER

This topic describes elements of ROOFER's entry page, explains main menu options and user links, and introduces the navigation tree.

After you log in to the ROOFER website, a screen similar to the one shown below will appear:

The user interface has the following main areas, each of which is described in greater detail below:

  • User Links (upper right, not shown in illustration above)
  • Main Menu
  • Main Toolbar
  • Navigation Pane
  • Content Area
  • Content Menu

User Links

The User Links area (upper right) offers these options:

  • User Preferences. This option allows you to change the unit of measure; to clear all the locks on the database records; or to change the password to your database.
  • Logout. Selecting this option causes you to log out of (sign off from) ROOFER. After logging out of ROOFER, you will no longer be able to view or edit data until you are logged back in.
  • Help. This option gives you access to a sub-list of multiple user Help options:
    • Context-Sensitive Help. Select this to launch a context-sensitive help file pertaining to the current ROOFER screen.
    • Table of Contents. This is the Contents for the online help documentation. To reach it, first select Index in the dropdown list of Help options. This will bring you to a Welcome page. In the navigation pane at the left you can choose between (1) Contents, (2) the Index, and (3) a Glossary.
    • Index. Selecting this option from the dropdown list of Help options will bring you to a Welcome page. In the navigation pane at the left you can choose between (1) Contents, (2) the Index, and (3) a Glossary.
    • Glossary. To reach the Glossary, first select Index in the dropdown list of Help options. This will bring you to a Welcome page. In the navigation pane at the left you can choose between (1) Contents, (2) the Index, and (3) the Glossary.
    • Contact Help. This option will open an Outlook window for submitting a ticket to ROOFER support. Alternatively, it will enable some other form of access as determined by your specific ROOFER support representative.
    • About ROOFER. Select this to open a popup window showing version information pertaining to your ROOFER software. To close the popup window, select the OK button in the popup.

Main Menu

The main menu has the following options:

  • Work Configuration. This option drops down a sub-menu with two options:
    1. The Prioritization Schemes option allows you to configure prioritization schemes that are used to generate work items.
    2. The Funding option allows you to manage settings for Expected Funds, Funding Restrictions, and Funding Precedence for multiple years into the future.
    In each of these areas, the Return button, located above the navigation pane, will return you to the Main Menu page.
  • Work Plan A multi-year collection of work items and/or projects planned for a Site or Organization.. Selecting Work Plan A multi-year collection of work items and/or projects planned for a Site or Organization. opens a window used to:
    1. Develop work plans by generating and prioritizing work items;
    2. Fund work items (given sufficient permissions level); and
    3. Develop projects based on the generated work items.
    The Return button, located above the navigation pane, will return you to the Main Menu page.
  • Reports. This button on the main menu bar is one of two methods for accessing ROOFER reports available at the current hierarchical level. See ROOFER Reports for more detail about reports.
  • Tools. This option opens a sub-menu with two options. Administration, accessible only by users with Administrator A user with Administrator Role can administer users, assign users to Roles, and change software configuration settings.A user with BUILDER or ROOFER Administrator Role can administer users, assign users to Roles, and change software configuration settings. Role, provides administrative tools for application settings, general security and user settings. Library provides functions to edit cost, inflation and service life data. In each of these areas, the Return button, located above the navigation pane, will return you to the Main Menu page.

Toolbar

The options displayed on the toolbar will vary, depending on the currently active level (Organization, Site, Complex, etc.).

Navigation Pane / Navigation Tree

The left-hand viewing area is the navigation pane. It contains the ROOFER navigation tree.

The navigation tree allows you to select the different levels of the tree and to view data related to that level in the content area.

Shown below is an example of the inventory tree:

Here, the tree can be expanded by activating the "+" next to an inventory level or collapsed by activating the "-" next to an inventory level.

An item in the tree can be selected by clicking on it. Detail information corresponding to that item will appear in the content area. An icon menu appears in the toolbar above the navigation pane; the available selections correspond to the actions that can be taken at the selected level or on the selected item within the tree. (If an icon is grayed out, it means that you do not have the permission level authorized to perform the action.) See Recognizing Common Icons for the icon definitions.

Content Area

The content area displays specific information pertaining to the current item selected in the navigation tree. This area is used to input, edit and work on the items relevant data. The input forms are accessed by selecting folder tabs at the top of the content area. The tabs are specific to the item type selected in the tree or the action selected on the menus.
