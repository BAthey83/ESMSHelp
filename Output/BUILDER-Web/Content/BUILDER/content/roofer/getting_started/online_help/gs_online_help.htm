How to Use Online Help

Online Help Table of Contents

Open Table of Contents

To open the table of contents for the online help documentation,

  1. Activate Help in the User Links area at the upper right of the main screen.
  2. Select "Index" from the dropdown list.

    This will open a Welcome page that has, at the left in the navigation pane, tabs for (1) the online help Contents, (2) the Index, and (3) the Glossary. The Contents tab is open by default.

  1. To expand the table of contents, activate any closed blue book icon to expand that area of the Contents. By contrast, activating a page icon will open that topic.

Hide or Unhide Table of Contents

Hide

To hide the navigation pane containing the table of contents (or index or glossary), activate the arrow button halfway down the window, between the navigation pane and content area:

The content area will expand into the navigation pane space, and a right-pointing arrow button will appear halfway down the left edge of the window.

Unhide

To re-open the table of contents, activate the right-pointing arrow button.

Context-Sensitive Help

Selecting Context-Sensitive Help from the Help dropdown list options should take you directly to a documentation topic that is helpful for a likely task that users might want to perform at the current location. Additionally, the Help Table of Contents will be displayed at the left.

Index

Open Online Help Index

To open the Index,

  1. Activate Help in the User Links area at the upper right of the main screen.
  2. Select "Index" from the dropdown list.

    This will open a Welcome page that has, at the left in the navigation pane, tabs for (1) the online help Contents, (2) the Index, and (3) the Glossary. The Contents tab is open by default.

 

Search the Online Help Index

You can look for an item in the online help index in either of two ways:

  1. Activate the Search box (where the text "Search Index" is initially entered) and type in the item you are looking for.
  2. Alternatively, browse the alphabetical list of index entries.

Selecting a term in the index entries that has a right-pointing arrow to the left of it will show sub-entries for that term. (The white arrow will turn black and tilt when the sub-entries are shown.)

Glossary

To open the Glossary,

  1. Activate Help in the User Links area at the upper right of the main screen.
  2. Select "Index" from the dropdown list.

    This will open a Welcome page that has, at the left in the navigation pane, tabs for (1) the online help Contents, (2) the Index, and (3) the Glossary. The Contents tab is open by default.

  1. To use the Glossary, select the Glossary tab, then enter or select the acronym or term you wish to see defined.

Contact the Help Desk

To send a help request to your SMS Sustainment Management SystemSustainment Management System support representative,

  1. Activate Help in the User Links area at the upper right of the main screen.
  2. Select "Contact Help" from the dropdown list.

Generally a help request is sent through Microsoft Outlook, but the procedure may vary depending on the support procedures your SMS Sustainment Management SystemSustainment Management System provider follows. If you don't have Microsoft Outlook active on your device or computer, a wizard will help you activate it.

See Version Information

To see the version number for your product, and for other "About" information, select the About BUILDER option in the list of Help options. A popup window with the information will appear.

Note: The popup window also provides a link to the SMS Sustainment Management SystemSustainment Management System Support website.

To close the popup, activate the OK button.
