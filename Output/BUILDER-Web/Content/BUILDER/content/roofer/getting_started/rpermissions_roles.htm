Overview of ROOFER Permissions Roles

WARNING: The privileges assigned to each user Role tend to change somewhat over time. Therefore, although the capabilities listed below under each Role will be mostly correct, they might not be 100% accurate at this time.

Each user is assigned to one or more Roles. Roles are allowed varying degrees of program access rights in the software system. The possible Roles include:

Administrator

A ROOFER Administrator A user with BUILDER or ROOFER Administrator Role can administer users, assign users to Roles, and change software configuration settings. can perform the following functions relating to system administration and user management:

  • Administer users
  • Assign users to Roles
  • Change ROOFER application settings
  • Design the specifications to be met by user passwords

To reduce the chance of data loss, the following functions can only be performed by an Administrator A user with BUILDER or ROOFER Administrator Role can administer users, assign users to Roles, and change software configuration settings.:

  • Delete an Organization
  • Delete a Site
  • Delete a Complex

Inventorior

An Inventorior A user with Inventorior Role can add and edit records below the Building level. can perform the following functions:

  • Add a Building
  • Add and delete roof sections
  • Input and edit inventory information at the Inventory screens (all levels)
  • Generate reports
