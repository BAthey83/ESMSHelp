BUILDER Account Setup

A BUILDER Administrator A user with Administrator Role can administer users, assign users to Roles, and change software configuration settings. A user with BUILDER or ROOFER Administrator Role can administer users, assign users to Roles, and change software configuration settings.will set you up with a user account that has a particular level of user privileges. For details on the levels of user privilege, see About BUILDER Permissions Roles.

Once you have an account and are logged in, you can access User Preferences via a link at the upper right hand corner of the home screen to do any of the following:

  • change the unit of measure
  • clear all the locks on the database records
  • change the default view in the navigation pane of BUILDER's main page
  • change the email address associated with your account
  • change the password to your account
  • register a CAC for use to log into BUILDER

To do any of these things, refer to User Preferences.
