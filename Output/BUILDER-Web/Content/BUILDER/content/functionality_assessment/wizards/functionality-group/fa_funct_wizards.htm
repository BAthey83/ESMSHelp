Functionality Wizards

As part of the Functionality assessment provided in the default configuration of BUILDER, there are multiple Wizards to help with assessing specific functionality questions. To access one or more of these, open an existing Functionality assessment, or else start a new one as follows:

  1. Activate the Functionality ribbon in the Navigation pane (it may be semi-hidden at the bottom of your screen).
  2. Navigate to and select the desired Building/Facility or Functional Area.
  3. Verify that you are at the Assessments tab in the content area of the screen.
  4. Activate the button. The "New Assessment" popup will appear.
  5. Select the Functionality assessment set by activating the radio button next to Functionality.

  6. Activate the button at the bottom of the popup. The Functionality assessment will appear. The Functionality assessment is made up of subsections, some of which have a wizard associated with them.

  7. Once your new assessment is created, you can activate one or more wizards from the Response column. Individual wizards in the Functionality group are described below.

The wizards currently available are:

  • Egress Wizard
  • ADA Wizard
  • Seismic Wizard. This wizard is based on the FEMA Federal Emergency Management Agency-154 data collection form.

Egress Wizard

The Egress Wizard is under the Access section of the Functionality assessment and addresses the question "Is exit (egress) from the building quick and easy?"

Activate the Assessment Wizard button in the Response column to activate the wizard.

Expanding and Contracting Sub-Areas

Once you are in the Egress Wizard, if you find that any of the wizard's sub-areas have been compacted (i.e., the arrow points directly to the right, instead of angled down), you can activate the arrow to expand that sub-area and show the questions.

Entering Responses

In the Egress Wizard, most of the responses call for you to select between "Compliant," "Non-Compliant," and "N/A."

Detailed background information for each question, such as tables showing number of exits needed per level of occupancy, can be found at Egress Checklist.

Comments

In the Action column is a button showing a clipboard. To enter a comment, activate the button, enter (and/or edit) the comment, then activate Save at the bottom of the comment pop-up.

If there is a green checkmark next to the clipboard on the button, it means that there is already some comment content.

Save

Save the wizard by activating the Save button at the bottom. Also consider activating the Functionality assessment's Save button (at the top of the assessment) periodically as you work your way through the assessment.

ADA Wizard

The ADA Wizard is in the ADA section of the Functionality assessment and addresses the question "Is the building ADA compliant?"

Activate the Assessment Wizard button in the Response column to activate the wizard.

Entering Responses

In the ADA Wizard, most of the responses call for you to select between "Compliant," "Non-Compliant," and "N/A."

Expanding to see Compliance Requirements

In the ADA Wizard, if you see that the arrow next to a numbered question points directly to the right, this question can be expanded to show all of the requirements needed for the item to be "Compliant." Activate the arrow to expand.

Seismic Wizard

CAUTION: This wizard is under construction. You can browse the list of items on the FEMA Federal Emergency Management Agency-154 data collection form and enter comments (make sure to Save).

The Seismic Wizard is under the Structural Adequacy section of the Functionality assessment and addresses the question "Is the building structurally adequate for all seismic conditions?"

Activate the Assessment Wizard button in the Response column to activate the wizard.

Entering Responses

Currently the only form of entering responses in this wizard is to enter comments (make sure to Save).

Comments

If there is a green checkmark next to the clipboard on the button, it means that there is already some comment content.
