Functionality Assessment Wizards Overview

Building-Level Wizards

As part of the Functionality assessment provided in the default configuration of BUILDER, there are multiple Wizards to help with assessing specific functionality questions. To access one or more of these Wizards, open an existing Functionality assessment, or else start a new one:

  1. Activate the Functionality ribbon in the Navigation pane (it may be semi-hidden at the bottom of your screen).
  2. Navigate to and select the desired Building/Facility.
  3. Verify that you are at the Assessments tab in the content area of the screen. Any existing assessments will show in a table in the content area.

  4. To open an existing assessment, activate the assessment's row in the table, then refer to the assessment type at one of the headings below (no more instructions from this list are needed).
  5. To start a new assessment, activate the button at the upper left of the table. The "New Assessment" popup will appear.
  6. Select Cultural Resources or Functionality by activating the radio button next to it (HPSB High Performance and Sustainable Building assessment does not have a Wizard).

  7. Activate the button at the bottom of the popup. You will be brought to a new instance of the selected assessment type. The types with associated Wizards are described below.

Cultural Resources

If you open an existing Cultural Resources assessment, or create a new one, the following screen appears. Activate the Assessment Wizard button to enter the Wizard. See the Cultural Resources Wizard topic for more details.

Functionality

If you open an existing Functionality assessment, or create a new one, the following screen appears. This assessment type contains multiple Wizards. Activate the Assessment Wizard buttons to enter the associated Wizards. See the Functionality Wizards topic for more details.

Functional Area-Level Wizards

In BUILDER, both the Cultural Resources Wizard and the functionality Wizards associated with the Functionality type functionality assessment (see Functionality Wizards) are available the Functional Area level.
