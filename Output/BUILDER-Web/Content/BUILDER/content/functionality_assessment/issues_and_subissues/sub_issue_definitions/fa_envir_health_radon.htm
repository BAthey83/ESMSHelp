Environment/Health - Radon

Functionality Issue: Environment/Health

Sub-Issue: Radon

Definition:

Radon is present in the building (or functional area).

Severity Levels:

Red - Radon is present in the building (or functional area), posing a health problem.

Amber - Radon is present in the building (or functional area), but health is not a problem.

Green (+) - No radon is present in the building (or functional area).

Examples:

  • Radon is present in the building (or functional area), and it is above the threshold of safety.
  • Radon is present in the building (or functional area), but it is below the threshold of safety.
