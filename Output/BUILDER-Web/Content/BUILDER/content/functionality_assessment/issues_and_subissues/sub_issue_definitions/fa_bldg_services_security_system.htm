Building Services - Security System

Functionality Issue: Building (or Functional Area) Services

Sub-Issue: Security System

Definition:

The security system in the building (or functional area) is inadequate for the associated threat level.

Note: If the building (or functional area) does not have or require a security system, rate this sub-issue "N/A."

Severity Levels:

Red - There is no security system or the security system is inadequate for threat.

Green (+) - The security system is adequate for threat.

Examples:

  • There are an inadequate number of intruder detection devices in use.
  • There is no alarm system, and one is required.
  • There locations of the security cameras in the building (or functional area) are inadequate.
