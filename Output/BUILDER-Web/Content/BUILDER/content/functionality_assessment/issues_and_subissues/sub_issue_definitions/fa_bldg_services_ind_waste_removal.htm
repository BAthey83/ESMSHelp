Building Services - Industrial Wastewater Removal

Functionality Issue: Building (or Functional Area) Services

Sub-Issue: Industrial Wastewater Removal System

Definition:

The industrial wastewater removal system is inadequate in the building (or functional area).

Severity Levels:

Red - There is no industrial waste removal system, or the industrial waste removal system violates code or is out of date/undersized.

Amber - The industrial waste removal system is exempt from current compliance, but future modernization will be required.

Green (+) - The industrial waste removal system is adequate.

Examples:

  • The industrial wastewater removal system has an inadequate capacity.
  • The industrial wastewater removal system is in violation of local codes
  • The industrial wastewater removal system is obsolete.
