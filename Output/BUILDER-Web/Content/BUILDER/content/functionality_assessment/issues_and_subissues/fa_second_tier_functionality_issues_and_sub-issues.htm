Second Tier Functionality Issues and Sub-Issues

Functionality Assessment Set

As stated in the Functionality Assessment Overview, the second tier of functionality assessment is a methodical assessment of a Building's Functional Areas. As in the first tier of assessment, questions are divided into functionality issues (italicized headings), which are further divided into sub-issues. The sub-issues are rated when Perform a Second Tier Functionality Assessment .

The second tier of assessment addresses 13 functionality issues (Cultural Resources and HPSB High Performance and Sustainable Building are first tier only), each with a varying number of sub-issues. Each issue and its associated sub-issues is listed below.

Activate any sub-issue to see its definition.
