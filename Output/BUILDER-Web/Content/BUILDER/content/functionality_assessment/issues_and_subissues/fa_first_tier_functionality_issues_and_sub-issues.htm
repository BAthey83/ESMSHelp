First Tier Functionality Issues and Sub-Issues

Functionality Assessment Set

As stated in the Functionality Assessment Overview, the first tier of the functionality assessment presents a limited set of questions that address general functionality issues throughout the Building. These questions are divided into functionality issues (italicized headings), which are further divided into sub-issues. The sub-issues are rated when performing a first tier functionality assessment.

The first tier of assessment addresses a total of 13 functionality issues (numbered from 1 to 13 below), each with a varying number of sub-issues. Each issue and its corresponding sub-issues are listed below.

Activate any sub-issue to see its definition.

Cultural Resources Assessment Set

This assessment set is separate from the basic Functionality assessment set. It helps determine if significant historic or cultural resources are present.

The functionality issue is Cultural Resources; its sub-issues are listed below.

Activate any sub-issue to see its definition.

  • Programmatic Agreement/MOA/MOU
  • National Register Eligibility
  • Contributing Resource
  • Near a Historic Property
  • Archeological Site
  • NAGPRA Sacred Site
