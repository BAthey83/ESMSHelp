Add Inventory to a Functional Area

Adding BUILDER inventory to a Functional Area allows users to see how various portions of their facility are performing, both in the functionality index as well as the condition index. The Functionality interface allows you to add specific Component-Sections, specific Components, or even entire Systems to a Functional Area.

Prerequisite

Before inventory can be added to a Functional Area, the relevant part of the inventory tree must be filled in down to the Component-Section level.

Add Inventory to a Functional Area Using BUILDER

To add inventory items into a Functional Area,

  1. Select the Functional Area.
  2. Make sure you are at the Components tab. What should I see?A list of Systems associated with the Functional Area's Use Type will appear in the right window pane.
  3. To add one or more System(s), mark the checkbox(es) next to each desired System(s). If this is all you need, skip to the last step.
  4. To add one or more Component(s) not in a selected System, expand the list of Systems as needed to navigate to and select the desired Component(s). How?(1) Activate the arrow left of a System's checkbox to reveal its Components.
    (2) Mark the checkbox(es) next to the desired Component(s).
    (3) Repeat until all desired Components are selected.
  5. To add one or more Component-Section(s) not in a selected Component, expand the list as needed to navigate to and select the desired Component-Section(s). How?(1) As needed, activate the arrow left of a System's checkbox to reveal its Components.
    (2) Activate the arrow left of a Component's checkbox to reveal its Component Sections.
    (3) Mark the checkbox(es) next to the desired Component Section(s).
    (4) Repeat until all desired Component Sections are selected.
  6. When your selections are complete, activate Save.

Add Inventory to a Functional Area Using BuilderRED

To add Inventory items into a Functional Area,

  1. Enter Functionality mode, using either (a) the View option on the BuilderRED toolbar or (b) the Inventory/Inspections/Functionality toggle in the leftmost position on the BuilderRED toolbar.
  2. Activate the "+" next to the Building containing the relevant Functional Area.
  3. Select the Functional Area.
  4. Make sure you are at the Components tab.

  1. To add one or more System(s), mark the checkbox(es) next to each desired System(s). If this is all you need, skip to the last step.
  2. To add one or more Component(s) not in a selected System, expand the list of Systems as needed to navigate to and select the desired Component(s). How?(1) Activate the arrow left of a System's checkbox to reveal its Components.
    (2) Mark the checkbox(es) next to the desired Component(s).
    (3) Repeat until all desired Components are selected.
  3. To add one or more Component Section(s) not in a selected Component, expand the list as needed to navigate to and select the desired Component Section(s). How?(1) As needed, activate the arrow left of a System's checkbox to reveal its Components.
    (2) Activate the arrow left of a Component's checkbox to reveal its Component Sections.
    (3) Mark the checkbox(es) next to the desired Component Section(s).
    (4) Repeat until all desired Component Sections are selected.
  4. When your selections are complete, activate Save.
