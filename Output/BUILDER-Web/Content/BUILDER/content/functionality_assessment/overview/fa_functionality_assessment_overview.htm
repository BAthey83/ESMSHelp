Functionality Assessment Overview

BUILDER has the ability to perform functional assessments on Buildings to measure their "functionality" state. The functionality state relates to the Building's suitability to function as intended and required for mission. The functionality state is distinct from, and determined independently from, the physical condition state that is evaluated using condition assessment.

Although the functionality assessment is not a detailed engineering assessment, it does satisfy the requirements necessary for routine Building facility management activities, including long range budgeting and modernization planning. It also helps to quickly identify problem areas that need more detailed assessments. There may be times when detailed engineering assessments may be required to diagnose specific problems.

Functionality Assessment Tiers

A functional assessment is structured according to a three-tiered, top-down approach that narrows the focus and is able to provide, at each lower level, greater assessment detail to the problem issues identified in the level above it. This approach saves effort, reduces cost, and focuses attention where needed. It also allows the process of identifying and developing modernization requirements to efficiently flow from a broad strategic planning phase to a detailed execution phase. The assessment approach encompasses the three levels shown below.

In BUILDER3.6, functionality assessments can be performed at the first and second tiers (Building and Functional Area levels). Later versions will include a third tier (Component-Section level) for functionality assessments.

First Tier Assessment: Building Level

The first tier of the functionality assessment is a fast, scoping Building-level assessment. Using a limited set of questions (listed in First Tier Functionality Issues and Sub-Issues) that address general functionality issues throughout the Building, the first tier provides a quick and accurate way to

  • Rule out from further assessment issues that are not a problem,
  • Identify issues that will require further assessment at the next tier level, and
  • Compute the Building Functionality Index (BFI Building Functionality Index - computed from the results of a functionality assessment.Building Functionality Index - computed from the results of a functionality assessment.).

This level of assessment should be performed when an overall indication of the functionality of the Building is desired and when areas that will require further assessment need to be identified.

Second Tier Assessment: Functional Area Level

The second tier of the functionality assessment is a methodical Building Functional Area assessment. This level of assessment produces more accurate results than the first tier, but it is also more time consuming and expensive to perform because it is performed multiple times due to multiple Functional Areas. In the second tier, a complete list of functionality questions (see Second Tier Functionality Issues and Sub-Issues) is used to

  1. Identify the specific functionality deficiencies in each Functional Area,
  2. Compute the Functional Area Functionality Index, and
  3. Refine the BFI Building Functionality Index - computed from the results of a functionality assessment.Building Functionality Index - computed from the results of a functionality assessment..

This level of assessment should be used when the functionality deficiencies within Functional Areas need to be identified and when the results of the first tier need to be refined.

Functionality Assessment Sets

At any assessment tier level, when you create a new assessment, the BUILDER Functionality module will ask which type of Assessment Set to create. Each Assessment Set created represents a snapshot of the facility's functionality state of a certain type at a point in time.

The following assessment sets are built in to BUILDER Version 3.6:

  1. Functionality. This is the baseline assessment set in BUILDER.
  2. Cultural Resources.
  3. HPSB High Performance and Sustainable BuildingHigh Performance and Sustainable Building Facility. This assessment set, High Performance and Sustainable Building, examines the area of energy efficiency.

These assessment sets can be created and assessed at either the Building or Functional Area tier (first or second tier).

Functionality / Condition / Performance Index

The functionality index is one of two indexes used to calculate another index called the performance index (the other contributing index is the condition index). The performance index is what determines the color (green, amber, or red) shown for an inventory asset.

In a nutshell, BUILDER calculates the performance index by first collecting the condition index value and the functionality index value applicable to the target inventory asset. Next, the lower index value is multiplied by 60 percent and the higher index value is multiplied by 40 percent. Finally, the two numbers are added together to determine the performance index value for that inventory asset.

Note: Until a functionality assessment is performed on a Building, the functionality index for that Building is 100 percent by default.
