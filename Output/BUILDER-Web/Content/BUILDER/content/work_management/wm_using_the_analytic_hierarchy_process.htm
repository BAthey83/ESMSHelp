Using the Analytic Hierarchy Process

The Analytic Hierarchy Process (AHP) is a mathematical method developed by Dr. Thomas Saaty in the early 1980s to support setting priorities and making decisions. It is a widely used method for multi-criteria decision making. The key advantage of AHP is that it provides a mathematical approach to computing relative weights by using pairwise comparisons based on subjective judgments. BUILDER uses AHP to aid in the process of establishing the weights of objectives and the point values (which are essentially weights) for measure field values.

To use AHP when editing relative weights on the Objective Evaluation screen or when editing point values on the Measure Evaluation screen, mark the Use Analytic Hierarchy Process checkbox and activate the EDIT AHP ANALYSIS button.

A screen similar to the one shown below will appear.

When you first choose to use AHP, all of the records for the comparisons must be created, so please allow some time for this operation. In this example, the 8 possible service life ranges are listed in pairs for comparison. Note that the number of combinations is (8 * 7)/2 = 28. Because of the combinatorics involved in creating all of the pairs to be compared and performing the necessary calculations in refreshing the weight, AHP cannot reasonably be used if you have more that 15-20 items (anything over ~100 combinations). BUILDER will disable AHP when there are more than 25 items. Also note that you will notice a considerable decline in performance speed as the number of combinations increases.

Required Data

Each row of the grid contains a pair of objectives or field values to be compared with each other. Your task is to:

  • Identify which item of the pair is more important, signified by checking the corresponding checkbox of the more important item.
  • Indicate the degree of importance of the checked item over the unchecked item by checking one of the 9 boxes in the right-hand side of the row using the following interpretations;

1 - Value #1 and Value #2 are equally important.
3 - Chosen value is weakly more important than the other.
5 - Chosen value is strongly more important than the other.
7 - Chosen value is very strongly more important than the other.
9 - Chosen value is extremely more important than the other.

Use the even numbers to interpolate between the values above.

When you have completed all of the comparisons, activate the View Scores button to see the results of the AHP computations of weights/points. It is important to note that if you save data at this point, the weights you may have already entered manually will be overwritten and not retrievable.
