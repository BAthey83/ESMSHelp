View a Work Plan or Work Item

View a Work Plan

The individual work item records, when grouped by fiscal year, become an annual work plan that can be viewed in the Work Items window. To view a work plan, first select Work Plan A multi-year collection of work items and/or projects planned for a Site or Organization. Provided by Mike Grussing to AJF in email June 17, 2019. from the main menu.

The Work Items window will appear, initially showing just the top level Organization.

 

Navigating the Work Items

Toolbar

  • Return. Closes the Work Item A discrete work activity applied to a single managed object such as a Component-Section, intended to improve its life cycle performance. Provided by Mike Grussing to AJF in email June 17, 2019. screen.
  • Generate Items. Automatically generates work items.
  • Prioritize. Prioritizes the work items for a work plan.
  • Rankings.Shows rankings of the work items for a work plan.
  • Fund Items. Allocates funding to the work items in a work plan.
  • Reports. Launches the Report Selection tool, which gives you access to a list of standard reports relevant to the work items. See How to Use the Report Viewer.

Changing the Work Items View

Initially, what is shown in the Work Items window is the work plan for the current fiscal year for the entire inventory. To view the work plan for a particular inventory level, navigate the tree at the left side of the window to the desired level. The work plan for the selected inventory level will be shown for the current fiscal year.

Year

To view the work plan for past or future fiscal years, select the desired fiscal year from the dropdown list at the top of the window:

Filter

You can filter by any column or by multiple columns having a blank field and a filter icon below the column header. Enter a search term in the blank field, then activate the filter icon to select a condition such as "Contains" or "StartsWith" or "EqualTo".

Sort

Once the desired work plan year is shown, you can sort it using the column headings. To sort in alphabetic or ascending numeric order, click once on the column you wish to sort by. To sort in reverse-alphabetic order or descending numerical order, click on the column a second time. Initially, the work plan should be sorted by name in ascending numerical order.

View a Work Item

To view details of an individual work item, activate the hyperlink in the Details column.

If the work item is a Section-level work item, you will see the Section-Level Work Item Details. If the work item is a Building-level work item, you will see the Building-Level Work Item Details. If the work item is a project, you will see the Project Details for that project.
