Edit a Preventative Maintenance Inspection

This topic covers how to edit an existing PM inspection Preventative Maintenance InspectionPreventative maintenance inspection that does not use sampling.

Note: An inspection may be viewed at any time. Editing, however, may only be performed within a specified number of days after the inspection date. The number of days is typically 30 by default in BUILDER as delivered to customers, but can be configured by a BUILDER Administrator A user with Administrator Role can administer users, assign users to Roles, and change software configuration settings.A user with BUILDER or ROOFER Administrator Role can administer users, assign users to Roles, and change software configuration settings..

To open an inspection for viewing or editing,

  1. If you are not already in the condition tree, activate the Condition ribbon.
  2. Navigate to the Building containing the desired inspection.
  3. Expand the Building's By System folder.
  4. Navigate to the desired Section and tap (if touch is enabled) or click on its name. In the content area you should see a table listing at least one inspection.

    Note: The oldest inspection was automatically generated by BUILDER as a direct rating when the Section was added to the inventory. No inspector name will be listed for this inspection.

    Because an inventory item is assumed to be new at installation, BUILDER assigns a Green+ rating to this automatically generated inspection.

  5. In the Inspection Date column, activate an inspection date link. The inspection should open. The radio button under the Type heading indicates what type of inspection you are viewing or editing.

Edit PM Inspection Data

The task list in the (vertically) middle portion of the content area is not editable. However, the subcomponent table for the selected Section, in the lower portion of the PM inspection Preventative Maintenance InspectionPreventative maintenance inspection content area, can be edited.

Note: The table displays a description of subcomponents that could be associated with the type of Section being inspected. Not all of the subcomponents listed will necessarily apply to every instance of that Section type.

You have a variety of options to consider as you enter or edit inspection data:

  • Defect Free. If the entire Section is defect free, check the Defect Free checkbox above the table of subcomponents. Once you have done so, skip to Comments and Attachments, below.

    Note: Section-level Defect Free designation takes priority over a designation of N/A (next option in this list).

  • N/A. If one or more of the listed subcomponents are not present in the Section you are inspecting, mark the checkbox in the N/A column for each of these.
  • Direct Rating on Subcomponent. To apply a Direct Rating to a subcomponent, select the appropriate color rating from the dropdown list in the Direct Rating column. After applying a direct rating to each subcomponent desired, proceed to the next option in this list if there are still more subcomponents to be inspected.

    Note: If both a Direct Rating and Criteria are applied to the same subcomponent, the direct rating has priority.

  • Criteria. Using the checkboxes in the Criteria column, you can mark each aspect of the subcomponent that is defective. BUILDER will then calculate the deduct values for each, adjust for the number of deducts as needed, and use these calculations to produce a CI Condition IndexCondition Index value.

    Note: If both a Direct Rating and Criteria are applied to the same subcomponent, Direct Rating has priority.

Comments and Attachments

IMPORTANT: Save your rating and/or Criteria BEFORE entering a comment or adding an attachment. Clicking or tapping elsewhere on the screen before saving can cause you to lose your entries or edit, This includes activating the Comment button on the toolbar.

You can add or modify comments pertaining to the Section being inspected, as outlined in Inspection Comments, and add attachments as described in Inspection Attachments.

Save

IMPORTANT: Save your rating and/or Criteria BEFORE entering a comment or adding an attachment. Clicking or tapping elsewhere on the screen before saving can cause you to lose your edits, and this includes activating the Comment button on the toolbar.

Activate Save on the toolbar to save the inspection.

If you decide to add or edit a comment, save again after doing so.
