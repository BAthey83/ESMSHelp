Building Functionality Assessment Detail

For each Building in the selected scope, this report shows

  • number and name of the Building (gray bar)
  • a brief summary of some general information about the Building, including the functionality index for the Building as a whole (BFI Building Functionality Index - computed from the results of a functionality assessment.Building Functionality Index - computed from the results of a functionality assessment.)
  • a listing of functionality issues, with the Issue FI Functionality IndexFunctionality Index for each.
  • within each issue, a listing of sub-issues and the color rating for each.

Scope

This report is available at the Site, Complex, and Building inventory levels.
