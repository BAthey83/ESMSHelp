Building Metrics Summary

The metrics supplied in this report are the BCI Building Condition Index - a condition rating for the overall Building. For each Building, the BCI is computed by taking the average of its Systems' CIs, weighted by replacement cost.Building Condition Index - a condition rating for the overall Building. For each Building, the BCI is computed by taking the average of its Systems' CIs (SCIs), weighted by replacement cost., BFI Building Functionality Index - computed from the results of a functionality assessment.Building Functionality Index - computed from the results of a functionality assessment., BPI Building Performance Index - a measure of a Building's overall performance, derived from a weighted combination of the BCI and BFI, where the lower of the two values is given two-thirds of the weight and the higher of the two values is given one-third of the weight. If no functional assessment has been performed, the BFI is assigned a value of 100.Building Performance Index - a measure of a Building's overall performance, derived from a weighted combination of the BCI and BFI, where the lower of the two values is given two-thirds of the weight and the higher of the two values is given one-third of the weight. If no functional assessment has been performed, the BFI is assigned a value of 100., FCI, and MDI Mission Dependency Index - measures the relative importance of a BuildingMission Dependency Index - measures the relative importance of a Building. If the report is run at the Site level, the information will be shown Complex by Complex. In the example presented below, "Unassigned" is the Complex name shown in the gray bar because none of the Buildings have been assigned to a Complex.

Note: The Complex name will be shown even if only one of the Buildings in the Complex is shown in the report due to the report being run at the Building level.

Scope

This report is available at the Site, Complex, and Building inventory levels.
