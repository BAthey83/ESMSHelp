Condition Index Performance

This report shows, first in a graph and then in a table, the Condition Index performance over time. The title within the shaded box identifies the subject asset; the Site name is listed upper left. Two samples are shown below: one for the lowest asset level (Component) and one for the highest asset level (Site) available.

Scope

This report is available at the Site through Component inventory levels.
