Condition Index Detail

For each Component-Section included in the scope of the report, this report shows the last inspection date (rightmost column), and the CI Condition IndexCondition Index not only for the Component-Section but also for the Component, System, and Building it is contained in.

Regardless of the level at which the report is generated (Site, Complex, etc.), the name and number of the Site is listed in the upper left of the report, and the name in the gray bar is the Building.

Troubleshooting Tip: If you export this report, then select a different scope to regenerate the report on a different asset, the previously exported report might appear in the "Report Viewer" popup instead of the new report. If this happens, close the report popup and generate the desired report again.

Scope

This report is available at the Site through Component inventory levels.
