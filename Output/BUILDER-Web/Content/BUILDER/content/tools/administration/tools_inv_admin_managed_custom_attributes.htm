What are Admin Managed Attributes?

Admin managed attributes are a collection of pre-named asset attributes that have been determined by the BUILDER instance's owner (for example, a corporation, or the Air Force, or the Army). They are templated or common attributes that are seen as likely to be used often. Some or all of these attributes are made available from the outset after installing BUILDER. Add an Admin Managed Attribute

Under Admin Managed Attributes at the bottom half of the Attributes screen, you can add, edit, or delete custom Admin managed attributes for this level of inventory. Also, in the Unassigned Admin Managed Attributes column, you can choose to assign an existing but unassigned Admin managed attribute to the asset whose data screen you have open.

When you add an Admin managed attribute, it will automatically be assigned to the asset where you are adding it. For all other assets at the same inventory level, the new attribute will be available in the Unassigned Admin Managed Attributes column.

To add an Admin managed attribute,

  1. Directly under the Admin Managed Attributes header at the lower right of the Attributes screen, activate Create New Managed Attribute. This will generate a new row in the Admin Managed Attributes table.
  2. Enter an Attribute Name and Value for your attribute into the table.
  3. If you want to apply an architect name, press the Enter key at this point . The popup "Add Architect" will appear. Enter the name of the architect, then select OK.
  4. When finished with the entry, activate the blue checkmark icon at the left side of the row to save the entry. Alternatively, to cancel and delete the entry before saving, activate the red circle with a slash icon (located to the right of the blue checkmark).

    CAUTION: Your new attribute will not be saved if you (a) exit without activating the blue checkmark or (b) initiate adding a new custom inventory attribute (by activating the Add New Attribute button) before saving with the blue checkmark.

Edit an Admin Managed Attribute

To edit an attribute,

  1. Activate the pencil icon in the attribute's table row, and wait for the row to change appearance.
  2. Make the desired changes.
  3. Finally, activate the blue checkmark icon at the left to save the changes (or the red Cancel icon to cancel the change).

Delete an Admin Managed Attribute

When you "delete" an Admin managed attribute it is removed from the Admin Managed Attributes table for the asset, and moved to Unassigned Admin Managed Attributes.

To delete an attribute,

  1. In the table of Admin managed attributes, locate the row containing the attribute you wish to delete.
  2. To the right of the Value field, activate the Delete icon (red X). A confirmation popup will appear, asking "Delete record?"
  3. Select OK to delete, or Cancel to stop the process.

After deletion, the attribute will be transferred to the Unassigned column for that asset.

Assign an Unassigned Attribute to Admin Management

You may see one or more items listed under the header Unassigned Admin Managed Attributes at the lower left of the Attributes screen. If so, these are available to be applied to the selected asset.

To assign an asset to Admin management,

In the Unassigned Admin Managed Attributes column, activate Assign to Asset.

This will immediately populate the attribute to the Admin Managed Attributes table instead.

The assignment action is reversible by deleting that attribute from the Admin Managed Attributes table. See Delete an Admin Managed Attribute for instructions.
