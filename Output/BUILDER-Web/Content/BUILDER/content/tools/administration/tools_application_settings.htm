When you create a new inventory database, you should set the preferences in the Application Settings window before proceeding with any additions to inventory. To edit the application settings, select Tools > Administration > Application Settings starting at the main menu.

The "Application Settings" popup window will appear. Subsequent sections explain the following tabs in the Application Settings window:

  • Settings
  • Systems
  • License
  • Advanced
  • User Agreement
  • Security Label
  • Email
  • Session Info
  • Building Templates

Settings Tab

Tip: If you do not see a screen similar to the one above, try selecting a different tab and then coming back to the Settings tab.

At the Settings tab, you can set the following preferences for the database:

  • User Category
  • User's Default Unit of Measure
  • Fiscal Year Start Date
  • Critical Work Item Autocreation

User Category

After you designate a user category, BUILDER will present an agency-specific list of Building Use types when inventory is being added. The available user categories in BUILDER are:

  • Army
  • Navy
  • Air Force
  • Civilian
  • Department of Energy
  • All

Choosing the All category allows you to enter Building Use types across multiple categories.

Note: Once your inventory contains Buildings from more than one category, the only choice here is All.

After changing user category, activate Save on the toolbar. If finished with application settings, then activate Close.

User's Default Unit of Measure

Here is where you select the default unit of measure (English or Metric) to be used for the inventory for all users. This is the unit of measure that will initially be shown for all users, but each user can change his or her units of measure at any time with the User Preferences tool when they are logged in.

After changing the default unit of measure, activate Save on the toolbar. If finished with application settings, then activate Close.

Fiscal Year Start Date

At this tab, enter the fiscal year start date. When running simulations using Scenarios, the fiscal start date determines (1) when items are estimated to be completed, and (2) when new work items will be evaluated.

After entering or changing the date, activate Save on the toolbar. If finished with application settings, then activate Close.

Clear All Record Locks

After reading the warning note below, activate the Clear All Record Locks button to clear all the locks placed on Building and inspection records by individual users and BuilderRED (BRED BUILDER Remote Entry Database (BuilderRED)BUILDER Remote Entry Database (BuilderRED)). Whereas the Clear Locks button in the User Preferences window only clears locks on records locked by that user, the Clear All Record Locks button on this administrative screen unlocks all locked records, regardless of how they were locked.

WARNING: To protect data accuracy, be sure that the following two steps are implemented BEFORE the record locks are cleared!

  1. All users should exit from the BUILDER program.
  2. All BRED BUILDER Remote Entry Database (BuilderRED)BUILDER Remote Entry Database (BuilderRED) files should be imported.

Record locking is used to protect data from inadvertent overwrites by multiple users. BUILDER also locks records when Buildings are exported to BuilderRED, to prevent data from being changed in the BUILDER program while inspection is in progress. To protect data accuracy, all users should exit from the BUILDER program and all BRED BUILDER Remote Entry Database (BuilderRED)BUILDER Remote Entry Database (BuilderRED) files should be imported before the record locks are cleared.

Configure Cost Modifiers

Custom Flag Option

The Enable Custom Flag checkbox allows you to toggle on/off a customized true/false column displayed for cost modifiers. If the Enable Custom Flag checkbox is marked, an additional column will be displayed in both the Cost Modifier Library window and at the Cost Modifiers tab of Sites, Complexes, Buildings, Systems, Components, or Sections. The column heading will be the label text entered in the Custom Flag Label field.

Marking the checkbox and entering a label does not affect cost modifier calculations; it is simply a way to record a Boolean value (true/false) that organizations can use as a flag of their choosing. For example, if an organization wants to track whether a cost modifier has received management approval, they might enable a custom flag with a label of "Approved."

Tip: You can retain the label in the Custom Flag Label field even if the Enable Custom Flag checkbox is not marked. The customized column will not appear for cost modifiers, but if you want to reinstate the column, the label is ready.

After making any changes to the checkbox or label field, activate Save on the toolbar. When finished with application settings, activate Close.

Disable Cost Modifier Features Checkbox

Marking this checkbox will disable cost modifier features. After marking or unmarking, activate Save on the toolbar. When finished with application settings, activate Close.

Email Tab

At the Email tab are options that configure automated response to users who have difficulty logging in and activate the request to change their password or to have their account unlocked.

The large text box is the location where you input the text to be sent in automated email to the user. Variables that can be used in the text are as follows:

  • <%ApplicationName%> = name of the application ( BUILDER or ROOFER)
  • <%Url%> = link for the user to follow to complete the password action
  • <%SupportEmail%> = email address of the application support representative

If you edit the automated response message, activate Save on the toolbar. When finished with application settings, activate Close.

Session Information Tab

The Session Information tab (abbreviated Session Info) shows session information such as number of users logged in.

Building Templates Tab

At this tab you can configure the percentage difference allowed between the square footage of a Building template and the square footage of the Building it will be applied to.

After changing the percentage, activate Save on the toolbar. When finished with application settings, activate Close.
