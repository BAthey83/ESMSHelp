Perform Manual Condition Rollup

This topic explains how to perform a global manual condition rollup that updates all condition index values for the entire inventory. A global rollup may only be performed by a BUILDER administrator.

When an inventory item is "rolled up," BUILDER calculates current CI Condition IndexCondition Index values for all inventory contained in that asset, from the Component-Section level on up. (For a description of this process, see the Condition Assessment Overview topic.)

A global manual condition rollup, covering all inventory in the tree, may only be performed by a BUILDER administrator. As an administrator you should perform this rollup calculation frequently, but especially when:

  • You have not performed the rollup within the last 6 months
  • You have entered a large number of new condition assessments
  • You need the most current condition index measures to develop a new work plan.

Steps

CAUTION: Depending on the quantity of inventory, this process can take many hours.

To perform the global manual rollup process,

  1. Select Tools > Administration > Manual Condition Rollup from the main menu.

  2. When presented with the confirmation popup, select OK. (Confirmation popup is new starting in BUILDER Version 3.6.1.)

Results

The rollup process will begin for all inventory in the database, and the progress popup will appear.

After the rollup process completes, the progress popup will close and you will be returned to the main BUILDER screen. Building inventory displays will show recalculated current condition indices for all elements.

What Does a Global Rollup Do?

The following processes will occur during the rollup:

  • Retrieve the latest condition assessment records and estimate a current CI Condition IndexCondition Index for all Component-Sections.
  • Estimate the current remaining service life (RSL Remaining Service LifeRemaining Service Life) for all Component-Sections. The RSL Remaining Service LifeRemaining Service Life is adjusted by the rate of deterioration in the CIs.
  • Estimate the current CI Condition IndexCondition Index for all Components by calculating the average of their Component-Sections' CIs, weighted by replacement cost.
  • Estimate the current CI Condition IndexCondition Index for all Systems by calculating the average of their Components' CIs, weighted by replacement cost.
  • Estimate the current CI Condition IndexCondition Index for all Buildings by calculating the average of their Systems' CIs, weighted by replacement cost.
  • Estimate the current CI Condition IndexCondition Index for all Complexes, Sites, and Organizations by calculating the average of their Buildings' CIs, weighted by replacement cost.
  • Retrieve the latest functionality assessment records and determine Building FIs and Functional Area FIs based on the results of the assessments.
  • Estimate the current FI Functionality IndexFunctionality Index for all Complexes, Sites, and Organizations by calculating the average of their Buildings' FIs, weighted by replacement cost.
  • Estimate the current PI Performance Index. See also: BPI.
