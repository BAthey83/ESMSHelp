Manage Inflation Rate Data Using Inflation Books

When costs are computed by the BUILDER cost module, an adjustment is made to account for inflation.

The inflation data used by the cost module is contained in one or more inflation books. A default inflation book is provided and is all that is required. Optionally, multiple customized inflation books can be associated with different Organizations or Sites in the tree; this topic will explain how to do this.

For more information about the cost module, start with the topic About Cost Books.

About Inflation Books

The inflation assumptions used by the cost module are contained in one or more inflation books. Multiple inflation books can be associated with different Organizations or Sites in the tree, as desired.

Default

The default inflation book provided with BUILDER is named "Reference." It cannot be edited, and it is associated with the top level of the asset tree. The scope of its data, therefore, is the entire asset tree.

The data in the Reference set matches the inflation rates used by the Department of Defense for operations and maintenance costs. As time goes on and the economy's ups and downs yield different forecasts of inflation, you should create new inflation sets to more accurately predict out-year costs.

Custom

To use different inflation data than what is provided in the Reference module, you can create one or more custom inflation books. A new inflation book can be based on Reference, or based on another existing inflation book. Another option for using different inflation data is to edit an existing inflation book (except for Reference).

Depending on the scope you designate for an inflation book when you assign it to an Organization or Site, you can apply custom values to either all or part of the asset tree.

Managing Inflation Rates

To manage the inflation rates, select Tools > Library > Inflation Books starting at the main menu.

The Inflation Set Management window will appear.

Expand the tree in the left portion of the window to show the available inflation sets in the database. As stated above, only the References inflation set is available initially. The Reference inflation set cannot be deleted or edited and serves as the foundation for all other inflation sets you create. You are free to use the Reference inflation set if you like and not add any data of your own.

Adding and Editing Inflation Rates

If you would like to edit the inflation rates, you must add your own data set first. To do so, choose the Organization or Site where it applies from the tree, then activate the Add button on the toolbar. The New Inflation Set window will appear:

Enter the name of the new inflation set, choose the set to copy inflation rates from, and activate Add. You will be returned to the Inflation Set Management window, the name of the new book will appear in the Inflation Set dropdown list, and you will see the data corresponding to the book you copied from. You may know edit the inflation rates on a year-by-year basis for the new set in the grid. To edit different inflation sets in your inventory, select them in the tree and their data will appear in the grid.

Choosing the Default Inflation Rate Data Set

After a new inflation set is created, it is available from the inventory screen to use for a given Site, region, or Organization.
