Delete a Cost Modifier Library

To delete a cost modifier library,

  1. Select Tools > Library > Cost Modifiers from the main menu.
  2. Expand the navigation tree at the left to show existing cost modifier libraries.
  3. Select the library you wish to delete. The Cost Modifier Library tab will be displayed in the content area.
  4. Activate the Delete button on the toolbar.
  5. In the confirmation popup window, activate the OK button. (Alternatively, activate Cancel to abandon the deletion.)
