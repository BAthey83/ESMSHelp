Family Housing

Outdated-material.

The Family Housing window is can be opened from the building inventory screen by activating the button adjacent to the Family Housing checkbox.

Toolbar

  • Close. Use this button to close the Family Housing Information window.
  • Save. Use this button to save changes made.
  • New. Use this button to create a new family housing record. Once a family housing record exists for this building, the New button will be disabled, i.e. a building can have at most one family housing record.
  • Delete. Use this button to delete the current family housing record.
  • Comment. Use this button to add, edit, and view comments about the family housing.
  • Help. Use this button to launch the help file associated with family housing.

Family Housing Data

The data, all of which is optional, that can be entered on this screen includes:

  • Type of Dwelling. Select the type of family housing:
    • Single Family Dwelling
    • Multi-Family Unit
    • Apartment Building.

If a non-single family dwelling is chosen, you will have an opportunity to identify each dwelling unit and to list each unit's room counts and appliances.

  • Classification. Select the housing classification from the dropdown list.
  • Type. Enter a description of the type of dwelling, usually style or standard design description. Limit: 15 characters.
  • House Number. Enter this building's housing number if you use a local numbering system for housing. This may differ from the building number.
  • Unit Number. Enter the unit number or apartment number of the current unit. This field only appears if the Type of Dwelling is not single family housing.
  • Number of Rooms
  • Total Rooms. Enter the total number of rooms in a single family dwelling. If the building is multi-family or an apartment building, you will be describing each dwelling unit.
  • Bedrooms. Enter the total number of bedrooms in a single family dwelling. If the building is multi-family or an apartment building, you will be describing each dwelling unit.
  • Full Baths. Enter the total number of full baths in a single family dwelling. If the building is multi-family or an apartment building, you will be describing each dwelling unit.
  • Half Baths. Enter the total number of half baths in a single family dwelling. If the building is multi-family or an apartment building, you will be describing each dwelling unit.
  • Appliances. List the appliances in the current dwelling unit of the building. Use the ADD APPLIANCE and DELETE APPLIANCE buttons at the top of the grid to add and delete appliances.
  • Type. Select the type of appliance in the dwelling unit from the drop-down list.
  • Make. Enter the make/manufacturer of the appliance, limited to 10 characters.
  • Year. Enter the 4-digit year in which the appliance was installed.
  • Warranty Date. Enter the warranty date of the appliance using the dropdown calendar
  • Comment. Enter a comment regarding the current appliance.
