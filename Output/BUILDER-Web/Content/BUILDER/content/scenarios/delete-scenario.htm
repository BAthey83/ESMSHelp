To delete a scenario,

  1. Select Scenarios > Management from the main menu.
  2. In the left pane, navigate to the desired level in the Scenarios tree.
  3. In the table of scenarios in the content area, select the scenario you wish to delete.
  4. On the toolbar, activate the Delete button.
  5. In the confirmation popup, respond to "Are you sure you want to delete the selected scenario?" by activating the Yes button.
