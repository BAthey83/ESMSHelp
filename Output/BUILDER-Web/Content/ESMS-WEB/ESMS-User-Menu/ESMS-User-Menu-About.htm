Understanding the User Preferences Menu Options

This topic explains the options available from the User Preferences menu at the far right on the main menu.

To see the options available at the user menu,

  1. Navigate to the far right of the main menu to where you see the icon.
  2. Click the icon to activate the popout menu and view the available options.

See the sections below for explanations of each option.

Save Default Workspace

To make your current setup of site and domain(s) the default that will be used for future logins,

  1. Activate the "Save Default Workspace" option in the user menu. A confirmation popup will appear, reporting "Saved current workspace settings.
  2. Activate OK in the popup.

Even if you change your site and/or domain selections before you log out, when you log in again you will have the selections that were in force when you activated "Save Default Workspace".

Application Sync Settings

The "Application Sync Settings" option is where you choose which permitted site (or group) will be the one whose information is transferred to and from your Desktop or Tablet version of Enterprise SMS. This is called your "Desktop sync target". If, for example, you choose Site 2 as your Desktop sync target, then that is the site that will open for you in your when you open your Enterprise SMS Desktop instance.

IMPORTANT: You login with the same username and password for Enterprise SMS Web, Desktop and Tablet.

Option A: Site as Sync Target

To set a site as your Desktop sync target,

  1. Activate the "Application Sync Settings" option in the user menu. The "Select Desktop Sync Target" popup will appear.
  2. Activate the button to the right of "Site". This will be either the name of the currently selected site, or None Selected.

    Note: The "Select Site" popup that appears is the same as the one used to select a site at the main menu, and the same search options are available if you want to narrow the selection displayed in the "Select Site" popup.

  3. Select the desired sync site.
  4. Activate the Set Sync Target button at the bottom of the popup. (Alternatively, activate Cancel to cancel selection.) You will be returned to the "Select Desktop Sync Target" popup.

    Troubleshooting Tip: If the Select button is grayed out, a site is perhaps not selected.

Option B: Group as Sync Target

A mutually exclusive option to selecting a site as a sync target is to select a group as your sync target. To set a group as your Desktop sync target,

  1. Activate the "Application Sync Settings" option in the user menu. The "Select Desktop Sync Target" popup will appear.
  2. Activate the button to the right of "Group". This will be either the name of the currently selected group, or None Selected. The "Select Group" popup will appear.
  3. Select the desired group to be your sync target.
  4. Activate the Select button at the bottom of the popup. (Alternatively, activate Close to cancel selection.) You will be returned to the "Select Desktop Sync Target" popup.

    Troubleshooting Tip: If the Select button is grayed out, a group is perhaps not selected.

  5. If, at the Select Desktop Sync Target popup, you want to cancel the whole sync target operation, activate Cancel. Or, if you want to cancel just the group selection, activate the "x" button to the right of the group name to deselect it:
  6. To proceed with setting the sync target, activate the Set Sync Target button.

Option C: Work Plan as Sync Target

To set a work plan as a Desktop sync target,

Activate the Application Sync Settings option in the user menu. The "Select Desktop Sync Target" popup will appear.

  1. Activate the button to the right of Work Plan. The Select Plan popup will appear.
  2. Enter the name of the plan you wish to use as your sync target and select Go.
  3. Choose the plan you want from the list of results and click Select at the bottom of the popup. (Alternatively, activate Close to cancel selection.) You will be returned to the "Select Desktop Sync Target" popup.

Troubleshooting Tip: If the Select button is grayed out, a group is perhaps not selected.

Download (Desktop) Application

To download either the Desktop version of Enterprise SMS, select the "Download Application" option in the user menu, then follow instructions in the topic Download Enterprise SMS Desktop or Tablet.

SMS Support

If you have any questions regarding Enterprise SMS, you can contact SMS Support via the "SMS Support" option in the user menu. After selecting this option, you will be redirected to the SMS Support website.

Click the "Contact Us" button in the upper-right corner and complete the form with as many details as possible. The SMS Support team will be happy to assist you.

Logout

To log out of Enterprise SMS Web, select the "Logout" option in the user menu.
