NOT IN USE: Managing Users and Roles (Security Screen)

This topic introduces tasks you can perform at the Security screen and provides links to greater detail for each.

What Can I Do Here?

At the Security screen you can use the Security Editor to add and delete users, create user roles with custom security permissions, and assign roles to users.

Security Tasks

Permissions Note: These security tasks require Enterprise SMS admin privilege.

At the Security screen you can

  • Add a user
  • Delete a user
  • Create a role
  • Assign a role to a user
