Manage Search Tags

This topic explains how to

  • add a tag to the pool of available document search tags
  • delete a tag from the pool of available document search tags

For how to associate a tag with a document stored in Enterprise SMS or how to remove the association, see the topic Tag Documents.

Enterprise SMS maintains a list of user-created tags that can be used to classify and search for documents. In this documentation, this list is referred to as the tag pool.

When you associate tags with documents at the Selection Bulk Edit accordion button, the tag pool governs what tag options will be presented. Therefore, if you don't see a tag you want to use, you can go to the Tag Editor accordion button and follow the directions at Add a Tag to the Tag Pool below to add the desired tag to the tag pool.

At the Tag Editor, you can also delete a tag from the tag pool (after deleting any associations it has with one or more documents).

Add a Tag to the Tag Pool

A few tips about tags:

  • Strive for consistency, e.g., singular vs. plural ("report" vs. "reports")
  • Using all lower case for tags is standard practice.
  • Keep tags short--one or two words is ideal, though not always possible.

Note: As of 3/29/2018, a one-word tag automatically generates as soon as you click on the space bar

 

To add a tag to the tag pool,

  1. Activate the Tag Editor accordion button if that section isn't already open. A list of currently available tags should appear, followed by "+Tag".
  2. Activate "+Tag". A blinking cursor should appear at the beginning of the "+".
  3. Type the name of a new tag.

    Tip: If using multiple words (max of two is recommended), do not use a space between them.

  4. Press Enter. The new tag should appear in the list.
  5. Refresh the browser to see the update (In Windows, Ctrl-R will refresh).

Edit a Tag in the Tag Pool

There is no functionality for changing the name of a tag in the tag pool.

If the number of documents associated with the undesired name is small enough to display on just one screen, you can keep them on the display and do the following:

  1. At Selection Bulk Edit, Tag Documents from each document (as of 2018, this needs to be done one record at a time).
  2. Open the Tag Editor.
  3. Add a Tag to the Tag Pool.
  4. Checkmark the documents that you want the replacement search tag applied to.
  5. With the documents checkmarked, activate the replacement tag under the heading "Multi-Select Tags" at Selection Bulk Edit.
  6. Refresh the browser to see the update (In Windows, Ctrl-R will refresh).

Delete a Tag from the Tag Pool

Delete Associations

Note: To delete a tag from the tag pool, you must first delete wherever that tag is associated with a document.

To delete all instances where a tag about to be deleted is associated with a document,

  1. Enter the name of the tag in the "Tag" field under the "Document Search" heading.
  2. Activate the Search button. This will list all documents that are tagged with that term.
  3. Checkmark the first document in the list.
  4. Activate the Selection Bulk Edit accordion button if that section is not already open.

Delete the Tag from Tag Pool

After all associations of the tag with documents have been removed, continue as follows:

  1. Activate the Tag Editor accordion button if that section is not already open.
  2. Activate the "X" portion of the tag you want to delete from the tag pool.
  3. Refresh the browser to see the update (In Windows, Ctrl-R will refresh).

Troubleshooting tip: If the tag reappears after the browser refresh, it probably still has an association with one or more documents.
