Tooltips for Home tab

Load Database:

To load a database you'll need the following for Microsoft SQL Server Login:

  1. Server Name
  2. Database Name
  3. Authentication type (Window Account or custom credentials)
  4. Username
  5. Password

 

Sync:

Sync with Enterprise SMS Web version

 

Import:

Current Import options are:

  1. GIS
  2. Trimble
  3. Windows Tablet

 

Export:

Current Export options are:

  1. Trimble
  2. Windows Tablet
  3. MUFD (Mass Upload File Definition)

 

Facilities:

Clicking Facilities here will create a new Navigation tab below.

If there is already a Facilities tab under Navigation, clicking Facilities will do nothing.

 

Groups:

Clicking Groups here will create a new Navigation tab below.

If there is already a Groups tab under Navigation, clicking Facilities will do nothing.
