What Can the Enterprise SMS Field Application Do?

Using the field application version of Enterprise SMS, you can

  • enter real property inventory data (Desktop)
  • associate images with inventory at the component level (Desktop)
  • perform assessment plans in the field (Tablet)
  • enter assessment data (Desktop)
  • search data
  • run reports
  • generate work items based on criteria $$ M2--this accurately describes Desktop functionality too, Yes??

The field application is not intended for

  • administering permissions for individuals and teams
  • document storage and retrieval other than associating images with inventory at the component level

Topical Overview of Web and Desktop Capabilities

  • Real Property Inventory
    1. For searching and viewing real property inventory, Enterprise SMS Web will suffice if all the inventory you need is kept up-to-date on your Web instance by other parties.
    2. However, in order to add or change inventory entries, you definitely need to use Enterprise SMS Desktop in addition to Enterprise SMS Web.
  • Associating Documents with Components
    1. Documents such as images can be imported into Enterprise SMS and "attached" to the relevant components in inventory only in Enterprise SMS Web. $$no, I don't think this is true...change in Enterprise SMS Web doc'n, too, as needed.....
    2. Enterprise SMS Desktop also has functionality for storing documents in association with components. These are synchronized to and from Enterprise SMS Web via the server (see Integration Between Enterprise SMS Web, Desktop, and Tablet).
  • Associating Components with Groups
    1. Enterprise SMS Web has no tools for associating components with groups. This is done in Enterprise SMS Desktop.
    2. In Enterprise SMS Desktop, a component can be associated with one or more groups in the "Add Component" popup when creating a new component.
  • assessment plans
    1. assessment plans can be viewed in Enterprise SMS Web.
    2. To create assessment plans, you need to use the Enterprise SMS field application. assessment plans can be entered into Enterprise SMS Desktop, or performed in the field using Enterprise SMS Tablet.
  • Work Items
    1. Work items can be generated in Enterprise SMS Web, in the Work task area, by specifying threshold levels for condition index and remaining service life.
    2. To enter a work item manually, you need to use Enterprise SMS Desktop.

Integration Between Enterprise SMS Web, Desktop, and Tablet

As shown in Diagram 1, Enterprise SMS Web, Enterprise SMS Desktop, and Enterprise SMS Tablet are each connected to a central server. In the case of Enterprise SMS Web, synchronization of data occurs automatically. With Enterprise SMS Desktop and Enterprise SMS Tablet, however, you must initiate a synchronization to upload information (such as inventory that has been entered, or data on assessment plans that have been performed), or to download the latest information.
