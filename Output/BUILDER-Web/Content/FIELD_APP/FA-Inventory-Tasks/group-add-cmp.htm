Add Component to a Group

First, choose from the following methods to add a component to a group:

  • add while creating the component
  • edit the component's properties listing
  • use the group editor (recommended for adding multiple components to a group)

Add Component to Group During Creation

To add a component to a group while adding the component to inventory,

  1. In the "Add Component" popup, navigate to the Groups fieldset and expand it.
  2. In the group-type field that contains the desired group, click the dropdown arrow at the right. This will display the groups belonging to that group type. If no options appear below the group-type field, this means no groups of that type have been established.
  3. Select the desired group.
  4. Repeat the previous two steps as needed if you want to add the component to additional groups. Because a component can belong to only one group in any given group type, each additional group must belong to a different group type.
  5. When finished with that as well as any remaining data entry for the new component, activate the Add button at the bottom of the popup.

Note: A component can belong to only one group in any given group type.

Add Component by Editing the Properties Listing

To add a component to a group by editing the component's properties listing,

  1. Select any one of the top tabs.
  2. In the navigation pane (open it if it is auto-hidden), activate the Systems tab.
  3. In the asset tree, double-click the component you want to add to a group (Alternatively, select the component, then activate Edit Component in the ribbon). The component's View Component tab will appear in the content area.
  4. At the View Component tab, expand the Groups fieldset.
  5. Activate the dropdown arrow for the desired group type (The example below shows the dropdown list for the group type "Area"). This will display the groups. If no text appears in the group-type field or the dropdown, that means no groups of that type have been established.

  6. Select the desired group from the dropdown list. The selected group will be highlighted.
  7. Activate the Save button (located above the Basic Properties fieldset).

Note: A component can belong to only one group in any given group type.
