About Groups and Group Types

What Is a Group?

$$brief description here

Some features of groups:

  • Currently, only components can be added to a group.
  • A group must always be created as a member of a group type.
  • A group is not limited by either system or facility boundaries; members of a group can be selected from a variety of systems and from more than one facility.
  • A component can be added to more than one group. For example, a fire alarm component might be added to a group named Safety, and also to a group consisting of all components located on the second floor of a particular building.
  • Users can create and delete groups, but not group types.

What Is a Group Type?

Group types are created and used for organizing groups.

Permissions Note: Only an Administrator can create or delete group types. This is done in Enterprise SMS Web.

Group Tasks

Tasks you can perform on a group are

  • View
  • Add
  • Add Component to Group
  • Edit
  • Delete
