Add Group

To add a group,

  1. Select the Tools tab.
  2. Open the Group Editor from the ribbon.
  3. At the Group Editor tab, select the appropriate group type (see the red "1" in the screenshot). The selected group type will be indicated by a right-facing triangle in the leftmost column of the content pane.

  4. In the "Group Name" field, type a name for the group (see the red "2" in the screenshot).
  5. Activate the Add Group button (see the red "3" in the screenshot).
