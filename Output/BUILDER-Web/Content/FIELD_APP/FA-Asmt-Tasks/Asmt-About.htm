About assessment plans

The following assessment types are offered in Enterprise SMS:

  • Direct Rating
  • Functional Assessment

Functional assessment plans are customized to the organization using Enterprise SMS Web. Administrator status is required for composing the assessment questions and the answer options.

Direct ratings use a color-coded system that make evaluation quick and easy for an experienced assessor. In the software, each color corresponds to a specific numerical value on a scale of 1 to 100. This value is the component's condition index (CI). Each component's CI will aggregate in weighted fashion (($$ engineer question: weighted by $ value of the component, right? Or am I driving by the rear-view mirror?)) up through whatever groups, system, facility, and site the component is a member of, to influence the CI of those assets as well.
