Purpose of this Guide

The BUILDERROOFER Inventory Guide assists BUILDERROOFER™ users with their tasks as they enter roofing inventory information.


About the Guide Series

This guide is first in a series of user guides covering the capabilities and resources provided by the BUILDERROOFER™ module of the Sustainment Management System (SMS). The SMS is under development by the U.S. Army Corps of Engineers at its Engineering Research and Development Center-Construction Engineering Research Laboratory (ERDC-CERL) in Champaign, IL.

The series of guides will cover these aspects of BUILDERROOFER:

  • Inventory
  • Condition Assessment
  • Work Configuration
  • Work Planning
  • Reference Books
  • Scenarios

For system administrators, the SMS Installation Guide and SMS System Administration Guide are available.

This Guide

The purpose of this guide is to assist BUILDERROOFER users with their tasks as they enter roofing inventory information into BUILDERROOFER, along with details about the associated Buildings, Site(s), and/or Organization(s). The guide is intended for users with a permissions role of Inventorior or above.

This guide:

  • Presents an in-depth examination of the hierarchical levels available in BUILDERROOFER's information storage configuration.
  • Describes actions the user can perform at each of these levels, and the properties to be entered at each of these levels.
  • Provides a reference chapter for BUILDERROOFER administrators that covers application settings, setting password policy, and administering users and permissions roles.

The next release of the ROOFER Inventory Guide is expected to contain a chapter that provides tips for designing an information structure customized to your organization's needs.

Version Note: This version of the Inventory Guide for BUILDERROOFER has been prepared based on BUILDERROOFER Version 3.3.10.
