Set Up Fields and Filters

This topic shows how to (1) set up fields to control what kinds of data you want to see in your search results and (2) set up filters to refine your searches.

Fields

The Fields button at the Search screen allows you to configure the presentation of search results. The example screenshot below shows the fields available for configuring information presented at the Work tab. Checkboxes that are marked are those that will appear on the results screen.

Open Fields Popup

To open the fields popup, activate the Fields button at the left side of the Search screen.

Change Field Selection

If you want to clear all the fields and start from scratch,

  1. Activate the Clear Fields button.
  2. Mark desired checkboxes.
  3. Activate the Apply Fields button.

Otherwise,

  1. If there are fields you want to see in the search results that are not already marked, mark the checkboxes for these.
  2. If any fields you don't care to see in the search results are marked, unmark the checkboxes for these.
  3. Activate the Apply Fields button.

Apply Changes

After you make your field selections, and apply sorting specifications if desired, click on the Apply Fields button near the bottom of the popup.

Note: If you've made no changes since activating the popup or since last activating Apply Fields, the button will be grayed out.

Close Fields Popup

To close the "Fields" popup, activate the Fields button again.
