Select Site and Domain

To select which site is to be the scope of your tasks, see "Select Site(s)" in the topic Get Started with Enterprise SMS Web. If you don't select a site, all sites you have permission for will be included.

To select the domain(s) to work in, see "Select Domain(s)" in the topic Get Started with Enterprise SMS Web. This step is required.

View Individual Inventory Items

To examine the details of a specific inventory item,

  1. Navigate to the Search page if not already there.
  2. Select "Facilities" or "Groups" or "Components" on the Search Tasks bar.
  3. From the inventory grid, select one item.
  4. The Facility Details, Group Details, or Component Details screen will appear.

Create a Filtered View of Inventory

To create a view showing just those inventory items that contain your search term,

  1. Navigate to the Search page if not already there.
  2. Type a search term (case sensitive!) in the search box on the main menu, and hit Enter.
  3. Only results matching the search term will remain on the display.
  4. To clear the search term, activate the "x" to the right of the search box. Full display will be restored.

View assessment plans

To view all assessment plans related to a specific inventory item,

  1. Navigate to the Search page if not already there.
  2. Select "Facilities" or "Groups" or "Components" on the Search Tasks bar.
  3. From the inventory grid, select one item.
  4. The Facility Details, Group Details, or Component Details screen will appear.
  5. At the inventory item's Details page, activate the assessment plans accordion button. (Note: Available accordion buttons will differ depending on inventory level.)

Find a Document

To search for a document,

  1. Navigate to the Documents page via the dashboard.
  2. Enter the document name and/or a search tag.

    IMPORTANT: If you fill in both the "Document Name" and the "Tag" fields, the search will find all documents containing either.

  3. Activate the Search button.

More details and screenshots are available in the Understanding the Documents Screen topic.
