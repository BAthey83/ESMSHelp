Welcome to Enterprise SMS

The Enterprise version of the Sustainment Management System (SMS) is the newest version under development by the U.S. Army Corps of Engineers at its Engineering Research and Development Center–Construction Engineering Research Laboratory (ERDC-CERL) in Champaign, IL.

In contrast to legacy versions of the Sustainment Management System, which continue to be offered side by side with Enterprise SMS, the Enterprise version combines into one software program the information and capabilities formerly contained in separate products. Users can import and access real property information from multiple domains formerly isolated from each other in separate products such as BUILDER™, RAILER™, and ROOFER™. This information can be utilized one domain at a time, or information from two or more domains can be viewed simultaneously.

Enterprise SMS is currently available in two formats:

  1. Enterprise SMS Web (ESMS Web). This version is optimized for information search and for generating work items using condition and remaining service life criteria.
  2. ESMS Field Application. The field application, which can be downloaded from the ESMS Web site, contains both a Desktop application and a Tablet application. The Desktop version of Enterprise SMS allows users to create and access both real property inventory and assessment information, and to create and manage assessments for real property repair and replacement. The Tablet version allows users to perform assessments in the field using a tablet.

Introduction to ESMS Web can help users determine which format will be more useful, or whether both are needed.
