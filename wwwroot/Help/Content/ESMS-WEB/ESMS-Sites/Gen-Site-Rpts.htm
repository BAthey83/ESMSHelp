Generate Site Reports

This topic shows how to locate, select, and generate Site-level reports.

There are two locations where you can generate reports:

  1. At the main menu bar. The selection of report data listed here is divided into categories: CI Condition Index, FI Functionality Index., PI Performance Index. The Performance Index is a weighted combination of the CI value and the FI value, with the lower of the two values being given two-thirds of the weight and the higher of the two values being given one-third of the weight. If no functional assessment has been performed, the FI is assigned a value of 100., FCI Facility Condition Index., and EFCI.
  2. At the content area toolbar.

The first section below provides instructions for generating Site-level reports from the Site's content area toolbar. Reports generated here will be restricted in scope to the selected Site.

The section that follows provides instructions for generating a report starting at the main menu.

Generate Report at Building Data Screen

Scope of Report

When you generate a report while you are at the Site level, the report will contain only data that applies to Buildings in the currently selected Site.

How to Access

To access a report from a Site's inventory data screen,

  1. Select a Site from the Site Search screen.
  2. Open the selected Site to observe the report data.
  3. Additionally, you can also apply one or more filters at the Site Search screen to obtain a Site's report data.

Generate Report from Main Menu

Scope of Report

By default, unless you have used filters to further restrict what is included in the report, the scope of the report will be the Organization, Site, Building, System, Component, or Component-Section where you are located within the Enterprise SMS framework.

How to Generate

To generate a report from the main menu,

  1. Select the Reports button from the main menu.
  2. Choose an Organization and Report from the Reports popup menu.
  3. Select the Sites tab.
  4. Click the Site dropdown menu to pick a site in order to access the desired report.
  5. Optionally, you can then select one or more filters at the Organization or Domain levels. For example, you can select the Fuels domain to only show Sites which host Liquid Fueling and Dispensing Facilities.
  6. After setting any desired filters, and selecting a Sort order, you can then access the Report data.
  7. From the More Options popup menu, you can choose to export the data as a CSV file or show it as a table. You can spotlight data, get insights, sort data in ascending or descending order, or sort by installation name, site name, facilities, or PRV Plant Replacement Value. This number will be the aggregate of the PRV of all inventory items contained within the current inventory item..
