WBS Details Screen

This Screen shows attributes of a WBS item selected from the Assets tab of the Project Details screen.

The WBS Details tab displays the following attributes:

  • Component name
  • Activity Type
  • Estimated Cost
  • Actual Cost
  • Status
  • Work Classification
  • Modification Number
  • Work Order Number
  • Year
  • CLIN
  • Created Date
  • RTO/RTI Planned Date
  • RTO/RTI Actual Date
  • Technically Complete Date
  • Reference
  • Source
  • Comment

For the WBS items in the Fuels domain the following attributes are displayed in the Attributes expandable seection of the tab

  • Current Situation
  • Impact
  • Scope Of Work
  • Regulatory Requirement
  • Classification

For the users with the Update Project privileges this screen allows to edit the WBS
