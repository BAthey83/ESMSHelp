Project Details Screen

This Screen shows attributes of a project selected from the Project List Screen. These project attributes are grouped into the Details, Assets, Funding, Milestones, and Documents tabs.

The Details tab displays the following attributes:

  • Composed Project Number (A concatenation of the Project Type, Year and Number)
  • Construction Agent
  • Construction Agent PM
  • RPUIDs
  • Installations
  • Organizations
  • COCOMs
  • Domain
  • % Complete
  • Status
  • Sites
  • Site PM
  • Work/Service Order Number
  • Planned Execution Year
  • Date Created
  • Created By
  • Last Edited Date
  • Last Edited By
  • Comment History

For the projects in the Fuels domain, the following attributes are displayed in the Attributes expandable section of the tab:

  • Pillar
  • CA Tracking Number

For users with Update Project privileges, this screen allows them to Edit Project Details, as well as add and update comments.

For users with Delete Project privileges, this screen allows them to delete the project authored by the user.
