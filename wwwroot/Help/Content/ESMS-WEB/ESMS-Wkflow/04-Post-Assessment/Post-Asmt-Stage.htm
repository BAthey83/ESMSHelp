Post-Assessment Stage

This topic describes the Post-Assessment stage of designing an ESMS assessment. This is the fifth of six stages, to be performed after assessment information and associated work items have been synchronized to ESMS Web at the end of the Desktop Sync / Field Assessment stage. Here you can create, search, or delete work items. You can edit work items, for example to modify the estimated cost.

—Quick Links—

Tasks available at this stage are to

  • manually add work items at the DSR tab if desired
  • edit work items at the DSR tab
  • select a component and view and/or edit the assessment question(s) results associated with that component at the Questionnaire tab
  • review and/or edit the assessment questionnaire results at the Questionnaire Summary tab
  • transition to the next stage when done.

DSR

The DSR (Deficiency Survey Report) tab displays work items that were generated in ESMS Desktop during the Desktop Sync / Assessment stage. At this tab you can edit previously created work items, or create new ones using the Add Work Item button.

Filters

Using the filter fields above the columns, you can filter the work items by Work Description, Component, RPUID and/or Facility Number.

Add Work Item

To add a work item at the DSR tab,

  1. Activate the Add Work Item button, located above the row of filters (see screenshot above). The "Create/Edit Work Item" popup will appear, and the fields will be unpopulated.

  2. In the popup, fill out the fields. Details about the fields in the "Create/Edit Work Item" popup are found in the ESMS Desktop documentation.
  3. You can upload images to be associated with the work item by activating the Upload Images button near the bottom left of the work item popup. Attached images will be displayed as thumbnails along the bottom of the popup. To view an image, click on its thumbnail.

    Note: Any images uploaded here will also show at the ESMS Documents area accessed by the Documents tile on the dashboard.

  4. When finished adding work item data, activate Save. Alternatively, activate the Close button to cancel your additions.

Edit Work Item

Based on your permission level, you can add, delete, and/or change data in a previously existing work item.

To edit a work item,

  1. Activate the work item's row in the Work Items table. The "Create/Edit Work Item" popup will appear.

  2. If you don't see the desired work item in the table, adjust the filters and repeat the step above.
  3. In the popup, edit fields as desired. Details about the fields in the "Create/Edit Work Item" popup are found in the ESMS Desktop documentation.
  4. You can upload images to be associated with the work item by activating the Upload Images button near the bottom left of the work item popup.
  5. If there are already images associated with the work item, they will be displayed near the bottom of the popup. To view an image, click on its thumbnail. You can delete images, or upload more images.

    Note: Any images uploaded here will also show at the ESMS Documents area accessed by the Documents tile on the dashboard.

  6. When finished with edits, activate Save. Alternatively, activate the Close button to exit without saving your changes.

Delete Work Item

To delete a work item,

  1. Navigate to the Work Items table at the DSR tab.
  2. In the rightmost column, activate the Delete button in the row describing the work item you wish to delete.
  3. Provide confirmation as prompted.

Questionnaire

At the Questionnaire tab, you can view or edit assessment questions associated with a particular component if (1) you have appropriate permissions and (2) you have been assigned to the particular discipline the question is associated with. Questionnaire Summary

The Questionnaire Summary table (shown below) will display a list and quick summary of answered questions related to any work item created using the Questionnaire. This is another location where you can edit the assessment information. To do so, click on the information in the field and edit it. The change will appear momentarily, without you needing to activate a button.

Please note that work items created in the DSR will not display in the Questionnaire Summary.

You can filter what questions are displayed by using the Facility, Component, Type, Question and/or Answer filter fields above the columns.

Transition

When entries are complete and any required approvals have been obtained (this is governed by the scheme; see Transitions in Understanding Assessments), you can proceed to the next and final stage, the Packages Stage.
