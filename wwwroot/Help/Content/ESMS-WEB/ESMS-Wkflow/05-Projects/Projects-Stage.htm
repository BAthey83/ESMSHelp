Packages Stage

This topic describes the Packages stage of designing an ESMS assessment. This is the last of six stages, to be performed after any necessary approvals have been obtained to progress from the Post-Assessment stage.

To complete the Packages stage, assign work items to appropriate packages. You can also have individual work items that are not assigned to any package if that is acceptable under the procedures followed by your organization. A work item can be in 0 packages or in 1 package. The same work item can not be in more than one package.

—Quick Links—

Tasks available at this stage are to

  • add a work package
  • edit a work package
  • assign work items to work packages

Add Work Package

Available work packages are listed to the left of the table of work items. If you have work items in the table that need to be assigned to one or more work packages that do not appear in the list of work packages, you can create one or more new work packages to be added to the list.

To add a work package to the list of work packages at the left,

  1. Activate the plus sign located above the list of work packages. The "Package Editor" popup will appear.
  2. Enter a name for the new work package in the Name field.
  3. Fill out all applicable remaining fields.
  4. Activate Save.

Edit Work Package

To edit an existing work package,

  1. Activate a package name in the list of work packages. The "Package Editor" popup will appear.
  2. Edit fields as desired.
  3. Activate Save.

Assign Work Item to a Package

This section describes how to assign a work item to a package that is listed at the left side of the screen. If you don't see an appropriate package, you can add a new one to the list as described in Add Work Package.

To assign a work item to an existing package,

  1. Activate anywhere in the work item's entry in the table of work entries at the right. The "Assign Packages" popup will appear.
  2. Activate the dropdown arrow at the right side of the Work Package field.

    Note: The Work Package field is the only editable field here.

  3. If there are no work packages listed, or if none of the packages listed apply to the selected work item, exit the popup and follow instructions at Add Work Package to add one or more packages to the list of available work packages. Then repeat the steps above.
  4. From the dropdown list of available packages, select the one you wish to assign the work item to.
  5. Activate the Okay button.
