View Assessment

To view or work with an existing assessment,

  1. Navigate to the table of assessments. From the Dashboard, this is done by activating the Assessments tile. Alternatively, from within a different assessment, you can activate the browser's back arrow button until you reach the Assessments screen with the table of assessments.

  2. In the table of assessments, activate the line for the desired assessment.

View the Assessment Data Page

After completing the steps above, you will be brought to the Details tab in the Charter stage for that assessment. This tab is essentially the assessment's data page, showing assessment level data such as the assessment name, budget, etc. For explanations of the assessment data fields, see Add Assessment.

Navigate to Other Stages and Tabs

To view other tabs and stages of the assessment, activate the Workflow button/flyout tab to open the Workflow navigation tool.

The Workflow navigation tool is located in one of two places:

  • On a wide desktop screen, this is activated via a flyout tab at the upper left of the screen.
  • On a narrower screen, activate the navy blue bar with the word "Workflow" appearing in white, below the title/menu bar.

After you activate, the Workflow navigation tool will expand as shown below:

Note: What you see and what you can activate in the Workflow menu depends on multiple factors. Depending on your permissions, you might not see the full selection of navigation buttons shown here. If approvals are outstanding, some of the options may be grayed out. Also, if Workflow was not activated for the assessment, this navigation tool will show only Charter / Details and Assessment / Draft Work Item.

Using the expanded navigation tool, you can jump directly to a tab in one of the Workflow stages. (Scroll down as needed if not all of the stages are visible.) Then when you click outside the tool, it will close.
