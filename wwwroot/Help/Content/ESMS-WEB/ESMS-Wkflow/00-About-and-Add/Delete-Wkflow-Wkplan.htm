Delete Assessment

To delete an assessment,

  1. Navigate to the table of assessments. (If you are inside of an assessment, you can return to the table of assessments by activating the browser's back arrow button until you reach the table.)
  2. Select the assessment you want to delete. The assessment's Assessment Details will be displayed.
  3. At the lower right corner will be two buttons: Save and Delete. Activate the Delete button.
  4. In the popup box asking "Are you sure you want to delete this Assessment?", activate the OK button. (Alternatively, activate Cancel to cancel the deletion.)
