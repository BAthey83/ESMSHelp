Manage Workflow Permissions

To manage Workflow permissions, you can add permissions roles, and you can delete roles that are not in use. Permissions roles are used when a "required permission" is specified in a scheme transition.

When required permissions are present in a scheme, then when a user adds an assessment based on that scheme, one or more users must be assigned to each of the permissions roles specified, at the time the assessment is created.

About Workflow Permissions

Workflow permissions are not the same as ESMS permissions; they have a different scope. Workflow permissions operate only within the scope of (1)the Workflow editor and (2)assessments that have Workflow enabled. ESMS permissions are broader.

Also with respect to scope, Workflow permissions are above the level of a scheme. Any listed Workflow permissions role can be used by one or more schemes.

Add Permissions Role

To add a Workflow permissions role,

  1. Activate Workflow Permissions on the Workflow menu bar. The "Workflow Permissions" popup will appear.
  2. Activate the + Add button. Another popup, "Create Permission" will appear.

  3. Enter a name for the permissions role.
  4. Activate Create to finish creating the permissions role. (Alternatively, activate Cancel to abandon creating the role.)

Delete Permissions Role

You can delete a permissions role listed in the "Workflow Permissions" popup only if the role is not being used in a scheme. When you select a role for deletion, the program will determine whether that role is in use or not.

To delete a Workflow permissions role,

  1. Activate the x to the right of the role name.
  2. If the role is deleted, that means it's not being used in a scheme. Return to Step 1 if you wish to delete another role; otherwise, activate Close and you are finished.
  3. If the role is being used in a scheme, an error message will be displayed. Activate OK to close the error message.

Assign a Permissions Role to a User

A Workflow permissions role can be assigned to a user only during the process of adding an assessment. See Permissions in the topic Add Assessment for instructions.
