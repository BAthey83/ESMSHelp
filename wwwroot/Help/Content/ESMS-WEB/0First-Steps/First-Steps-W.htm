Get Started with Enterprise SMS Web

This topic lists steps to take to get started with your Web instance of Enterprise SMS (ESMS).

This topic presents steps to getting started in this order:

  1. Get username and password
  2. Log in
  3. Select domain(s)
  4. Select site
  5. Select task area
  6. Log Out

You also have the option to delay steps C and D (select site and select domains) until after you select a task area in step E.

A. Get Username and Password

Your ESMS administrator can set you up with the following:

  1. Username and password. Username and password need to be the same for Web, Desktop, and Tablet sign-ins (whichever of the three you use).
  2. Designated permissions roles and the scope for each.

B. Log In

Of the two login options, Password Login is currently operational.

To log in at the Enterprise SMS login page,

  1. Enter your username and password into the corresponding fields.
  2. Activate the Login button.

The login screen is shown below.

C. Select Domain(s) (Required)

Using the dropdown at the Domains button on the main menu bar, you can pick and choose which category or categories of infrastructure you wish to view or work with during your current login. These categories are called domains in ESMS.

  • Selecting at least one domain is required.
  • The number in parentheses after the word "Domains" indicates how many domains are currently selected.

Select a Domain (Required)

To select one or more domains,

  1. Activate anywhere on the Domains button in the main menu bar. The dropdown of domain options will appear.
  2. Select one or more domain checkboxes.
  3. Activate Apply.

Change Domain Selection

To change your selection of domain(s),

  1. Activate the Domains button on the menu bar.
  2. In the dropdown, activate the checkbox next to each domain desired, and de-activate any currently checked boxes next to domains whose data you do not wish to view.
  3. Activate the Apply button in the dropdown.
  4. The number in parentheses on the Domains button indicates how many domains are currently selected.

D. Select Site (Optional)

Selecting a single site to work with is optional. If you don't select a site, the scope of your actions will be all sites you have permission for (which is the same as the list of sites initially presented in the "Site Selection" popup).

To select the site to work with,

  1. On the main menu bar, activate the site button, which either (a) lists the currently selected site or (b) displays Select Site.

  2. The "Select Site" popup window will appear, listing available sites.

  3. (Optional) If you wish to trim down the list of selections via search, follow these steps:
    1. Enter a search string in the "Site Name" field.
    2. Activate the Go button to see only those site names containing the search string.

      Tip: To see the full list of sites again after using the search field, clear the search field, then activate the Go button.

  4. Click on one site in the site list.
  5. Activate the Select button at the bottom of the popup. The popup should disappear. If it does not, and the Select button is grayed out, a valid site has not been properly selected.
  6. Alternatively, activate the Close button to cancel site selection and exit the popup.

More detail about selecting and changing a site is in the topic ESMS Entry-Level Tasks.

E. Select Task Area

After successful login, you will be brought to the Enterprise SMS dashboard where you can choose between the task areas shown below by double-clicking the one you want.

F. Log Out

When you are ready to log out,

  1. Expand the user menu dropdown at the far upper right, on the main menu bar.

  2. Select Logout.
