Get Started with Enterprise SMS Web

This topic lists steps to take to get started with your Web instance of Enterprise SMS (Enterprise SMS).

This topic presents steps to getting started in this order:

  1. Get Username and Password
  2. Logging In
  3. Welcome Screen
  4. Select Domain(s) (Required)
  5. Enterprise SMS Dashboard Task Options
  6. Logging Out

You also have the option to delay step D (select domains) until after you select a task area in step E.

Get Username and Password

Your ESMS administrator can set you up with the following:

  1. Username and password. Username and password need to be the same for Web, Desktop, and Tablet sign-ins (whichever of the three you use).
  2. Designated permissions roles and the scope for each.

Logging In

There are two login options: Azure Active Directory Login and EADS-A Login.

To log in at the Enterprise SMS login page,

If you have an existing Azure account:

  • You can log in by selecting the Azure button, and you will be automatically redirected to the ESMS home page. The account may be locked at this stage. If so, please contact SMS Support to unlock your account.

If you do not have an existing Azure account:

  • Select the EAMS-A (Enterprise Access Management Service-Army) button to create a new user account. Click “PKI Login” once you are directed to the EAMS website and complete the form on the following page.

Welcome Screen

After you log in, you will be directed to the Welcome screen. Here, you can visit your favorite and frequent pages, access reports, and quickly go to your most recently visited and favorite pages, as shown below.

Select Domain(s) (Required)

Using the dropdown at the Domains button on the main menu bar, you can pick and choose which category or categories of infrastructure you wish to view or work with during your current login. These categories are called domains in Enterprise SMS.

  • Selecting at least one domain is required.
  • The number in parentheses after the word "Domains" indicates how many domains are currently selected.

Select a Domain (Required)

To select one or more domains,

  1. Activate anywhere on the Domains button in the main menu bar. The dropdown of domain options will appear.
  2. Choose Select All or select one or more individual domain checkboxes.
  3. Activate Apply Selected Domains.

Change Domain Selection

To change your selection of domain(s),

  1. Activate the Domains button on the menu bar.
  2. In the dropdown, activate the checkbox next to each domain desired, and de-activate any currently checked boxes next to domains whose data you do not wish to view.
  3. Activate the Apply Selected Domains button in the dropdown.
  4. The number in parentheses on the Domains button indicates how many domains are currently selected.

Logging Out

When you are ready to log out,

  1. Expand the user menu dropdown at the far upper right, on the main menu bar.
  2. Select Log Out
