Search

To perform a search at the Groups tab,

  1. If not already selected, select one or more domains.
  2. If desired and if not already selected, select a single site to search in.
  3. Activate the Groups tab.
  4. Configure what data fields you will want to see in the results. See Set Up Columns and Filters for instructions.
  5. To narrow the scope of the search by properties such as parent site, parent facility, or asset class, follow instructions for setting up filters in Set Up Columns and Filters.
  6. Finally, enter a search term in the search field on the main menu bar and press Enter.

Export Results

You can export your results to CSV by activating the Download to CSV button at the top of the screen:

The CSV file can be opened with Excel. It will be exported to your standard downloads location, and the file name will incorporate the download date.

Details

Click anywhere in a row of data in the results table to see more details about that item. For more information, see the topic Group Details.
