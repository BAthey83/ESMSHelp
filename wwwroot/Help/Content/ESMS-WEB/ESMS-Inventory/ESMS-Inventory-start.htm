Understanding the Export Screen

This screen is the place for exporting data from the external sources. It is accessible from the main menu to users with Administrator level privileges.

To access the Export screen, select the Import/Export dropdown menu. Click the Export button, and the following screen appears:

Here, users can select the Export Type (such as CSV). Primarily, this screen is used to request Exports. Additionally, Exports can be downloaded by clicking the arrow button, or deleted by selecting the X button.

You can click the plus sign to add or edit an export configuration. Once the configuration is added or edited, the Export Mapping details will appear, as well as the Export Level and associated Export Detail. Once the information is ready, select the Request Export button, or push Close to cancel the export request.
