Navigating the Asset Search Screen

This topic provides a navigation overview of the information available on the Enterprise SMS Asset Search screen. Separate topics (linked below at the end of this topic) show the information available at each screen at the Details level.

After you have selected one or more domains and, if applicable, a site, the initial display at the Asset Search screen displays:

  • the number of facilities, groups, or components that are in the active site(s)
  • what site and domain(s) are selected (on the main menu)
  • a data table of assets, at the facilities level

Once you understand the components of the Asset Search screen and how they work, you will be able to:

  • control the kind of data that is displayed by designating which fields are active for each asset level
  • narrow searches using filters
  • navigate to pages that show detailed information about assessment plans, work items, and individual assets

Components of the Asset Search Screen

The major components of the Asset Search screen, as numbered in the illustration below, are:

  1. main menu
  2. content area, which will display the results of the most recent search
  3. Download as CSV , which lets you see the results in spreadsheet form
  4. show map, which displays map results
  5. show graphs, which displays graph results
  6. select columns, which lets you choose which field columns appear in the content area

Main Menu

At the main menu, you can

  • return to the home page, using the Enterprise SMS logo button (site and domains will be cleared)
  • Select Domain(s) (Required) to work with (Required)
  • select from options in the user preferences menu: (a) announcements; (b) user preferences; (c) save your current workspace as the default; (d) download a Desktop version of ESMS; (e) change sync settings for synchronizing with ESMS Desktop; (f) clear application cache; (g) see the application and software versions; (h) contact SMS support); or (i) logout.

Content Area

The content area of the Asset Search screen shows one of the following:

  • Depending on which tab is selected on the tab bar, the content area shows a table listing the facilities, groups, components, assessment plans, or work items that are (1) associated with either your selected site or all of your permitted sites (depending on whether a site has been selected) and (2) within your selected domain(s). The column headings in the table are determined by the fields selected via the Select Columns button.
  • When you perform a search, a results table showing the search results is displayed in the content area. The column headings for the results table are shown directly below the task selection bar. These column headings present are determined by the fields selected via the Select Columns button.

Download as CSV

To control which data columns will be displayed in the content area,

  1. Activate the Select Columns button.
  2. Select and/or deselect checkboxes.
  3. Activate the Update button.

Note: The options shown in the Select Columns window will depend on what is selected on the main menu (Facilities, Groups, Components, Assessment Plans, or Work items).

Expanded information about setting up fields is in the topic Set Up Columns and Filters.

Filters Selection

Filter selection can be accessed using the Filter button(s) next to each column.

To apply filters to your search,

  1. Activate the Filter button directly to the right of each column.
  2. In the popup, fill in the desired filter information.
  3. Activate the check button to hide the Filter box.

Note: The options shown in the Filter selection box will depend on what topic is selected on the main menu (Facilities, Groups, Components, Assessment Plans, or Work Items).

Expanded information about setting up filters is in the topic Set Up Columns and Filters.
