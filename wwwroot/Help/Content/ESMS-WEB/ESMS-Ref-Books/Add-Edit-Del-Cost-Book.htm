Add, View, Edit, or Delete Books

This topic details how to add, view, edit, or delete a Book in Reference Books.

Note: These processes require Administrative permissions. Consult your system administrator for details.

Before you add, edit, or delete a Book:

  1. Choose an Organization from the Current Organization button.
  2. Select the Reference Books dropdown, followed by the button for your chosen book type. In this example, we will be referring to Cost Books.
  3. Pick a book from the subsequent Search window.
  4. The book's window will display, with the name of the book at the top. You can now proceed with adding, editing, or deleting books.

Add a Book

To add a book,

  1. Click the "+" icon that appears above the search bar. The Create Book window will appear.


  1. Complete the fields in the window as directed in the View and Edit a Book topic.
  2. Once you have named your book in the Cost Book Name column, select a Source, which identifies the source of the cost information.
  3. If you wish to import existing book data to your new book, select a file from the Copy Existing Book dropdown menu.

    4. Note: leave the Copy Existing Book field blank if you wish to create a custom book from scratch.

  4. Click Create to add your new custom book, or select Cancel to stop the process.

View and Edit a Book

To view a book,


  1. Open a book from the Search window.
  2. Your book data will appear in the above window.

To edit a book,

The Cost Record screen displayed above appears when you select a book. The Work Action, Work Activity Type, Managed Object Type, and Specification fields cannot be edited from this screen. However, the other fields can be configured and updated within the Cost Record screen.

Once you have added the Cost Record data, select the Update button. If you do not want to save the Cost Record data, select Close.

Note: You must Save your changes (via the floppy disk icon highlighted above) before viewing another book or your updates will be lost if you navigate away from this page.

Features and Search Bar

  • Work Action: Identifies the item obtained from concatenating the Specification and the Activity.
  • Specification: The item selected for editing within the book.
  • Activity: The action required for the book item.
  • Object Type: A generalized item classification compared to the Specification. For example, a Specification may be "Pipe, Copper, 8-12 Inch." The Object Type will simply be "Pipe."
  • Domain: The domain applied to the book, such as Utility-Gas, Waterfront, or Facility-Building.
  • Description: Text entered by the user to aid in identifying the item.
  • Level: The condition for the specified item.
  • Parent: A key sub-classification of the Object Type.
  • Current Cost: The latest cost of the specified book item(s).
  • Original Cost: The initial cost of the specified book item(s).
  • Cost Year: The year at which the item cost is set.
  • UoM Unit of Measure: The Unit of Measurement for the specified item(s).

    • (Note: the available UoM Unit of Measures depend upon the selected cost book and user requirements).

  • Minimum Cost: The base-level value of the book.
  • Materials: The value of materials used to execute the book.
  • Labor: The value of labor used to execute the book.
  • Equipment: The value of equipment used for the book.
  • Bare Total: The estimated total value of the book.
  • Source: The basis of the book information, which is user-entered.
  • Frequency: Describes how often repeated work described by the book needs to be performed.
  • Crew: The number of people who are using the book.
  • Daily Output: The book's output value per day.
  • Total Output: The book's overall value for the fiscal year.

Delete a Book

To delete a book,

  1. Click on the book you want to delete in the Search menu.
  1. Select the trashcan icon just above the Search menu.
  1.  The Confirm Action popup will appear. Click the Confirm button to delete your book. Alternatively, select Cancel to keep your book.
