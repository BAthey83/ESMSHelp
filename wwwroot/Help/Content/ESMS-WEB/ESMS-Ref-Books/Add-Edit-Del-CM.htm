Add, Edit, or Delete a Cost Modifier

This topic details how to add, edit, or delete a Cost Modifier in Reference Books.

Note: These processes require Administrative permissions. Consult your system administrator for details.

—Quick Links—

  • Add a Cost Modifier
  • Edit a Cost Modifier
  • Delete a Cost Modifier

Before you add, edit, or delete a Cost Modifier:

  1. Choose an Organization from the Current Organization button.
  2. Select the Reference Books dropdown, followed by the Cost Modifiers button.
  3. Pick a Cost Modifier book from the Search window.
  4. The Cost Modifiers window will display, with the name of the book at the top. You can now proceed with adding, editing, or deleting cost modifiers.

Add a Cost Modifier

    To add a cost modifier,

    Select the Create Modifier button in the Cost Modifier window. The Cost Modifier popup window will appear.

    Enter the following information:

    • Type - Indicates whether the cost modifier is a Multiplier or an Adder.
    • Scope - Indicates whether cost modifier's scope is for a Component, Group, Asset, or Site.
    • Value - Indicates the overall cost of the cost modifier.
    • PRV Plant Replacement Value. This number will be the aggregate of the PRV of all inventory items contained within the current inventory item. - Indicates whether the Plant Replacement Value has been factored into the cost modifier.

    The Name and Description fields are populated with the information entered in the stage.

    Select the Create button to add your cost modifier.

    Edit a Cost Modifier

    To edit a cost modifier,

    1. Select a modifier to edit in the Cost Modifier window.
    2. Within the popup window, edit the Name, Description, Type, Scope, Value or PRV Plant Replacement Value. This number will be the aggregate of the PRV of all inventory items contained within the current inventory item. information as shown in Add a Cost Modifier.
    3. Click the Update button to complete the edit.

    Delete a Cost Modifier

    To delete a cost modifier,

    1. Open the Cost Modifier window and select the modifier you wish to delete.
    2. Select the trashcan icon located under the Clear column to delete the cost modifier.

      3.  Note: Deleting a modifier will also remove all assignments of that modifier in Enterprise SMS.

    3. The modifier is now deleted. Select the Save Changes icon (the floppy disk icon displayed above) to complete the process.
