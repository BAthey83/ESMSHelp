View Documents

This topic outlines how to:

  • Search for a Document
  • View a Document
  • View Component Associations
  • Download a Document

Search for a Document

At the Documents screen, accessed from the dashboard, you can search for one or more documents by (either or both):

  • name or name fragment
  • associated search tag

A sample of the Documents screen, before any of the search fields are filled in, is shown below.

Types of Document Search

Search by Document Name

To search by document name,

  1. Type the name or name fragment in the "Name" field.
  2. The results will load automatically in the display window below.

The display window will change to list only those documents meeting the search requirement.

Search by Document Type

To search by document type, check the document type(s) you want in the "Type" field.

Search by Tag

To search by tag,

  1. Click the drop-down arrow on the right side of the "Tag" field.
  2. Select the Tag(s) you wish to search by clicking the respective checkbox(es).
  3. If you wish to view all tags, click "Select All" at the top of the field. To view select tags, click "Deselect All" and highlight the tags you wish to view.
  4. Click the drop-down arrow again to view your results.

The display will change to list only those documents meeting the search requirement(s).

Combination Search

If you fill in both the "Name" and the "Tag" fields, the search will find only those documents meeting both criteria.

Clear a Search

To clear a search and return to the full listing of documents, activate the X button.

View a Document

Once you have found a relevant document via Search for a Document, you can view it by activating its View Document icon.

If the document record has no View Document icon, follow these steps:

  1. Activate the Download icon.
  2. Select the document from the Downloads window in your web browser. (This step might not be needed if you have already selected the option to "Always open files of this type" for its file type.)

View Component Associations

To ascertain whether a document is associated with one or more components,

  1. Checkmark the document in the list of documents.
  2. Activate the Document Associations and Tags Editor if that section isn't already open.
  3. Look in the "Associated Components" section of the Document Associations and Tags Editor.
  4. If there are component associations, they will be listed under the "Associated Components" tab.
  5. If the document is not associated with select components, you will see a message: "No Associated Components." If the document is not associated with any components, you will see a message: "No Associated Components for Selected Documents".

Productivity Tip: You can leave the Document Associations and Tags Editor open while you uncheck and check different documents.

Caution: If you select more than one document at a time at the table of documents, there won't be any indication of which document is associated with what component. You will need to select documents one at a time to see the precise matching.

Download a Document

To download a document, activate its download icon:

The document will appear in the Downloads folder in your web browser.

Productivity Tip: You can use the "Show in folder" option on the document's button to jump to its download location.
