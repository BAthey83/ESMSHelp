Work Analysis

This topic introduces tasks you can perform at the Work Analysis screen, providing detailed descriptions for each task.

What Can I Do Here?

At the Work Analysis screen, you can:

  • Add, duplicate, edit, publish, or delete Policies
  • Create, edit, or delete policy standards
  • Add, duplicate, or delete Priorities
  • Add, duplicate, or delete Profiles
  • See and compare work profiles
  • Compare profiles to Work items
  • Download profiles in Excel format
  • Copy profiles as work items
  • Delete assessments
  • Explore work items in your Queue
  • Select and compare Reports

Beginning the Work Analysis Process

To begin the Work Analysis process, choose a policy standard by using the Search window or clicking on a policy standard. If the policy standard is not present, you can perform one of the following options from the Work Analysis menu:

  • Add New Policy: Select the plus icon at the far left to add a new policy standard.
  • Duplicate Selected Policy: Choose the dual paper icon (the second icon) to duplicate an existing policy standard.
  • Edit Selected Policy: Pick the pencil and square icon (the third icon) to edit an existing policy standard.
  • Return to Draft: Select the "no" icon (the fourth icon) to return to the original draft.
  • Delete Selected Policy: Pick the "X" icon at the far right to remove the policy standard from the possible options.

You can execute any of these actions across all of the following tabs:

  • Policies
  • Priorities
  • Profiles
  • Work
  • Queue
  • Reports

Policies

The Policy tab allows you to create, organize, edit, publish, and delete policy standards by adding rules or rule clauses. Choose from the following options:

  • Standard Name - The policy standard's title, which typically indicates the standard's priority.
  • Global Repair - Choose this option for repairs that are necessary for all work items.
  • Maintain - This option is for adding rule clauses to maintenance-related values.
  • Inspect - Select this option to add rule clauses to inspections.
  • Replace - Pick this option for add value thresholds for work items.
  • Local Repair - Choose this option for repairs that are necessary for a small selection of work items.
  • Standard Scope - Use this option to set limits for your policy standard(s).

Note: You can click the reorder icon to change a standard's priority. The quantity of each available rule is indicated in parentheses by each option.

Priorities

The Priority tab allows users to draft and publish multiple prioritization schemes. Data for each scheme can be customized, including: 

  • Objective Name
  • Weight
  • Age
  • Importance Index
  • CI
  • RSL
  • Years Since Last Inspection

Profiles

The Profile tab lets users generate a work profile based on the following information:

  • Cost Book
  • Inflation Book
  • Policy
  • Prioritization
  • Domains
  • Scopes
  • Funding Sources

Here, you can also select checkboxes to Ignore Work Items and Roll Over Unused Funds.

Work

The Work tab lets you review and execute several options related to work profiles, including:

  • Show Profile Details - Additional information about an individual profile.
  • Compare to Other Profiles - Choose up to two assessments to compare them.
  • Compare to Work Items - Compare an individual profile to different work items.
  • Download Excel - Import all of the data from your work profile as an Excel file.
  • Copy as Work Items - Copy work profile data from an individual year as work items.
  • Delete Assessment - Remove a selected assessment from the work profile.

In addition, you can view the date the profile was created, who created it, the date it was last managed, who managed it, and the profile's status.

You can see the following information about your work profile here as well, such as:

  • Object Name
  • Activity
  • Priority
  • Cost
  • Base Cost
  • Modifier Cost
  • Asset Name
  • Domain
  • Type
  • Site
  • Year
  • Funding

You can filter these options (except for Priority, Cost, Base Cost, and Modifier Cost) to modify the data even further.

Queue

The Queue tab lists the work profiles in your queue for easier access.

Note: If you do not see the work profile you are looking for, select the Refresh button and it will appear.

Reports

The Reports tab lets you choose and view the complete report with all your Work Analysis data included.
