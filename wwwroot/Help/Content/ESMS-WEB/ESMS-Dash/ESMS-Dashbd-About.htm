Understanding the ESMS Dashboard

This topic introduces the purpose of the ESMS Dashboard and provides a brief overview of each of the task areas.

What Can I Do Here?

  • You can transfer from one task area to another via the dashboard because the dashboard serves as the central hub connecting all of the task areas in ESMS Web.
  • You can initiate a search from the dashboard without having to navigate to the Search screen first. (Before searching, you must first select one or more domains and, if desired, select a site.)
  • Via the user dropdown menu in the upper right of the screen, you can select from the following options: (a) save your current workspace as the default; (b) change sync settings for synchronizing with ESMS Desktop; (c) download a Desktop version of ESMS; or (d) logout.
  • The dashboard will also list your team memberships:

    Note: If you don't have permission to administer a team, the Teams icon won't appear on your dashboard.

Overview of Task Areas

Each task area in ESMS Web is represented by a tile on the dashboard. Below is a brief description of each of the task areas.

Search

At the Search screen, you can search for real property assets, assessments, and work items that meet your search criteria. The search process is driven by

  • filters (selected in a context-sensitive popup)
  • your search term (typed in the field on the main menu bar).

Results will show dynamically in the content area of the Search page.

Starting from the search page, you can also navigate to pages showing detailed information about assessments, work items, and individual assets.

For more detail about the Search screen, see Navigating the Search Screen and Understanding the Search Screen.

Reports

As of July 2019, no reports have been adapted for use with ESMS Web. After this takes place, you will be able to select from among report types that are tailored for the different domains in ESMS, then generate and view the report.

Documents

Consuming Documents

At the Documents screen, you can

  • search for a document
  • search for documents with a particular tag
  • view a document
  • download a document

The tasks above are described in View Documents.

Managing Documents

If you are actively engaged in uploading documents and adding or grooming search tags, you can also do the following:

  • add documents in bulk (Upload)*
  • associate documents with a component*
  • delete a document
  • associate search tags with documents
  • add or delete tags from the pool of search tags

*This can also be performed at the relevant Components Details screen, which is accessed via the Search screen.

Teams

At the Teams tab on the Teams screen you can

  • add and delete teams
  • add and delete team members
  • assign roles (security permissions) to teams
  • rename teams

At the Roles tab you can

  • add and delete customized security permissions roles
  • edit roles
  • rename roles

Work

At the Work Generation screen, you can automatically generate work items by first selecting target site(s) and then specifying thresholds for condition index (CI) and remaining service life (RSL).

Specification

The Specifications area of ESMS allows you to group objects by type and create attribute/value specifications for the group.

Costs

At the Costs screen, you can work with cost books and cost modifiers.

Assessments

At the Assessments screen, subject to permissions, you can view, add, edit, or delete aa assessment. Optionally, an assessment can be associated with a workflow specification (called a scheme) that governs progress through the work plan and designates what conditions need to be met or what permissions are needed from whom before progressing from one stage to the next in the work flow.

Workflow Editor

At the Workflow editor screen, existing workflow schemes can be edited and new schemes can be created. (See Assessments for what a scheme is.)

Note: The Workflow Editor tile will appear on your ESMS Dashboard only if your User Details at the ESMS Administrator's Security screen include the global permission "WorkflowRole."

Not yet implemented.

Other

Additional icons such as Catalog and Security will appear if you have ESMS administrator-level permissions.
