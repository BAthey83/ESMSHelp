Edit Funding Restriction

Upon clicking on the button on the Funding Restriction List screen brings up the Edit Funding Restriction dialog.

Edit Funding Restrictions

The dialog includes the following options:

  • Category (Read only)
  • Property (Read only)
  • Condition : This drop-down allows to select "equal to" or "not equal to" relational operator to apply funding restriction.
  • Select Site or Domain Name (Read only)
