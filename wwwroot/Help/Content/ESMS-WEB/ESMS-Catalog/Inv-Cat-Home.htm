Inventory Catalog

This topic introduces tasks you can perform at the Inventory Catalog and provides links to greater detail for each.

Component Types

General Tab

On the General tab, you can add information for the following fields:

  • General: Root Type or System Type.
  • Domain: Select any of the 22 available Domains.
  • UoM Category: Choose from Area, Number, Energy, Volume, Flowrate, Power, Depth, Each, Length, N/A, or Weight.
  • Parent Components: Here, you can select as many components that appear in the left column as needed.
  • Subcomponents: Choose from any of the subcomponents that appear in the left column as needed.

Custom Attributes Tab

For the Custom Attributes tab, you can create a root attribute for your inventory catalog, followed by a category, to which you can attach additional attributes. Select the plus icon to add a root attribute, the folder icon to add a new category, or the folder with arrow icon to add an attribute.

With the Specifications tab, you can add, edit, duplicate, or remove specifications and related attributes, as well as view specification attribute counts. Select the plus icon to add a new specification, the pencil icon to edit an existing specification, the clipboard with arrow icon to duplicate a specification, or the X icon to delete a specification.

Units of Measure

Measurement Unit Manager

The Measurement Unit Manager allows you to add a New Measurement Unit, search Abbreviations, search Names, search Measurement Category Names, or delete Measurement Units related to individual components. Select the plus icon to add a New Measurement Unit, enter phrases into the respective search boxes to look for Abbreviations, Names, or Measurement Category Names, or click the trashcan icon to delete a Measurement Unit.
