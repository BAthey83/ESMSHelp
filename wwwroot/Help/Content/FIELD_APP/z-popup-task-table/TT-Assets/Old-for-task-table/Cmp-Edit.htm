Edit a Component

To edit a component,

  1. Select the Assets page.
  2. In the asset tree, select the component to be edited.
  3. Activate Edit Component in the ribbon. The View Component tab will appear in the content area.
  4. Edit information as desired in the fields. More detail about these fields is given in the Component Data topic.
  5. Activate the Save button (located above the Basic Properties set of fields).
