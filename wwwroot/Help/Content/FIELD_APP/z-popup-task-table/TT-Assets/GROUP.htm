Working with Groups

Operations you can perform with groups are

  • View
  • Add a component to a group
  • Add a group to a group
  • Delete a component or group from a group
  • Edit the membership of a group

Note: Any component or group can belong to more than one group. However, it cannot belong to more than one group of a given group-type.

To manage the groups themselves, see the Group Manager tool at the Tools page.

 

View Available Groups

To see what groups are available,

  1. Add the Groups tab to the navigation pane if it's not already there (at the Home page, activate Groups in the ribbon).
  2. In the navigation pane, select the Groups tab.
  3. Expand the list. A quick way to show group-types and groups, as well as their member assets, is to use the button. Alternatively, click the arrow to the left of the site name to show the list of group-types; then click the arrow left of the desired group-type to view its groups.

View Group Membership

To see what assets are in a particular group,

  1. Add the Groups tab to the navigation pane if it's not already there (at the Home page, activate Groups in the ribbon).
  2. In the navigation pane, select the Groups tab.
  3. At the site name, click the arrow at the left to display the list of group-types.
  4. At the appropriate group-type, click the arrow at the left to display the member groups.
  5. At the appropriate group, click the arrow at the left to display the group's member assets.

Associate a Component with a Group

To associate a component with a group while adding the component to inventory,

  1. In the "Add Component" popup, navigate to the Groups fieldset.
  2. In the group-type field that contains the desired group, click the down arrow at the right. This will display the groups. If no text appears in the group-type field, that means no groups of that type have been established.
  3. Select the desired group.
  4. When finished with that as well as any remaining data entry for the new component, activate the Add button at the bottom of the popup.

To associate an existing component with a group,

  1. Double-click on a component in the asset tree. The View Component tab should appear in the content pane, displaying the component data.
  2. Expand the "Groups" fieldset. This will display the list of group-types.
  3. In the group-type field that contains the desired group, click the down arrow at the right. This will display the groups. If no text appears in the group-type field, that means no groups of that type have been established.
  4. Select the desired group.
  5. Activate the Save button at the upper left of the content pane.
