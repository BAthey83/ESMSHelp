Get Started with the Enterprise SMS Field Application

This topic lists steps to take to get started with the Enterprise SMS field application, which includes Enterprise SMS Desktop and Enterprise SMS Tablet. Desktop has a full suite of non-administrator functions (administrator functions are performed using Enterprise SMS Web). Enterprise SMS Tablet allows either (a) Direct Ratings or (b) customized assessment plans to be performed in the field.

If you have not yet downloaded Enterprise SMS Desktop, first follow instructions at "Download the Enterprise SMS Field Application" immediately below.

1. Download the Enterprise SMS Field Application

(Skip this step if you have already downloaded the field application.)

Download

To download the Enterprise SMS field application to your computer, first do the following in Enterprise SMS Web:

  1. Open Enterprise SMS Web.
  2. Navigate to "Hello, {username}" at the right side of the menu bar.
  3. Open the dropdown, then activate the "Download Application" option.
  4. Activate (click or tap) the desired desktop version.
  5. Wait until the folder "EnterpriseDesktop_....zip" has finished downloading .
    (Look at the lower left corner of the Enterprise SMS window    ; the download progress status should appear below the folder name, then the status indicator may disappear shortly after the download is complete.)

Extract and Store

Next, follow these instructions to unzip the application folder and store it in the desired location:

  1. Right-click on "EnterpriseDesktop_....zip" in the lower left corner.
  2. Activate "Show in folder" in the list of options.
  3. Move the .zip folder to a new location, if desired.
  4. Right-click on the .zip folder.
  5. Select "Extract all."
  6. When extraction is finished, rename the folder if desired.

2. Open the Enterprise SMS Field Application

To open the Enterprise SMS field application,

  1. Navigate to the location in your file system where you stored the EnterpriseDesktop (or renamed) folder.
  2. Activate Enterprise SMS.FieldApp.exe.

  3. If this is your first time entering Enterprise SMS Desktop, you'll be required to synchronize your desktop instance to the sync site designated in your Web instance of Enterprise SMS. For more information about synchronizing, see Synchronize Enterprise SMS.

4. Select Domain(s)

Using Enterprise SMS, you can pick and choose which category or categories of infrastructure you wish to deal with. These categories are called domains in Enterprise SMS. The selections currently listed are Buildings, Fuels, Pavements, Rails, Roofs, and Utilities.

Note: Data currently contained in the Fuels domain already crosses category boundaries, because the Defense Logistics Agency (DLA) maintains diverse real property assets associated with its mission to store and deliver fuels.

The Domains button allows you to select one or more domains to view and work with. Selecting at least one domain is required. The number in parentheses after the word "Domains" indicates how many domains are currently selected.

To select or change one or more domains,

  1. Activate the Domains button on the menu bar.
  2. In the dropdown, activate the checkbox next to each domain desired, and de-activate any currently checked boxes next to domains whose data you do not wish to view.
  3. Activate the Apply button in the dropdown.
  4. The number in parentheses on the Domains button indicates how many domains are currently selected.

6. Select a Task

The task selection bar, directly below the menu bar, is where you will select whether to work with assessment plans, work items, or a level of inventory:

  • Facilities. Use the Filters button to select facilities to be viewed, and the Fields button so select what information to view.
  • Groups. Use the Filters button to select facilities to be viewed, and the Fields button so select what information to view.
  • Components. Use the Filters button to select facilities to be viewed, and the Fields button so select what information to view.
  • assessment plans. **This documentation is under development March 2018 pending creation of assessment plans in desktop version and their transfer to Web version.
  • Work. Select the Work button to see work items. There are no work items in the sample data at present.
