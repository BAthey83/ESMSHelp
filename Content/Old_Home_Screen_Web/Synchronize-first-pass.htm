Synchronize

To make sure you have the latest data that may have been input using one or more desktop instances of Enterprise SMS, use the synchronize (sync) feature in the dropdown under "Hello, {your name}". You have the option to either accept the current sync site, or select another one.

View Sync Site

To see what is currently selected in the system as the sync site:

  1. Activate the dropdown list under "Hello, {your name}", at the upper right of the screen.

    Note: This dropdown will be referred to in the documentation as the user options list.

  2. The site listed above the other options is the current default for synchronization. It will have the sync icon to the left of the site name.

    Troubleshooting tip: If no site name appears above the "Preferences" option, this means that no site is currently selected.

Select Current Sync Site

If the site shown in the user options list is the one you want to sync with, do the following:

  1. Hover over the sync site name in the user options list.
  2. Then activate the Sync Site button that appears left of the dropdown:

    Troubleshooting tip: The Sync Site button appears and disappears with each successive highlighting of the site name in the user options list, so if you don't see it, just navigate away from the site name, then hover over it again.

  3. If choosing the sync site shown, you are now finished with the synchronization.

Change Sync Site

If you want to sync with a different site than the one shown in the user options list–or if no sync site is listed–then follow these instructions:

  1. If a site name shows in the menu bar, activate the button with the white "X" to the right of it. The Select Site button will appear in place of the site name.
  2. Activate the Select Site button, and follow instructions at Synchronize to select the site you wish to sync.
  3. Go to View Sync Site; follow instructions there and at Select Current Sync Site.
