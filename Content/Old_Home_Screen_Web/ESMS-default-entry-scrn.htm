Welcome

Welcome to the March 2018 draft version of Enterprise SMS, developed by the U.S. Army Corps of Engineers at its Engineering Research and Development Center Construction Engineering Research Laboratory (ERDC-CERL) in Champaign, IL.

Using Enterprise SMS Web, you can create customized views of inventory, condition assessment plans, and work projects or work items.

Getting Started

To learn how to get started, you can do either or both of the following:

  1. Press **activation key to be determined** to see an overlay explaining the items on the screen.
  2. Read documentation explaining how to Get Started with Enterprise SMS Web and Navigating Enterprise SMS Web .

Importing Data

Real property data can be brought into Enterprise SMS either by syncing with the desktop or tablet version of Enterprise SMS, or by selecting the "Import" option in the Tools dropdown on the main menu. Currently, either Fueler SMS file or comma-separated value (CSV) files can be imported.
