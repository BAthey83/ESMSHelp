Navigation overview

The major components of the home screen are, from top to bottom on the screen:

  • Main Menu
  • Tab Bar
  • Fields
  • Content
  • Filters

Here's how the Enterprise Web entry screen is laid out:

1. Main Menu

The main menu is persistent throughout Enterprise SMS. From it, you can:

  • select the domain and the site to view
  • open up a tool (Import, Role Permission, Work Generation, Libraries, Policies)
  • search the site or find a document
  • change login procedure or other user preferences
  • sign out

2. Tab Bar

Directly below the Main Menu, the tab bar lets you select

  • inventory (facilities, groups, or components)
  • condition assessment, or
  • work items

(entry screen overview continues on the next slide)

More detail about each of these will be provided under Home Screen Tasks.**

Tab Bar

Directly below the Main Menu on the home screen is the tab bar that allows you to select which of the following to view:

  • Facilities
  • Groups
  • Components
  • assessment plans
  • Work (work items)

Each of these views will be explained **

Fields

Directly below the **task selection bar, the Fields line displays column headings for the content area. You control which data fields will be displayed as follows:

  1. Activate the vertical Fields button (outlined in red below).
  2. Select and/or deselect checkboxes.
  3. Activate the Apply Fields button.
  4. Activate the vertical Fields button again to hide the Fields selection box; or simply navigate to and activate a different part of the screen.

    Note: The options shown in the Fields selection box will depend on what is selected at the upper level in the Fields area (Facilities, Groups, Components, assessment plans, or Work items.

Content

Immediately below the column headings in the Fields area is the content area where data is shown.

Filters

Filter selection can be accessed using the Filters button on the left at the top of the content area.

Filters can be applied as follows:

  1. Activate the vertical Filters button directly left of the content area.
  2. Fill in the desired filter information.
  3. Activate the Apply Filters button.
  4. Activate the vertical Filters button again to hide the Filters selection box; or simply navigate to and activate a different part of the screen.

Note: The options shown in the Filters selection box will depend on what is selected at the upper level in the Fields area (Facilities, Groups, Components, assessment plans, or Work items).

The first step at the home screen is to select one or more domains whose data you wish to work with. Note the tool tip at the center top of the screen:

To select domain(s),

  1. Activate the down arrow to the right of Domains.
  2. Select one or more checkboxes.
  3. Activate the Apply button.
