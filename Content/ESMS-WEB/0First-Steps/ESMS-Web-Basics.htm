Select Site

About Selecting a Site

The Select Site button on the menu bar allows you to select one site to view and work with. If you either (a) don't select a site or (b) cancel site selection, you will be accessing all of the sites you have permission for (which is also an option). Your choice is between one site and all of the sites. When all of your permitted sites are part of your active view, the data on the screen, search results, etc., take all of these sites into account.

Note: If the menu bar button reads "Select Site" instead of listing a site name, then you're viewing all sites.

Select Site

To select the site to work with,

  1. On the main menu bar, activate the site button, which either (a) lists the currently selected site or (b) displays Select Site.

  2. The "Select Site" popup window will appear, listing available sites.

  3. (Optional) If you wish to trim down the list of selections via search, follow these steps:
    1. Enter a search string in the "Site Name" field.
    2. Activate the Go button to see only those site names containing the search string.

      Tip: To see the full list of sites again after using the search field, clear the search field, then activate the Go button.

  4. Click on one site in the site list.
  5. Activate the Select button at the bottom of the popup. The popup should disappear. If it does not, and the Select button is grayed out, a valid site has not been properly selected.
  6. Alternatively, activate the Close button to cancel site selection and exit the popup.

Change or Cancel Site Selection

Change Site Selection

To change your selection of site, follow the same instructions as for Select Site.

Cancel Site Selection

To cancel your site selection and instead, see results from all sites, activate the "X" button to the right of the site name on the main menu.

Select or Change Domain(s)

This section provides (1) details about domain selection, (2) instructions for selecting a domain , and (3) instructions for changing the domain selection.

About Selecting Domains

Domains represent the various categories of infrastructure used in ESMS. You can restrict your view of the information stored in ESMS to just one domain, or you can select multiple domains to be active at one time in your information searches and other tasks. The number in parentheses after the word "Domains" indicates how many domains are currently selected.

Note: Selecting at least one domain is required.

Select a Domain

To select a domain (or domains),

  1. Activate the Domains button on the main menu bar. A dropdown will display the list of domains.

  2. Mark the checkbox next to each domain you want to see in your current session.
  3. Activate Apply.

Change Domain Selection

To change your selection of domain(s),

  1. Activate the Domains button on the menu bar.
  2. In the dropdown, activate the checkbox next to each domain desired, and de-activate any currently checked boxes next to domains whose data you do not wish to view.
  3. Activate the Apply button in the dropdown.
  4. The number in parentheses on the Domains button indicates how many domains are currently selected.
