Manage Scheme Parameters

Parameters are variables that are global to the scheme.

View Parameters

To see existing parameters, activate the Scheme Parameters button on the Workflow editor menu bar. The "Scheme Parameters" popup will appear.

Note : If no parameters have been created yet, only the + Create button will appear. Follow instructions in Add Parameter to add one or more parameters.

Add Parameter

To add a parameter,

  1. Activate the Scheme Parameters button on the Workflow editor menu bar. The "Scheme Parameters" popup will appear.
  2. Activate the + Create button.
  1. Enter a name for the parameter in the Name column.
  2. Enter the display name in the next column.
  3. Using the dropdown list in the Type column, select the parameter type (Approve, Boolean, Decimal, Integer, or String) for the variable.
  4. Activate the Save button to save the parameter. (Alternatively, activate Cancel.)

Edit Parameter

To edit a parameter,

  1. Activate the Scheme Parameters button on the Workflow editor menu bar. The "Scheme Parameters" popup will appear.
  2. Edit entries as desired in the Name, Display Name, and/or the dropdown Type column(s).
  3. Activate the Save button to save the parameter. (Alternatively, activate Cancel.)

Delete Parameter

To delete a parameter,

  1. In the "Scheme Parameters" popup, activate the gray "x" at the right of the parameter you wish to delete.
  1. Confirm the deletion in the "Delete Parameter" popup. (Alternatively, activate Cancel to keep the parameter and not delete it.)
