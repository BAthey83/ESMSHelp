Generate Work Items

Generate

To generate work items,

  1. Make sure that you have set standards (threshold values) for work generation and selected one or more target sites. (Selecting a domain is not required as of August 17, 2018.)
  2. Activate the Generate Work Items button, which is the last item on the Work Generation screen.
  3. Review the results popup window that shows how many work items have been generated:

  4. Activate the OK button to close the results popup.

View Results

To view the results,

  1. Navigate to the Search screen.
  2. Select the site you generated work items for. (If you generated work items for multiple sites, you will need to repeat steps starting here.)
  3. Activate the Work tab.

  4. Summary information about each work item (one work item per row) should appear on the Work screen. In this example, there is just the name of the component to which the work item applies.

  5. To change what data columns are shown for each work item, see the Search screen topic Set Up Fields and Filters.
  6. To see more detail about an individual work item, double click on it and you will be brought to the Work Item Details page.
