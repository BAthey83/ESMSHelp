Organizations Screen

This screen shows the Organizations information. It is accessible from the Enterprise SMS main menu.

Organization Search

You can look for Organizations using the Search screen shown above. Type in the Organization in the search bar, or select one from the resulting menu.

Review and Edit Organization Information

Once you have selected an Organization, you can review and edit information relevant to the Organization on the following screen(s) :

General Tab

The General tab contains the key information for an Organization, including the overall Value, CI, FI, PI, FCI, and EFCI. The tab also displays information related to the FCI by Domain for each Site in the Organization.

In addition, the General tab for many Organizations features a color-coded Performance Records chart below the FCI by Domain information that displays the progress of CI, FI, PI, FCI, and EFCI data, year-by-year.

You can also roll up Organization data on this tab. Simply select the Rollup Organization button located above the General information portion of the page.

Funding Tab

The Funding tab contains information related to an Organization's annual projected budget. In addition to reviewing existing funding information, users can Add New Funding by selecting the + button next to Projects.

When the Add New Funding popup appears, you can edit and adjust budget information for as many years as needed.

Note: If you need to add data for more years besides the initial five that appear, select Add Year.

Once you have inserted your funding information, select the Save button. If you wish to discard it, click Cancel.

The data you added or edited will then appear in the following fields of the Funding window:

  • Projects: Indicates the number of Projects using the funding information.
  • Name: Indicates the funding source.
  • Years: Indicates the time span of the funding.
  • Total Funds: Indicates the funding's full dollar amount.
  • Obligated: Indicates how much of the funding is set aside for mandatory use.
  • Committed: Indicates how much of the funding has been allocated.
  • Remaining: Indicates the amount of funding that is still available to use.

In addition, you can edit existing funding instances by selecting the pencil icon or delete them by selecting the trashcan icon .

To rearrange the order of your Organization's funding instances, select the icon, and then drag and drop the instances in your preferred order.

Permissions Tab

The Permissions tab is where you can view and edit user roles within Enterprise SMS.

Note: Your ability to view and edit Permissions will depend on which Permissions you can access. If you have any questions regarding Permissions, consult your system Administrator.

Once you have updated Permissions settings, click the Save button. If you need to remove your changes, select Reset.

For more details about the Permissions hierarchy, refer to the Permissions topic.

Reference Books Tab

The Reference Books tab houses the Reference Book data for the following book types: Service Life, Cost, Cost Modifier, Inflation, and CII.

You can choose the Reference Book type from the corresponding dropdown menu.

For more information on Reference Books, refer to the Reference Books topic.
