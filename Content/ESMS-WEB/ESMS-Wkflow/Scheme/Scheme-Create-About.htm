About Creating Schemes

If your workflow needs to follow a scheme that does not yet exist, you can use the Scheme Editor to create a new scheme.

A scheme defines the workflow process, the steps to be followed in creating an assessment. Each scheme is created by someone in the user's organization who has appropriate permissions to do so.
