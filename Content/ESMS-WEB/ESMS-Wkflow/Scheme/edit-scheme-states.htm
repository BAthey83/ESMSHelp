Edit Scheme States

To edit details about a state, activate the state box in the diagram and the corresponding "State" popup will appear with the details:

  • Name - Name of the state, which appears in the state box on the diagram. This can be changed.
  • Initial / Final - If the state is to be the first state in the work flow, mark the Initial checkbox; if the state is to be the last state in the work flow, mark the Final checkbox .
