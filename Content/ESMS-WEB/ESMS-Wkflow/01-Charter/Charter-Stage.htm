Charter Stage

This topic describes the Charter stage of developing an ESMS assessment.

After you have added an assessment into the Workflow Engine, you can start the process of designing an assessment. There are six stages to be navigated in the development process, with transitions between the steps that may require approvals. The stages are listed in Understanding Assessments. The transitions are guided by the user-defined scheme.

—Quick Links—

Tasks at this stage are to

  • enter or edit assessment details, at the Details tab
  • view read-only assessment information that was set during assessment creation but can also be viewed, at the Details tab
  • select assets to be assessed, at the Assets tab
  • select Groups to work with, at the Groups tab
  • select planning team members, at the Pre-Assessment Team tab
  • transition to the next stage when done.

Details Tab

The Details tab under Charter is the default tab that will show when you either (a) activate the Charter stage of developing an assessment or (b) have just finished creating an assessment instance in Workflow.

Editable Fields

The Details tab displays the following fields that can be edited:

  • Name of the assessment
  • Project Charter
  • Budget
  • Planned Assessment Date - To enter or change the planned assessment date, you can either enter the date into the field, or navigate into the field and activate the larger down triangle at the right side of the field to use the calendar tool. If the field contains real contents (rather than the mm/dd/yyyy format indicator), you can clear its contents by clicking the "x" in the field.

At the bottom right of the screen is the Save button, which will save the assessment details. Next to the Save button is the Delete button. This will delete the entire assessment (after confirmation).

Read-Only Fields

The following fields can be viewed but not edited; they were entered when the assessment was created as described in Add Assessment

  • Site - Shows the Site that the assessment pertains to
  • Domain - Shows the Domain that the assessment pertains to
  • Scheme - Shows the scheme being used. If you want to know more about what a scheme is, see Scheme in Understanding Assessments
  • Pre-Assessment Template - Shows the pre-assessment template being used
  • Assessment Template - Shows the assessment template being used

At the bottom right of the screen is the Save button, which will save the assessment details. Next to the Save button is the Delete button. This will delete the entire assessment (after confirmation).

Assets Tab

At the Assets tab, a listing of assets selected for assessment will appear. Here you can add assets that you want to assess and/or generate work against. You can also remove assets from the listing.

Add Asset to the List

To select one or more additional assets,

  1. Activate the Assets button below the listing. The "Asset Select" popup will appear.

  2. Select one or more assets in the listing on the popup.

    Note: Activating an asset will select it if it is not selected, or deselect it if it is selected. An asset will remain selected when you scroll, and will remain selected if you browse to other pages in the listing using the page ladder navigation below the listing of assets.

  3. To narrow the selection of assets that will appear, you can enter search text in the box at the top of the popup, and then activate the Search button. More detail about this search is included in the next step immediately below.
  4. By default, the search is performed only on the Names of the assets. However, if you want to search by another field such as Facility or RPUID, you can activate the gear button and designate a different field for the search. Only one field can be searched at a time.

  5. When you are satisfied with your selection, activate the Select button at the bottom of the popup. (Alternatively, you can activate Close to cancel the selection.) You will be returned to the Assets tab on the Charter screen, showing the Asset Selection containing the original list of assets plus your additions.

Remove Asset from the List

To delete an asset from the Asset Selection listing,

  1. Navigate to the Assets tab on the Charter screen.
  2. In the Asset Selection list, activate the "x" button to the right of each asset you wish to remove from the list.

Groups Tab

At the Groups tab, a listing of groups selected for assessment will appear. Here you can add groups that you want to assess and/or generate work against. You can also remove groups from the listing.

Add Group to the List

The process of adding a group to the list happens in two sequential popups. In the first popup you filter by group type. In the second popup you select one or more groups from the list of groups that passed the filter.

To select one or more additional groups,

  1. Activate the Groups button below the listing. The "Group Type Filter" popup will appear.
  2. First, filter by group type: Select one or more group types in the listing on the popup.

    Note: Activating a group type will select it if it is not selected, or deselect it if it is selected. A group type will remain selected when you scroll, and will remain selected if you browse to other pages in the listing using the page ladder navigation below the listing of group types.

  3. Activate Next at the bottom of the popup when you are satisfied with your selection of group types. The "Group Select" popup will appear.
  4. Next, especially if the list of results is longer than a page, filter by search text: Enter search text into the field at the top of the popup, and activate Go.

  5. Select one or more groups in the list.

    Note: As with selecting group types, activating a group will select it if it is not selected, or deselect it if it is selected. A group will remain selected when you scroll, and will remain selected if you browse to other pages in the listing using the page ladder navigation below the listing of group types.

  6. When you are satisfied with your selection, activate the Select button at the bottom of the popup. (Alternatively, you can activate Close to cancel the selection.) You will be returned to the Groups tab on the Charter screen, showing the group selection containing the original list of groups plus your additions.

Remove Group from the List

To delete a group from the Group Selection listing at the Groups tab,

  1. Navigate to the Groups tab on the Charter screen.
  2. In the "Group Selection" list, activate the "x" button to the right of each group you wish to remove from the list.

"Pre-Assessment Team" Tab

Assessors selected for the Pre-Assessment stage are listed at the Pre-Assessment Team tab. Here you can add or remove team members from the list, and assign each of them to one or more appropriate disciplines (Civil, Electrical, Structural, etc.)

Add Assessor to the Team List

To add an assessor to the pre-assessment team,

  1. Activate the Add button. The "User Search" popup will appear.

  2. Enter search text in the Name field at the top of the popup, then activate the Go button at the right side of the field.

    Note: You will get all results where the search text appears in the user name, even if it is in the middle or at the end of the user name.

  3. If too many results appear, or if no results appear, you may want to change the search text and try again.
  4. Once the list appearing in the "User Search" popup is sufficiently filtered, select one user name from the filtered list. You can only select one name per Add activation.
  5. To finalize the selection, activate the Select button at the bottom of the popup. (Alternatively, activate Close to cancel your selection.) You will be returned to the Pre-Assessment Team tab.

Edit Assignment of Disciplines

To assign an assessor to one or more disciplines (Civil, Electrical, Structural, etc.),

  1. Activate the Manage Disciplines button in the row for the assessor. The "Manage Disciplines" popup will appear.
  2. Mark the checkbox for each desired discipline.
  3. To unmark a checked box, activate the box again.
  4. When you are satisfied with your selection, exit the "Manage Disciplines" popup. The selection will be preserved.

Remove Assessor from the Team List

To remove an assessor from the list at the Pre-Assessment Team tab, activate the Remove button in the corresponding line of the table.

Transition

When entries are complete and any required approvals have been obtained (this is governed by the scheme; see Transitions in Understanding Assessments), you can proceed to the next stage, Pre-Assessment Stage.
