Desktop Sync / Assessment Stage

After the Review stage is complete, the work flow will transfer to the ESMS field application (Desktop or Tablet) for the assessment.

This topic is not intended as comprehensive instructions for assessment, but simply to provide a high-level glimpse at what happens in the interim where the action passes temporarily outside of ESMS-Web.

Pre-Synch

To perform pre-assessment synchronization,

  1. On the main menu in ESMS Web, activate the dropdown of user options. img
  2. Select Application Sync Settings from the dropdown.
  3. In the popup, select the Workflow Plan to be made accessible to ESMS Desktop or Tablet for the assessment that has been prepared for with asset selection, group selection, assessment team selection, and Assessment Questionnaire.
  4. Mark the Sync Images checkbox if you want images as well as data to be synchronized.
  5. Next, log in to ESMS Desktop or Tablet and activate the Sync button to synchronize.

Assessment Questionnaire

After synchronization, execute the assessment questionnaire, either in the field with a tablet or else in ESMS Desktop using field notes.

Work Item Generation

Generate work items in ESMS Desktop, based on answers to the assessment questionnaire. After re-synchronization, these work items will appear in the Work Items table at the Post-Assessment stage in ESMS Web.

Re-Synch

When the assessment has been completed, synchronize again using the Sync button to make the assessment results available to ESMS Web.
