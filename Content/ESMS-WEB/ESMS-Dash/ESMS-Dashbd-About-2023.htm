Understanding the Enterprise SMS Menu

This topic introduces the purpose of the Enterprise SMS Menu and provides a brief overview of each of the task areas.

What Can I Do Here?

  • Because the menu serves as the central hub connecting all of the task areas in Enterprise SMS Web, you can transfer from one task area to another quickly and easily.
  • You can initiate a search from the menu without having to navigate to the Asset Inventory screen first. (Before searching, you must select one or more domains and, if desired, select a site.)
  • Via the user dropdown menu in the upper right of the screen, you can select from the following options: (a) view Enterprise SMS announcements, (b) set user preferences, (c) save your current workspace as the default; (d) download a Desktop version of Enterprise SMS; (e) change sync settings for synchronizing with Enterprise SMS Desktop; (f) clear the application cache; or (g) log out.

Note: If you don't have permission to administer a team, the Teams icon won't appear on your menu.

Overview of Task Areas

Each task area in Enterprise SMS Web is represented by an icon on the main menu. Below is a brief description of each of the task areas.

Asset Inventory

At the Asset Inventory screen, you can search for real property assets, assessment plans, and work items that meet your search criteria. The asset inventory process is driven by

  • filters (selected in a context-sensitive popup)
  • your search term (typed in the field on the main menu bar).

Results will show dynamically in the content area of the page.

Starting from the Asset Inventory page, you can also navigate to pages showing detailed information about assessment plans, work items, and individual assets.

For more detail about the Asset Inventory screen, see Navigating the Asset Inventory Screen and Understanding the Asset Inventory Screen.

Facilities

At the Facilities screen, you can:

  • search facilities at a given site
  • search facilities at a single domain
  • see the properties of a given facility
  • review domain-specific facility properties
  • create components, assessments and work items
  • upload documents
  • search the cost modifier book
  • review performance records

Groups

At the Groups screen, you can perform a search and see the following properties:

  • group name
  • group type
  • group site
  • parent group
  • group description

Components

At the Components screen, you can see the following properties:

  • component name

  • component type

  • parent site

  • facility number

  • quantity

  • year installed

  • operational status

  • CI Condition Index

  • MDI Mission Dependency Index

  • UNIFORMAT II system

  • UNIFORMAT II component

  • UNIFORMAT II section

  • functional area

  • facility name

  • (facility) RPUID Real Property Unique Identifier. This is a facility-level property, described in JIRB 2004-07-19, Department of Defense Real Property Inventory (RPI) Initiative, dated July 19, 2004.

  • repair date

  • expected service life

Assessments

At the Assessments screen, you can see the following properties:

  • date of assessment
  • date updated
  • site name
  • assessor name
  • result
  • completion percentage

Work

At the Work screen, you can see the following properties:

  • work item name
  • current situation
  • deficiency
  • impact
  • completion date
  • target component
  • regulatory requirement
  • scope of work
  • cost
  • reference

Map Search

At the Map Search screen, you can:

  • select a site map
  • choose between grayscale, satellite view or street view
  • select the option to add CI Condition Index indicators, separated by color
  • (Optional) color pavement components based on PCI or CI Condition Index

Assessements Work

At the Assessments Work screen, you can perform work analyses, as well as create and update existing work plans. This screen also allows you to search and update projects, create and update workflow schemes and review tasks related to your work plans.

Work Analysis

At the Work Analysis screen, you can:

  • choose a policy
  • review priorities
  • select a profile
  • configure a work plan
  • review your queue
  • compare reports

Assessments

At the Assessments screen, you can:

  • create a new assessment
  • navigate specific steps in an existing assessment

Projects

At the Projects screen, you can:

  • filter and view a list of available projects
  • export the filtered list of projects
  • add projects
  • view and edit project details

The Projects screen is accessible from the main menu to users with Project privileges.

Workflows

At the Workflows screen, you can:

  • view, add, or edit workflow schemes to be used with assessments
  • manage scheme parameters
  • view scheme details

Permissions Note: Only users with "WorkflowRole" permission set at an ESMS Administrator's Security screen can access the Workflow editor screen.

Tasks

At the Tasks screen, you can:

  • generate work items
  • specify threshold levels for CI Condition Index and RSL Remaining service life.

Libraries

At the Libraries screen, you can:

  • add custom component and group types to the ESMS catalog
  • design templates used to create assessment questionnaires
  • develop books for CII Component Importance Index, service life, inflation, cost, cost modifiers, and work actions

Catalog

At the Catalog screen, you can:

  • manage the inventory catalog
  • create category templates
  • search templates for components and assets
  • create and update PowerBi reports
  • create and delete ArcGis endpoints
  • create and configure mapping sets
  • import or export configurations

Reference Books

At the Reference Books screen, you can:

  • select or create new CII Component Importance Index books
  • select or create new service life books
  • select or create new inflation books
  • add, edit, duplicate, or delete cost books
  • add, edit, or delete cost book modifiers
  • add, delete, or duplicate work action books

Administration

Note: the Administration screen is only viewable by ESMS administrators. It will not appear on the menus of users who are not administrators.

At the Administration screen, you can:

  • manage users and roles
  • create and delete teams
  • add and delete permissions and roles
  • search and configure organizations

Users

At the Users screen, you can:

  • view user details
  • add a user
  • edit user details
  • delete a user
  • assign a role to a user
  • create a role or copy an existing role

Teams

At the Teams tab on the Teams screen, you can:

  • add and delete teams
  • add and delete team members
  • assign roles (security permissions) to teams
  • rename teams

Permissions

At the Permissions screen, you can:

  • add a security permissions role
  • delete a security permissions role
  • edit a security permissions role
  • change a role's name

Permissions Note: These tasks require permission to administer a team.

Organizations

At the Organizations screen, you can:

  • search organizations
  • rollup organizations
  • view, add, or edit funding
  • define multiple funding sources
  • create funding restrictions
  • review FCI Facility Condition Index. data by domain
  • find associated sites
  • check performance records

Import/Export

At the Import/Export screen, you can:

  • import ArcGIS and CSV data from external sources
  • export ArcGIS and CSV data from external sources

Note: the import/export screens are only visible to users with administrator privileges.

Import Configuration

At the Import Configuration screen, you can:

  • open ArcGIS import requests
  • request ArcGIS imports

Export

At the Export screen, you can:

  • open CSV export requests
  • request CSV exports

Documents

At the Documents screen, you can:

  • search for a document
  • search for documents with a particular tag
  • view a document
  • download a document

Reports

At the Reports screen, you can:

  • (Optional) narrow the list of available reports to those tailored for just one of the Enterprise SMS domains
  • select a report to be generated
  • view the report

Note: Some icons such as Catalog will only appear if you have ESMS administrator-level permissions.
