Understanding the Enterprise SMS Dashboard

This topic introduces the purpose of the Enterprise SMS Dashboard and provides a brief overview of each of the task areas.

What Can I Do Here?

  • Because the dashboard serves as the central hub connecting all of the task areas in Enterprise SMS Web, you can transfer from one task area to another via the dashboard.
  • You can initiate a search from the dashboard without having to navigate to the Inventory screen first. (Before searching, you must select one or more domains and, if desired, select a site.)
  • Via the user dropdown menu in the upper right of the screen, you can select from the following options: (a) save your current workspace as the default; (b) change sync settings for synchronizing with Enterprise SMS Desktop; (c) download a Desktop version of Enterprise SMS; (d) set your user preferences; (e) view Enterprise SMS announcements; (f) set the current page as your homepage; or (g) logout.

Note: If you don't have permission to administer a team, the Teams icon won't appear on your dashboard.

Overview of Task Areas

Each task area in Enterprise SMS Web is represented by a tile on the dashboard. Below is a brief description of each of the task areas.

Inventory

At the Inventory screen, you can search for real property assets, assessment plans, and work items that meet your search criteria. The inventory process is driven by

  • filters (selected in a context-sensitive popup)
  • your search term (typed in the field on the main menu bar).

Results will show dynamically in the content area of the Inventory page.

Starting from the Inventory page, you can also navigate to pages showing detailed information about assessment plans, work items, and individual assets.

For more detail about the Inventory screen, see and .

Maps

At the Maps screen you can

  • select a site map
  • choose between grayscale, satellite view or street view
  • select the option to add CI indicators, separated by color
  • (Optional) color pavement components based on PCI or CI

Reports

At the Reports screen you can

  • (Optional) narrow the list of available reports to those tailored for just one of the Enterprise SMS domains
  • select a report to be generated
  • view the report

Teams

At the Teams tab on the Teams screen you can

  • add and delete teams
  • add and delete team members
  • assign roles (security permissions) to teams
  • rename teams

At the Roles tab you can

  • add and delete customized security permissions roles
  • edit roles
  • rename roles

Work Analysis

At the Work Analysis screen you can

  • choose a policy
  • review priorities
  • select a profile
  • configure a work plan
  • review your queue
  • compare reports

Work Plans

At the Work Plans screen you can

  • create a new work plan
  • navigate specific steps in an existing work plan

Tasks

At the Tasks screen you can

  • generate work items
  • specify threshold levels for CI and RSL
