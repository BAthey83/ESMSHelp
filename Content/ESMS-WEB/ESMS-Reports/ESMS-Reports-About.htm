Understanding the Reports Screen

This topic introduces tasks you can perform at the Reports screen and provides links to greater detail for each. It also provides an overview of navigating the Reports screen.

What Can I Do Here?

You can select among report types that are tailored for the different domains in Enterprise SMS, then generate and view the report.

Reports Tasks

At the Reports screen you can:

  • edit and review system and component data for one or multiple Enterprise SMS domains via the engineering library
  • expand or narrow the list of available reports to those tailored for one or multiple Enterprise SMS domains
  • select a report to be generated
  • view the report

Engineering Library Report

The Engineering Library contains the relevant System and Component information for items within an Organization. You can minimize or expand the scope of information for Inventory, Inspection, and Work domains as needed.

Navigating the Reports Screen

Before you choose a domain, you must select an Organization and a Report.

Organization options include:

  • Air Force
  • Department of Defense (DoD)
  • Army Active

Report options include:

  • Engineering Library
  • ESMS Real Property
  • Site Report Test
  • SRS Test
  • USA RAIL

At the Reports screen, you can choose from three separate domain tabs: Inventory, Inspection, and Work.

Note: Not all Organizations have the same Report options available. Please contact your system administrator for details.

Filters on Visual

Open the Filters on Visual list for any Filters or slicers applied that may change the Report results. If none are applied, you will see the following screen:

Focus Mode

Focus Mode allows you to customize the Systems and Components to as few (or as many) as you need.

More Options

In addition, the Reports screen offers the following functions, which can be accessed by clicking on the ellipsis under the Inventory domain:

  • Search: Search for the optimal Systems and Components that you need for your report.
  • Export Data: You can export data in .xlsx and .csv formats. Export options include:
    1. Data with Current Layout: Select this option for table and matrix visuals.
    2. Summarized Data: Choose this feature to export the data used to create your visual (such as sums, averages, and medians).
    3. Underlying Data: Use this option if the visual does not have aggregates or measures.
  • Show as a Table: Select this option to view the report options in table format.
  • Spotlight: Choose this option to focus solely on the chart nearest the domain tab.
  • Get Insights: Select this option to remove the results from all charts except for the Component / Specification Name / UM chart.
  • Sort Descending: Choose this option to sort results in descending order.
  • Sort Ascending: Select this option to sort results in ascending order.
  • Sort By: Click this option to sort results by Class, Subclass Type, or CompTypeName.
 

System & Component Options

System options include:

  • Electric
  • Gas
  • Stormwater
  • Thermal
  • Wastewater-Industrial
  • Wastewater-Oily
  • Wastewater-Sanitary
  • Water

  • Each System contains multiple Component options. For example, the Electric system consists of several components, including Airfield Lights and Capacitor Banks.

Once your selections are confirmed, you can choose one of the following domains:

  • Inventory Domain
  • Inspection Domain
  • Work Domain

You can select individual items in the charts found in each domain to customize your Report results.

Note: If you select an option from any of the charts instead of checking an option from the System & Components section, the Report will automatically populate with the data relevant to your selection. This feature applies to all three tabs.

Inventory Domain Tab

The Inventory Domain tab contains the following charts:

  • System Description
  • Component / Component Description / Classification_SDSFIE / SDSFIE Entity / SDSFIE Enumerant
  • Component / Specification Name
  • Subcomponent / Subcomponent Description
  • Component / Attribute / # / Data Type / Description
  • Picklist Value / Lower / Upper

Inspection Domain Tab

The Inspection Domain tab includes the following charts:

  • Assessment Type / Description
  • Assessment Type / Rating / Rating Criteria
  • Assessment Type / Defect Name / Distress / Severity / Defect Category / Description

Work Domain Tab

The Work Domain tab includes the following charts:

  • Subcomponent
  • Component / Activity Name / UM / Narrative
  • Task Number / Task Name / UM / Duration / Crew / Description

ESMS Real Property Report:

Sites Button:

The Real Property Sites button lets you review the PRV and Facility data for as few or as many Installations as needed.

You can narrow down the Installation data that appears in the chart by Facility type. Facility types include:

  • (Blank)
  • Administrative
  • Hospital & Medical
  • Housing & Community
  • Land
  • Maintenance & Production
  • Operation & Training
  • Research, Development, Test, and Evaluation
  • Supply
  • Utility & Ground Improvements

Each Facility type holds several submenus to tailor the Installation data to any Facility type. (For example, the Maintenance & Production submenu contains multiple Maintenance Facilities and Production Facilities to create a Report).

Once you have selected your Facility type(s), the PRV and Facility data will appear for the chosen Installation(s) in the main window.

Facilities Button:

The Real Property Facilities button allows you to review Facility data for as few or as many Facility types as you require.

You can narrow down the Facility data with dropdown menus for:

  • RPUID
  • Facility Type
  • Operational Status

There is also a Year Built section that can be modified by entering the year range in the two text boxes, or by using the slider underneath.

Alternatively, you can choose a Facility type in the top chart to obtain the necessary report data.

Once you have input your data, the following results will appear in the bottom chart on the main window:

  • RPUID
  • Facility Number
  • Facility Name
  • MDI
  • Year
  • Status
  • Quantity
  • Unit of Measure (UM)
  • PRV

Lastly, the Real Property Facilities button offers large text boxes which display the relevant Facility Count and Total PRV data.

Site Report Test:

Select the Site Report Test to create a Site Report example that you can explore before the final Report is produced.

SRS Test Report:

The SRS (Software Requirement Specification) Test Report provides users with a foundation for their final report.

USA RAIL Report:

The USA RAIL Report implements the familiar features from RAILER, but with more customization and visualization options. There are nine (9) tabs for USA RAIL data:

  • Facilities
  • Network Info
  • Zones
  • Components
  • Inspection Info
  • Work Plan
  • Component Library
  • Defect Library
  • Work Action Library

All of these tabs allow you to input the Domain, Organization, and Site Name required for your report via drop-down menus.

Facilities Tab:

At the Facilities tab, you can review the following details:

  • Total Track Miles
  • Number of RPUIDs
  • Total PRV
  • Average TSCI
  • Asset Status

There are dropdown menus to select the RPUID and Status. You can also type in the Date Built range, or use the slider underneath to create the range.

There are also charts for the following items:

  • Railroad Network Locations
  • Real Property Assets

Network Info Tab:

The Network Info tab lets you customize the Area Name, Track Name, Segment Name, Construction Type, Category, and Use options so you can review the associated data.

Here, you can also review the following information:

  • Segments
  • PRV
  • Total Length (in Feet)
  • Repair Needs
  • Average TSCI
  • Track Category
  • TSCI by Length (in Feet)
  • Track Areas
  • Tracks
  • Track Segments

Zones Tab:

The Zones tab features the following options: Area Name, Track Name, Segment Name, Zone Type, and Zone Name.

Under the Consolidated Zones button, you can input data that you want to review for up to six (6) Attributes and Attribute Values. The results appear in the Track Zones window.

The Bridge Details button displays the Railroad Segment, Railroad Curve, Railroad Bridge, Railroad Clearance Restriction, and Railroad Grade data on an easy-to-view chart. The Railroad Bridges window lists additional relevant data by track.

Select Curve Details to review items such as a track curve's length, curvature type, and maximum speed.

Choose Clearance Details to explore information related to a railroad's clearance restrictions.

Components Tab:

The Components tab contains dropdown menus to customize the Area Name, Track Name, Segment Name, Component Type, and Component Name data. In addition, you can input data that you want to review for up to six (6) Attributes and Attribute Values. The Track Components window displays the results.

  • The Consolidated Inventory button lists the Component Type, Component Name, Quantity, and Units.
  • Track Structure Details include Rail Weight, Rail Section, Tie Size, and Tie Material.
  • Turnout Details consist of the Turnout Name, Orientation, Direction, Frog Type, and Frog Size.
  • Grade Crossing Details include Road Name, Length, Surface Material, Flangeway, Protection, and Bolted Joints.
  • Track Appliance Details contain the Station and Appliance Type data.
  • Track Drainage Details are comprised by the Drainage Type, Material, Length, Diameter, Height, and Width.

Inspection Tab:

In addition to the full list of Defects, the Inspection tab lets you review the Defect Count by Restriction, the Defect Count by Component Type, and Frequently Occurring Defects.

There are drop-down menus for the Area Name, Track Name, Segment Name, Component Name, Inspection Type, Inspection Sub-Type, and the Recent Inspection level. You can adjust the Inspection Date range either by inputting the dates or by adjusting the slider below.

Work Plan Tab:

The Work Plan tab contains the Total Repair Cost by Restriction, the Cost by Component, and the Top Work Items by Cost information. The Work Items chart lists the Segment, Begin and End Stations, Component and Sub-Component, Work Action, Quantity, Units, Unit Cost, Total Cost and Defect(s).

Drop-down menus let you customize data for the Work Plan Name, the Work Project ID, the Area Name, the Track Name, the Segment Name, and the Component.

Component Library Tab:

The Component Library tab features dropdown menus that allow you to input the Item Class, Object Type, Attribute, and Attribute Value.

After inputting your data, the results appear in two separate charts. Chart one (1) lists the Object, Object Type, Unit of Measurement (UM), and Attribute. Chart two (2) displays the Attribute Value.

Defect Library Tab:

The Defect Library tab comes with dropdown menus to input the Assessment Type, the Assessment Template, the Observation Type, and any relevant Defect or Restriction.

Once this information is updated, you can see the resulting data in two separate charts. Chart one (1) lists the Assessment Template, Assessment Question, Defect, Units, and Restriction data. Chart two (2) lists the Work Action Description.

Work Action Library Tab:

The Work Action Library tab offers dropdown menus to input the Component Type, the Work Action Description, and the Level. In addition, you can adjust the Unit Cost in dollars either by inputting the data in text boxes or via the slider (located directly underneath the text boxes).

Once the data is added, you can review the following data in chart one (1): Level, Work Action Description, Component Type, Unit Cost and Unit of Measurement (UM). Chart two (2) describes the Defect(s) that require attention.
