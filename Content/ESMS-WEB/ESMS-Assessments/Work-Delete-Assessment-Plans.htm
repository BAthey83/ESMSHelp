Delete Assessment Plan

To delete an assessment plan,

  1. Navigate to the table of assessment plans. (If you are inside of an assessment plan, you can return to the table of assessment plans by activating the browser's back arrow button until you reach the table.)
  2. Select the assessment plan you want to delete. The assessment plan's Details will be displayed.
  3. At the lower right corner, there are two buttons: Save and Delete. Activate the Delete button.
  4. In the popup box asking "Are you sure you want to delete this assessment plan?", activate the OK button. (Alternatively, activate Cancel to cancel the deletion.)
