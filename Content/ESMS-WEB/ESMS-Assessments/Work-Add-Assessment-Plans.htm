Add Assessment Plans

Permissions Needed for Assessment Plans/Workflow

  • To edit or view information in stages of an assessment plan after it has been created, you need the ESMS permission named "Assessment Plan Editor". This can be set up either (a) as a scoped permission / role (restricted to a particular site), or (b) as a global permission, to edit or view any assessment plan.
  • To create an assessment plan, one either with or without workflow, you need "Assessment Plan Role" permission. This permission allows you to activate the Save button in the "Create Assessment Plan" popup. It can be set up either (a) as a scoped permission / role (restricted to a particular site), or (b) as a global permission.
  • To see the Workflow Editor tile on the menu and to edit or create schemes that govern the workflow through an assessment plan, you need "WorkflowRole" permission. The "WorkflowRole" permission is global only.

These permissions are granted by an ESMS administrator at the Security tile, Users tab, Global Permissions and/or User Workspace section.

Create Assessment Plan

To add a new assessment plan, activate the Create Assessment button above the Assessment Plan table.

Next you will enter data about the assessment plan. If applicable, this will include assigning individuals to permissions roles specified in the scheme.

Enter Assessment Plan Data - All Assessment Plans

All assessment plans need the following at the "Create Assessment Plan" popup:

  1. Site. To initiate selecting a site, first activate the Select Site button. A new popup will appear.

Note: Only one site may be selected for an assessment plan.

  1. Check one and only one box to select the site.
  2. The popup will close and you will be back at "Create Assessment Plan."
  3. Domain. Select a domain from the dropdown list.
  4. Budget. Enter the budget amount for the assessment plan here.
  5. Name. Enter a name for the assessment plan.
  6. Charter . Some text for the assessment plan's charter must be entered here. You can later edit or expand on it at the Details tab in the Charter Stage.
  7. Planned Assessment Date. Enter or select the planned assessment date here.

Tips: To select, navigate into the field box and activate the larger down arrow to bring up a calendar tool. To clear the date, navigate into the field box and activate the Clear button (the Clear button will appear only if a date has been entered or selected).

  1. Operator and Assessor Templates. Select the checkbox to enable the Operator and Assessor Templates.
  2. Do one of the following based on whether the Workflow tool will be used for the assessment plan:
  1. If Workflow will not be used, skip to Save and Create in this topic.
  2. If Workflow will be used for the Assessment Plan, continue at Enter Workflow Data.

Enter Workflow Data

Note: If Workflow will not be used for the assessment plan, skip to Save and Create after completing the steps above in Enter Assessment Data - All Assessment Plans.

  1. To continue, you must first mark the Enabled checkbox to activate the remaining Workflow fields. (Shaded fields will become white.)
  2. Scheme. Select a scheme from the dropdown.

    IMPORTANT: If you need to edit a scheme in order for it to appropriately guide the workflow of a given assessment plan, make sure to do all edits necessary BEFORE assigning the scheme to the assessment plan at the "Create Assessment Plan" popup. The assessment plan will function from a snapshot of the scheme taken at the time of creating the assessment plan, and will not reflect any changes made later to the scheme.

  3. Pre-Assessment Plan Template. Select a pre-assessment plan template from the dropdown.

    4. Productivity Tip: For selecting this template and the next one, you can type the first part of the template name in the blank field, to bring up a match.

  4. Assessment Plan Template. Select a assessment plan template from the dropdown.
  5. Permissions. Do one of the following based on whether the Permissions button is grayed out or active:
  1. If the Permissions button is grayed out, no permission roles need to be assigned. Skip to Save and Create.
  2. If the Permissions button is active (not grayed out), this means that the scheme you selected requires one or more people to be designated for roles that approve transitions from one stage to the next in the workflow. See the following list of instructions for how to designate permissions roles.

Permissions

Note: An assessment that does not use Workflow (and thus has no scheme) will not need Permissions.

Depending on the scheme selected, role permissions may need to be set for the new assessment plan. If the Permissions button near the bottom of the "Create Assessment Plan" popup is active (not grayed out), this means that the scheme you selected requires one or more people to be designated for roles that approve transitions from one stage to the next in the workflow.

If you need to know more about Workflow permissions or how to manage them, see the topic Manage Workflow Permissions in the documentation about the Workflow editor.

To assign users to permissions roles,

  1. Activate the Permissions button at the bottom of the "Create Assessment Plan" popup. The "Set Permissions" popup should appear.

Note: Before you can set permissions, all the previous fields in the "Create Assessment Plan" popup must be filled in.

  1. In the "Set Permissions" popup, activate the Add More button in a permissions category you want to add one or more users to. A popup will appear with a list of users to select from.
  2. If the user list spans multiple pages, you may need to bring appropriate names into view for selection as follows:
    1. Use the Name search field and the Go button to narrow down the selection list.
    2. As needed, use the pagination display at the bottom of the popup to step through the list of users.
  1. Select one or more users directly from the list as displayed in the popup. Activate an unselected name to select it, and activate a selected name to deselect it.
  2. Activate the Select button. You will be returned to the "Set Permissions" popup. You may repeat the previous three steps and this one as needed to select more users into the category.
  3. As needed, move on to another category in the "Set Permissions" popup and repeat the process, until you are finished with selections.
  4. When permissions are set for all desired users, activate the Close button in the "Set Permissions" popup. The "Set Permissions" popup will close, and your selections will be retained.

IMPORTANT: All permissions must be set at this stage before saving.

  1. Next, be sure to save and create the assessment plan (or cancel, if desired), as described in Save and Create.

Save and Create

To save the assessment plan data and create the assessment plan, activate the Save button at the lower right of the "Create Assessment Plan" popup.

Result: If all needed information has been entered, the assessment plan will appear on a row of the Assessment Plans table.

Next Step

Next you can select the assessment plan in the table and you will be brought to the Details tab of the Charter Stage. To access other tabs and stages, use the Workflow navigation tool (see Assessment Plan Navigation for instructions).
