Understanding the Work Screen

This topic introduces tasks you can perform at the Work screen and provides links to greater detail for each. It also provides an overview of navigating the Work screen.

What Can I Do Here?

At the Work Generation screen, you can automatically generate work items by first selecting target site(s) and then specifying thresholds for condition index (CI) and remaining service life (RSL).

Work Tasks

  • Select site(s) to generate work for
  • Set standards for work generation
  • Generate work items

Navigating the Work Screen

In the left column of the work generation table, you select what site or sites to generate work items for.

In the right column of the work generation table, you set the standards (thresholds) for what assets need work.

After configuring work generation in the table, you activate creation of work items using the Generate Work Items button, below the table.
