Work Items Tab

To generate work items,

  • Select the plus (+) icon under the Work Items menu.
  • The Create Work Item popup menu will appear.
  • Complete the following data fields:
    • Description:
    • Domain:
    • Asset Class:
    • Group:
    • Reference:
    • Year:
    • Category:
    • Activity Type:
    • Classification:
    • Estimated Cost:
  • Once you have filled in the data fields, you can select Create, Create And Navigate to Work Item, or Cancel.
