View Assessment Plans

To view or work with an existing assessment plan,

  1. Navigate to the table of assessment plans. From the menu, this is done by activating the Assessment Plans tile. Alternatively, from within a different assessment plan, you can activate the browser's back arrow button until you reach the assessment plans screen with the table of assessment plans.
  2. In the table of assessment plans, activate the line for the desired assessment plan.

View the Assessment Plan Details Page

After completing the steps above, you will be brought to the Details tab in the Charter stage for that assessment plan. This tab is essentially the assessment plan's data page, showing assessment level data such as the assessment plan name, budget, etc. For explanations of the assessment plan data fields, see Add Assessment Plans.
