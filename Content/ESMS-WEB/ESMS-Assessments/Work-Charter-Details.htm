Details Tab

The Details tab is the default tab that will show when you activate the Charter stage of developing an assessment plan.

Editable Fields

The Details tab displays the following fields that can be edited:

  • Name of the assessment plan
  • Project Charter
  • Budget
  • Planned Assessment Date - To enter or change the assessment plan date, you can either enter the date into the field, or navigate into the field and activate the larger down triangle at the right side of the field to use the calendar tool. If the field contains real contents (rather than the mm/dd/yyyy format indicator), you can clear its contents by clicking the "x" in the field.

At the bottom right of the screen is the Save button, which will save the assessment plan details. Next to the Save button is the Delete button. This will delete the entire assessment plan (after confirmation).

Read-Only Fields

The following fields can be viewed but not edited; they were entered when the assessment plan was created as described in Add Assessment Plans

  • Date Created - Shows the Date the assessment plan was made
  • Site - Shows the Site that the assessment plan pertains to
  • Domain - Shows the Domain that the assessment plan pertains to
  • Scheme - Shows the scheme being used. If you want to know more about what a scheme is, see Scheme in Understanding Assessment Plans
  • Pre-Assessment Template - Shows the operator pre-assessment plan template being used
  • Assessment Plan Template - Shows the assessment plan template being used
  • Assessment Management - Shows the management types and the manager assigned to them
  • Created By - Shows the author of the Assessment Plan

At the bottom right of the screen is the Save button, which will save the assessment plan details. Next to the Save button is the Delete button. This will delete the entire plan assessment (after confirmation).
