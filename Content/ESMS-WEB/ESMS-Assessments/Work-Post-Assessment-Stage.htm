Post-Assessment Plan Stage

This topic describes the Post-Assessment Plan stage of designing an ESMS assessment plan. This is the fourth of five stages, to be performed after assessment plan information and associated work items have been synchronized to ESMS Web at the end of the Desktop Sync / Assessment. Here you can create, search, or delete work items. You can edit work items, for example, to modify the estimated cost.

—Quick Links—

Tasks available at this stage are to

  • manually add work items at the Work Items tab if desired.
  • edit work items at the Work Items tab.
  • select a component and view and/or edit the assessment plan question(s) results associated with that component at the Field Questionnaire tab.
  • review and/or edit the assessment plan questionnaire results at the Field Questionnaire Summary tab.
  • transition to the next stage when done.

Work Items

The Work Items tab displays work items that were generated in ESMS Desktop during the Desktop Sync / Assessment. At this tab you can edit previously created work items, or create new ones using the Add Work Item button.

Filters

Using the filter fields above the columns, you can filter the work items by Work Description, Estimated Cost, Date, Component, RPUID, or Facility, who the work item was Created By, or Delete the work item.

Add Work Item

To add a work item at the Work Items tab,

  1. Activate the Add Work Item button, located above the row of filters (see screenshot above). The "Create/Edit Work Item" popup will appear, and the fields will be unpopulated.
  2. In the popup, fill out the fields. Details about the fields in the "Create Work Item" popup are found in the ESMS Desktop documentation.
  3. When finished adding work item data, activate Create or Create and Navigate To Work Item. Alternatively, activate the Cancel button to cancel your additions.

Edit Work Item

Based on your permission level, you can add, delete, and/or change data in a previously existing work item.

To edit a work item,

  1. Activate the work item's row in the Work Items table. The "Edit Work Item" popup will appear.
  2. If you don't see the desired work item in the table, adjust the filters and repeat the step above.
  3. In the popup, edit fields as desired. Details about the fields in the "Edit Work Item" popup are found in the ESMS Desktop documentation.
  4. If there are already images associated with the work item, they will be displayed near the bottom of the popup. To view an image, click on its thumbnail. You can delete images, or upload more images.
  5. When finished with edits, activate Saveor Save And Navigate To Work Item. Alternatively, activate the Cancel button to exit without saving your changes.

Delete Work Item

To delete a work item,

  1. Navigate to the Work Items table at the Post-Assessment tab.
  2. In the rightmost column, activate the Delete button in the row describing the work item you wish to delete.
  3. Provide confirmation as prompted.

Field Questionnaire

At the Field Questionnaire tab, you can view or edit assessment plan questions associated with a particular component if (1) you have appropriate permissions and (2) you have been assigned to the particular discipline the question is associated with. Field Questionnaire Summary

The Field Questionnaire Summary table will display a list and quick summary of answered questions related to any work item created using the Questionnaire. This is another location where you can edit the assessment plan information. To do so, click on the information in the field and edit it. The change will appear momentarily, without you needing to activate a button.

You can filter what questions are displayed by using the Facility, Component, Type, Question and/or Answer filter fields above the columns.

Transition

When entries are complete and any required approvals have been obtained (this is governed by the scheme; see Transitions in Understanding Assessment Plans), you can proceed to the next and final stage, the Packages.
