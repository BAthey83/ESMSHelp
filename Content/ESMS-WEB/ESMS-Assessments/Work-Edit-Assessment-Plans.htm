Edit Assessment Plan

To edit an existing assessment plan,

  1. Navigate to the table of assessment plans by one of these means: (a) from the main menu, activate the Assessment Plans tile; or (b) from within a different assessment plan, activate the browser's back arrow button until you reach the Assessment Plans screen with the table of assessment plans.
  2. In the table of assessment plans, activate the line for the desired assessment plan. You will be brought to the Charter stage for that assessment plan.

Assessment plan level data such as the assessment plan name, budget, etc. can be edited at the Details tab in the Charter stage. For explanations of the assessment plan data fields, see Add Assessment Plans.

Use the assessment plans button/flyout menu at the upper left to navigate to other tabs and stages. (For instructions, see Assessment Plan Navigation.)
