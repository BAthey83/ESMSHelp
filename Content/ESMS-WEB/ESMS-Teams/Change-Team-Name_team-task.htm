Change Team Name

This task requires that you have permission to administer a team.

To change the name of a team,

  1. At the Teams ("Team Editor") screen, select the team you want to change the name of. The team's Team Details should appear in the content area to the right.
  2. In the Team Name field, do one of the following:
    1. Edit the old team name,
    2. Delete the old team name and replace it with the new name, or
    3. Select the old name (shown below) and overwrite it with the new name.
  3. Activate the Save button at the bottom of the screen.
