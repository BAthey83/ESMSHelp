Creating a New Import Request

Create an import request to import data from a BUILDER environment.

Note:BUILDER environments are configured by the ESMS administrators.

Prerequisites

Before starting to import data from a BUILDER environment:

  • Obtain user credentials for that BUILDER environment
  • Map BUILDER sites to ESMS sites

To Create a New Import Request

  • Click on the Request Import button on the Buildings Import Screen to bring up the New Import Request dialog
  • In the dialog select the environment you wish to work with.
  • Click on theSelect Site button to select a site to import data from. The selected site list includes mapped ESMS sites only.
  • Multiple sites may be selected. Please note that increasing the number of sites increases the amount of data and, potentially, the amount of time it takes to perform the request.
  • Enter user credentials for the selected BUILDER environment and click the Check Credentials button to verify that the credentials are valid.
  • If you wish to be able to edit the request later, click the Create button. If you wish to start importing immediately, click the Create And Submit button. In both scenarios the request will be created and added to the list on the Buildings Import Screen with the status of New for the former and Sent for the latter scenario.
