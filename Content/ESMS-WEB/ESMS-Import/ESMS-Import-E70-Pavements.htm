Understanding the Pavements Import Screen

This screen is the place for importing site data from the E70 (Pavements) screen.

How to Import

To import data from the E70 (Pavements) screen, you will need to create an import request specifying which sites you wish to be imported. This request is queued and processed. You can check on the status of the request to verify if it has been completed. The request status is New when the request is created but not yet submitted. The status becomes Sent when the request is submitted and is being processed on the E70 side. Once the E70 data is received, the request status switches to In Progress and then to Done when processing is complete. If the status does not change from Sent to In Progress for a while, that may be an indication of a problem on the E70 side. To verify, use the Check Status function on the Import Request Details dialog. Error status indicates when there is a problem processing the request on the ESMS side. Contact Support if there is an error either on the E70 side or the ESMS side.

The following data gets imported: Asset Inventory, Assessment Plans, Work Items, and some reference data. The items are added or updated if they already exist.

Pavements Import Screen

From the Import Screen it is possible to

  • Create a new import request
  • View the list of previous import requests, if any, ordered by request date in descending order
  • View import request details and monitor request status
