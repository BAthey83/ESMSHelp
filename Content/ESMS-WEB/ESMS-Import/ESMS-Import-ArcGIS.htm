Understanding the Import from the ArcGIS Screen

This screen is the place for importing site data from the ArcGIS screen.

How to Import

To import data from the ArcGIS screen, you will need to Creating a New Import Request specifying which sites you wish to be imported. This request is queued and processed. You can check on the status of the request to verify if it has been completed. The request status is New when the request is created but not yet submitted. The status becomes Sent when the request is submitted and is being processed on the ArcGIS side. Once the ArcGIS data is received, the request status switches to In Progress and then to Done when processing is complete. If the status does not change from Sent to In Progress for a while, that may be an indication of a problem on the ArcGIS side. To verify, use the Check Status function on the Import Request Details Screen dialog. Error status indicates when there is a problem processing the request on the ESMS side. Contact Support if there is an error either on the ArcGIS side or the ESMS side.

The following data gets imported: Asset Inventory, Assessment Plans, Work Items, and some reference data. The items are added or updated if they already exist.
