Generalized Imports

This topic introduces tasks you can perform at the Generalized Imports screen and provides links to greater detail for each.

Import Configurations

At the Import Configurations menu, you can:

  • Creating a New Import Configuration - Select the Plus icon. Refer to the steps in the next section to learn how to create a new configuration.
  • Run an Import Configuration - Pick the Arrow icon. Select an Import Source in Excel or Access format and run the configuration.
  • Delete an Import Configuration - Click the Trashcan icon to remove the Import Configuration, or select Cancel to the deletion.
  • Import History - Choose the clock Icon to see the date the import was initiated, and its source details.

Creating a New Import Configuration

Edit the following fields when you create a new Import Configuration:

  • Name - The title of your Import Configuration.
  • Abort on Error - Select this option to stop the Import Configuration from being created if there is an error.
  • Source Schema - Import or generate a Source Schema and the corresponding Record Sets in Excel or CSV format.
  • Recordset Mappings - Choose Add to select the following options: Destination Type, Source Key, and Record Locator. You can also choose to Abort on Error or Create Record If No Destination Found, as well as add Field Mappings.

Once you have finished your Import Configuration, select the Save button to keep your updates.

Run an Import Configuration

To Run an Import Configuration, select an Import Source format (in Excel or Access), and choose a file to run. Once selected, click the Run Import button and view the results.

Delete an Import Configuration

To Delete an Import Configuration, select the trashcan icon, followed by the Delete button in the subsequent popup menu. Click Cancel to keep the Import Configuration.

Import History

To review an Import's configuration history, select the clock icon. Here, you can see the date the Import Configuration was initiated, along with the Import's Source Details.
