Mapping BUILDER Sites to ESMS Sites

Mapping BUILDER sites to ESMS sites is a prerequisite for importing BUILDER data into ESMS.

Note: BUILDER environments are configured by the ESMS administrators.

Prerequisites

Before starting to map sites, obtain user credentials for the BUILDER environment you want to import from.

To Map a BUILDER Site to an ESMS Site

  • Open the Map BUILDER Sites to ESMS Sites dialog accessible by clicking the Edit Builder Site Mapping button on the Buildings Import Screen.
  • In the dialog select the environment you wish to work with.
  • Enter credentials and click the Get BUILDER Sites button. This action requests a list of available sites for the selected BUILDER environment and opens the next screen.
  • The next screen presents a paged list of sites already mapped, if any. By clicking on the list entry, you can view the last 10 import requests for that mapping. If the mapping has not been used, you can remove it by clicking on the "x" on the right side of the list entry.
  • To create a new mapping, select the BUILDER site and select an existing ESMS site or create a new one by checking the Create New ESMS Site check box. Edit the site name if desired.
  • Click the Map button.
  • Repeat as needed.
