Associate Document(s) with a Component

Enterprise SMS Web allows documents such as images, Word documents, and PDFs to be associated with individual components. There are two methods for associating a document stored in Enterprise SMS to its related component. It can be done:

  1. at the component details screen for the component, or
  2. by bulk Upload Files at the documents screen.

See Which Method To Use? immediately below for advice which method to use.

Which Method To Use?

If there is only one or a small number of related documents, and if they are available at the time of creating the component, it's usually best to do the association at that time, at the Component Details screen, because the documents will be automatically associated with the component.

However, if there are a large number of documents, it's better to use the bulk Upload Files button at the Documents screen so that multiple documents can be selected at one time using the checkboxes.

Associate Documents at the Component Details Screen

Note: This section contains instructions for uploading a document and associating it with a component at the component's Details screen. The other method for associating a document with a component is done at the documents screen.

In this method you will start at the desired component and search for documents to be associated with it, one at a time.

To associate documents (images, etc.) with a component using this method,

  1. Select the Inventory drop-down menu.
  2. Select the Components button.
  3. As needed, use Filters to focus the relevant component(s) that appear in the results table.
  4. Select the component. You will be brought to the component's Component Details screen.
  5. Activate the Documents accordion button if that section isn't already open.
  6. Activate the Upload Files button.
  7. Navigate to and select a document to be associated with the component. Repeat this step as many times as needed. Associated files will appear under "Name", above the Choose File button.
  8. When finished, you can select the Documents accordion button to close it.

Associate Documents at the Documents screen

Note: This section contains instructions for associating a document with a component by using the bulk Upload Files button at the Documents screen. The other method for associating a document with a component is done at the component's detail screen.

In this method, you will start with one or more documents selected at the Documents screen, then search for components to be associated with the document(s). This method will only work for documents that have already been uploaded.

To associate documents (images, etc.) with a component using this method,

  1. Navigate to the Documents screen.
  2. As needed, use the document search tabs (Name, Type, Tags, Upload Date) to display relevant documents in the results.
  3. Using the checkboxes in the table of documents, select one or more of the documents listed.
  4. If you wish to add new documents, select the bulk Upload Files button to open it.
  5. Choose the file(s) you wish to upload from your internal File Explorer/directory.
  6. Repeat the process as needed until all the files you wish to associate with a component are in the queue (which can be opened with the drop-down menu).

    7. Note: if the documents you wish to associate already appear in the search results, you only need to perform steps 1-3 before proceeding to steps 7 and 8.

  7. After you select a document, its name will appear in the Document Associations and Tags Editor at the right side of the screen. Be sure to select the Associations tab and Component in the Select Association Type drop-down menu. At the right side of each component, there is a minus – symbol. You can activate the minus – symbol to delete your selection from the field; the components search results will refresh accordingly.
  8. When the desired component appears in the field, activate the Select Sites to Associate button under Associate Selected Documents. The component name will appear under Associated Components.
