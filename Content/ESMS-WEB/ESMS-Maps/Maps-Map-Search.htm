Map Search

This topic introduces tasks you can perform at the Map Search screen, providing detailed descriptions for each task.

What Can I Do Here?

  • find and review maps for sites you have selected.
  • filters search results related to the site maps you are viewing.
  • switch back to list view.

Maps Tab

At the Maps Tab, you can:

  • Select a Site to search for its map.
  • Choose what Layer of map you wish to see: Open Street Map, Gray, Satellite, or Streets.
  • Select to accentuate the map's Color by CI.

Filters Tab

At the Filters tab, you can filter map search results by:

  • RPUID
  • Number
  • Name
  • Site
  • Operational Status
  • Category Code
  • Quantity
  • Year Constructed
  • PRV
  • CI
  • FCI
  • EFCI
  • FI
  • PI
  • MDI

List View

Note: You can switch from Map view to the List View by selecting this icon () in the bottom left corner of the screen.
