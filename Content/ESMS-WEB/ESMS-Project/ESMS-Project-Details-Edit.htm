Edit Project Details

For the users with the Update Project privileges the Project Details screen allows to edit some attributes by engaging the Edit Details button.

The following attributes are editable regardless of the project status:

  • Status (for non-workflow-enabled projects only. See updating workflow status for updating the status of a workflow-enabled project)
  • % Complete
  • Pillar (Fuels project only)
  • CA Tracking Number (Fuels project only)

The following attributes are editable only when the project status is Project Development:

  • Title
  • Type
  • Year
  • Number
  • Construction Agent
  • Site(s)
  • Domain
  • Assigned PM
  • Planned Execution Year
  • Is Workflow
  • Scheme (workflow-enabled projects only)
  • Permissions (workflow-enabled project only and if the selected scheme requires permissions)

If there is any change in the Site or Domain attribute, all asset-related data (including funding and WBS items) that were previously defined for the project will be lost
