Exporting the List of Projects

The Export button on the Project List Screen exports the filtered list of projects into a .csv file.

The .csv file is downloaded automatically into the local Downloads folder.

The following project attributes are exported:

  • Title
  • Project Number
  • Assigned PM
  • Installations
  • RPSUIDs
  • Sites
  • Domain
  • Status
  • Workflow-Enabled
  • Construction Agent
  • Construction Agent PM
  • Organizations
  • COCOMs
  • Work/Service Order Number
  • Planned Execution Year
  • % Complete
  • Create Date
  • Created By
  • Last Edited Date
  • Last Edited By
  • Total Cost
  • Total Funding
  • Contract Award Planned Date
  • Contract Award Actual Date
  • Design Start Planned Date
  • Design Start Actual Date
  • Design Finish Planned Date
  • Design Finish Actual Date
  • Construction Start Planned Date
  • Construction Start Actual Date
  • Construction Completed Planned Date
  • Construction Completed Actual Date
  • Commissioning Planned Date
  • Commissioning Actual Date
  • BOD/RTO Planned Date
  • BOD/RTO Actual Date
  • Financial Closeout Complete Date

When the project list is filtered so that only the Fuels domain is included, the following project attributes get exported in addition:

  • Pillar
  • CA Tracking Number
