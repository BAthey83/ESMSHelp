Main Menu Search

 

By designating fields, you control the kind of data that is displayed. By designating filters and selecting a search term, you control the content.

Note: A search from the Main Menu search box will persist through changes to domain or site. Activate "x" next to the search box to cancel the search.

When you search from the Main Menu, you need to check the number displayed by each item in the tab bar (Facilities, Groups, Component, assessment plans, and Work) to see which of them contain results.
