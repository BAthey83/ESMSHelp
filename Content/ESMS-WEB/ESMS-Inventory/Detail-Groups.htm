Navigating the Group Search Screen

When you select a row in the group results table at the Group Search screen, the detail screen for that group will appear.

For example:

The screen layout consists of (1) a listing of group properties, followed by (2) a series of accordion buttons providing access to additional aspects of group management.

Group Functions

The Group Details screen lets you perform the following functions:

Rollup Group: Select this function to roll up your selected group data.

Create New Group: Choose this option to add a new group to the inventory. Details you can create for the new group include the Classification, Quantity, Operational Status, and Construction Type. Click the Create button to create the group, or select Cancel to discard your updates.

Edit Group: Select this button to update or revise existing group data. Items you can edit here include the Group Type, Classification, Status, and Construction Type.

Delete Group: Choose the button to remove a group and its data from Inventory.

Group Properties

At the Group Properties screen, you can see the following properties for the selected group, if the properties have been entered into Enterprise SMS:

  • Group Name
  • Group Type
  • Parent Site
  • Parent Facility
  • Parent Group
  • Status
  • Quantity
  • Units
  • Classification
  • Date Built
  • Construction Type
  • Domain
  • CI (Condition Index)
  • FI  (Functionality Index)
  • FCI  (Facility Condition Index)
  • EFCI (Extended Facility Condition Index)
  • PI (Performance Index)

Accordion Button Options

At the bottom of the screen, you can activate any of the accordion buttons to toggle the section open or closed. An explanation of each button's capabilities follows.

Group Associations

Shows any group associations for the selected group.

Components

Shows what components are in the selected group.

Domain-Specific Properties

Shows the domain-specific properties of the selected group.

Child Groups

Shows any child groups that may be in the selected group. This should include:

  • groups associated with all components in the selected group, and
  • groups associated with all components in all children groups of the selected group.

Assessments

Shows assessments associated with the current group. This should include:

  • assessments associated with all components in the selected group, and
  • assessments associated with all components in all children groups of the selected group.

Work Items

Shows work items associated with the current group. This should include:

  • work items associated with all components in the selected group, and
  • work items associated with all components in all children groups of the selected group.

Cost Modifiers

Shows the cost modifiers associated with the selected group.

Performance Records

Shows the performance records associated with the selected group. This should include:

  • CI (Condition Index)
  • FI (Functionality Index)
  • FCI (Facility Condition Index)
  • EFCI (Extended Facility Condition Index)
  • PI (Performance Index)
