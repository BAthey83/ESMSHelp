Navigating the Group Details Screen

When you select a row in the group results table at the Search screen (single-click suffices), the detail screen for that group will appear. For example:

The screen layout consists of (1) a listing of group properties, followed by (2) a series of accordion buttons providing access to additional aspects of facility management.

Group Properties

At the Group Details screen, you can see the following properties for the selected group, if the properties have been entered into Enterprise SMS Web:

  • Group Name
  • Group Type
  • Parent Site
  • Parent Facility
  • Parent Group
  • Status
  • Quantity
  • Units
  • Classification
  • Date Built
  • Construction Type
  • Domain
  • CI (Condition Index)
  • FI  (Functionality Index)
  • FCI  (Facility Condition Index)
  • EFCI (Extended Facility Condition Index)
  • PI (Performance Index)

Accordion Buttons

At the bottom of the screen you can activate any of the accordion buttons to toggle the section open or closed. An explanation of information available at each button follows.

Group Associations

Shows any group associations for the current group.

Components

Shows what components are in the current group.

Domain-Specific Properties

Shows the domain-specific properties of the current group.

Child Groups

Shows any child groups that may be in the current group. This should include

  • groups associated with all components in the current group, and
  • groups associated with all components in all children groups of the current group.

Assessments

Shows assessments associated with the current group. This should include

  • assessments associated with all components in the current group, and
  • assessments associated with all components in all children groups of the current group.

Work Items

Shows work items associated with the current group. This should include

  • work items associated with all components in the current group, and
  • work items associated with all components in all children groups of the current group.

Cost Modifiers

Shows the cost modifiers associated with the current group.

Performance Records

Shows the performance records associated with the current group. This should include

  • CI (Condition Index)
  • FI (Functionality Index)
  • FCI (Facility Condition Index)
  • EFCI (Extended Facility Condition Index)
  • PI (Performance Index)
