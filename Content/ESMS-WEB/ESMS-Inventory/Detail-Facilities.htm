Facility Details

Note: "Facility" and "Asset" may be used interchangeably throughout the following information.

Navigating the Facility Details Screen

When you select a row in the facility results table at the Search screen, the detail screen for that facility/asset will appear. For example:

The screen layout consists of (1) a listing of facility properties, followed by (2) a series of accordion buttons providing access to additional aspects of facility/asset management.

Facility Functions

The Facility Details screen lets you perform the following functions:

Rollup Facility: Select this function to roll up your selected facility data.

Create New Facility: Choose this option to add a facility/asset to the inventory. Details you can input for the new facility/asset include the Category Code, Quantity, Construction Date, and Operational Status. Click the Create button to add the facility, or select Cancel to discard the facility data.

Edit Facility: Select this option to update or revise facility/asset data. Items you can edit here include the Domain, the Facility Number, the Unit of Measure, the PRV, and the Construction Date.

Delete Facility: Choose this button to remove a facility/asset and its data from Inventory.

Facility Properties

At the Facility Details screen, you can see the following properties for the selected facility/asset if the properties have been entered into Enterprise SMS Web:

  • Installation
  • Site
  • Facility
  • RPUID (Real Property Unique Identifier)
  • RPSUID (Real Property Site Unique Identifier)
  • Domain
  • Category Code
  • Type
  • Operational Status
  • Quantity
  • UoM (Units of Measure)
  • PRV (Plant Replacement Value)
  • Construction Date
  • Date Rebuilt
  • PoC (Point of Contact)
  • MDI (Mission Dependency Index)
  • CI (Condition Index)
  • FI  (Functionality Index)
  • PI (Performance Index)
  • FCI  (Facility Condition Index)
  • EFCI (Extended Facility Condition Index)

Accordion Button Options

At the bottom of the screen, you can activate any of the following accordion buttons to toggle the section open or closed.

Components

This button identifies what components are in the current facility/asset. If there are no components for the facility/asset, you can create a new one by selecting the + button.

The Create Component popup appears. Here, you can input data related to items such as the component type, classification, quantity, and the replacement cost. Once ready, select Create or Create and Navigate to Component. If you wish to discard the data, select Cancel.

 

 

Assessments

Identifies assessments associated with the current facility/asset. You can create a new assessment by selecting the + button.

The Create Assessment popup appears. Here, you can input data related to items such as the site, asset class, object type, and component. Once ready, select Create or Create and Navigate to Assessment. If you wish to discard the data, select Cancel.

 

Work Items

Identifies work items associated with the current facility/asset. You can add new work items by selecting the + button.

The Create Work Items popup will appear. Here, you can input data related to items like the asset class, object type, category and activity type. Once ready, select Create or Create and Navigate to Work Item. If you wish to discard the data, select Cancel.

Additionally, you can edit or delete existing work items. Simply select a work item under the Work Item dropdown to edit.

Once it appears, select Edit Work Item as highlighted above (or Delete Work Itemto remove the work item) and proceed with your changes on the following screen:

After you have completed your edits, select the Save button to retain your changes or Cancel to discard them.

 

Documents

Identifies documents associated with the current facility/asset. If no documents are available, you can add files by selecting the Upload Files button.

Cost Modifiers

Identifies cost modifiers associated with the current facility/asset. You can select a cost modifier book by clicking the "Book" dropdown menu.

Performance Records

Identifies performance records associated with the current facility/asset. Select the button to view data in line graph format.
