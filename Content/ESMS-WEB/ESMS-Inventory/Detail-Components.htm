Component Details

Navigating the Component Details Screen

When you select a row in the results table at the Component screen, the detail screen for that component will appear.

For example:

The screen layout consists of (1) a listing of component properties, followed by (2) a series of accordion buttons providing access to additional aspects of component management.

Component Functions

The Component Details screen lets you perform the following functions:

Create New Component: Select this button to add a new component. Details you can input for the new component include the Domain, Type, Classification, Quantity, Operational Status, and Cost Source. Click the Create button to add the component, or select Cancel to discard the component data.

Edit Component: Select this button to update/revise your component data. Items you can edit here include the Component Name, the Component Type, the Date Installed, the Replacement Cost, and MDI. Click Save to retain your component data, or select Cancel to discard the data.

Delete Component: Select this button to remove your component and its data from Inventory.

Component Properties

At the Component Details screen, you can see the following domain properties for the selected component, if the properties have been entered into Enterprise SMS:

  • Name
  • Site
  • Facility
  • Asset Class
  • Status
  • Type
  • Replacement Value
  • Quantity
  • UoM
  • Year
  • CI
  • Design Life
  • RSL
  • MDI
  • Asset RPUID
  • Facility Number
  • UNIFORMAT Code
  • Classification Type

Accordion Button Options

At the bottom of the screen, you can activate any of the following accordion buttons to toggle the section open or closed.

Custom Attributes

Shows the attributes that are custom-matched to the selected domain.

Component Details

Shows details associated with the selected component.

Select the + button, and the Create Details popup menu appears. Details you can input for the new component include the Capacity, Equipment Type, Serial Number, ID Number, and Equipment Make. Click the Create button to add the component, or select Cancel to discard the component data.

Building Section Properties

Shows the properties associated with the selected component.

Assessments

Shows assessments associated with the selected component.

Click the + button, and the Create Assessment popup menu appears. Details you can input for the assessment include the Site, Asset Class, Object Type, and Assessment Type. Select the checkbox at the bottom of the popup to Enable Samples. Click the Create button to add the assessment, or select Cancel to discard the assessment data.

Work Items

Shows work items associated with the selected component.

Inspection Schedule

Shows the inspection schedule established for the selected component.

Child Components

Shows any child components that are linked to the selected component.

Click the + button, and the Create Child Component popup appears. Details you can input for the child component include the Domain, Type, Classification, Operational Status, Number, and Cost Source. Click the Create button to add the child component, or select Cancel to discard the child component data.

Documents

Shows documents (images, diagrams, etc.) associated with the selected component. Here, you can also upload additional documents to be associated with the component.

Select the Upload Files button to add any documents for the component.

Cost Modifiers

Shows the cost modifiers associated with the selected component. You can select a cost modifier book by clicking the "Book" dropdown menu.

Performance Records

Shows the performance records associated with the selected component. Select the button to view data in line graph format.
