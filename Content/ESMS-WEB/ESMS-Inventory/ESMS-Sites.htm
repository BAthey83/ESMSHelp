Select a Site

Before you select a Site,



  1. You must select an Organization from the Current Organization tab.
  2. Once you have chosen an Organization, you can select a Site from the subsequent screen, as shown above.

Select Columns

To Select Columns on the Site screen,

  1. Click the Select Columns button (the three-dash button to the right of the Filterbutton).
  2. The Select Columns popup screen will appear. Here, you can customize your column fields by selecting or deselecting checkboxes.

The following fields of Site data can be customized on the Select Columns screen:

  • RPSUID: The six-digit Real Property Site Unique Identifier for the selected site.
  • Site Code: The six-digit ID code for the selected site.
  • Name: Identifies the selected site.
  • Installation Name: The Installation ID for the selected site.
  • Installation Code: The five-digit identifier for the selected site.
  • Organization Name: Identifies the Organization where the selected site is located.
  • CONUS: Identifies whether the site is located in the Continental U.S.
  • Country: Identifies the Country where the selected site is located.
  • State: Identifies the State where the selected site is located.
  • ZIP Code: Identifies the Zone Improvement Plan code for the selected site.
  • CI: Identifies the Condition Index rating for the selected site.
  • PI: Identifies the Performance Index rating for the selected site.
  • EFCI: The Extended Facility Condition Index rating for the selected site.
  • Replacement Value: The Plant Replacement Value, or PRV of the selected site.
  • ACF: The Area Cost Factor for the selected site.
  • Climate: Identifies the environmental conditions related to the selected site.
  • Seismic: Identifies the condition of the selected site related to earthquake durability.

Once you have chosen the site fields you wish to include in your Site data, click Update. If you do not want to keep your updates, click Cancel.

Review and Edit Site Information

The Site screen integrates the fields chosen in the Select Columns page. In addition, the screen lets you review the site data as a chart for easy reference. Selecting the Show Graphs button highlighted above the site list to view the site data in a chart format.

Along with the chart data, you can also view site maps by selecting the Show Map button next to the graph button mentioned above. Another feature you can use is to download the site data in a CSV chart. Just click the arrow button to the left of the Show Map button.

In addition, the Site Search screen allows you to filter data to further customize your search. Simply select the Filter button next to any column where you wish to filter data.

Site Details

If you select a Site, you will see the Site Details screen, as shown above. On the General tab, you can see the following information:

  • Code
  • Name
  • RPSUID
  • Value
  • Parent Organization
  • CI
  • FI
  • PI
  • FCI
  • EFCI

Clicking on the Permissions tab will allow users with Administrative rights to add, edit, delete, or reset user permissions.

(Note: If you have any questions about your user permissions, contact your system administrator).

On this tab, you can assign site permissions to users related to Assets and Inventory.

Clicking an up arrow by a site permission block will expand a dropdown menu.

The four permissions you can assign to users on this page are:

  • Curator
  • Assessor
  • Work Planner
  • Assessment Supervisor

You can add a new team to a site by selecting the Add Team button.

To discard a team's site permission, select the Remove Team icon.

Select the Save button to save a team's updated site permission.

You can also revert your changes by clicking the Resetbutton.

In addition, you can rollup the site on this page by selecting the Rollup Site button.

Review the Permissions topic for more information.
