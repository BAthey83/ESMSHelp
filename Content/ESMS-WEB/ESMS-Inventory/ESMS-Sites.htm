Sites Details

This screen shows the Site Details information. It is accessible from the Enterprise SMS main menu.

Select a Site

Before you select a Site,

 

  1. You must select an Organization from the Current Organization tab.
  2. Once you have chosen an Organization, you can select a Site from the subsequent screen.

Select Columns

To Select Columns on the Site screen,

  1. Click the Select Columns button (the three-dash button to the right of the Filterbutton).
  2. The Select Columns popup screen will appear. Here, you can customize your column fields by selecting or deselecting checkboxes.

The following fields of Site data can be customized on the Select Columns screen:

  • RPSUID: The six-digit Real Property Site Unique Identifier for the selected site.
  • Site Code: The six-digit ID code for the selected site.
  • Name: Identifies the selected site.
  • Installation Name: The Installation ID for the selected site.
  • Installation Code: The five-digit identifier for the selected site.
  • Organization Name: Identifies the Organization where the selected site is located.
  • CONUS: Identifies whether the site is located in the Continental U.S.
  • Country: Identifies the Country where the selected site is located.
  • State: Identifies the State where the selected site is located.
  • ZIP Code: Identifies the Zone Improvement Plan code for the selected site.
  • CI: Identifies the Condition Index rating for the selected site.
  • PI: Identifies the Performance Index rating for the selected site.
  • EFCI: The Extended Facility Condition Index rating for the selected site.
  • Replacement Value: The Plant Replacement Value, or PRV of the selected site.
  • ACF: The Area Cost Factor for the selected site.
  • Climate: Identifies the environmental conditions related to the selected site.
  • Seismic: Identifies the condition of the selected site related to earthquake durability.

Once you have chosen the information fields you wish to include in your Site data, click Update. If you do not want to retain your updates, click Cancel.

Review and Edit Site Information

The Site screen integrates the fields chosen in the Select Columns page. In addition, the screen lets you review the site data as a chart for easy reference. Selecting the graph button highlighted above the site list to view the site data in a chart format.

Along with the chart data, you can also view site maps by selecting the Flag button next to the graph button mentioned above. Another feature you can use is to download the site data as an Excel spreadsheet. Just click the arrow button to the left of the Flag button.

In addition, the Site Search screen allows you to filter data to further customize your search. Simply select the Filter button next to any column where you wish to filter data.
