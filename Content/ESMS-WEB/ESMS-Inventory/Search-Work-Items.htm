Search

To perform a search at the Work tab,

  1. If not already selected, select one or more domains.
  2. If desired and if not already selected, select a single site to search in.
  3. Activate the Work Items tab.
  4. Configure what data fields you will want to see in the results. See Set Up Fields and Filters for instructions.
  5. To narrow the scope of the search by properties such as status, priority, or work activity type, follow instructions for setting up filters in Set Up Fields and Filters.
  6. Finally, enter a search term in the search field on the main menu bar and press Enter.
