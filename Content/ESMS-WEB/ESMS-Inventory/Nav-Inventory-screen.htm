Navigating the Inventory Screen

This topic provides a navigation overview of the information available on the Enterprise SMS Inventory screen.

—Quick Links—

  • Components of the Inventory Screen
  • Related Topics: Detail Screens

After you have selected one or more domains and, if applicable, a site, the initial display at the Inventory screen displays:

  • the number of Organizations, Sites, Facilities/Assets, Groups, or Components that are in the active site(s)
  • what Site(s) and Domain(s) are selected (on the main menu)
  • a data table of assets, at the Facilities level

Once you understand the components of the Inventory screen and how they work, you will be able to:

  • control the kind of data that is displayed by designating which fields are active for each asset level
  • narrow searches using filters
  • navigate to pages that show detailed information about each Inventory level

Components of the Inventory Screen

The major components of the Inventory screen, as numbered in the illustration below, are:

  1. main menu
  2. content area, which will display the results of the most recent search
  3. download as CSV, which lets you see the results in spreadsheet form
  4. show map, which displays map results
  5. show graphs, which displays graph results
  6. select columns, which lets you choose which field columns appear in the content area

Main Menu

At the main menu, you can

  • return to the home page, using the Enterprise SMS logo button (the sites and domains will be cleared).
  • select organization(s).
  • select domain(s).
  • select the following options in the user preferences menu: (a) announcements; (b) user preferences; (c) save your current workspace as the default; (d) clear application cache; (e) see the application and software versions; (f) contact SMS support; or (g) logout.

Content Area

The content area of the Inventory screen shows one of the following:

  • Depending on which button is selected on the taskbar, the content area shows a table listing the organizations, sites, facilities/assets, groups, or components that are (1) associated with either your selected site or all of your permitted sites (depending on whether a site has been selected) and (2) within your selected domain(s). The column headings in the table are determined by the fields selected via the Select Columns button.
  • When you perform a search, a results table showing the search results is displayed in the content area. The column headings for the results table are shown directly below the task selection bar. The column headings which appear are determined by the fields selected via the Select Columns button.

Download as CSV

  • Select the Download as CSV button to download Inventory data as a spreadsheet.

Show Map

  • Click the Show Map button to display the map related to your selected Inventory data.

Show Graphs

  • Click Show Graphs to display charts featuring your chosen Inventory data.

Select Columns

To control which data columns will be displayed in the content area,

  1. Activate the Select Columns button.
  2. Select and/or deselect checkboxes.
  3. Activate the Update button.

Note: The options shown in the Select Columns window will depend on what Inventory button is selected on the main menu (Facilities, Groups, Components, Assessment Plans, or Work items).

Filters Selection

Filter selection can be accessed using the Filter button(s) next to each column.

To apply filters to your search,

  1. Activate the Filter button directly to the right of each column.
  2. In the popup menu, fill in the desired filter information.
  3. Activate the check button to hide the Filter box.

Note: The options shown in the Filter selection box will depend on what Inventory button is selected on the main menu: (Facilities, Groups, Components, Assessment Plans, or Work Items).
