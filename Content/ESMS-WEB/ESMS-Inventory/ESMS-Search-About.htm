Understanding the Search Screen

This topic introduces tasks you can perform at the Search screen and provides links to greater detail for each.

What Can I Do Here?

The Search screen is where you search for facilities, groups, components, assessments, or work items, applying filters if desired and narrowing your search to applicable sites and domains. You can also configure the resulting data by selecting the information fields of interest to you.

Note: Before you begin a search, you must select at least one domain from the main menu bar dropdown. Selecting a site is optional (all sites permitted to you are active if you don't select a site).

Once the results are delivered to the screen, you can access the detail pages for the resulting facilities, groups, components, assessments, or work items.

Alternatively, you can just begin at the starting page (which shows facilities by default) and navigate/drill down to the desired information.

Search Screen Tasks

Tasks you can perform at the Search screen are:

  • search facilities
  • search groups
  • search components
  • search assessments
  • search work items

Search Screen Information

A great deal of information is available via the Search screen once you have results and display the corresponding detail pages, :

Facility Details

At the Facility Details page you can see

  • a list of components in the facility
  • a list of assessments for all components in the facility
  • a list of work items for all components in the facility

Group Details

At the Group Details page you can see

  • what components are in the group, if any
  • what child groups are in the group, if any
  • group associations
  • a list of assessments for all components contained in the group or its child groups
  • a list of work items for all components contained in the group or its child groups

Component Details

At the Component Details page you can see

  • component attributes
  • a list of assessments performed on the component
  • a list of work items associated with the component
  • the component's inspection schedule
  • associated documents, such as images

As mentioned at Search Screen Tasks, you can also upload documents, which will be automatically associated with that component.

Assessment Details

At the Assessment Details page you can see

  • assessment attributes
  • assessment score
  • identification of the component corresponding to the assessment, plus the facility that the component is in.

Work Item Details

At the Work Item Details page you can see the work item's attributes.
