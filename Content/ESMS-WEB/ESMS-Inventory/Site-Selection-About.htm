About Selecting a Site

The Select Site button on the menu bar allows you to select one site to view and work with. If you don't select a site, you will be accessing all of the sites you have permission for (which is also an option). Your choice is between one site and all of the sites. When all of your permitted sites are part of your active view, the data on the screen, search results, etc., take all of these sites into account.

Note: If the menu bar button reads "Select Site" instead of listing a site name, then you're viewing all sites.
