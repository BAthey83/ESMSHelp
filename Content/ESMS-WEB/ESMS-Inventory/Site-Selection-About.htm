About Selecting a Site

The Select Site button on the menu bar allows you to select one site to view and work with. If you don't select a site, you will be accessing all of the sites you have permission for (which is also an option). Your choice is between one site and all of the sites. When all of your permitted sites are part of your active view, the data on the screen, search results, etc., take all of these sites into account.

Note: If the menu bar button reads "Select Site" instead of listing a site name, then you're viewing all sites.

Select Site

To select the site to work with,

  1. On the main menu bar, activate the site button, which either (a) lists the currently selected site or (b) displays Select Site.

  2. The "Select Site" popup window will appear, listing available sites.

  3. (Optional) If you wish to trim down the list of selections via search, follow these steps:
    1. Enter a search string in the "Site Name" field.
    2. Activate the Go button to see only those site names containing the search string.

      Tip: To see the full list of sites again after using the search field, clear the search field, then activate the Go button.

  4. Click on one site in the site list.
  5. Activate the Select button at the bottom of the popup. The popup should disappear. If it does not, and the Select button is grayed out, a valid site has not been properly selected.
  6. Alternatively, activate the Close button to cancel site selection and exit the popup.

Change Site

If you want to change which site you are working with, do the following

  1. To the right of the site name on the Main Menu, activate the button with the white "X", as shown below. This will clear your site selection.

  2. If you want to be viewing all permitted sites, ignore remaining instructions in this list.
