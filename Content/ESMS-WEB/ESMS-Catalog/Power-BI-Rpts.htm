PowerBI Reports

This topic introduces tasks you can perform at the PowerBI Reports screen and provides links to greater detail for each.

To access the PowerBI Reports in Enterprise SMS, you can select the Catalog dropdown menu, followed by the PowerBI Reports button.

At the main screen, you can open an existing PowerBI Report, create a new PowerBI Report by selecting the plus button, or delete an existing PowerBI Report by selecting the X button.

Creating a PowerBI Report

The following fields can be edited on the Create PowerBI Report popup screen:

  • Organization: Select any of the Organizations you can access, such as Air Force Active or Army Active.
  • Display Name: Complete the Display Name field with the title of your PowerBI Report.
  • Type: Choose any of the following report types: General, Work Comparison, or Project.
  • Report ID: Add, select, or edit the Report ID information here.
  • Group ID: Add, select, or edit the Group ID information here.
  • Dataset ID: Add, select, or edit the Dataset ID information here.
  • Filters: Add filters for the Table Name, Column Name, or Default Value here.

Once you have added this information, be sure to click the Save button at the bottom of the page.
