Import/Export Configurations

This topic introduces tasks you can perform at the Import/Export Configurations screen and provides links to greater detail for each.

Import Catalog Manager

At the Import Catalog Manager, you can add, edit, or delete Endpoints, Mapping sets, and Export Configurations.

Endpoints Tab

At the Endpoints tab, there are multiple fields you can edit, including the Name and Base URL. In addition, you can select Manually Add Token to create a new access token for your Endpoint.

Mappings Tab

The Mappings tab features two tabs you can edit and update: General and Component Specific.

General Sub-Tab

The General Sub-Tab shows displays the Mapping Set's Name and lets you select the Fields Source.

When you add a new Field Configuration, you can add information to the following fields:

  • ESMS Table/Field
  • Data Type

    • In addition, data can be customized for the following fields:

  • Import From
  • Default Value
  • Map

You can also delete any Field Configuration as needed.

You can also add and edit Group Type Configurations. Select the plus sign at the bottom of the page to select a new Group Type to add.

Component Specific Sub-Tab

The Component Specific Sub-Tab lets you add and edit Component Type Mapping. Fields you can edit include:

  • Component Type
  • GIS Feature Name
  • Mapped Fields

Export Configurations Tab

At the Export Configurations tab, you can add, edit, and delete export configurations for your catalog entries. In addition, field configurations can be incorporated into your export configurations.

To add a new export configuration, select the plus sign. To edit an existing export configuration, select one of the existing configurations that appear in the subsequent list. To delete an export configuration, click the X button.

To add a new field configuration, select the plus sign that appears above the ESMS Table/Field column. On this screen, you can also edit existing field configurations. Tables and fields can be displayed in ascending or descending order, or select None in the Order By column to keep tables/fields in the order they appear.

In addition, you can create an Export Order for your tables and fields by entering your custom order in the text boxes shown above. To delete any field configuration from the list, select the X button at the far right of each field configuration.
