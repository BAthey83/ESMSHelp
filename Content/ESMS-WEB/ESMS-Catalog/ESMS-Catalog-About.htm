Understanding the Catalog Screen

Note: The Catalog screen is under development and is not currently linked to the Enterprise SMS Web dashboard.

This topic introduces tasks you can perform at the Catalog screen and provides links to greater detail for each.

What Can I Do Here?

At the Catalog screen, you can add custom component and group types to the Enterprise SMS catalog. You also have the option to design an assessment template for each new type.

Catalog Tasks

Custom Component Type

Tasks involved in creating a custom component type are to

  • Add the new component type to the catalog
  • Assign subcomponents to it as needed
  • List the relevant properties for the new component type.

Custom Group Type

  • Add the new group type to the catalog
  • Assign subcomponents to it as needed
  • List the relevant properties for the new group type.

Assessment Template

  • Design an assessment template for any of the custom components as desired, using the assessment template editor

Navigating the Catalog Screen

At the left side of the Catalog screen is where you add a new component or group type. A popup will gather information such as the name of the new type.

At the right side of the Catalog screen you can select the new type by name and flesh out the design by assigning properties and subcomponents. Separate tabs are provided for subcomponent assignment and for listing properties.

Also at the right side of the Catalog screen is where you can design an assessment template for the new type, if desired, by selecting the Assessment Template Editor tab.
