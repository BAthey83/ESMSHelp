What Can Enterprise SMS Web Do?

Using the Web version of Enterprise SMS, you can

  • import real property data
  • explore the data and search for specific data items
  • store and retrieve documents
  • generate work items based on criteria
  • run reports
  • administer permissions for individuals and teams (requires Team Leader or Administrator permission level)

Enterprise SMS Administrators can also

  • create component types and assign user-defined fields for the new types
  • create group types
  • create templates for customized assessment plans
  • create new user accounts

Enterprise SMS Web is not intended for

  • real property inventory entry
  • performing assessment plans

Do I Need Enterprise SMS Web, the Enterprise SMS Field Application, or Both?

Note: The Enterprise SMS field application offers two modes, Desktop and Tablet. Tablet functionality is restricted to performing assessment plans and navigating asset information.

Work Flow

The typical work flow interaction between Enterprise SMS Web and the Enterprise SMS field application typically follows these steps:

  1. Inventory and assessment plans are entered into Enterprise SMS Desktop. Assessment plans may alternatively be entered in the field using Enterprise SMS Tablet.
  2. Synchronization is performed between Enterprise SMS Desktop and Enterprise SMS Web.
  3. Work items are generated in Enterprise SMS Web, in the Work task area, by specifying threshold levels for condition index and remaining service life.

Topical Overview of Web and Desktop Capabilities

  • Real Property Inventory
    1. For searching and viewing real property inventory, Enterprise SMS Web will suffice if all the inventory you need is kept up-to-date on your Web instance by other parties.
    2. However, in order to add or change inventory entries, you definitely need to use Enterprise SMS Desktop in addition to Enterprise SMS Web.
  • Associating Documents with Components
    1. Documents such as images can be imported into Enterprise SMS and "attached" to the relevant components in inventory only in Enterprise SMS Web.
    2. Enterprise SMS Desktop also has functionality for storing documents in association with components. These are synchronized to and from Enterprise SMS Web via the server (see Integration Between Enterprise SMS Web, Desktop, and Tablet).
  • Associating Components with Groups
    1. Enterprise SMS Web has no tools for associating components with groups. This is done in Enterprise SMS Desktop.
    2. In Enterprise SMS Desktop, a component can be associated with one or more groups in the "Add Component" popup when creating a new component.
  • Assessment Plans
    1. Assessment plans can be viewed in Enterprise SMS Web.
    2. To create assessment plans, you need to use the Enterprise SMS field application. Assessment plans can be entered into Enterprise SMS Desktop, or performed in the field using Enterprise SMS Tablet.
  • Work Items
    1. Work items can be generated in Enterprise SMS Web, in the Work task area, by specifying threshold levels for condition index and remaining service life.
    2. To enter a work item manually, you need to use Enterprise SMS Desktop.
