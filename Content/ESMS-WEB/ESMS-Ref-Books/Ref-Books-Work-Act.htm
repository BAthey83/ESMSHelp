Work Actions

This topic introduces tasks you can perform at the Work Actions screen, providing detailed descriptions for each task.

Note: Only users with Administrative access will be able to view or make changes to Work Actions. If you have any questions about your access privileges, contact your system administrator.

Access to Work Actions

To access the Work Actions screen,

  1. Open the Reference Books drop-down menu.
  2. Select the Work Actions button, as highlighted above.

Features and Search Bar

The following details are available as individual columns on the Features and Search Bar:

  • Name: The process(es) being executed, and the work action item listed under Specification.
  • Specification: The item selected for editing within the work action.
  • Object Type: Describes the work action item.
  • Domain: The domain applied to the work action, such as Utility-Gas, Waterfront, or Facility-Buildings.
  • Description: Identifies the work action. This is generated from the Specification.
  • Level: The condition for the specified item.
  • Parent: The parent organization for the specified item(s).
  • Cost: The starting cost for the specified item(s).
  • Cost Year: The year at which the item cost is set.
  • UoM (Each): The Unit of Measurement for the specified item(s).
  • Minimum Cost: The base-level value of the work action.
  • Materials: The value of materials used to perform the work action.
  • Labor: The value of labor used to execute the work action.
  • Equipment: The value of equipment used for the work action.
  • Bare Total: The estimated total value of the work action.
  • Source: The cost book used as the foundation for the work action.
  • Frequency: Describes how often the work action is updated.
  • Crew: The number of people who are using the work action.
  • Daily Output: The work action's output value per day.
  • Total Output: The work action's overall value for the fiscal year.
  • Clear: Choose this icon to reset your specified item search.
  • Filter: Select this icon to focus the specified item(s) appearing in your search.

Note: Any updates you make to the above options will not be added to the Specification unless you select the Save Changes button (the floppy disk icon shown above).

You can focus the search results by clicking on the Filter () icon next to each column. In addition, you can view results in ascending or descending order by selecting the minus symbol next to each column. Click it once to switch to ascending order (the up arrow icon), twice to switch to descending order (the down arrow icon), or three times to revert back to the original order (the minus icon).

Selecting a Work Activity

The following options can be selected from the Work Activity drop-down menu:

  • Paint: Identifies whether the work item requires additional painting.
  • Corrective Maintenance: Identifies whether the work item requires common, everyday repairs.
  • Stop Gap: Identifies if the Work Activity is a temporary repair or adjustment.
  • Global Repair: Identifies whether all instances of the work item needs fixing.
  • Inspection - Direct Rating: Identifies whether a Direct Rating was assigned during the inspection stage.
  • Maintain: Identifies whether the work item is standard maintenance.
  • Remove: Identifies whether the work item is going to be withdrawn.
  • Inspect: Identifies if the work item has been officially inspected.
  • Replace: Identifies whether the work item needs to be changed out.
  • Repair: Identifies whether the work item requires fixing.
  • Demolish: Identifies whether a work item will be torn down.
  • Inspect: Identifies a work item that requires an inspection.
  • Alter: Identifies a work item that requires modifications.
  • Local Repair: Identifies a single work item which requires fixing.
  • Modernize: Identifies a work item that needs to be updated to current standards.
  • Inspection - Distress Survey: Identifies whether a Distress Survey was executed during the inspection stage.
  • None: Identifies that no Work Activities are required for the work item.
  • Add/Install: Identifies if a work item needs to be set up at a site.
  • Inspection - Distress Survey with Quantity: Identifies whether a Distress Survey with Quantity was assigned during the inspection stage.

Editing Work Actions

The following options are found within the Edit Work Action screen:

Note: These Edit Work Action options are also displayed on the Features and Search Bar.

  • Name: The process(es) being executed, and the work action item listed under Specification.
  • Work Activity Type: The work action process selected from the Work Activity drop-down menu .
  • Managed Object Type: Describes the work action item.
  • Specification: The item selected for editing within the work action.
  • Description: Identifies the work action. This is generated from the Specification.
  • Unit Cost: The starting cost for the specified item(s).
  • Unit of Measure: The Unit of Measurement for the specified item(s).
  • Cost Year: The year when the item cost is originally set.
  • Material Cost: The value of materials used for the work action.
  • Labor Cost: The value of labor used to execute the work action.
  • Equipment Cost: The value of the equipment used for the work action.
  • Bare Total Cost: The estimated total value of the work action.
  • Cost Source: The cost book used as the foundation for the work action.
  • Frequency: Identifies how often the work action is performed.
  • Crew: The number of workers performing the work action.
  • Daily Output: The work action's output value per day.
  • Total Work Output: The work action's overall value for the fiscal year.

The options that cannot be edited under Edit Work Actions are: Name, Work Activity Type, Managed Object Type, and Specification.

Note: Any changes you make to the Edit Work Action screen will not be saved unless you activate the Update button. Select the Save icon (the floppy disk icon highlighted above) before choosing another work action from the Work Activity dropdown menu.
