Access to Work Actions

To access the Work Actions screen,

  1. Open the Reference Books drop-down menu.
  2. Select the Work Actions button, as highlighted above.

Next, you can select a Work Activity.

Selecting a Work Activity

To select a Work Activity, click the Work Activity drop-down menu. Here, the following options can be selected:

  • Paint: Identifies whether the work item requires additional painting.
  • Corrective Maintenance: Identifies whether the work item requires common, everyday repairs.
  • Stop Gap: Identifies if the Work Activity is a temporary repair or adjustment.
  • Global Repair: Identifies whether all instances of the work item need fixing.
  • Inspection - Direct Rating: Identifies if a Direct Rating was assigned during the inspection stage.
  • Maintain: Identifies if the work item is standard maintenance.
  • Remove: Identifies whether the work item is going to be withdrawn.
  • Inspect: Identifies if the work item has been officially inspected.
  • Replace: Identifies if the work item needs to be changed out.
  • Repair: Identifies whether the work item requires fixing.
  • Demolish: Identifies if a work item will be torn down.
  • Inspect: Identifies a work item that requires an inspection.
  • Alter: Identifies a work item that requires modifications.
  • Local Repair: Identifies a single work item which requires fixing.
  • Modernize: Identifies a work item that needs to be updated to current standards.
  • Inspection - Distress Survey: Identifies whether a Distress Survey was executed during the inspection stage.
  • None: Identifies that no Work Activities are required for the work item.
  • Add/Install: Identifies if a work item needs to be set up at a site.
  • Inspection - Distress Survey with Quantity: Identifies whether a Distress Survey with a Quantity was assigned during the inspection stage.

After you select a Work Activity, you can proceed to the Features and Search Bar.

Features and Search Bar

The Features and Search Bar is where you can review and edit data for existing Work Action items.

  • Save Changes: Select the floppy disk icon on the left to retain your edits to an existing Work Action.
  • Filter: Select the center icon to add a filter to your Work Action search results.
  • Select Columns: Choose the triple-dash icon on the right to select which data columns to view in your Work Action.

Fields on the Features and Search Bar can be customized as needed. To add or remove fields from the Features and Search Bar, click the Select Columns button, as indicated by the triple-dash icon highlighted above. The Select Columns to View popup will appear in a new window.

At the Select Columns to View popup, you can choose as few or as many fields as you need for your Work Action. Once you have made your selections, click the Update button to save them, or click Cancel to discard them.

The following fields can be selected on the Features and Search Bar:

  • Name: Identifies the item obtained from concatenating the Work Activity and the Specification.
  • Specification: The item selected for editing within the work action.
  • Object Type: A generalized item classification compared to the Specification. For example, a Specification may be "Pipe, Copper, 8-12 Inch." The Object Type will simply be "Pipe."
  • Domain: The domain applied to the work action, such as Utility-Gas, Waterfront, or Facility-Buildings.
  • Description: Text entered by the user to aid in identifying the item.
  • Level: The condition for the specified item.
  • Parent: A key sub-classification of the Object Type.
  • Cost: The starting cost for the specified item(s).
  • Cost Year: The year at which the item cost is set.
  • UoM: The Unit of Measurement for the specified item(s).

    • (Note: the available UoMs depend upon the selected Work Action and user requirements).

  • Minimum Cost: The base-level value of the work action.
  • Materials: The value of materials used to perform the work action.
  • Labor: The value of labor used to execute the work action.
  • Equipment: The value of equipment used for the work action.
  • Bare Total: The estimated total value of the work action.
  • Source: The basis of the cost information, which is user-entered.
  • Frequency: Describes the rate at which work needs to be performed.
  • Crew: The number of people who are using the work action.
  • Daily Output: The work action's output value per day.
  • Total Output: The work action's overall value for the fiscal year.

You can select Filters by each field column () to narrow or expand your search. Results can be organized in ascending or descending order by selecting the minus icon by each field column. Click the minus icon once to get the Up caret, which outputs results in ascending order, click it twice to get the Down caret, which outputs results in descending order, or click it three times to get the minus icon again, which will revert the Work Action items to their original order.

If a Work Action has been overridden or updated, you will see the Remove button (the trashcan icon) at the far right of the item row. Select this button to delete a Work Action.

Editing Work Actions

The following Edit Work Actions fields cannot be edited: Name, Work Activity Type, Managed Object Type, and Specification.

Note: Any changes you make to the Edit Work Action screen will not be saved unless you click the Update button.

Note: You must Save your changes (via the floppy disk icon highlighted above) or your work will be lost if you select another Work Activity from the dropdown or navigate away from this page.

Access to Global Work Actions

Note: Only select users with Administrative access will be able to make changes to Global Work Actions. If you have any questions about your access privileges, contact your system administrator.

Features and Search Bar

  • Name: The process(es) being executed, and the work action item listed under Specification.
  • Specification: The item selected for editing within the work action.
  • Object Type: Describes the work action item.
  • Domain: The domain applied to the work action, such as Utility-Gas, Waterfront, or Facility-Buildings.
  • Description: Identifies the work action.
  • Level: The condition for the specified item.
  • Parent: The parent organization for the specified item(s).
  • Cost: The starting cost for the specified item(s).
  • Cost Year: The year at which the item cost is set.
  • UoM Unit of Measure: The Unit of Measurement for the specified item(s).
  • Min. Cost: The base-level value of the work action.
  • Materials: The material cost within the work action.
  • Labor: The labor cost for the work action.
  • Equipment: The equipment cost for the work action.
  • Bare Total: The estimated total value of the work action.
  • Source: The cost book used as the foundation for the work action.
  • Frequency: Describes how often the work action is updated.
  • Crew: The number of people who are using the work action.
  • Daily Output: The work action's output value per day.
  • Total Output: The work action's overall value for the fiscal year.
  • Clear: Choose this icon to reset your specified item search.
  • Filter: Select this icon to scale down the specified item(s) in your search.

Note: any updates you make to the above options will not be added to the Specification unless you select the Save Changes button.
