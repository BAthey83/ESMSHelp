Add a Cost Modifier Library

This topic details how to add a Cost Modifier library. It also details the tasks that can be performed here.

A cost modifier library can be created for any Organization or Site in Enterprise SMS. Note: this process requires elevated permissions.

To add a cost modifier library,

  1. Choose an Organization from the Current Organization button.
  2. Select the Reference Books dropdown, followed by the Cost Modifiers button.
  3. Choose the Add Book icon that appears above the Search CM Books window.
  4. The Create Cost Modifier Book popup window will appear. Here, you will enter the Cost Modifier Book Name and Source information. In addition, you can also Copy an Existing Book if you want to use existing cost modifier data.

    5. Note: the Organization information is automatically populated by ESMS.

  5. Click the Update button to add your Cost Modifier library.

The Cost Modifiers window will display, with the name of the library at the top. You can now edit the library's name, create new cost modifiers to include in the library, or edit existing modifiers.

The columns of the cost modifier library are described in View a Cost Modifier Library.

To add, edit, or delete a cost modifier record, see Add, Edit, or Delete a Cost Modifier.
