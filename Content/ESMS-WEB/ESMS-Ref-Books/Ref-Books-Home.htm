What Can I Do Here?

Reference Books allow Enterprise SMS users to calculate current and future replacement values, estimate a building's Remaining Service Life (RSL), and apply higher or lower importance rankings to selected Component-Sections.

Component Importance Index

The Component Importance Index (CII) reference book is used to specify the importance level for individual Component-Sections.

Service Life

The Service Life reference book specifies the expected service life for Component-Sections. This data, as well as data from inspections, is used to calculate the Remaining Service Life and produce deterioration curves.

Inflation

The Inflation reference book specifies yearly inflation rates into the future. Enterprise SMS uses these rates to calculate future replacement costs.

Cost

The Cost reference book specifies the replacement costs for Component-Sections and Buildings.

Cost Modifiers

The Cost Modifiers reference book specifies the multipliers or adders that can be applied at multiple Enterprise SMS levels in order to adjust Section replacement costs. Modified costs will roll up through the different levels, and will also affect CI roll-up weights, work plan item costs, and costs output by Scenario simulations.

Work Actions

The Work Actions reference book specifies the costs required to carry a work item through to completion. Work Action costs are calculated using the Component Importance Index, the work item's service life, the Inflation reference book, the Component-Section cost reference book and cost modifiers.
