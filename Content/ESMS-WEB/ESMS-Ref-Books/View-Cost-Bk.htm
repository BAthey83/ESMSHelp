View a Cost Book

This topic covers how to view the default cost book or a custom cost book. The topic Edit Cost Book Data explains how to add or edit data in a cost book.

Permissions Note: Depending on the operation (e.g. adding vs. deleting a book), configuring cost book data requires either Work Planner or Master Planner permission for the Organization applied to the cost book. The ability to assign a custom cost book to an Organization depends on your permissions for that Organization.

To view a cost book,

  1. Select an Organization to assign to the Cost Book in the Current Organization menu.
  2. Open the Reference Books drop-down menu. Select the Cost button.

  3. Open a Cost Book from the Search Cost Books window.
  4. Your Cost Book data will appear. For details on how to edit your Cost Book, see Edit Cost Book Data. If you need to add a Cost Book, see Add a Custom Cost Book.

Cost Book Menu Buttons

The following buttons are available on the Cost Book page:

  • Add Book: Select the Plus icon on the left to add a new Cost Book.
  • Delete Book: Select the Trashcan icon in the center to delete a Cost Book.
  • Edit Book: Select the Pencil and Square icon on the right to Edit a Cost Book.
  • Save Changes: Select the floppy disk icon on the left to keep any additional edits to an existing Cost Book.
  • Filter: Select the center icon to filter your Cost Book search results.
  • Select Columns: Choose the three-bar icon on the right to select which columns of data you wish to view in your Cost Book.

Cost Book Fields

The following columns are available under the Select Columns window. Many of these options can be edited within Cost Records for individual items.

  • Work Action: The process(es) being executed, and the Cost Book item listed under Specification.
  • Specification: The description for the Cost Book item being inventoried.
  • Activity: The action that the Cost Book item requires: Maintain, Inspect, Replace, Local Repair,or Global Repair.
  • Object Type: Describes the Cost Book item.
  • Domain: The Domain applied to the Cost Book/Cost Book item, such as Waterfront, Utility-Electrical, or Facility-Buildings.
  • Description: Identifies the Cost Record.
  • Level: Identifies where the Cost Book is ranked in the hierarchy.
  • Parent: The Cost Book's assigned parent Organization.
  • Original Cost: The initial value of the specified Cost Book item.
  • Cost Year: Indicates the inflation value of the specified Cost Book item.
  • UoM: Unit of Measure The Unit of Measure used by the Cost Book item.
  • Minimum Cost: The base-level value of the Cost Record.
  • Materials: The Material cost within the Cost Record.
  • Labor: The Labor cost within the Cost Record.
  • Equipment: The Equipment cost within the Cost Record.
  • Bare Total: The estimated total cost value of the Cost Record.
  • Source: The Cost Book used as the foundation for the Cost Book item.
  • Frequency: Describes how often the Cost Book item is updated.
  • Crew: The number of people who are using the Cost Book item.
  • Daily Output: The Cost Book output value per day.
  • Total Output: The overall Cost Book's output value for the fiscal year.

Additional Cost Book Functions

You can select Filters by each field column to narrow or expand your search. Results can be organized in ascending or descending order by selecting the minus icon by each field column. Choose the Up caret to see results in ascending order, the Down caret to see results in descending order, or revert back to the minus icon to keep the Cost Book items in their original order.

If an Cost Book item's data has been overridden, you will see a Remove button (the trashcan icon at the far right of the Cost Book item row). Select it to delete an item from your Cost Book.

Cost Records

The Cost Record screen displayed above appears when you select a Cost Book item. The Work Action, Work Activity Type, Managed Object Type, and Specification fields cannot be edited from this screen. However, the following fields (many of which correspond to the Cost Book fields listed above) can be configured and updated within an item's Cost Record screen:

  • Description
  • Unit Cost
  • Cost Year
  • Minimum Cost
  • Material Cost
  • Labor Cost
  • Equipment Cost
  • Bare Total Cost
  • Cost Source
  • Frequency
  • Crew
  • Daily Output
  • Total Work Output

Once you have added the Cost Record data you wish to add, select the Update button. If you do not wish to save the Cost Record data, select Close.
