Delete a Cost Modifier Library

This topic details how to delete a Cost Modifier library. It also details the tasks that can be performed here.

Note: this process requires Administrator permissions.

To delete a Cost Modifier library,

  1. Select the Reference Books dropdown menu.
  2. Click the Cost Modifiers button.
  3. Choose the Cost Modifier library you wish to delete from the Search window.
  4. Click the trashcan icon highlighted above to delete the selected Cost Modifier library.
  5. In the confirmation popup window, activate the Confirm button to complete the deletion (alternatively, select Close to prevent the deletion).
