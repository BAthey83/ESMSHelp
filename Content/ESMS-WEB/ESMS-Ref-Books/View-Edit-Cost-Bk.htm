Viewing and Editing A Cost Book

This topic covers how to view and edit a custom cost book.

To view a cost book,

  1. Select an Organization to assign to the Cost Book in the Current Organization button.
  2. Open the Reference Books drop-down menu. Select the Cost button.

  3. Open a Cost Book from the Search Cost Books window.
  4. Your Cost Book data will appear. For details on how to edit your Cost Book, see Editing Cost Records. If you need to add a Cost Book, see Add a Custom Cost Book.

Cost Book Menu Buttons

The following buttons are available on the Cost Book page:

  • Add Book: Select the plus icon on the left to add a new Cost Book.
  • Delete Book: Select the trashcan icon in the center to delete a Cost Book.
  • Edit Book: Select the pencil and paper icon on the right to change the Name, Description, or Source.
  • Save Changes: Select the floppy disk icon on the left to retain your edits to an existing Cost Book.
  • Filter: Select the center icon to add a filter to your Cost Book search results.
  • Select Columns: Choose the three-bar icon on the right to select which data columns to view in your Cost Book.

Additional Cost Book Functions

You can select Filters by each field column to narrow or expand your search. Results can be organized in ascending or descending order by selecting the minus icon by each field column. Choose the Up caret to see results in ascending order, the Down caret to see results in descending order, or the minus icon to revert the Cost Book items to their original order.

If an Cost Book item's data has been overridden or updated, you will see the Remove button (the trashcan icon) at the far right of the Cost Book item row. Select the button to delete an item from your Cost Book.

Editing Cost Records

The Cost Record screen displayed above appears when you select a Cost Book item. The Work Action, Work Activity Type, Managed Object Type, and Specification fields cannot be edited from this screen. However, the other fields can be configured and updated within the Cost Record screen.

Once you have added the Cost Record data, select the Update button. If you do not want to save the Cost Record data, select Close.

Note: You must Save your changes (via the floppy disk icon highlighted above) before viewing another Cost Book or your updates will be lost if you navigate away from this page.

Features and Search Bar

  • Work Action: Identifies the item obtained from concatenating the Specification and the Activity.
  • Specification: The item selected for editing within the cost book.
  • Activity: The action required for the cost book item.
  • Object Type: A generalized item classification compared to the Specification. For example, a Specification may be "Pipe, Copper, 8-12 Inch." The Object Type will simply be "Pipe."
  • Domain: The domain applied to the cost book, such as Utility-Gas, Waterfront, or Facility-Buildings.
  • Description: Text entered by the user to aid in identifying the item.
  • Level: The condition for the specified item.
  • Parent: A key sub-classification of the Object Type.
  • Current Cost: The latest cost of the specified cost book item(s).
  • Original Cost: The initial cost of the specified cost book item(s).
  • Cost Year: The year at which the item cost is set.
  • UoM: The Unit of Measurement for the specified item(s).

    • (Note: the available UoMs depend upon the selected cost book and user requirements).

  • Minimum Cost: The base-level value of the cost book.
  • Materials: The value of materials used to execute the cost book.
  • Labor: The value of labor used to execute the cost book.
  • Equipment: The value of equipment used for the cost book.
  • Bare Total: The estimated total value of the cost book.
  • Source: The basis of the cost book information, which is user-entered.
  • Frequency: Describes how often repeated work described by the cost book needs to be performed.
  • Crew: The number of people who are using the cost book.
  • Daily Output: The cost book's output value per day.
  • Total Output: The cost book's overall value for the fiscal year.

Delete Cost Records

To delete a cost record,

  1. Find and select a cost record to delete from the custom cost book.
  2. Click the trashcan icon under Remove to delete the cost record.
