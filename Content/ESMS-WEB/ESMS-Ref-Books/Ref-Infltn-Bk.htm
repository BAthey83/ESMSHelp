About Inflation Books

Enterprise SMS is equipped with an embedded inflation module that factors inflation into work costs and Component replacement costs over time.

The inflation module contains a default inflation book. However, custom inflation books can be added to suit an Organization's needs or to simulate the effect multiple inflation rates may have on work costs over time.

Inflation Data

The inflation data used by the inflation module is contained in one or more inflation books. Multiple custom inflation books can be associated with different Organizations.

Default Inflation Book

The default inflation book is dependent on the selected Organization as well as the default data imported from Work Actions.

Note: the Work Actions page can only be viewed by users with Administrator privileges.

You can choose to depend exclusively on the base inflation data for your selected Organization, or you can create one or more custom inflation books for the chosen Organization.

Custom Inflation Book(s)

To create custom inflation data, you can either:

  1. Edit an existing inflation book.
  2. Create one or more new inflation books. You can use an existing inflation book as the starting point for the new inflation book.

By assigning a custom inflation book to an Organization, you can control the scope of that inflation book, applying custom values to all or part of the site.

Note: You cannot edit the inflation data from past years. You can only edit the inflation data for the current year, or for future years.

Inflation Factor Used in Cost Books

The inflation factor used in cost books is computed from the base year of the cost data and the inflation rates given in the Organization's default Inflation book or in Work Actions. If the cost data corresponds to the data in the Inflation cost book, then the cost module uses a fixed year for the base year. If you add your own cost book and enter unit cost(s) different from the base year, then the cost module saves the year in which you input that data as the base year for inflation computations.
