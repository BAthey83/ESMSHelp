Deleting Cost Books

This topic explains how to delete all types of custom cost books, including standard cost books, Component Importance Index (CII) books, Service Life books, and Inflation books. The topic Viewing and Editing A Cost Book explains how to view or edit data in a cost book.

To delete a cost book,

  1. Choose an Organization from the Current Organization menu.
  1. Select the Reference Books dropdown menu. Choose a relevant cost book type (such as Cost, as highlighted above).
  2. Click on the cost book you want to delete in the Search menu.
  3. Select the trashcan icon just above the Search menu.
  4.  The Confirm Action popup will appear. Click the Confirm button to delete your cost book. Alternatively, select Cancel to keep your cost book.

    Delete a Cost Book

    This topic explains how to delete a custom cost book. The original Reference book cannot be deleted.

    Permissions Note: Deleting a cost book requires Master Planner permission for the Organization or Site applied to the cost book.

    To delete a cost book,

    1. Select the Reference Books dropdown menu. Choose the Cost button.
    2. Click on the custom cost book you want to delete in the Search Cost Books menu.
    3. Select the trashcan icon that is just above the Search Cost Books menu.
    4.  The Confirm Action popup will appear. Click the Yes button to delete your custom cost book.
