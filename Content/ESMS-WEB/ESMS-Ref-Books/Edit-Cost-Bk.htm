Edit Cost Book Data

This topic covers how to add or edit data in a custom cost book. If the custom book still needs to be created, refer to the Add a Custom Cost Book topic.

Permissions Note: Editing a custom cost book requires Master Planner permission level or above at the target Organization.

Tasks:

  1. Open a Custom Cost Book
  2. Navigate Cost Book Fields
  3. Edit Cost Book Data
  4. Save Changes

Steps for each of the tasks are listed below.

Open a Custom Cost Book

To open a custom cost book,

  1. Select the Reference Books dropdown menu. Click the Cost button. The available custom cost books will appear.
  2. Choose a custom cost book from the menu by clicking and highlighting it with your mouse. If you do not see the cost book you want, type your book title in the Search Cost Books window.

Navigate Cost Book Fields

You can use one or a combination of the navigation options described below to bring cost book records into view: 

  • Work Action: The process(es) being executed, and the Cost Book item listed under Specification.
  • Specification: The description for the Cost Book item being inventoried.
  • Activity: The action that the Cost Book item requires: Maintain, Inspect, Replace, Local Repair,or Global Repair.
  • Object Type: Describes the Cost Book item.
  • Domain: The Domain applied to the Cost Book/Cost Book item, such as Waterfront, Utility-Electrical, or Facility-Buildings.
  • Description: Identifies the Cost Record.
  • Level: Identifies where the Cost Book is ranked in the hierarchy.
  • Parent: The Cost Book's assigned parent Organization.
  • Original Cost: The initial value of the specified Cost Book item.
  • Cost Year: Indicates the inflation value of the specified Cost Book item.
  • UoM: The Unit of Measure used by the Cost Book item.
  • Minimum Cost: The base-level value of the Cost Record.
  • Materials: The Material cost within the Cost Record.
  • Labor: The Labor cost within the Cost Record.
  • Equipment: The Equipment cost within the Cost Record.
  • Bare Total: The estimated total cost value of the Cost Record.
  • Source: The Cost Book used as the foundation for the Cost Book item.
  • Frequency: Describes how often the Cost Book item is updated.
  • Crew: The number of people who are using the Cost Book item.
  • Daily Output: The Cost Book output value per day.
  • Total Output: The overall Cost Book's output value for the fiscal year.

Edit Cost Book Data

Current Cost Data

To change the current cost data for an item,

  1. Select the item you wish to modify.
  2. The Cost Record window will appear.
  3. Add the new Unit Cost value to the Unit Cost field.
  4. In addition, you can modify the Cost Year, Minimum Cost, Material Cost, Labor Cost, Equipment Cost, Bare Total Cost, Cost Source, Frequency, Crew, Daily Output, and Total Work Output fields.

Save Changes

To save your changes to the various cost fields, click the Update button. If you wish to remove your cost data changes, select the Close button.

IMPORTANT: Do not forget to select the Save icon at the top of the Cost Book screen before selecting a new Cost Book, or your changes will be deleted.
