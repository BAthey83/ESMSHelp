Add, Edit, or Delete a Cost Modifier

This topic details how to add, edit, or delete a Cost Modifier to Reference Books.

—Quick Links—

  • Add a Cost Modifier
  • Edit a Cost Modifier
  • Delete a Cost Modifier

Add a Cost Modifier

To add a cost modifier,

Select the Create Modifier button in the Cost Modifier window. The Cost Modifier popup window will appear.

Enter the following information:

  • Type - Indicates whether the cost modifier uses a Multiplier or an Adder.
  • Scope - Indicates whether a cost modifier's scope is for a Component, Group, Asset, or Site.
  • Value - Indicates the overall price of the cost modifier.
  • PRV - Indicates whether the Plant Replacement Value has been factored into the cost modifier.

Note: The Name and Description fields are automatically populated with the information entered in the Add a Cost Modifier Library stage.

Select the Create button to add your cost modifier, or click Cancel to discard your additions.

Edit a Cost Modifier

To edit a cost modifier,

  1. Select a modifier to edit in the Cost Modifier popup window.
  2. Within the popup window, edit the Name, Description, Type, Scope, Value or PRV information as shown in Add a Cost Modifier).
  3. Click the Update button to complete the edit.

Delete a Cost Modifier

To delete a cost modifier,

  1. Open the Cost Modifier popup window and select the modifier you wish to delete.
  2. Select the trashcan icon under the Remove column to delete the cost modifier.

    3.  Note: Deleting a cost modifier will also remove all assignments of that modifier in Enterprise SMS.
