Adding Cost Books

This topic details how to add cost books of all types, including custom cost books, Component Importance Index (CII) books, Service Life books, and Inflation books. The topic Viewing and Editing A Cost Book explains how to view or edit data in a cost book.

To add a cost book,

  1. Choose an Organization from the Current Organization menu.
  2. Select one of the options (Cost, Cost Modifiers, Component Importance, Service Life, Inflation, or Work Actions) found within the Reference Books dropdown menu.
  3. Click the "+" icon that appears above the search bar, as highlighted above. The relevant Create Book screen appears in a separate window.
  4. Complete the fields in the window as directed in the Editing Cost Records topic.
  5. Once you have named your cost book in the Book Name column, select a Source, which identifies the source of the cost information.
  6. If you wish to import existing cost book data to your new cost book, select a file from the Copy Existing Book dropdown menu.

    7. Note: Leave the Copy Existing Book field blank if you wish to create a cost book from scratch.

  7. Click Create to add your new custom cost book, or select Cancel to stop the process.

Note: Once you have created or canceled your cost book, save your changes by clicking the floppy disk icon, as highlighted above. Otherwise, your changes will not be retained when you open another page.

Adding Cost Books

This topic covers how to add cost books, regardless of type. The topic Viewing and Editing A Cost Book explains how to view or edit data in a cost book.

Note: the following steps can be applied to any cost book type, including custom Cost Books, Component Importance Index (CII) books, Service Life Books, or Inflation Books.

Permissions Note: Adding a cost book requires Administrative permissions. Consult your system administrator for details.

To add a cost book,

  1. Choose an Organization from the Current Organization menu.

 

  1. Select a Reference Book type from the Reference Books dropdown menu. For example, if you would like to add a cost book, select the Cost button.
  2. Click the "+" icon that appears above the search bar. The Create Cost Book window will appear.
  3. Complete the fields in the window as directed in the Editing Cost Records topic.
  4. Once you have named your Cost Book in the Cost Book Name field, fill in the Source, which identifies the root of the cost information.
  5. If you wish to import existing cost book data to your new Cost Book, select a file from the Copy Existing Book dropdown menu.

    6. Note: Leave the Copy Existing Book field blank if you wish to create a Cost Book from scratch.

  6. Click Create to add your new Cost Book, or select Cancel to stop the process.
