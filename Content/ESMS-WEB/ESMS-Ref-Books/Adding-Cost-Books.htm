Adding Cost Books

This topic covers how to add cost books, regardless of type. The topic Viewing and Editing A Cost Book explains how to view or edit data in a cost book.

Note: the following steps can be applied to any cost book type, including custom Cost Books, Component Importance Index (CII) books, Service Life Books, or Inflation Books.

Permissions Note: Adding a cost book requires Administrative permissions. Consult your system administrator for details.

To add a cost book,

  1. Choose an Organization from the Current Organization menu.

 

  1. Select a Reference Book type from the Reference Books dropdown menu. For example, if you would like to add a cost book, select the Cost button.
  2. Click the "+" icon that appears above the search bar. The Create Cost Book window will appear.
  3. Complete the fields in the window as directed in the Editing Cost Records topic.
  4. Once you have named your Cost Book in the Cost Book Name field, fill in the Source, which identifies the root of the cost information.
  5. If you wish to import existing cost book data to your new Cost Book, select a file from the Copy Existing Book dropdown menu.

    6. Note: Leave the Copy Existing Book field blank if you wish to create a Cost Book from scratch.

  6. Click Create to add your new Cost Book, or select Cancel to stop the process.
