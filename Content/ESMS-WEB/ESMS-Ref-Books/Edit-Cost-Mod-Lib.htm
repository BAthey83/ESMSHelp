Edit a Cost Modifier Library

This topic details how to edit a Cost Modifier library. It also details the tasks that can be performed here.

Note: this process requires Administrator permissions.

To edit a Cost Modifier library,

  1. Select the Reference Books dropdown menu.
  2. Click the Cost Modifiers button.
  3. Choose the Cost Modifier library you wish to edit from the Search window.
  4. Select a Cost Modifier book from the library. The Cost Modifier popup will appear.

Here, you can edit the following fields:

  • Type - Indicates whether the Cost Modifier is a Multiplier or an Adder.
  • Scope - Indicates whether Cost Modifier's scope is for a Component, Group, Asset, or Site.
  • Value - For an adder, the value is a dollar amount. For a multiplier, it is the rate at which the cost should be increased.
  • PRV - Select to factor the Plant Replacement Value into the Cost Modifier.

The Name and Description fields are populated with the information entered in the Add a Cost Modifier Library stage. The Times Assigned column is automatically populated as well.

  1. Once you have finished editing the Cost Modifier fields, select the Update button to add them to the library. To delete these changes, select Cancel.

The columns of the Cost Modifier library are described in View a Cost Modifier Library.

For information on how to add, edit, or delete a Cost Modifier record, see Add, Edit, or Delete a Cost Modifier.
