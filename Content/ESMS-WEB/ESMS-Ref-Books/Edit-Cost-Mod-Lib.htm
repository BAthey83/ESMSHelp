Edit a Cost Modifier Library

Note: this process requires elevated permissions to access.

To edit a cost modifier library,

  1. Select the Reference Books dropdown menu.
  2. Click the Cost Modifiers button.
  3. Choose the Cost Modifier library you wish to edit from the Search CM Books window.
  4. Select a cost modifier book from the library. The Cost Modifier popup will appear.

Here, you can edit the following fields:

  • Type - Indicates whether the cost modifier is a Multiplier or an Adder.
  • Scope - Indicates whether cost modifier's scope is for a Component, Group, Asset, or Site.
  • Value - Indicates the overall cost of the cost modifier.
  • PRV - Indicates whether the Plant Replacement Value has been factored into the cost modifier.

The Name and Description fields are populated with the information entered in the Add a Cost Modifier Library stage. The Times Assigned column is automatically populated by the Cost Modifier Book as well.

  1. Once you have finished editing the Cost Modifier fields, select the Update button to add them to the library.

The columns of the Cost Modifier library are described in View a Cost Modifier Library.

To add, edit, or delete a cost modifier record, see Add, Edit, or Delete a Cost Modifier.
