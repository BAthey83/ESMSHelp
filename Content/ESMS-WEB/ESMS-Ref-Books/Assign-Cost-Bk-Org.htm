Assign a Cost Book to an Organization

This section discusses how to assign a Cost Book to an Organization.

You can create multiple cost books, and assign a separate book to any Organization within your permissions scope, as desired. This is done as follows:

  1. Go to Current Organization and select the Organization to assign to the cost book.
  2. Open the Reference Books drop-down menu.
  3. Select the Cost button.
  4. Choose an existing Cost Book for that particular Organization.

  5. If no Cost Book exists for your Organization, you can create a new Custom Cost Book for it. Select the Add Book button (highlighted in red above) near the Search window to add a new custom Cost Book. Refer to the Add a Custom Cost Book topic for additional information.
  6. Click the Save icon to retain your revisions/updates.

Note: If a Cost Book is not chosen for an Organization, the default/parent data travels up the Enterprise SMS hierarchy until a Cost Book is found. For additional details, refer to the About Cost Books topic.
