Assign a Cost Book to an Organization

This section discusses how to assign all Cost Book types to an Organization.

Note: Only users with Administrative permissions can assign a Cost Book to an Organization.

To assign a Cost Book to an Organization,

  1. Select the Inventory drop-down menu and click Organizations.
  2. Choose an Organization from the Search Organizationsscreen.
  3. In the subsequent menu, scroll to the Reference Books tab. The Organization Reference Books screen will appear.
  4. Select one or multiple custom Reference Book(s) available for your Organization from the dropdown menu(s).
  5. Once you have made your choice(s), click Save, which is located at the bottom of the page.

    Assign a Cost Book to an Organization

    This section discusses how to assign a Cost Book to an Organization, with detailed steps on the process.

    You can create multiple cost books, and assign a separate book to any Organization within your permissions scope, as desired. This is done as follows:

    1. Go to Current Organization and select the Organization to assign to the cost book.
    2. Open the Reference Books drop-down menu.
    3. Select the Cost button.
    4. Choose or create a Cost Book for that particular Organization.
    5. Click the Save icon to retain your revisions/updates.
