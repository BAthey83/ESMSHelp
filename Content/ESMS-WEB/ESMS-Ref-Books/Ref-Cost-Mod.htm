Cost Modifier Basics

The option to apply cost multipliers and adders provides great flexibility and control in accounting for factors that affect costs. Common factors that change costs include requirements related to security, health and safety, and containment.

Cost Modifiers are created and managed in cost modifier books. These cost modifier books can be created at the Organization level. The modifiers in a book can then be assigned within the Organizations page, available from the Inventory menu.

Simply select an Organization, go to the Reference Books tab and select an available Cost Modifier book.

When a Cost Modifier is assigned, that modifier is calculated for all child Sections below the parent level at which the modifier was assigned. A modifier can also be assigned at the Section level, but it only applies to that specific Component-Section.

A Section can inherit one or more Cost Modifiers from a parent as well as have its own modifier applied. For example, if the Building multiplier is 2, the System multiplier is 2, and the Section multiplier is 2, the Section cost will be multiplied by a factor of 8 (2x2x2).

Accessing Cost Modifiers

To access Cost Modifiers,

  1. Choose an Organization from the Current Organization button.

  1. Select the Reference Books dropdown, followed by the Cost Modifiers button.
  1. Pick a Cost Modifier book from the Search window.
  2. The Cost Modifiers window will display, with the name of the library at the top. You can now proceed with cost modifiers.

Cost Modifier Types

The Cost Modifier types listed below are applied in addition to Area Cost Factor, the General Multiplier for the cost book, and any specific multipliers for the cost book.

The cost multiplier is a multiplying factor applied to all child Sections. For example, if a Building has a 10% cost multiplier, then the 10% multiplier will be applied to all Sections below the Building.

Cost Adder

The cost adder is a fixed cost applied to all child Sections. For example, if a Building has a $2700 cost adder, then all Sections within that Building will be assigned a $2700 cost adder.

Cost Multiplier

The cost multiplier is a fixed cost applied to all child Sections. For example, if a Building has a cost multiplier, then all Sections within that Building will be assigned a cost multiplier.

Assigning Cost Modifiers

Users with elevated permissions can create and name multiple Cost Modifier books, defining separate book as needed for different Organizations.

Once an active Cost Modifier book has been selected for an Organization, the modifiers can be assigned to the Enterprise SMS inventory.
