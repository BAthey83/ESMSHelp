Add a Custom CII Book

This topic explains how to create a custom Component Importance Index (CII) book for an Organization.

Permissions Note: Adding a custom CII book requires Work Planner permissions or above at the target Organization.

 

To add a custom CII book,

 
  1. Go to Current Organization on the main menu to choose the Organization to assign to your custom CII book.
 
  1. At the main menu, select the Component Importance button under the Reference Books category.
  2. Click the "+" sign icon above the "Search CI Books" search box.
  3. The Create CII Book window will appear. Here, you will enter your CII Book Name and Source. Once the fields have been filled in, select the Create button.

    4. Note: If you need to delete your new CII Book, select the trashcan icon next to the "+" sign at the top of the window.

  4. After creating your CII book, you can filter search results to include a specific domain (such as Fuels).
  5. Next, you can add/update CII rating(s) to any items in your search results. If you wish to delete any CII rating(s), select the Clear Filters icon to the right of the CII rating(s).
  6. Once you have added the CII rating(s) you wish to add, click the Save icon.
