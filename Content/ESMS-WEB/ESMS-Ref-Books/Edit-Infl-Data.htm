Edit Inflation Data

This topic demonstrates how to edit inflation data in Enterprise SMS, with detailed descriptions for each task.

Permissions Note: Editing a custom inflation book requires Work Planner permission level or above at the target Organization.

To edit a custom inflation book,

  1. Select a custom inflation book at the Search Inflation Books window. If there are no custom inflation books available, you can create a new custom inflation book. (Note: For details on how to create a new custom inflation book, see Add a Custom Inflation Book).
  2. Open the custom inflation book. Here, you can add custom inflation rates for each year that appears in the inflation book.
  3. If you wish to see inflation rates for previous years, you can select Yes at the Show Past Years? dropdown menu.
  4. Once you have reviewed and added all the necessary data, click the Save button to complete the editing process.
