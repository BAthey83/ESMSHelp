Add a Custom Cost Book

This topic covers how to add a custom cost book. The topic Viewing and Editing A Cost Book explains how to view or edit data in a cost book.

To add a cost book,

  1. Choose an Organization from the Current Organization menu.
 
  1. Select the Cost button within the Reference Books dropdown menu.
  2. Click the "+" icon that appears above the "Search Cost Books" search bar. The Create Cost Book window will appear.
  3. Complete the fields in the window as directed in the Editing Cost Records topic.
  4. Once you have named your Custom Cost Book in the Cost Book Name column, select a Source, which identifies the source of the cost information.
  5. If you wish to import existing cost book data to your new Custom Cost Book, select a file from the Copy Existing Book dropdown menu.

    6. Note: Leave the Copy Existing Book field blank if you wish to create a Custom Cost Book from scratch.

  6. Click Create to add your new Custom Cost Book, or select Cancel to stop the process.
