View a Cost Modifier Library

This topic discusses how to view a cost modifier library. It also details the tasks that can be performed here.

Note: Administrator permissions are required to View a Cost Modifier Library.

To view a Cost Modifier library,

  1. Open the Reference Books dropdown menu.
  2. Select the Cost Modifiers button.
  3. Go to the Search window and select a Cost Modifier book from the available options.

The following functions are available on the Cost Modifiers page:

  • Add Book. Select the Add Book icon (the plus icon highlighted above) to create a new Cost Modifier book in the library.
  • Delete Book. If the Delete Book icon (the trashcan icon highlighted above) is activated, it will delete the Cost Modifier book.
  • Edit Book Name. Select the Edit Book Name icon (the pencil and paper icon highlighted above) to modify the cost modifier book name and/or the cost modifier book source.

The table of cost modifiers has the following columns:

  • Name. Identifying name assigned to the modifier.
  • Description. Text input by the user to identify the cost modifier.
  • Multiplier? Indicates whether a multiplier or adder has been applied.
  • Scope. Shows the scope where the cost modifier applies, whether it is Component, Group, Asset, or Site.
  • Times Assigned. The number of times the modifier has been assigned.
  • Value. For an adder, the value is a dollar amount. For a multiplier, it is the rate at which the cost should be increased.
  • PRV. Select this to factor the Plant Replacement Value in the cost modifier.
  • Remove. This field lets you remove any or all filters applied to the cost modifier.

Each of the columns allows you to apply a filter on any or all of the columns to potentially reduce the number of cost modifier records displayed on the screen.
