User Details

User details are found in the Security Editor, and pertain when you view, add, or edit a user at that screen. (For details how to perform user management tasks, see Managing Users and Roles (Security Screen)).

User Details include the following fields:

General Tab

  • General - Mark or unmark checkboxes to select one or more of the following:
    1. Administrator - gives the user Administrator rights
    2. Can Add Users via Invite - allows the user to use Invite to add other users
    3. API Developer - gives the user API Developer rights
    4. Locked - locks the account and disables login
  • UserName - User's account name/email address
  • Email - User's email
  • Phone - User's phone
  • Default Organization - Marking one or more default organizations here can save the user time when logging in
  • Default Domains - Marking one or more default sites here can save the user time when logging in
  • Organization Memberships - This is where you can add or delete organization memberships
  • Account Management - This is where you as administrator can add or delete the user's account

Roles Tab

  • User Workspace - This is where the administrator assigns organization, site, and/or group roles to the user. See "Assign Role to User" in Managing Users and Roles (Security Screen).
