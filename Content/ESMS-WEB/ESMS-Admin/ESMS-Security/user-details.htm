User Details

User details are found in the Security Editor, and pertain when you view, add, or edit a user at that screen. (For details how to perform user management tasks, see Managing Users and Roles (Security Screen).)

User Details include the following fields:

  • General - Mark or unmark checkboxes to select one or more of the following:
    1. Administrator - gives the user Administrator rights
    2. Can Add Users via Invite - allows user to use Invite to add other users
    3. Locked - locks the account and disables login
  • Email - User's email
  • Phone - User's phone
  • Password - This is where you as administrator can change the user's password
  • Site to Sync - Specify the site that the field application will sync when the user activates sync in Desktop or Tablet. To do so, activate the None Selected button, then select the desired site using the popup that appears.
  • Sync Behavior - The default behavior is Full Site Sync. To use ESMS with tablets, activate Assessments only.
  • Sync Assessment - Not currently functional.
  • Sync Direct Rating - Not currently functional.
  • Default Site - This is a selection that can save the user time if they use only one site all or most of the time. Establishing a default site, which activates when the user logs in, saves the user the step of selecting a site each time they log in while still leaving them the option to change to a different allowed site (using the Select Site button on the menu bar) as needed
  • Default Domains - As with Default Site, marking one or more default sites here can save the user time when logging in
  • User Workspace - This is where the administrator assigns site and/or group roles to the user. See "Assign Role to User" in Managing Users and Roles (Security Screen).
