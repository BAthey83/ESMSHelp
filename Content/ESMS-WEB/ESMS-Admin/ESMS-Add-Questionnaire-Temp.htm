Add Questionnaire Template

This topic shows how to add an assessment questionnaire template at the Catalog Manager screen, and how to design the questions to be contained in the template.

Note: Questionnaire templates are added and designed at the Catalog Manager screen, accessed by activating the Catalog tile at the ESMS Dashboard. The Catalog Manager screen is viewable by ESMS administrators only; the Catalog tile will not appear on the Dashboard of a non-administrator.

What Can I Do Here?

At the Assessment Templates tab on the Catalog Manager screen, Enterprise SMS enables you to create templates that can be used to assess component types.

You can make multi-level questionnaires, and the program automatically assigns the correct numbering.

WARNING : Hierarchy levels in the questionnaire template cannot be changed, and the questions located at a particular hierarchical level in the questionnaire template cannot be moved to another level.

Best Practice Tip: When creating a questionnaire template with more than one level of question category, you should carefully plan out the structure and questions in advance of making entries in the assessment template.

Permissions Note: Accessing questionnaire template tasks requires ESMS admin privilege.

Add the Template

Note: In the popup associated with creating a template, a template is called "assessment set template." "Set" refers to the set of questions.

To add a template,

  1. In the list at the left hand side of the screen, activate the component type that should be associated with the new template. The Assessment Templates menu bar should appear.
  2. Activate the Add Assessment Set Template button on the menu bar. The "Create a new Assessment Set Template" popup will appear.
  1. In the "Create Set Template" popup,
    1. Enter a Name for the template.
    2. Activate Create. Alternatively, activate Cancel to close the popup without creating the assessment set template.

  1. Continue with instructions under Add Questions to the Template.

Add Questions to the Template

Best Practice Tip: For a questionnaire with more than two levels (question categories at level one, plus a second level with actual questions), you might want to plan out the structure and questions in advance of making entries in the template.

To add a question or question category to an assessment set template,

  1. If you are coming directly from instructions under Add the Template above, skip to Step 5.
  2. Otherwise, if you are not already in the Catalog Manager, select the Catalog tile on ESMS Dashboard.
  3. Activate the Assessment Templates tab in the Catalog Manager.
  4. Activate component type in the list at the left. If the assessment template still needs to be created for that component type, a message will pop up to this effect. After you select a component type, the blue menu bar of options will appear below the Assessment Templates tab.
  5. Click the down arrow to the right of the Add button in the menu bar. A dropdown will show the types of questions available:
    1. Question category (not itself a question; it creates an organizing container for other questions)
    2. Multi-select question
    3. Single-select question
    4. Subcomponent question category
    5. Distress question
    6. User Input question
  1. Select Question Category or a question type (Multi-Select, Single-Select, Distress, User Input) from the dropdown. Question categories and types, and how to enter them, are explained below.

Question Types

Question Category

Question categories are an optional organizing device for questions contained in the template. Categories can be nested as described in About Nesting and Presentation Order, theoretically indefinitely.

To create a question category,

  1. As outlined in Add Questions to the Template above, select Question Category from the dropdown at the Add button on the Assessment Templates menu bar. If successful, a box containing the fields for designing the question category will appear in the content area. Each question category and each question will have a separate, numbered box.

Troubleshooting Tip: If no template set has been created for the component type selected in the left pane, the fields for designing the question category will not appear in the content area.

  1. You can select an Engineering Discipline from the dropdown at the upper right of the box, if a list of engineering disciplines has been configured into ESMS for your organization.
  2. Type a name for the question category in the Category field.
  3. Activate Save (lower right).

Note: The content area of the screen might briefly go blank during the save process.

Question Type: Single-Select

Question type is different from question category. Question category is defined by the user and is optional. Selecting a question type is mandatory. There are two types: (a) Single-select and (b) multi-select. Single-select is described here.

For a single-select question, you enter the question, and then add one or more custom answers.

To create a single-select question, first enter these non-repeating fields:

  1. Question (Required). Type the text of the question here.
  2. Comment (Optional). If you want a comment--for example, further explanation about the question--to be displayed to the assessor/user, enter that comment into this field.
  3. Reference. A reference may be added in this field.

Note: Reference is optional with respect to the software, but may be required by your organization.

Next, create the answer blocks:

  1. Enter Answer (Required). Type in the text of one of the answers that can be selected.
  2. Enter Points. Enter the number of points to be added (or subtracted, if a negative number is selected) when the user selects this answer. Omit if points are not applicable.
  3. Enter Comment (Optional). If you want a comment to be displayed, enter that comment into this field.
  4. Add Followup Question (Optional). If you want a particular followup question presented when the user select the answer you entered above, then activate the Add Followup Question button. Select between Multi-Select Question and Single-Select Question, then fill in the appropriate Question, Comment(s), Reference, and Answer fields for the followup question, including Points for each answer if applicable.
  5. To add another answer that can be selected, activate the Add Answer button and repeat the steps above.
  6. If you are finished entering answers, activate Save.

Question Type: Multi-Select

Question type is different from question category. Question category is defined by the user and is optional. Selecting a question type is mandatory. There are two types: (a) Single-select and (b) multi-select. Multi-select is described here.

A multi-select question has a fixed format; the user responds by selecting between the three options: "Yes," "No," and "N/A."

To create a multi-select question, first enter these non-repeating fields:

  1. Question (Required). Type the text of the question here.
  2. Comment (Optional). If you want a comment--for example, further explanation about the question--to be displayed to the assessor/user, enter that comment into this field.
  3. Reference. A reference may be added in this field.

Note: Reference is optional with respect to the software, but may be required by your organization.

Next, fill in the Yes, No, and N/A answer blocks as needed. Make sure to Save!

  1. Enter Points. Enter the number of points to be added (or subtracted, if a negative number is selected) when the user selects this answer. Omit if points are not applicable.
  2. Add Followup Question (Optional). If you want a particular followup question presented when the user select this answer, then activate the Add Followup Question button. Select between Multi-Select Question and Single-Select Question, then fill in the appropriate Question, Comment(s), Reference, and Answer fields for the followup question, including Points for each answer if applicable.
  3. Repeat the steps above until all the answer options are complete.
  4. Activate Save.

Change Presentation Order

To change the order that the questions will be presented in,

  1. Use the up and/or down arrows to the right of each question.
  2. Activate Save when finished.

Note: You can move questions only within a given hierarchical level in the questionnaire templates; you cannot move them between levels.

For more detail, see About Nesting and Presentation Order.
