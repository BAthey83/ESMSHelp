Understanding Questionnaire Templates

This topic introduces questionnaire templates and provides a launching pad to a variety of template tasks. The primary purpose of questionnaire templates is to produce assessment questionnaires to be used by inspectors.

Note: Questionnaire templates are added and designed at the Configuration Manager screen, accessed by activating the Catalog tile at the ESMS Dashboard. The Configuration Manager screen is viewable by ESMS administrators only; the Catalog tile will not appear on the Dashboard of a non-administrator.

What Can I Do at the Assessment Templates Tab?

At the Assessment Templates tab in the ESMS Configuration Manager, you can create different kinds of assessment questions, and organize them using question categories and nesting (nesting and question categories are both optional). How to do this will be explained in Add Questionnaire Template. The completed set of questions is called an assessment set template, or template set. The ESMS program automatically assigns the correct numbering of questions within the template set.

If one or more template sets have been created, you can select an existing one using the menu option Choose Template Set.

When viewing a template set, you may find navigating through it to be more efficient if you activate Collapse All in the menu, to view just the individual questions without details about answers. Then you can expand individual questions as desired, or activate Expand All to expand all of the questions at once.

Navigating at the Assessment Templates Tab

To activate the Assessment Templates tab menu,

  1. If not already in the configuration manager, activate the Catalog tile on the Enterprise SMS Dashboard to reach the configuration manager.
  2. Activate the Assessment Templates tab. A list of component types will appear in the left pane.
  3. To make the menu appear, activate a component type in the list. You can use the search field (labeled "Search component types here") at the top of the list to help find a desired component type. Once a component type is activated, the menu bar will appear below the Assessment Templates tab:

The menu bar at the Assessment Templates tab allows you to work with template sets and to view and add questions in a template.

Note: The menu will not appear until a component type is selected in the list at the left.

Menu options are:

  • Add. After you select an assessment set, this button allows you to add questions or question categories to the template
  • Add Assessment Set Template. Activating this button introduces a popup for creating a new template
  • Expand All. If the content area has been collapsed to show just each question category (where one has been assigned) or question, activating Expand All will change the view to show full detail
  • Collapse All. Activate Collapse All to see just each question category (where one has been assigned) or question in the content area
  • Choose Template Set. Activate the down arrow to display a dropdown of template sets to choose from
