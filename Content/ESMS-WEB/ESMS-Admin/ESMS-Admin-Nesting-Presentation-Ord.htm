About Nesting and Presentation Order

This topic

  • Explains nesting option for questions and question categories in a questionnaire template
  • Explains how to change the order of questions

Nest Questions in a Questionnaire Template

Question Categories

Number of Sub-Levels: Unlimited

One way to nest questions in a questionnaire template is to use question categories to group questions. When you create a question category, you will first be prompted to specify an engineering discipline from a dropdown list, if your questionnaire has been configured with engineering disciplines. Then, in the Category field, enter the name of the category. Once you have created the category, initiate the Add button to initiate adding either (a) a question or (b) a subcategory. Fill in the appropriate information, then Save.

WARNING: Hierarchy levels in the questionnaire template cannot be changed, and the questions located at a particular hierarchical level in the questionnaire template cannot be moved to another level.

Best Practice Tip: When creating a questionnaire template with more than one level of question category, you should carefully plan out the structure and questions in advance of making entries in the assessment template.

Followup Questions

Number of Sub-Levels: One

Another way to nest questions is to add one or more followup questions to a question. One followup question is allowed (but not required) for each potential answer to the source question.

Note: In contrast to the (theoretically) unlimited nesting abilities of categories, you are limited to one sub-level of nesting in a question using this technique.

Using the Add Followup Question button, you have the option to designate a followup question to be presented when the user selects the answer associated with the followup question. As with other questions, the followup question can be either a single-select or multi-select question type. However, the answer or answers to a followup question do not present the option for a followup question.

Change Presentation Order in a Questionnaire Template

After inserting and composing questions in the assessment template, you can change the order in which the questions will be presented.

Questions can be re-ordered within the template set as follows:

  1. At the far right of each template element (category or question), there will be a down arrow, a down arrow plus an up arrow, or an up arrow. The arrow selection depends on the element's relative position: the first element at a given hierarchical level in the template will only have a down arrow; the last question or category will only have an up arrow.
  2. To move the element down, activate the down arrow. The element will move below the element after it.
  3. To move the element up, activate the up arrow. The element will move above the element before it.
  4. When finished, activate the Save button.
