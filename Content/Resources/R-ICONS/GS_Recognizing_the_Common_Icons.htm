Recognizing the Common Icons

Listed below are the common command, toolbar, and menu buttons you will encounter when using ROOFER™. It is a good idea to become familiar with these icons before using the program.

Navigation Group Icons:

  • - ADD ORGANIZATION: Click to add a new organization to the inventory.
  • - DELETE ORGANIZATION: Click to delete an organization selected on the navigation tree.
  • - ORGANIZATION: Click this icon on the navigation tree to access data about a particular organization. Colors (Gray, Green, Yellow, Red) reflect the conditions of the organizations.
  • - ADD SITE: Click to add an new site under an organization in the inventory.
  • - DELETE SITE: Click to delete a selected site on the navigation tree.
  • - SITE: Click this icon on the navigation tree to access data about a particular site. Colors (Gray, Green, Yellow, Red) reflect the conditions of the sites.
  • - ADD COMPLEX: Click to add a complex to an organization.
  • - DELETE COMPLEX: Click to delete a complex selected on the navigation tree, from an organization.
  • - Complex: Click this icon on the navigation tree to access data about a particular complex. Colors (Gray, Green, Yellow, Red) reflect the conditions of the complexes.
  • - ADD BUILDING: Click to add a new building to a site.
  • - DELETE BUILDING: Click to delete a selected building from a site.
  • - BUILDING: Click this icon on the navigation tree to access data about a particular building. Colors (Gray, Green, Yellow, Red) reflect the conditions of the buildings.
  • - ADD ROOF SECTION: Click to add a roof section to a building.
  • - DELETE ROOF SECTION: Click to delete a selected roof section from a building.
  • - ROOF SECTION: Type: Steep Roof. Click to access data for a particular steep roof, roof section. Colors (Gray, Green, Yellow, Red) reflect the conditions of the roof sections.
  • - ROOF SECTION: Type: Membrane. Click to access data for a particular Membrane, roof section. Colors (Gray, Green, Yellow, Red) reflect the conditions of the roof sections.
  • - ROLLUP: Manually rolls up performance metrics for a building, complex, site, or organization.
  • - REFRESH: Refresh the inventory tree to reflect the latest data edits. Note that refreshing the tree collapses all expanded nodes.

Content and Form Icons:

  • - SAVE: Click to permanently save and store changes to the data in content area or popup form.
  • - COMMENT: Click to add a comment about a particular inventory item.
  • - REPORTS: Click to access available reports for a selected item in the content area.
  • - NEW: Click this button to create a new record. The type of record to be created is determined by where the NEW button is located.
  • - DELETE: Click this button to delete a selected record. The record to be deleted is determined by where the DELETE button is located.
  • - CLOSE: Click to close a particular popup window or form.
  • - HELP: Click to access context specific help within a particular window or form.
  • - CALENDAR: Click to select a date from a calendar to populate a date type field.
  • - GENERATE: Click this button to generate analysis results. This may be for inspection schedules, work items, or Scenarios.
