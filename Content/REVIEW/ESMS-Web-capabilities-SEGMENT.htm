What Can Enterprise SMS Web Do?

The Web version of Enterprise SMS is intended for

  • importing real property data
  • data search and exploration
  • document storage and retrieval
  • running reports
  • administering permissions for individuals and teams
  • generating work items based on criteria.

It is not intended for

  • real property inventory entry
  • performing assessment plans.

Inventory is imported into the Web version either

  1. via the Import screen (comma-separated-value files are accepted), or
  2. by synchronizing with the Desktop version of Enterprise SMS.

assessment plans are transferred into the Web version by synchronizing with the Desktop version.
