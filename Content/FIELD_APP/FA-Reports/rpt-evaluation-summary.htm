Functional Assessment Summary Report ("Evaluation Summary Report")

The Functional Assessment Summary Report presents a summary of the selected functional assessment.

Generate Report

To generate an evaluation summary report,

  1. Activate Evaluation Summary in the ribbon at the Reports tab.
  2. In the Parameters pane on the left side of the print preview popup, activate the dropdown arrow at the right of the "Assessor" field.

    Troubleshooting Tip: If the Parameters pane is not showing, select Parameters in the ribbon at the Preview tab.

  3. Select one of the dropdown options.
  4. Activate the Submit button. The report should appear in a print preview popup.

  5. To see the summary of a different functional assessment, expand the dropdown, select another assessment from it, and activate Submit.
  6. Repeat the step above as desired.

Save Report

To save a report,

  1. While the report is showing in the print preview popup, activate Save in the ribbon at the Preview tab. (Alternatively, activate Save in the dropdown at the leftmost tab.)
  2. In the "Save As" popup, navigate to the desired save location.
  3. Give the file a more specific identifying name to replace EvaluationSummaryReport.prnx.
  4. Activate the Save button (at the lower right of the "Save As" popup).
