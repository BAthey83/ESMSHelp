Component Summary Report

Generate Report

To generate a component summary report,

  1. In the navigation pane, select the relevant component.
  2. Activate Component Summary in the ribbon at the Reports tab. The report should appear in a print preview popup.

    Note: if you activate Component Summary without having a component selected, the most recently generated component summary report will appear in the "Print Preview" popup.

Save Report

To save a report,

  1. While the report is showing in the print preview popup, activate Save in the ribbon at the Preview tab. (Alternatively, activate Save in the dropdown at the leftmost tab.)
  2. In the "Save As" popup, navigate to the desired save location.
  3. Give the file a more specific identifying name to replace .
  4. Activate the Save button (at the lower right of the "Save As" popup).
