Planning Study Report

Generate Report

To generate a planning study report,

  1. Activate Planning Study in the ribbon at the Reports tab.
  2. In the Parameters pane on the left side of the print preview popup, activate the dropdown arrow at the right of the "Real Property" field.

    Troubleshooting Tip: If the Parameters pane is not showing, select Parameters in the ribbon at the popup's Preview tab.

  3. Select a facility from the dropdown options.
  4. Activate the Submit button. The report should appear in a print preview popup.

Save Report

To save a report,

  1. While the report is showing in the print preview popup, activate Save in the ribbon at the Preview tab. (Alternatively, activate Save in the dropdown at the leftmost tab.)
  2. In the "Save As" popup, navigate to the desired save location.
  3. Give the file a more specific identifying name to replace $$.$$$.
  4. Activate the Save button (at the lower right of the "Save As" popup).
