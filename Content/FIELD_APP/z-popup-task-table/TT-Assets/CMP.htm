Working with Components

Operations you can perform on a component are

  • View
  • Add
  • Edit
  • Delete

Component properties are described in the topic Component Data.

View a Component

To view a component,

  1. Select any one of the top tabs except for the File tab.
  2. In the navigation pane, select the Systems tab above the asset tree table.
  3. Navigate in the asset tree to the desired component and double-click it.

    Note: After the first double-click, only a single click should be needed to view subsequently selected components.

  4. In the content pane, the component's information will appear at the View Component tab.

For more information about accessing and configuring the navigation pane and content pane, see Manipulating Navigation and Content Panes.

Add a Component

To add a component,

  1. Select the Assets page.
  2. Open the navigation pane if it is auto-hidden.
  3. In the asset tree, select an asset on the first or second level below the site level. The new component will be created below the level you have selected, as a component of that asset.
  4. Activate Add Component in the ribbon. The "Add Component" popup will appear.
  5. Enter the requested information in the fields. More detail is given in the topic Component Data.

For more information about accessing and configuring the navigation pane and content pane, see Manipulating Navigation and Content Panes.

Edit a Component

To edit a component,

  1. Select the Assets page.
  2. Open the navigation pane if it is auto-hidden.
  3. In the asset tree, select the component to be edited.
  4. Activate Edit Component in the ribbon. The View Component tab will appear in the content area.
  5. Edit information as desired in the fields. More detail about these fields is given in the topic Component Data.
  6. Activate the Save button (located above the Basic Properties set of fields).

Delete a Component

To delete a component,

  1. Select the Assets page.
  2. Open the navigation pane if it is auto-hidden.
  3. In the asset tree, select the component to be deleted.
  4. Activate Edit Component in the ribbon. The View Component tab will appear in the content area.
  5. Activate the Delete button (located above the Basic Properties set of fields).
