View a Functional Assessment

To view a functional assessment,

  1. Select the assessment plans page.
  2. Activate Functional Assessment on the ribbon.
  3. At the Functional Assessment tab in the content pane, select the functional assessment you wish to view.
  4. Activate the Load Assessment button. {{$$ results to follow, once I know what they are}}
  5. For more information about the functional assessment data shown, see {{afdo -link}} Functional Assessment Data.
