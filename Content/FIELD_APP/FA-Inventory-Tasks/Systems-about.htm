About Systems

From an engineering standpoint, real property in a site is decomposed into systems representing various functions. Every component created in Enterprise SMS must be assigned to a system.

Unlike components and groups, systems are typically manipulated (add, edit, delete) only by an Administrator.

Technically, in the software, a system is a specialized type of group. This will become evident in the process for adding a system.

System Tasks

Tasks you can perform on a system are

  • View
  • Add
  • Edit
  • Delete
