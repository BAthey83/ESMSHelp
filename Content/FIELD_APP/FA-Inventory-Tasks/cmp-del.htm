Delete Component

To delete a component,

  1. Activate the Assets tab.
  2. In the navigation pane (open it if it is auto-hidden), activate the Systems tab.
  3. In the asset tree, select the component to be deleted.
  4. Activate Edit Component in the ribbon. A View Component tab will appear in the content area.
  5. Activate the Delete button.
