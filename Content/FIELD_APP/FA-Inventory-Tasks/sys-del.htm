Delete System

WARNING: Deleting a system will also delete all components and subcomponents contained in that system. The popup requesting confirmation of the delete will show the number of components contained in the system.

To delete a system,

  1. At the Tools tab, activate Group Editor in the ribbon. A Group Editor tab will open in the content pane.
  2. In the list of group types (located at the left side of the content pane), activate "System". A listing of all systems will appear to the right, along with the importance factor percentage for each.

  3. Select the system you wish to delete: check that it is highlighted.
  4. Make note of the number in the "Importance Factor (%)" column for the system about to be deleted.
  5. Activate the Delete Group button (located in the cluster of buttons at the upper right of the content pane). The "Delete Alert" popup will appear, revealing the number of components contained in the system to be deleted.
  6. In the popup, activate Yes.
  7. In the column labeled "Importance Factor (%)" , adjust one or more numbers upwards to compensate for the number associated with the deleted system. This step is not finished until the total of all the numbers in the "Importance Factor (%)" column add to exactly 100.
  8. IMPORTANT: Before exiting, activate the Save Systems button. Even though you have confirmed, in the popup, your desire to delete the group, activating Save Systems button is still required for the deletion to take effect.
