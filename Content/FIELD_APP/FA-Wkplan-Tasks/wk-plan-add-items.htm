Add Work Item(s) to a Work Plan

At the Work Plan Manager tab, at the bottom of the screen in the content pane, arrow buttons between two columns help you select which work items to add into the work plan that is currently active on screen.

You can select either one item or all items in the "Component Work Items" column to be moved into the work plan (the "Work Plan Work Items" column).

Productivity Tip: If you want to move most but not all of the component work items into the work plan, you may find it most efficient to first move all work items into the plan using the double arrow pointing right, and then remove the few exceptions from the work plan individually, as described in Remove Work Item from a Work Plan.

Add One Work Item

To add one item to the work plan,

  1. View the work plan as described in View Work Plan.
  2. In the bottom section of the content pane, view the list of "Component Work Items" on the left.
  3. Select the work item you wish to add to the work plan. (When you select the work item, it will highlight.)

  4. Between the "Component Work Items" column on the left and the "Work Plan Work Items" column on the right, activate the right-facing single arrow.

  5. The selected item will be moved to the "Work Plan Work Items" column on the right.

Add All Available Component Work Items

To add all items to the work plan,

  1. View the work plan as described in View Work Plan.
  2. In the bottom section of the content pane, view the list of "Component Work Items" on the left.
  3. Between the "Component Work Items" column on the left and the "Work Plan Work Items" column on the right, activate the double right-facing single arrow.

  4. All of the work items will be moved into the work plan. If you wish to remove one or more of the work items from the plan, see Remove Work Item from a Work Plan.
