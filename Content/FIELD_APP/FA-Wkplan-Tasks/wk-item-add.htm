In the Enterprise SMS field application, all work items are generated manually.

To add a work item,

  1. In the navigation pane, select the component that the work item is to be associated with.

    Note: You can select the component either at the Systems tab or at the Facilities tab.

  2. At the assessment plans tab, activate Work Item in the ribbon. A Work Item tab will appear in the content pane.
  3. At the Work Item tab, activate the New button. Entry fields will become active.
  4. Enter the following work item data into the "Details" fieldset:
    • Description - Brief (maximum of 50 characters), identifying description of the work to be performed. After you finish entering this description, it will also appear under "Work Items" near the top of the content pane.
    • Reference - If applicable, enter a reference organization (such as ASTM or UFC) and the reference number from that organization that is related to the work needed. For example:

      Note: This field is often related to the "Regulatory Requirement" field that appears later in the entry form.

    • Cost Amount - The estimated cost to perform the work included in the work item. As you enter the cost amount it will also appear under "Work Items" near the top of the content pane.
    • Date - Enter today's date as the creation date of the work item.
    • Current Situation - Description of the component's current situation or condition that merits the repair or replacement called for in the work order.
    • Impact - Description of the adverse impact resulting from the current situation.
    • Scope of Work - Description of the work that needs to be done to mitigate the impact.
    • Regulatory Requirement - $$Description of any regulatory requirement(s) applicable to the work needed.
  5. If desired, attach an image by activating the Add Image button (immediately below "Work Items" near the top of the content pane), then browsing to and selecting the image to be attached.

    To remove an image, roll the mouse cursor onto the image; when the "x" turns red, activate it.

  6. IMPORTANT: Before exiting the work item, activate the Save button at the top of the content pane.
