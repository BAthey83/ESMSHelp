Add Work Plan

To add a work plan,

  1. If the Facilities tab is not showing in the navigation pane, navigate to the Home tab and activate Facilities in the ribbon.
  2. In the navigation pane, select the relevant facility for the work plan.
  3. At the Work Plans tab, activate Work Plan Manager in the ribbon. The Work Plan Manager tab will appear in the content pane.
  4. If there are no work plans are associated with the selected facility, you will see a message stating "There are no work plans for this item." The work plan you add will be the first one for this facility.
  5. If one or more work plans are associated with the facility, the first of these should be displayed in the content pane.

    Troubleshooting Tip: If there is a work plan associated with the selected facility (i.e., you do NOT see a message stating "There are no work plans for this item"), but the fields are not displaying as white and active), try (1) navigating to a different facility in the navigation pane, then (2) navigating back to the desired facility.

  6. Enter the following work plan data into the "Details" fieldset:
    • Title - Label the work plan with a title.
    • Project #(s) - Depending on the sponsoring organization, the fields provided for project numbers may vary in number and the titles may differ in the application interface.
    • Description - Provide a detailed description of the work plan here (the field size limit is over 3700 characters).
    • Status - Activate the dropdown arrow at the right end of the field and select from the following options:

    • Classification - Activate the dropdown arrow at the right end of the field and select from the following options:

    • Condition Code - Activate the dropdown arrow at the right end of the field and select from the following options:

    • Calculated Cost - This field will be automatically calculated from the total costs of constituent work items in the work plan.
    • FY Execution Year - Enter the anticipated fiscal year in which the work will be executed.
  7. To complete the work item data entry by selecting work items to be included into the work plan, see Add Work Item(s) to a Work Plan, and, if necessary, Remove Work Item(s) from a Work Plan.
