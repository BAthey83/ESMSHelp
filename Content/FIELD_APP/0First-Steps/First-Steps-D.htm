Get Started with the Field Application

This topic lists steps to take to get started with the Enterprise SMS (Enterprise SMS) field application.

The field application includes Enterprise SMS Desktop and Enterprise SMS Tablet. Desktop has a full suite of non-administrator functions (administrator functions are performed using Enterprise SMS Web). Enterprise SMS Tablet allows either (a) Direct Ratings or (b) customized assessment plans to be performed in the field. .

This topic presents the tasks involved in getting started, in this order:

  1. Get username and password
  2. Download the Enterprise SMS field application from Enterprise SMS Web
  3. Open the field application
  4. Verify site
  5. Get acquainted with navigation

Each main heading below will cover one of the tasks, broken down into steps.

B. Download the Enterprise SMS Field Application

(Skip this step if you have already downloaded the field application.)

Download

To download the Enterprise SMS field application to your computer, first do the following in Enterprise SMS Web:

  1. Open Enterprise SMS Web.
  2. Navigate to "Hello, {username}" at the right side of the menu bar.
  3. Open the dropdown, then activate the "Download Application" option.
  4. Activate (click or tap) the desired desktop version.
  5. Wait until the folder "EnterpriseDesktop_....zip" has finished downloading .
    (Look at the lower left corner of the Enterprise SMS window    ; the download progress status should appear below the folder name, then the status indicator may disappear shortly after the download is complete.)

Extract and Store

Next, follow these instructions to unzip the application folder and store it in the desired location:

  1. Right-click on "EnterpriseDesktop_....zip" in the lower left corner.
  2. Activate "Show in folder" in the list of options.
  3. Move the .zip folder to a new location, if desired.
  4. Right-click on the .zip folder.
  5. Select "Extract all."
  6. When extraction is finished, rename the folder if desired.

C. Open the Field Application

To open the Enterprise SMS field application,

  1. Navigate to the location in your file system where you stored the EnterpriseDesktop (or renamed) folder.
  2. Activate Enterprise SMS.FieldApp.exe.

     

Note: If this is your first time entering the field application, you'll be required to log in, and the program will synchronize with the server to collect up-to-date data. For more information about synchronizing, see .

The first time you open the field application, Enterprise SMS Desktop will load. After that, either Desktop or Tablet will load, depending on which of these you were in when closing the field application.

D. Verify Site

Once the sync process completes and the program opens, you will be able to see assets listed in the navigation pane at the left side of the screen (this is true for both Desktop and Tablet).

DO: Verify that the site listed at the top of the asset tree is the one you want to be working with.
